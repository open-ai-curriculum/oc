# Module 6 Reproducible Masters

## Purpose

This file defines the draft reproducible masters for Module 6.

These assets support blending, segmenting, controlled-word reading, controlled text rereading, and shared sentence work.

## Included Reproducibles

- sound-box mat
- blending arrow strip
- taught-word cards
- high-frequency routine word cards
- controlled text page set
- dictated-sentence frame
- end-of-module conference form
- family sound-to-word note

## Reproducible Master: Sound-Box Mat

Directions:

- say the sounds
- move one counter into each box
- slide your finger across and say the word

Layout:

| Box 1 | Box 2 | Box 3 |
| --- | --- | --- |
|  |  |  |

## Reproducible Master: Blending Arrow Strip

Teacher use:

- point under each sound
- sweep across the arrow while saying the whole word

Frame:

- /___/ /___/ /___/  ->  __________

## Reproducible Master: Taught-Word Cards

Use only module-approved taught words.

Each card should include:

- the printed word
- optional picture support for early introduction only

Teacher note:

- remove picture support once the word routine is established when appropriate

## Reproducible Master: High-Frequency Routine Word Cards

Use a very small set of routine words tied to controlled text.

Suggested card sections:

- word
- phrase use
- teacher note for introduction lesson

## Reproducible Master: Controlled Text Page Set

Each page must record:

- title
- taught patterns used
- routine words used
- lesson link

Teacher note:

- every controlled page must also exist in the module text approval log

## Reproducible Master: Dictated-Sentence Frame

- We can __________.
- I can __________.
- The __________ is __________.

Teacher note:

- students may contribute one sound, one letter, one routine word, or a full word with support

## Reproducible Master: Family Sound-To-Word Note

This week we are practicing how sounds come together to make simple words.

Try this at home:

- say a word slowly and listen for the sounds
- slide your finger as you put the sounds together
- reread one short teacher-sent text together
- keep the work short and encouraging
