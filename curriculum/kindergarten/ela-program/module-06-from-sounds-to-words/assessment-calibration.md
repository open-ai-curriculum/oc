# Module 6 Assessment Calibration

## Purpose

This file defines how Module 6 assessment evidence should be recorded and interpreted.

## Assessment Recording Form

Capture student performance in:

- oral blending
- oral segmenting
- controlled-word reading or identification
- shared sentence contribution

For each area record:

- secure
- developing
- not yet
- support used

## Decision Rules

- secure:
  - student completes the task with routine teacher language and little additional support
- developing:
  - student completes the task with visible scaffolds, repeated modeling, or reduced-choice options
- not yet:
  - student cannot demonstrate the skill even with routine module supports

## Exemplar Evidence

- secure blending:
  - student hears and combines a short sequence of taught sounds into a word without the teacher finishing it
- developing blending:
  - student blends correctly after the teacher repeats each sound and uses counters or boxes
- secure controlled-word reading:
  - student identifies or reads multiple taught words from the module set with minimal prompting

## Interpretation Notes

- do not score a child down for slow response time if the pattern is eventually demonstrated accurately
- separate oral articulation challenges from sound-sequencing understanding
- count dictated-sentence participation as valid writing evidence when the student contributes a meaningful sound, letter, or word

## Calibration Guidance

- assess only with previously taught words and patterns
- keep the order consistent: blend, segment, controlled word, sentence contribution
- compare borderline cases using one shared rubric discussion before recording final status
