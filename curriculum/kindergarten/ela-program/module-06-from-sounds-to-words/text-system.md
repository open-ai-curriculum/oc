# Module 6 Text System

## Purpose

This file operationalizes text quality, selection, and provenance controls for Module 6.

## Approved Text Types

- controlled word-card sets
- very short controlled decodable texts
- dictated sentence frames tied to taught patterns

## Approved Text List Parameters

Texts and assets for Module 6 must:

- align only to taught sound-symbol correspondences and routine words
- remain semantically sensible and orally natural
- be short enough for supported rereading
- avoid introducing hidden untaught complexity as the core challenge

## Text Selection Rubric

- pattern accuracy:
  - the text uses taught patterns consistently
- language quality:
  - the text sounds like natural child-directed language rather than a forced string of words
- meaning:
  - even controlled text should communicate an understandable idea or event
- developmental fit:
  - task load matches supported kindergarten decoding work
- inclusion:
  - names, images, and contexts are broadly welcoming and respectful

## Controlled Inventory

### Required Canonical Assets

- taught-word list for the module
- high-frequency routine word list
- controlled text inventory with lesson links
- dictated sentence stems aligned to weekly goals

### Controlled Text Inventory

- Week 21:
  - word-card routines only
- Week 22:
  - one short controlled text for launch and rereading
- Week 23:
  - one follow-up controlled text or revised version of the first text
- Week 24:
  - one cumulative review text using already taught patterns

## Rights And Provenance

- preferred source order:
  - program-authored controlled text
  - openly licensed controlled text reviewed for pattern integrity
  - externally licensed text with documented rights status
- each selected text must record:
  - title
  - author
  - source
  - rights status
  - taught patterns used

## Vocabulary And Language Rationale

Priority vocabulary:

- blend
- segment
- word
- sentence
- read
- write

Rationale:

- these are the core process words for the shift from sound work to supported word reading and writing
- they need to be used consistently across teacher talk, family communication, and assessment
