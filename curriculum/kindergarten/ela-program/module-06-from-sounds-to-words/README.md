# Module 6: From Sounds To Words

## Purpose

This package is the fourth fully authored module in the kindergarten ELA commercialization build.

It converts Weeks 21-24 of the annual ELA architecture into a daily four-week module centered on blending, segmenting, controlled text, routine high-frequency words, and dictated/shared sentence work.

## Included Artifacts

- `module.md`
- `standards-map.md`
- `review-record.md`
- `lessons/`

## Scope

This module covers Weeks 21-24 of the kindergarten ELA year and corresponds to:

- Module 6 in [module-structure.md](/Volumes/data/development/oc/curriculum/kindergarten/ela-program/module-structure.md)
- Weeks 21-24 in [lesson-inventory.md](/Volumes/data/development/oc/curriculum/kindergarten/ela-program/lesson-inventory.md)

## Status

Draft authored module. Human kindergarten literacy, foundational-skills, accessibility, and HQIM review are still required before classroom release.
