# Kindergarten ELA Module Structure

## Purpose

Define the full-year module sequence for the kindergarten ELA program.

## Module Sequence

| Module | Weeks | Working Title | Primary Strands |
| --- | --- | --- | --- |
| Module 1 | 1-4 | Launching Literacy Routines | print concepts, oral language, read-aloud, retell |
| Module 2 | 5-8 | Playing With Sounds And Language | phonological awareness, rhyme, syllables, alphabet launch |
| Module 3 | 9-12 | Stories, Characters, And Sequence | story comprehension, retell, shared reading, early writing |
| Module 4 | 13-16 | Letters, Sounds, And Shared Reading | alphabet knowledge, letter-sound work, shared reading, early encoding |
| Module 5 | 17-20 | Knowing More About The World Through Texts | informational read-aloud, vocabulary, oral explanation, drawing/writing |
| Module 6 | 21-24 | From Sounds To Words | blending, segmenting, decodable launch, sentence routines |
| Module 7 | 25-28 | Thinking, Talking, And Writing About Texts | comprehension, opinion and informative response, discussion, writing |
| Module 8 | 29-32 | Reading, Writing, And Knowledge In Connected Study | integrated narrative/informational work, stronger decoding, language |
| Module 9 | 33-36 | End-Of-Year Literacy Consolidation | cumulative application, benchmark synthesis, Grade 1 transition |

## Module Descriptions

### Module 1. Launching Literacy Routines

Focus:

- book handling
- print directionality
- listening routines
- answering and asking simple questions
- beginning retell

Existing pilot connection:

- `read-aloud-print-concepts`
- `oral-language-and-story-retell`

### Module 2. Playing With Sounds And Language

Focus:

- rhyme recognition and production
- syllables
- oral language play
- first alphabet routines

Existing pilot connection:

- `phonological-awareness-rhyme`
- `letter-recognition-alphabet-knowledge`

### Module 3. Stories, Characters, And Sequence

Focus:

- key details in narrative text
- beginning, middle, end
- characters and events
- oral retell with more detail
- shared drawing and dictation

### Module 4. Letters, Sounds, And Shared Reading

Focus:

- uppercase/lowercase automaticity
- letter-sound correspondences
- concept of word in shared text
- simple labeling and sound-linked writing

### Module 5. Knowing More About The World Through Texts

Focus:

- informational read-aloud
- vocabulary and knowledge building
- asking questions about a topic
- speaking and drawing to explain

### Module 6. From Sounds To Words

Focus:

- blending and segmenting
- high-frequency words in routine contexts
- simple decodable practice
- dictated and shared sentence work

### Module 7. Thinking, Talking, And Writing About Texts

Focus:

- text-based talk
- supported opinion and informative writing
- evidence from familiar text or topic
- stronger comprehension discussion

### Module 8. Reading, Writing, And Knowledge In Connected Study

Focus:

- integrated narrative and informational text study
- continued foundational-skills practice
- longer response routines
- knowledge-rich text sets

### Module 9. End-Of-Year Literacy Consolidation

Focus:

- cumulative review
- benchmark evidence
- stronger student independence with support
- transition language for Grade 1 expectations

## Module Build Priorities

The first authoring priorities should be:

1. Module 1 and Module 2, because they align with the strongest existing pilot strands
2. Module 4 and Module 6, because they establish the core foundational-skills sequence
3. Module 5 and Module 7, because they keep knowledge-building and writing from becoming secondary

## Module Count Rationale

Nine modules across 36 weeks gives the program enough annual structure to:

- support full-year planning
- cluster related literacy work coherently
- allow benchmark windows and review cycles
- avoid treating the kindergarten year as a handful of disconnected packets
