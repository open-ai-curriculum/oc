# Module 1 Text System

## Purpose

This file operationalizes text quality, selection, and provenance controls for Module 1.

## Approved Text Types

- one narrative picture book suitable for repeated interactive read-aloud
- one short predictable shared-reading chart created by the program
- one set of sequence-card images derived from the anchor narrative

## Approved Text List Parameters

The anchor narrative for Module 1 must:

- support repeated rereading across four weeks
- have a clear beginning, middle, and end
- include identifiable characters and events suitable for oral retell
- contain strong illustrations that support meaning without requiring decoding
- reflect varied identities and respectful classroom and family representations

The shared-reading chart must:

- contain one short sentence per line
- permit clear pointer tracking
- reuse familiar oral language from the read-aloud or classroom routines

## Text Selection Rubric

Use all criteria before adopting a text for Module 1:

- meaning and coherence:
  - the story is understandable and worth revisiting
- illustration support:
  - pictures materially support retell and questioning
- oral-language value:
  - the text contains repeated, discussable language
- developmental fit:
  - the text does not depend on independent decoding
- inclusion:
  - characters, homes, and settings avoid narrow or exclusionary framing
- routine fit:
  - the text supports book-part work, questioning, and sequencing

## Rights And Provenance

- preferred source order:
  - original program-created shared text
  - openly licensed text or image assets
  - externally licensed anchor picture book with documented rights status outside the repo
- required provenance note for each adopted text:
  - title
  - creator
  - source or publisher
  - rights status
  - approved use in repository artifacts

## Vocabulary And Language Rationale

Priority vocabulary:

- front cover
- title
- author
- illustrator
- first
- next
- last
- question

Rationale:

- these terms are core to early book orientation and story discussion
- they recur across later kindergarten literacy modules
- they can be supported with visuals, gestures, and family transfer language
