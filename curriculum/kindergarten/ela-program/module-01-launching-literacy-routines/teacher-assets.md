# Module 1 Teacher Assets

## Purpose

This file defines the canonical teacher-use asset layer for Module 1.

It is intended to close the gap between authored lesson pages and a teachable classroom module.

## Weekly Teacher Overview

### Week 1

- instructional priority:
  - launch read-aloud expectations, book handling, and first discussion routines
- teacher moves to emphasize:
  - model how to sit, watch, listen, point, and respond
  - narrate book-handling language repeatedly
  - keep oral responses short and supported
- key materials:
  - large-format anchor text
  - classroom routine visuals
  - book-part cards

### Week 2

- instructional priority:
  - establish where reading begins, how print moves, and how stories can be told in order
- teacher moves to emphasize:
  - track print slowly and consistently
  - rehearse first, next, last with pictures and gestures
  - prompt partner talk with modeled sentence starts
- key materials:
  - pointer
  - sequence cards
  - enlarged lines of shared text

### Week 3

- instructional priority:
  - deepen key-detail questioning, concept of word, and retell support
- teacher moves to emphasize:
  - return to the same text to show how rereading builds meaning
  - use wait time before calling for oral answers
  - accept gesture, picture, and teacher-recorded responses
- key materials:
  - story-picture cards
  - word-strip cards
  - sentence strip and pocket chart

### Week 4

- instructional priority:
  - consolidate routines and gather end-of-module evidence
- teacher moves to emphasize:
  - keep conferences short
  - record what students can do with support and without support
  - prepare families for at-home book talk routines
- key materials:
  - conference form
  - familiar read-aloud
  - family reading note

## Lesson Preparation Notes

- prepare one anchor read-aloud for repeated use across the module and one secondary text for comparison
- post or project routine visuals for listen, turn-and-talk, point, and retell
- pre-sort sequence cards and book-part visuals before each week begins
- rehearse the exact teacher language for book orientation and print directionality
- choose seating and movement patterns that allow every child to see the text and pointer clearly

## Observation And Conference Tools

### Daily Observation Focus

- book orientation
- response stamina during read-aloud
- use of sequence language
- ability to follow talk routines

### End-Of-Module Conference Form

- hold book correctly:
  - secure / developing / not yet
- identify front cover and title:
  - secure / developing / not yet
- show where reading begins:
  - secure / developing / not yet
- retell first, next, last:
  - secure / developing / not yet
- answer one simple question about a familiar story:
  - secure / developing / not yet
- ask one simple question about a story:
  - secure / developing / not yet

## Family Communication Asset

### Family Message

This month students are learning how books work, how readers listen and talk about stories, and how to tell a story in order. Families can help by naming the front cover, title, and pictures in a book and asking, "What happened first?" and "What happened next?"

## Intervention And Extension Guidance

### Intervention

- reduce the number of events to sequence from three to two
- allow pointing and acting before oral retell
- repeat the same anchor text rather than rotating too quickly
- pair book-part vocabulary with gestures and visual icons

### Extension

- invite students to add one detail to each retell step
- have students compare two covers or two versions of a familiar story
- add simple "who" and "where" questions to the end-of-module conference
