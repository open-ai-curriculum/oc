# Module 1 Assessment Calibration

## Purpose

This file defines how Module 1 assessment evidence should be recorded and interpreted.

## Assessment Recording Form

Record each student on the following dimensions:

- book handling
- book-part identification
- where reading begins
- retell sequence
- answer a question
- ask a question

For each dimension, record:

- secure
- developing
- not yet
- notes on support used

## Decision Rules

- secure:
  - student completes the task with minimal prompting and accurate demonstration
- developing:
  - student completes part of the task with moderate prompting, visuals, or partner rehearsal
- not yet:
  - student is unable to demonstrate the skill even with routine supports

## Exemplar Evidence

- secure retell:
  - student places or gestures to three events in order and labels them with simple oral language
- developing retell:
  - student sequences two events correctly and needs teacher prompting for the third
- not yet retell:
  - student cannot distinguish event order and needs the teacher to model the full sequence

## Interpretation Notes

- credit gesture-supported and picture-supported responses when the skill target is comprehension or sequencing
- do not down-score a student solely for short English utterances if understanding is evident
- distinguish between expressive-language limitation and lack of story understanding

## Calibration Guidance

- calibrate using the same familiar text and conference materials across the class
- score immediately after the conference while supports are still fresh in memory
- review one secure, one developing, and one not-yet example with another adult before high-stakes use
