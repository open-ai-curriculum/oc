# Module 2 Text System

## Purpose

This file operationalizes text quality, selection, and provenance controls for Module 2.

## Approved Text Types

- rhyme-rich songs and chants
- short shared oral texts with repeated sound patterns
- alphabet cards and picture-sort assets

## Approved Text List Parameters

Texts and assets for Module 2 must:

- foreground oral language over dense print
- include clear rhyme or sound patterns that are easy to hear in classroom use
- avoid culturally narrow or exclusionary examples
- use illustrations that make picture sorts unambiguous

## Text Selection Rubric

- phonological clarity:
  - target rhyme, syllable, or beginning-sound pattern is easy to hear
- developmental fit:
  - task does not require advanced decoding or spelling
- oral-language value:
  - chant or song is worth repeating and supports joyful participation
- dialect sensitivity:
  - examples do not depend on one narrow pronunciation pattern without teacher note
- visual clarity:
  - picture cards support, not confuse, the intended sound target

## Rights And Provenance

- program-created chants and picture cards are preferred
- any adopted song, chant, or picture set must include:
  - creator
  - source
  - rights status
  - approved classroom reuse note

## Vocabulary And Language Rationale

Priority vocabulary:

- rhyme
- syllable
- same sound
- first sound
- letter
- uppercase
- lowercase

Rationale:

- these terms anchor future phonological-awareness and alphabet instruction
- they recur in later modules and teacher conference language
