# Module 2: Playing With Sounds And Language

## Purpose

This package is the second fully authored module in the kindergarten ELA commercialization build.

It converts Weeks 5-8 of the annual ELA architecture into a daily four-week module focused on sound play, rhyme, syllables, beginning sounds, and alphabet launch.
