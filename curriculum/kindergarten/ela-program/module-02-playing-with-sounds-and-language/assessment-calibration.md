# Module 2 Assessment Calibration

## Purpose

This file defines how Module 2 assessment evidence should be recorded and interpreted.

## Assessment Recording Form

Record each student on:

- identifies rhyme
- produces or selects rhyme
- claps syllables
- notices beginning sound
- names and matches taught letters

For each area, capture:

- secure
- developing
- not yet
- support used

## Decision Rules

- secure:
  - student completes the task correctly with minimal repetition
- developing:
  - student completes the task with repeated modeling, reduced choices, or visual support
- not yet:
  - student cannot demonstrate the skill even with routine module supports

## Exemplar Evidence

- secure rhyme:
  - student identifies the rhyming pair and can echo an additional rhyme example
- developing rhyme:
  - student picks the correct rhyming pair from two choices after hearing the teacher repeat them
- secure syllables:
  - student accurately claps familiar two- and three-syllable words with the group and in conference

## Interpretation Notes

- do not confuse articulation differences with lack of sound awareness
- accept movement-based evidence when the target is syllable awareness
- note whether a child understands the routine but cannot yet verbalize the label

## Calibration Guidance

- use the same familiar picture cards for all students in the conference
- keep the task order stable: rhyme, syllables, beginning sound, letters
- calibrate judgments with a second adult using one borderline example per band
