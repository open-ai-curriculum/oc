# Module 2 Multilingual Supports

## Purpose

This file defines module-specific multilingual supports for Module 2.

## Core Design Principles

- treat oral-language play as access work, not as a barrier
- avoid assuming that rhyme and beginning-sound patterns operate identically across languages
- support listening and comparison before requiring English production

## Lesson-Type Scaffolds

### Rhyme And Chant Lessons

- model two examples and one non-example before asking for a response
- allow children to echo a line rather than generate original rhyme immediately
- use simple oral frames:
  - "These sound the same."
  - "These do not sound the same."

### Syllable Lessons

- pair clapping with picture naming
- allow students to repeat after teacher model before independent response
- keep vocabulary highly familiar

### Beginning-Sound Lessons

- note when home-language phonology may shift how the first sound is perceived
- use mouth modeling and picture support together
- accept pointing to matching pictures as evidence before oral naming

## Home-Language Bridge Opportunities

- encourage families to clap names and familiar words in any language
- invite home-language rhyme play even when rhyme does not transfer exactly to English examples
- ask families to notice letters in environmental print using the taught letter set

## Cross-Linguistic Notes For Teachers

- do not treat non-response on rhyme as a comprehension failure
- some beginning-sound contrasts may require extra modeling depending on language background
- keep the focus on noticing and participating, not perfect English production
