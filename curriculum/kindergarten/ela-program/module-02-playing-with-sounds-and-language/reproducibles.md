# Module 2 Reproducible Masters

## Purpose

This file defines the draft reproducible masters for Module 2.

These assets support rhyme, syllables, beginning-sound comparison, and early alphabet routines.

## Included Reproducibles

- rhyme picture pair cards
- rhyme / not-rhyme sort mat
- syllable clap picture strip
- beginning-sound picture-sort cards
- taught-letter card set
- sound-and-letter conference form
- family sound-play note

## Reproducible Master: Rhyme Picture Pair Cards

Card pairs:

- cat / hat
- bug / rug
- log / dog
- sun / bun

Teacher note:

- replace any pair that is not phonologically clear in the local speech community

## Reproducible Master: Rhyme / Not-Rhyme Sort Mat

Sort headings:

- sound the same
- do not sound the same

Directions:

- say the picture names aloud
- place the pair in the correct column

## Reproducible Master: Syllable Clap Picture Strip

Pictures:

- apple
- pencil
- banana
- robot

Directions:

- say the word
- clap the parts
- circle how many parts you heard

## Reproducible Master: Beginning-Sound Picture-Sort Cards

Use a small taught comparison set, such as:

- /m/: moon, map
- /s/: sun, sock

Teacher note:

- keep the sort set cumulative and limited

## Reproducible Master: Taught-Letter Card Set

Card types:

- uppercase cards
- lowercase cards
- matching-picture cue cards

Use:

- whole-group quick review
- partner match
- small-group conferencing

## Reproducible Master: Family Sound-Play Note

This week we are playing with rhyme, syllables, and letters.

Try this at home:

- sing a short rhyme
- clap the parts in a name or word
- find one taught letter in a book or sign
- play in any language and keep it joyful
