# Module 4 Text System

## Purpose

This file operationalizes text quality, selection, and provenance controls for Module 4.

## Approved Text Types

- cumulative shared-reading charts
- letter cards and picture cards
- simple labeling prompts connected to classroom objects or pictures

## Approved Text List Parameters

Texts and assets for Module 4 must:

- align to the taught letter set only
- include large clear print and dependable word spacing
- support repeated shared reading rather than independent decoding expectations
- use concrete images whose target initial sounds are phonologically defensible

## Text Selection Rubric

- taught-pattern alignment:
  - text does not rely on untaught letter-sound patterns for the main routine
- visual clarity:
  - fonts, spacing, and illustrations reduce confusion
- meaning and usefulness:
  - shared text is meaningful enough to reread and discuss
- developmental fit:
  - children can participate through tracking, talk, and supported contribution
- inclusion:
  - classroom objects, names, and examples are broad and respectful

## Controlled Inventory

### Focus-Letter Asset Inventory

- daily focus-letter cards
- uppercase/lowercase match cards
- picture exemplars for each taught sound
- one cumulative shared chart per week
- one labeling prompt set per week

## Rights And Provenance

- program-created shared charts and labeling prompts are preferred
- all picture assets must record source and rights status before publication
- if external images are used, maintain a separate licensing register outside the lesson page

## Vocabulary And Language Rationale

Priority vocabulary:

- letter
- sound
- word
- space
- label
- sentence

Rationale:

- these words support the bridge between oral language, shared reading, and early writing
- they remain central in later decoding and sentence-construction work
