# Module 4 Multilingual Supports

## Purpose

This file defines module-specific multilingual supports for Module 4.

## Core Design Principles

- support the alphabetic principle without collapsing multilingual language knowledge into deficit framing
- make oral rehearsal part of every print-linked task
- distinguish sound awareness from English naming accuracy

## Lesson-Type Scaffolds

### Letter And Sound Lessons

- present letter name, sound, mouth model, and picture together
- allow students to repeat the sound chorally before individual response
- use sentence frames:
  - "This is letter __."
  - "It can say __."

### Shared Reading Lessons

- preview one or two routine words before rereading
- use finger or pointer tracking with teacher narration
- accept pointing to words and spaces as valid evidence before oral explanation

### Labeling And Dictated Writing Lessons

- allow home-language oral rehearsal before English label contribution
- accept a first sound or first letter when the full English word is not yet stable
- keep teacher scribing visible and explicit

## Home-Language Bridge Opportunities

- families can notice taught letters in environmental print in any language script where helpful comparisons are possible
- families can talk about object names in home language and English while the child notices the first sound in English examples

## Cross-Linguistic Notes For Teachers

- some target English sounds may not exist in all home languages and may require extra modeling
- concept-of-word work should not be confused with ability to explain print terms in English
- teacher judgment should privilege demonstrated understanding over fluent terminology
