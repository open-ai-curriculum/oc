# Module 4 Teacher Assets

## Purpose

This file defines the canonical teacher-use asset layer for Module 4.

## Weekly Teacher Overview

### Week 13

- instructional priority:
  - strengthen taught letters and launch shared-reading pointer routines
- teacher moves to emphasize:
  - keep the letter set cumulative and visible
  - separate name practice from sound practice when students confuse the two
  - use large text and slow pointer tracking

### Week 14

- instructional priority:
  - connect letters and beginning sounds through pictures, labels, and dictated word work
- teacher moves to emphasize:
  - use clear exemplars for common sounds
  - keep labeling tasks oral first, then supported print
  - revisit shared text every day

### Week 15

- instructional priority:
  - deepen concept of word and routine high-frequency word noticing
- teacher moves to emphasize:
  - point to spaces and word boundaries
  - model rereading for fluency without expecting independence
  - let children contribute one sound or letter during shared writing

### Week 16

- instructional priority:
  - synthesize letters, sounds, shared reading, and early writing evidence
- teacher moves to emphasize:
  - use the same text and word set for cumulative review
  - record exactly which supports were needed
  - avoid grading handwriting neatness as the target

## Lesson Preparation Notes

- prepare a cumulative focus-letter wall with one consistent font
- create shared-reading charts with ample spacing and clear line breaks
- preselect picture examples with unmistakable target sounds
- stage pocket-chart materials and label cards before each lesson block

## Observation And Conference Tools

### Daily Observation Focus

- recognizes taught letters
- connects picture to taught sound
- tracks one line of shared text
- contributes sounds or letters during labeling

### End-Of-Module Conference Form

- uppercase/lowercase matching:
  - secure / developing / not yet
- common sound connection:
  - secure / developing / not yet
- shared-text tracking:
  - secure / developing / not yet
- sound or letter contribution in writing:
  - secure / developing / not yet

## Family Communication Asset

### Family Message

This month students are strengthening letter names, common sounds, shared reading, and simple labeling routines. Families can help by noticing letters in signs and books, naming a few taught letters, and inviting children to say the first sound in a familiar word.

## Intervention And Extension Guidance

### Intervention

- shrink the focus-letter set
- use oral sound practice before picture sorts
- provide finger tracking or hand-over-hand pointer support when needed
- allow students to dictate labels while the teacher records

### Extension

- invite students to identify a routine word in shared text independently
- have students compare two pictures that start with the same taught sound
- let students contribute two sounds or letters in shared writing
