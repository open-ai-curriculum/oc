# Module 4: Letters, Sounds, And Shared Reading

## Purpose

This package is the third fully authored module in the kindergarten ELA commercialization build.

It converts Weeks 13-16 of the annual ELA architecture into a daily four-week module centered on uppercase/lowercase fluency, letter-sound correspondences, concept of word, shared reading, and early encoding.
