## Standards and Objective

- Standards addressed:
  - CCSS.ELA-LITERACY.RF.K.1.D
  - CCSS.ELA-LITERACY.RF.K.3.A
- Lesson objective:
  - Students will review and name the current taught set of focus letters with growing fluency.

## Materials and Preparation

- Materials needed:
  - focus-letter cards
  - alphabet chart
- Teacher preparation:
  - select a limited taught set for the lesson

## Lesson Flow

### Opening

- Open with a brief alphabet routine using only the taught letters.

### Core Learning Experience

- Flash the focus letters one at a time and model smooth naming.

### Guided Practice / Discussion / Investigation

- Students respond chorally, with partners, and individually.
- Use a quick "find it" routine on the chart.

### Check for Understanding

- Students correctly identify multiple taught letters.

### Closing

- Review which letters the class knows well already.

## Accessibility and Inclusion Notes

- Use large high-contrast cards.
- Keep the taught set small.
- Allow pointing before oral naming.

## Assessment Notes

- This lesson formatively assesses letter-name fluency on the taught set.
- Evidence matches the objective because students directly identify target letters.

## Subject-Specific Notes

- Avoid untaught distractors.
- Keep pacing brisk but supportive.
