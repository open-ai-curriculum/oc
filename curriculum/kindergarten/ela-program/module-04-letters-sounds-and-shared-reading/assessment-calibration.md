# Module 4 Assessment Calibration

## Purpose

This file defines how Module 4 assessment evidence should be recorded and interpreted.

## Assessment Recording Form

Capture student performance in:

- uppercase/lowercase match
- common sound and picture match
- shared-text tracking
- writing contribution

For each area record:

- secure
- developing
- not yet
- support used

## Decision Rules

- secure:
  - student completes the task accurately with only routine teacher language
- developing:
  - student completes the task with visible prompts, reduced choices, or modeled rehearsal
- not yet:
  - student cannot show the skill even when routine supports are provided

## Exemplar Evidence

- secure letter matching:
  - student matches four to six taught uppercase/lowercase pairs consistently
- developing letter matching:
  - student matches a smaller subset or relies on immediate comparison support
- secure writing contribution:
  - student contributes an appropriate first sound or letter during labeling with little prompting

## Interpretation Notes

- do not score handwriting formation as the target unless explicitly assessing fine-motor production
- treat pointing to spaces and words as meaningful print-concept evidence
- interpret oral sound production with attention to multilingual phonology and articulation differences

## Calibration Guidance

- keep the same focus-letter set in all conference tasks
- score tracking on familiar shared text, not on new text
- calibrate "developing" examples carefully, especially when students know the concept but need expressive support
