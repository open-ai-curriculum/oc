# Module 4 Reproducible Masters

## Purpose

This file defines the draft reproducible masters for Module 4.

These assets support taught letters and sounds, shared reading, and early labeling and dictated writing.

## Included Reproducibles

- uppercase / lowercase match cards
- picture cue cards for taught sounds
- shared-reading pointer chart
- concept-of-word tracking strip
- labeling picture frames
- dictated-writing response strip
- end-of-module conference form
- family letter-and-sound note

## Reproducible Master: Uppercase / Lowercase Match Cards

Directions:

- match each uppercase letter to its lowercase partner
- say the letter name after matching

Teacher note:

- keep only the taught letter set in play

## Reproducible Master: Picture Cue Cards For Taught Sounds

Each taught sound should include:

- one primary picture exemplar
- one backup picture exemplar
- one teacher note if pronunciation may vary

## Reproducible Master: Shared-Reading Pointer Chart

Chart sentence frame:

- We can read together.

Use:

- pointer tracking
- word counting
- space noticing

## Reproducible Master: Concept-Of-Word Tracking Strip

Directions:

- touch each word one time
- stop at the end
- return to the beginning and reread

## Reproducible Master: Labeling Picture Frames

Prompt frames:

- school object picture
- classroom object picture
- familiar animal or action picture

Teacher note:

- students may contribute one sound, one letter, or the full word with support

## Reproducible Master: Dictated-Writing Response Strip

- I say: ____________________
- I hear: ____________________
- I can write: ____________________

Teacher note:

- teacher may scribe as the student points or contributes orally

## Reproducible Master: Family Letter-And-Sound Note

This week we are practicing letters, sounds, and shared reading.

Try this at home:

- notice one taught letter in a book or sign
- say the letter name
- say a word that starts with the same sound
- let your child point to words while you read together
