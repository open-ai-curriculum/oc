# Kindergarten Current-Vs-Target Gap

## Purpose

Compare the current kindergarten repository inventory to the commercial build targets so the next build order is explicit.

## Current Inventory Snapshot

As of 2026-04-02, the kindergarten workspace currently includes:

- 11 pilot packets
- 11 packet lessons
- 88 packet-level classroom-material files
- 5 assembled omnibus program volumes
- 5 print-ready omnibus core volumes

The current packet set is distributed as:

- literacy: 4 packets
- math: 1 packet

- health: 1 packet
- PE: 1 packet

## What The Comparison Means

### ELA

ELA is the best-developed kindergarten subject in the repository, but it is still only a pilot cluster. It should not yet be mistaken for a full-year literacy program.

### Math

Math currently has one packet and one lesson, while the commercial target is a daily full-year subject program. This is the second-most urgent build area after ELA.

### Non-literacy subjects

Science, social studies, arts, health, PE, and technology all currently exist as proof-of-coverage packets, not as year plans. Their current state is useful for architecture validation, but not for market-comparable scope.

## Prioritized Build Order

1. Kindergarten ELA commercialization package
2. Kindergarten Math commercialization package
3. Kindergarten Science and Social Studies annual module definitions
4. Kindergarten Health, PE, Arts, and Technology annual module definitions
5. Rebuild omnibus volumes from richer subject-program sources

## Immediate Build Implications

- no further large omnibus-book expansion should happen before ELA and math subject-program work catches up
- the current kindergarten books should be treated as draft compilations, not as evidence of full-year subject completeness
- subject-program build progress should be tracked against lesson and asset targets, not just against whether a packet exists
