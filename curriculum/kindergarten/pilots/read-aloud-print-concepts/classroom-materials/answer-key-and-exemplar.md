## Purpose

Provide a reference for what counts as correct or acceptable evidence in the live classroom task.

## Correct Or Acceptable Responses

### Book Parts

Acceptable evidence includes:

- pointing to the actual front cover
- identifying the title area when prompted
- showing where the book opens

### Reading Start

Acceptable evidence includes:

- pointing to the first line or first readable position on the page
- moving finger left to right across the line with teacher support

### Key-Detail Question

Acceptable evidence includes:

- naming one correct detail from the page
- pointing to a correct picture detail and explaining it briefly
- using a short phrase that matches the text or illustration

## Exemplar Student Responses

- "This is the front cover."
- "We start here."
- "The dog is running."
- pointing to the correct image while saying, "Here."

## Common Error Patterns

- pointing to the cover image instead of the title
- pointing randomly on the page
- answering from prior experience instead of the read-aloud
- naming a favorite detail that is unrelated to the teacher question

## Teacher Note

This lesson uses performance evidence, so the key is a guide to acceptable demonstrations rather than a fixed written-answer sheet.
