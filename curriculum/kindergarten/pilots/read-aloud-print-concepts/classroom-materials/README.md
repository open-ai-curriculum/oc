# Classroom Materials: Kindergarten Read-Aloud And Print Concepts

## Purpose

This directory contains the first teacher-use classroom-material set for the kindergarten read-aloud and print-concepts pilot.

It is intended to bridge the gap between:

- governed curriculum packet artifacts
- support materials
- actual teacher-deliverable lesson materials

## Included Artifacts

- `lesson-narrative.md`
- `classroom-assignment.md`
- `home-practice.md`
- `answer-key-and-exemplar.md`
- `scoring-guidance.md`
- `assessment-check.md`
- `assessment-key.md`

## Source Authority

These materials derive from:

- `k-lit-readaloud-print-concepts-unit`
- `k-lit-readaloud-print-concepts-lesson-01`
- the packet support materials in `../support-materials/`

## Status

Teacher-use draft. Not classroom-ready without human instructional-quality, accessibility, and usability review.
