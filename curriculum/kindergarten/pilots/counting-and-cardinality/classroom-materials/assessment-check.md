## Purpose

Provide a short teacher-led check of counting and total naming.

## Assessment Items

- Count a set of up to 5 objects.
- Tell how many are in the set.
- Compare a second set if appropriate.

## Accessibility Notes

- Use manipulatives and oral directions.
- Reduce set size for learners who need it.
