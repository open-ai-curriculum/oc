## Purpose

Define the live classroom task students complete to show they can count a set and name the total.

## Student Task

- Count one small set of objects by touching or moving each item once.
- Say the total after counting.
- Compare a second set as more, less, or equal if the teacher extends the task.

## Expected Evidence Of Learning

- student counts each object once
- student says the final number as the total
- student uses comparison language if prompted

## Accessibility Notes

- Allow pointing, touching, moving, or lining up objects.
- Use visuals for more, less, and equal.
