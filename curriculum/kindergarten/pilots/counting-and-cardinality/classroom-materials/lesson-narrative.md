## Purpose

Provide a teacher-ready script for modeling one-to-one counting and naming the total in a small set.

## Teacher Narrative

- Gather a small set of counters and say, "We will touch each object once and count carefully."
- Model counting slowly while moving one object at a time.
- After the final object, say, "The last number tells how many there are."
- Invite students to count a second set with you, then count a third set independently or with a partner.
- Ask, "How many?" after each set and prompt students to say the total clearly.

## Assessment Notes

- Watch for one-to-one correspondence, steady verbal counting, and whether students use the last number as the total.
- Accept touching, moving, or pointing as long as each object is counted once.

## Accessibility and Inclusion Notes

- Use large, easy-to-move objects and clear spacing.
- Allow oral counting with partner support or teacher echoing.
- Keep set sizes small for learners who need reduced cognitive load.
