## Standards and Objective

- Standards addressed:
  - CCSS.MATH.CONTENT.K.CC.A.1
  - CCSS.MATH.CONTENT.K.CC.B.4
- Lesson objective:
  - Students will count a small set of objects and state how many are in the set.

## Lesson Flow

- opening:
  - count together with a teacher model
- core learning:
  - count a small set with one-to-one correspondence
  - say the final number as the total
- check for understanding:
  - each student counts a set and answers “How many?”

## Accessibility and Inclusion Notes

- use manipulatives, clear spacing, and oral modeling
- allow touching, moving, and verbalizing counts
