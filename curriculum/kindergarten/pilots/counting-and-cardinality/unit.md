## Unit Purpose

- What is the unit about?
  - Students count objects with one-to-one correspondence, name how many are in a set, and compare small groups.
- Why does it matter for learners?
  - Kindergarten students need early number sense built through concrete counting and comparison, not number words alone.

## Desired Results

- Students should understand:
  - counting tells how many objects are in a set
  - the last number said tells the total
  - groups can be compared as more, less, or equal
- Students should be able to do:
  - count a small set accurately
  - answer “How many?”
  - compare two groups using concrete language

## Lesson Sequence

### Lesson 1

- objective: Count objects in a set and say how many there are.
- key activities:
  - count real objects with one-to-one pointing
  - stop and name the total in the set
  - compare two small sets

## Accessibility and Inclusion Notes
