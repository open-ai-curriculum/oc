# Kindergarten Literacy Pilot: Letter Recognition and Alphabet Knowledge

## Purpose

This package extends kindergarten literacy scope with a unit focused on recognizing uppercase and lowercase letters, connecting letter names to common sounds, and building alphabet knowledge through oral language, movement, and visual supports.
