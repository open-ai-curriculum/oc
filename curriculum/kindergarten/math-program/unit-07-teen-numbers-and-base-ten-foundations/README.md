# Unit 7: Teen Numbers And Base-Ten Foundations

## Purpose

This package is the seventh fully authored unit in the Kindergarten Math commercialization build.

It introduces teen numbers through counting beyond 10, ten-and-some-ones language, concrete grouping, frames, numeral matching, and comparison.
