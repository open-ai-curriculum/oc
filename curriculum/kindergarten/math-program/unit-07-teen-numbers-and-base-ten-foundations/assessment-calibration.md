# Unit 7 Assessment Calibration

## Assessment Recording Form

Record each student on:

- counts beyond 10 in sequence
- builds a teen number with a visible ten and some ones
- matches a teen numeral to a concrete representation
- explains a teen number as ten and some ones

For each area record:

- secure
- developing
- not yet
- support used

## Interpretation Notes

- accept pointing, touching, and teacher-recorded oral language as valid evidence
- do not require independent numeral writing for secure place-value understanding
- prioritize the visible ten-and-ones structure over rote counting performance
