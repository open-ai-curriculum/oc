## Unit Purpose

- What is the unit about?
  - Students identify, describe, compare, sort, and compose shapes while using positional language to describe objects and arrangements.
- Why does it matter for learners?
  - Geometry in Kindergarten should help students notice structure in the world, talk about attributes, and build larger forms from smaller ones through concrete exploration.

## Desired Results

### Students should understand:

- shapes can be named and described by attributes
- shapes can be sorted and compared
- bigger shapes can be built from smaller shapes
- position words help describe where objects are

### Students should know:

- names of common flat and solid shapes
- words such as above, below, beside, in front of, and behind
- that shape orientation does not change the shape name

### Students should be able to do:

- identify and name common shapes
- describe shapes by attributes
- sort shapes by a chosen rule
- compose pictures and larger shapes from smaller shapes
- describe object positions with clear language

## Essential Questions

- How do we know what shape something is?
- How can we describe how shapes are alike and different?
- How can shapes and position words help us talk about what we build and see?

## Assessment Evidence

### Summative evidence

- Students complete a short end-of-unit conference in which they:
  - identify common shapes
  - sort shapes by an attribute
  - compose a larger shape or picture
  - describe position using a simple positional phrase

### Formative evidence

- shape hunts
- sort-and-explain routines
- build-and-describe tasks
- positional language tasks with concrete objects

## Accessibility And Inclusion Notes

- use large shapes, real objects, and movement-based position work
- allow pointing, placing, sorting, tracing, and oral responses
- avoid requiring only fine-motor drawing as evidence

## Subject-Specific Quality Notes

- What discipline-specific risks need attention?
  - overemphasizing shape appearance instead of attributes
  - treating orientation change as a different shape
  - using positional language without concrete referents
- What content or task types require extra review?
  - shape sets
  - sort categories
  - composition tasks
