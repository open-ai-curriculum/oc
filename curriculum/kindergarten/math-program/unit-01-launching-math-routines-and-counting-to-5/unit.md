## Unit Purpose

- What is the unit about?
  - Students learn how math time works in kindergarten while counting objects to 5, matching number words to quantities, comparing small sets, and sorting objects by simple attributes.
- Why does it matter for learners?
  - Kindergarten mathematics requires stable routines for counting, touching, showing, explaining, and comparing. This unit establishes those routines using concrete objects and teacher-guided discussion.

## Desired Results

### Students should understand:

- counting tells how many objects are in a set
- each object is counted once
- the last number said tells the total
- two sets can be compared as more, less, equal, or not equal

### Students should know:

- number words to 5
- how to touch or move one object at a time when counting
- that objects can be sorted into groups using one clear rule

### Students should be able to do:

- count a set of up to 5 objects accurately
- answer "How many?"
- show a quantity that matches a spoken number word
- compare two small sets
- sort and count groups

## Essential Questions

- How do we count a set carefully?
- How do we know how many are in a set?
- How can we tell whether one group has more, less, or the same number?

## Assessment Evidence

### Summative evidence

- Students complete a short end-of-unit math conference in which they:
  - count a set of up to 5 objects with one-to-one correspondence
  - state how many are in the set
  - match a spoken number word to a small quantity
  - compare two small sets as more, less, or equal

### Formative evidence

- daily counting routines
- count-and-show responses
- comparison tasks with concrete objects
- teacher observations during sort and explain routines

## Lesson Sequence

### Week 1

- KMATH-001 through KMATH-004
- focus: launch math routines, count to 3 and 5, show quantities, compare two small sets

### Week 2

- KMATH-005 through KMATH-008
- focus: sort objects, count sounds and movements, match numerals to quantities, explain counting

### Week 3

- KMATH-009 through KMATH-012
- focus: count scattered objects, compare equal/not equal, sort and count groups, use a counting mat

### Week 4

- KMATH-013 through KMATH-016
- focus: count and circle totals, show 5 in different ways, review routines, complete unit check

## Accessibility And Inclusion Notes

- Representation supports:
  - real objects, spaced sets, counting mats, numeral cards, and visual comparison mats
- Engagement supports:
  - repeated routines, partner counting, movement counting, and short whole-group practice
- Action/expression supports:
  - touching, moving, pointing, circling, saying, and teacher-recorded explanation are valid
- Language/accessibility considerations:
  - keep directions short, use consistent math language, and allow oral rehearsal before response
- Inclusive examples or framing notes:
  - use familiar classroom objects and examples that reflect varied student experiences respectfully
