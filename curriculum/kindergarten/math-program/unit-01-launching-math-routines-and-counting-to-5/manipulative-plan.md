# Unit 1 Manipulative Plan

## Core Materials

- counters or small classroom objects
- numeral cards 1-5
- counting mats
- compare-two-sets mat
- sort trays or hoops

## Use Guidelines

- start with real objects before moving to printed representations
- use one manipulative set consistently for several lessons
- let students physically move counted objects when possible

## Accessibility Notes

- use large, easy-to-grasp objects
- keep sets visually separated
- permit pointing and teacher movement support when needed
