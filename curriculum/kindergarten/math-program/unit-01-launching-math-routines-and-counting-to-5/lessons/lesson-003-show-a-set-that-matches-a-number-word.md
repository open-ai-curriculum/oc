# Lesson: Show A Set That Matches A Number Word

## Lesson Metadata

- Lesson title: Show A Set That Matches A Number Word
- Lesson ID: `KMATH-003`
- Unit: `k-math-unit-01`
- Subject: Mathematics
- Grade / grade band: Kindergarten
- Estimated duration: 25 minutes
- Review status: Draft

## Standards and Objective

- Standards addressed:
  - CCSS.MATH.CONTENT.K.CC.B.4
  - CCSS.MATH.CONTENT.K.CC.B.5
- Lesson objective:
  - Students will build a set that matches a spoken number word to 5.

## Lesson Flow

- opening:
  - hear a number word and show it with fingers
- core learning:
  - build matching sets with counters
- check for understanding:
  - show one target quantity

## Accessibility and Inclusion Notes

- allow students to point, build, or show with fingers
