# Lesson: Count Up To 5 Objects With One-To-One Pointing

## Lesson Metadata

- Lesson title: Count Up To 5 Objects With One-To-One Pointing
- Lesson ID: `KMATH-002`
- Unit: `k-math-unit-01`
- Subject: Mathematics
- Grade / grade band: Kindergarten
- Estimated duration: 25 minutes
- Review status: Draft

## Standards and Objective

- Standards addressed:
  - CCSS.MATH.CONTENT.K.CC.A.1
  - CCSS.MATH.CONTENT.K.CC.B.4
  - CCSS.MATH.CONTENT.K.CC.B.5
- Lesson objective:
  - Students will count up to 5 objects using one-to-one correspondence.

## Lesson Flow

- opening:
  - review counting to 3
- core learning:
  - count sets up to 5
- check for understanding:
  - count one set and say how many

## Accessibility and Inclusion Notes

- allow moving objects into a counted space
- keep sets visually separated
