# Lesson: Sort And Count Groups

## Lesson Metadata

- Lesson title: Sort And Count Groups
- Lesson ID: `KMATH-011`
- Unit: `k-math-unit-01`
- Subject: Mathematics
- Grade / grade band: Kindergarten
- Estimated duration: 25 minutes
- Review status: Draft

## Standards and Objective

- Standards addressed:
  - CCSS.MATH.CONTENT.K.MD.A.3
  - CCSS.MATH.CONTENT.K.CC.B.5
- Lesson objective:
  - Students will sort a set and count one or more resulting groups.

## Lesson Flow

- opening:
  - sort a small mixed set together
- core learning:
  - count each group after sorting
- check for understanding:
  - sort and count one category
