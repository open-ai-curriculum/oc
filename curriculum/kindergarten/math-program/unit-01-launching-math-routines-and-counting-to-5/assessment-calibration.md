# Unit 1 Assessment Calibration

## Assessment Recording Form

Record each student on:

- counts up to 5 accurately
- uses one-to-one correspondence
- states how many after counting
- compares two sets

For each area record:

- secure
- developing
- not yet
- support used

## Decision Rules

- secure:
  - student completes the task with routine teacher language only
- developing:
  - student completes the task with visible prompting, object movement support, or reduced set size
- not yet:
  - student cannot demonstrate the skill even with routine supports

## Interpretation Notes

- do not confuse number-word production errors with inability to track quantity
- accept touching and moving objects as valid evidence
- note if the student understands comparison only after objects are rearranged
