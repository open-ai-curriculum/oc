# Unit 1: Launching Math Routines And Counting To 5

## Purpose

This package is the first fully authored unit in the Kindergarten Math commercialization build.

It converts the annual math architecture into an actual four-week unit with daily lesson pages.

## Scope

This unit covers Weeks 1-4 of the Kindergarten Math year and corresponds to:

- Unit 1 in [unit-structure.md](/Volumes/data/development/oc/curriculum/kindergarten/math-program/unit-structure.md)
- Lessons `KMATH-001` through `KMATH-016` in [lesson-inventory.md](/Volumes/data/development/oc/curriculum/kindergarten/math-program/lesson-inventory.md)
