## Unit Purpose

- What is the unit about?
  - Students compare objects by measurable attributes, describe differences with informal language, measure with nonstandard units, sort data from class questions, build simple picture graphs, and discuss what graphs show.
- Why does it matter for learners?
  - Kindergarten measurement and data work builds early mathematical language, observation habits, and visual interpretation. Students need concrete comparison and class-generated data before later formal graphing and measurement work.

## Desired Results

### Students should understand:

- objects can be compared by attributes such as length, weight, and capacity-like features
- data can answer class questions when it is sorted and displayed
- graphs help show which categories have more, less, or the same

### Students should know:

- informal comparison words such as longer, shorter, heavier, lighter, and taller
- that direct comparison is more reliable than guessing
- that a picture graph organizes data into categories

### Students should be able to do:

- compare two objects directly
- sort and count data from a class question
- build and read a simple picture graph
- measure an object with nonstandard units and talk about the result

## Essential Questions

- How can we compare two objects?
- What can a graph tell us?
- How can measurement and data help us describe the world?

## Assessment Evidence

## Accessibility And Inclusion Notes

- use real classroom objects, picture cards, physical graph spaces, and nonstandard units large enough to handle easily
- allow pointing, moving objects, placing graph cards, oral explanation, and teacher-recorded language
- use repeated sentence frames for comparison and graph interpretation

## Subject-Specific Quality Notes

- What discipline-specific risks need attention?
  - turning measurement into worksheet-only comparison without real objects
  - using graphs students did not help build
  - treating nonstandard measurement results as exact across inconsistent units
- What content or task types require extra review?
  - comparison prompts
  - graph categories and picture keys
  - nonstandard measurement tasks
