# Unit 8 Assessment Calibration

## Assessment Recording Form

Record each student on:

- compares two objects by a shared attribute
- uses informal comparison language
- helps read or build a simple picture graph
- answers a question about what the graph shows

For each area record:

- secure
- developing
- not yet
- support used

## Interpretation Notes

- accept pointing, moving graph pieces, and teacher-recorded oral language as valid evidence
- do not require independent written graph labels for secure understanding
- prioritize concrete comparison and graph interpretation over vocabulary perfection
