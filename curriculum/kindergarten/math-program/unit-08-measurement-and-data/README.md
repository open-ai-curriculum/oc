# Unit 8: Measurement And Data

## Purpose

This package is the eighth fully authored unit in the Kindergarten Math commercialization build.

It introduces direct comparison of measurable attributes, nonstandard measurement, class data routines, picture graphs, and teacher-supported interpretation of graphs.
