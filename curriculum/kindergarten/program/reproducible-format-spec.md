# Kindergarten Reproducible Format Spec

## Purpose

Define how kindergarten reproducibles should be formatted when converted from content masters into print-ready classroom assets.

## Requirements

- large type and simple labels
- high contrast with photocopy-safe design
- wide cut lines and clear card borders
- minimal text per card or strip
- space for pointing, circling, drawing, or teacher recording
- icon support where icons clarify meaning without creating dependency

## Asset Types

- center cards
- sort cards
- observation charts
- response strips
- family mini handouts

## Accessibility Notes

- avoid relying on color alone
- use simple repeated language
- keep visual clutter low
