Chapter 5: Counting Tells How Many

Page 9: Count One by One

Touch each object once. Say one number for each object.

Page 10: The Last Number Tells How Many

When you finish counting, say the last number again. That number tells how many are in the set.

Page 11: Compare Two Small Sets

We can ask which group has more, which has fewer, or whether the groups are the same.

Talk About It
  • Count a small group.
  • Tell how many.
  • Which group has more?
Teacher Note

Pair with count-and-touch mat and compare-two-sets strip from reproducibles.

Chapter 8: We Can Create and Share

Page 15: Art Can Show an Idea

We can create with drawing, color, music, and movement.

Page 16: Share Your Work

When we share, we can say what we made, what we noticed, and what it means to us.

Page 17: Art Can Use Many Materials

Artists can use crayons, paper, paint, music, and movement.

Talk About It
  • Make something simple.
  • Share one part of your work.
Accessibility

Allow gesture, sound, movement, drawing, and short oral explanation.
