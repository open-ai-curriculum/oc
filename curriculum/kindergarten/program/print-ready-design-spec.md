# Kindergarten Print-Ready Design Spec

## Purpose

Define the expected print-ready layout and production standards for the assembled kindergarten program volumes.

## Scope

- `textbook.md`
- `workbook.md`
- `teacher-edition.md`
- `grading-key.md`
- `assessment-pack.md`
- `reproducibles.md`

## Layout Principles

- prioritize large type, generous spacing, and uncluttered pages
- keep one main learning move per page spread where possible
- preserve oral, visual, gestural, and manipulative pathways in page design
- avoid dense text blocks that assume independent reading

## Textbook Layout Requirements

- large student-facing type
- high-contrast headings
- visual callouts for `Talk About It`
- clear page numbering and chapter tabs
- space for teacher pointing and shared reading

## Workbook Layout Requirements

- one practice demand at a time
- ample writing and drawing space
- clear cut-and-sort boundaries where needed
- room for teacher scribing or observational notes

## Teacher Edition Layout Requirements

- clear distinction between teacher directions and student-facing text
- reusable icons for observation, accessibility, family connection, and asset references
- explicit cross-references to workbook sections, reproducibles, exemplars, grading key, and assessment pack

## Grading Key and Assessment Pack Requirements

- compact evidence tables
- consistent use of `secure`, `developing`, and `not yet`
- dedicated space for prompting notes and observational context

## Reproducible Formatting Requirements

- copy-safe black-and-white friendly masters
- no visual dependencies that disappear when photocopied
- trim-safe margins
- large response boxes and card boundaries

## Production Checks Before Final QA

- stable page numbering
- consistent headers and footers
- resolved cross-references
- finalized icon system
- accessible contrast and spacing checks
