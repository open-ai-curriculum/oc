# Grade 1 Print-Ready Design Spec

## Purpose

Define the expected print-ready layout and production standards for the assembled Grade 1 program volumes.

## Scope

- `textbook.md`
- `workbook.md`
- `teacher-edition.md`
- `grading-key.md`
- `assessment-pack.md`
- `reproducibles.md`

## Layout Principles

- preserve readability while increasing independence from Kindergarten
- keep evidence, explanation, and response structures visually explicit
- support multimodal access without fragmenting page flow
- maintain consistent chapter navigation across all volumes

## Textbook Layout Requirements

- readable student text with clear chunking
- visible vocabulary, prompt, and `Talk About It` treatments
- page references that align with workbook and teacher-edition use

## Workbook Layout Requirements

- response spaces matched to Grade 1 writing stamina
- visible sentence frames where needed
- enough room for drawing, labeling, and equation work

## Teacher Edition Layout Requirements

- clear sidebars for teacher moves, observation, and accessibility
- explicit references to reproducibles, exemplars, grading criteria, and assessment tasks
- chapter-openers that identify paired textbook and workbook sections

## Grading Key and Assessment Pack Requirements

- concise calibration language
- explicit space for prompt level, independence, and evidence mode
- chapter-aligned assessment naming

## Reproducible Formatting Requirements

- print-safe charts, mats, cards, and organizers
- consistent visual treatment across literacy and non-literacy strands
- response areas large enough for Grade 1 writing and drawing

## Production Checks Before Final QA

- stable page numbering
- resolved chapter-to-chapter references
- unified icon and sidebar system
- accessibility checks on spacing, contrast, and reading load
