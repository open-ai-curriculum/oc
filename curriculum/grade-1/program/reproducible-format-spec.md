# Grade 1 Reproducible Format Spec

## Purpose

Define how Grade 1 reproducibles should be formatted when converted from content masters into print-ready classroom assets.

## Requirements

- readable student-facing labels
- sufficient space for short written response, drawing, and math representation
- photocopy-safe visuals and borders
- clear prompt hierarchy
- consistent header treatment across subjects

## Asset Types

- organizers
- response sheets
- mats
- quick checks
- scenario cards
- reflection slips

## Accessibility Notes

- avoid unnecessary copying load
- preserve sentence frames where they support access
- keep response formats flexible enough for oral or teacher-recorded use
