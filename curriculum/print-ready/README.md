# Print-Ready Export Templates

## Purpose

Provide shared HTML and CSS assets for export-ready curriculum layouts.

## Included Assets

- `print.css`
- `kindergarten-cover.html`
- `grade-1-cover.html`

## Usage

- Grade-level program directories contain volume-specific HTML templates that import `../../print-ready/print.css`.
- These templates are export-ready starting points for browser print, PDF generation, or later design-tool conversion.
- Repository markdown sources remain authoritative for content.
