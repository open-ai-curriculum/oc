# Implementation Plan: Kindergarten Math Commercialization

## Objective

Create the canonical commercialization package for Kindergarten Math so the repository can build a full-year commercial-grade math source product.

## Deliverables

- `curriculum/kindergarten/math-program/README.md`
- `curriculum/kindergarten/math-program/year-overview.md`
- `curriculum/kindergarten/math-program/unit-structure.md`
- `curriculum/kindergarten/math-program/manipulative-and-practice-architecture.md`
- `curriculum/kindergarten/math-program/assessment-architecture.md`
- `curriculum/kindergarten/math-program/standards-bands.md`
- `curriculum/kindergarten/math-program/lesson-inventory.md`

## Design Notes

- use ELA commercialization structure where helpful, but adapt to mathematics-specific needs
- keep mathematical representations, discussion, and concrete materials central
- define repeated instructional routines so later authored units can be consistent

## Risks

- building a math inventory that is too shallow to support commercial-grade claims
- letting the program become worksheet-heavy instead of representation-rich
- failing to define the manipulative and teacher-move layer early enough

## Exit Condition

Kindergarten Math has a canonical annual commercialization structure and can become the next major authored subject program.
