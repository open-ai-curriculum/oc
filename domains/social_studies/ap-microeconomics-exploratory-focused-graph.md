# AP Microeconomics Exploratory Focused Graph

## Purpose

Define the bounded social studies graph cut that most directly supports the current AP Microeconomics exploratory overlay.

This package exists to make the repository's weakest current AP-facing economics surface explicit rather than implicit.

## Focus Rule

This is a bounded exploratory graph cut. It documents what the current canonical graph can support, and just as importantly, where the graph is still too coarse for stronger course-specific claims.

## Included Core Nodes

### Economics Backbone

- `E1` needs, wants, choice, and exchange
- `E2` production, consumption, and local markets
- `E3` incentives, trade, and simple systems
- `E4` markets, labor, and public resource decisions
- `E5` macroeconomic patterns, global trade, and finance basics
- `E6` economic analysis, policy tradeoffs, and capstone modeling

### Shared Inquiry Support

- `Q3` supported explanation from evidence
- `Q4` investigation design, representation, and data use
- `Q5` evaluation of sources, models, and claims
- `Q6` synthesis, argument, and public communication

### Policy And Argument Bridge

- `C4` constitutional principles and public policy
- `C6` advanced civic argument, policy design, and participation
- `A1` interdisciplinary research and evidence synthesis

## Why These Nodes Are In Scope

This cut covers:

- choice and incentive reasoning
- basic market structure and decision-making
- policy intervention and public-resource tradeoff reasoning
- evidence-based economics argument

## Deliberately Out Of Scope

This cut does not attempt to model:

- firm structure as first-class separate nodes
- welfare-analysis detail as first-class separate nodes
- elasticity, cost-curve, and market-model substructures as governed course-level entities

## Minimal Dependency Logic

The core dependency flow inside this cut is:

`E1 -> E2 -> E3 -> E4 -> E5 -> E6`

`Q3 -> Q4 -> Q5 -> Q6`

`C4 -> policy intervention reasoning`

`C6 + E6 + Q6 -> advanced economics argument and tradeoff communication`

`A1` provides the bounded synthesis bridge when the cut is used for capstone-style applied economics work

## Main Current Strength

The graph already models incentives, markets, tradeoffs, policy reasoning, and evidence-based explanation strongly enough to support exploratory overlay work.

## Main Current Limitation

This package remains explicitly exploratory because the repository does not yet model firm behavior, welfare analysis, and course-specific microeconomics structures at sufficient resolution for stronger fidelity claims.

## Claim Boundary

This focused graph documents the current exploratory economics-facing assessment slice in the repository. It does not establish official blueprint verification, course equivalence, or exam-preparation approval.
