# AP Human Geography Focused Graph

## Purpose

Define the bounded social studies graph cut that most directly supports the current AP Human Geography overlay.

This package isolates the current strongest geography-facing assessment surface so spatial reasoning, migration, human systems, and policy application can be inspected together as one bounded view.

## Focus Rule

This is a bounded overlay-supporting geography cut, not a separate AP course graph. It remains governed by the canonical social studies graph.

## Included Core Nodes

### Geography Backbone

- `G1` place, maps, and environmental features
- `G2` human-environment interaction and regions
- `G3` spatial patterns, movement, and resources
- `G4` geographic reasoning with maps, data, and scale
- `G5` global interdependence, migration, and sustainability
- `G6` geospatial analysis and policy application

### Shared Inquiry And Argument Support

- `Q4` investigation design, representation, and data use
- `Q5` evaluation of sources, models, and claims
- `Q6` synthesis, argument, and public communication

### Economics And Advanced Bridge

- `E3` incentives, trade, and simple systems
- `E4` markets, labor, and public resource decisions
- `E5` macroeconomic patterns, global trade, and finance basics
- `A2` capstone argument, modeling, and public communication

## Why These Nodes Are In Scope

This cut covers:

- spatial concepts and geographic tools
- migration, population, and human systems
- economic and cultural spatial patterns
- sustainability, policy, and public argument

## Deliberately Out Of Scope

This cut does not attempt to model:

- AP Human Geography thematic subunits as first-class separate graph entities
- more detailed cultural, urban, and agricultural sub-branches than the canonical graph currently supports

## Main Current Strength

The graph already models spatial reasoning, movement, human-environment systems, and policy-facing geography strongly enough to support a bounded high-confidence overlay cut.

## Main Current Limitation

Thematic specificity remains compressed into broad geography and economics nodes, so this package is strongest for topology and coverage analysis rather than reviewed course-equivalence claims.

## Claim Boundary

This focused graph documents the current geography-facing assessment slice in the repository. It does not establish official blueprint verification or exam-preparation approval.
