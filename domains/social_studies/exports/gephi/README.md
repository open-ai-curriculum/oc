# Gephi Export

This directory contains Gephi-ready exports of the unified social studies operational graph.

## Included Layers

- capability nodes and prerequisite topology
- capability family nodes
- standards-native entities and alignment edges
- assessment overlay and assessment unit nodes
- artifact registry and artifact attachment nodes
- runtime slice and decision-rule nodes

## Files

- `social-studies-gephi-nodes.csv`: node table for Gephi CSV import
- `social-studies-gephi-edges.csv`: edge table for Gephi CSV import
- `social-studies-gephi-graph.gexf`: single-file graph export for direct Gephi import
