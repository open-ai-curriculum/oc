# AP World History: Modern Focused Graph

## Purpose

Define the bounded social studies graph cut that most directly supports the current AP World History: Modern overlay.

This package isolates the current world-history-facing assessment surface so global processes, historical comparison, migration, exchange, and argument can be inspected together as one bounded view.

## Focus Rule

This is a bounded overlay-supporting world-history cut, not a separate AP course graph. It remains governed by the canonical social studies graph and preserves the repository's current compression boundaries.
