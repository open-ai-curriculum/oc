# E1 Node Package

## Node

`E1` - Needs, wants, choice, and exchange

## Purpose

This package defines the entry economics node for the social studies domain.

It operationalizes the learner's ability to distinguish needs and wants, recognize that choices involve limits, and understand simple exchange before later market and incentive reasoning begins.
