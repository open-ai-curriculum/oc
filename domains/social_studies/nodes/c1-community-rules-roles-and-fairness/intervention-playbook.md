# C1 Intervention Playbook

## Purpose

Map `C1` failure modes to bounded intervention responses.

## Intervention Rules

### For C1-FM-01 Rule-Purpose Opacity

Primary interventions:

- pair each rule with a shared problem it addresses
- use what happens if there is no rule comparisons
- connect the rule to safety care or turn-taking outcomes

### For C1-FM-02 Role Confusion

Primary interventions:

- match roles to actions in simple scenarios
- use role cards for common community settings
- ask who is responsible and why

### For C1-FM-03 Fairness-As-Preference

Primary interventions:

- compare fair and unfair scenarios with explicit reasons
- sort examples by shared rule versus individual want
- use sentence stems for fair because

### For C1-FM-04 Language Or Context Familiarity Risk

Primary interventions:

- simplify scenario language
- begin with familiar school home or neighborhood contexts
- allow oral explanation or acting out before written response
