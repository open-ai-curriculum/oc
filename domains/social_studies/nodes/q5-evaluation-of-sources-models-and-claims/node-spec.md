# Q5 Node Specification

## Node Identity

- id: `Q5`
- name: `evaluation_of_sources_models_and_claims`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `social-studies-evaluation-node-package`
- evidence class: `Synthetic`

## Purpose

The learner evaluates the credibility, usefulness, and limits of sources, models, and claims and explains why some evidence is stronger than other evidence for a given question.

## Dependencies

- `Q4` investigation design, representation, and data use

## Exit Conditions

The learner is secure on `Q5` only when all of the following are true:

1. identifies relevant strengths or limits of a source, model, or claim
2. distinguishes stronger and weaker support for a question
3. explains the evaluation using source or model features
4. succeeds across changed evidence types or contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
