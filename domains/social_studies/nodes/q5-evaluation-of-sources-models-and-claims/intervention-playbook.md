# Q5 Intervention Playbook

## Purpose

Map `Q5` failure modes to bounded intervention responses.

## Intervention Rules

### For Q5-FM-01 Face-Value Acceptance

Primary interventions:

- require one reason for trusting or questioning the source
- compare a strong and weak example on the same question
- use check prompts for evidence support and limitation

### For Q5-FM-02 Feature-Notice Without Judgment

Primary interventions:

- connect each noticed feature to a judgment consequence
- use because this source has therefore it is frames
- compare one feature that matters and one that does not

### For Q5-FM-03 Stronger-Weaker Reversal

Primary interventions:

- sort sources into stronger and weaker with explicit criteria
- test the judgment against the question being asked
- compare confidence language with actual evidence strength

### For Q5-FM-04 Source-Load Or Language Risk

Primary interventions:

- reduce source length and preteach key terms
- use annotation plus oral reasoning
- verify whether comprehension supports materially change performance
