# H1 Node Package

## Node

`H1` - Chronology, identity, and community memory

## Purpose

This package defines the entry history node for the social studies domain.

It operationalizes the learner's ability to place experiences and shared events in basic temporal order, connect stories or records to people and communities, and recognize that communities remember the past through sources and traditions.
