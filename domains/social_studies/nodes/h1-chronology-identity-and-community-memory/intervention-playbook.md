# H1 Intervention Playbook

## Purpose

Map `H1` failure modes to bounded intervention responses.

## Intervention Rules

### For H1-FM-01 Time-Order Instability

Primary interventions:

- use concrete before-now-after sequences
- anchor events with visual timeline strips
- retell the same sequence with one order change to detect the error

### For H1-FM-02 Source-Community Disconnect

Primary interventions:

- pair sources with named people places or groups
- ask who might keep or use this source and why
- connect the source to a familiar community memory example

### For H1-FM-03 Personal-Past Collapse

Primary interventions:

- contrast personal memory with shared community record
- sort examples into mine, our community, and imagined story
- use simple language for source type and purpose

### For H1-FM-04 Language Or Background-Access Risk

Primary interventions:

- preteach essential vocabulary
- use familiar community examples before transferring to less familiar ones
- reduce expressive demand while preserving chronology reasoning
