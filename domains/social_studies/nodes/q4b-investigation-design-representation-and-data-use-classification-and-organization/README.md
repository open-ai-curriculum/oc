# Q4B Node Package

## Node

`Q4B` - Investigation Design Representation And Data Use Classification And Organization

## Purpose

This package defines a decomposition node extending `Q4` for the social studies domain.

It operationalizes the learner's ability to organize compare and classify elements of investigation design representation and data use.
