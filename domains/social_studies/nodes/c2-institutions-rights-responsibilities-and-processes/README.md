# C2 Node Package

## Node

`C2` - Institutions, rights, responsibilities, and processes

## Purpose

This package defines the second civics node for the social studies domain.

It operationalizes the learner's ability to identify basic institutions, connect rights and responsibilities to those institutions, and recognize simple civic processes.
