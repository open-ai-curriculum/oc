# G4C Failure Taxonomy

### G4C-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of geographic reasoning with maps data and scale but does not yet build a stable structure for the decomposition node

### G4C-FM-02 Organization Drift

- description: learner begins the explanation and application work but loses organization or classification discipline during the task

### G4C-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of geographic reasoning with maps data and scale but struggles to apply the work in a bounded new context

### G4C-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
