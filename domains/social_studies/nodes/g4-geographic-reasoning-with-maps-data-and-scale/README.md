# G4 Node Package

## Node

`G4` - Geographic reasoning with maps, data, and scale

## Purpose

This package defines the next geography node for the social studies domain.

It operationalizes the learner's ability to reason with maps, data displays, and scale in order to explain geographic patterns, comparisons, and consequences.
