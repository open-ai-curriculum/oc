# G4 Intervention Playbook

## Purpose

Map `G4` failure modes to bounded intervention responses.

## Intervention Rules

### For G4-FM-01 Map-Data Split

Primary interventions:

- connect one map feature to one data point explicitly
- annotate the two displays together
- compare separate reading with integrated reasoning

### For G4-FM-02 Scale Blindness

Primary interventions:

- compare local and broader scale views of the same issue
- ask what changes when the scale changes
- require one scale-based caution in the explanation

### For G4-FM-03 Evidence-Claim Mismatch

Primary interventions:

- require a map or data citation for each claim
- sort supported and unsupported claims
- narrow the conclusion to what the evidence can justify

### For G4-FM-04 Display Complexity Risk

Primary interventions:

- reduce the number of displays first
- preteach legend and axis features
- verify whether simplified displays change performance materially
