# E6B Node Package

## Node

`E6B` - Economic Analysis Policy Tradeoffs And Capstone Modeling Classification And Organization

## Purpose

This package defines a decomposition node extending `E6` for the social studies domain.

It operationalizes the learner's ability to organize compare and classify elements of economic analysis policy tradeoffs and capstone modeling.
