# H2C Node Specification

## Node Identity

- id: `H2C`
- name: `historical_sources_and_past_present_comparison_explanation_and_application`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `history-decomposition-node-package`
- evidence class: `Synthetic`

## Purpose

The learner develops historical sources and past present comparison through the explanation and application slice of the governed social studies graph.

## Dependencies

- `H2B`

## Exit Conditions

The learner is secure on `H2C` only when all of the following are true:

1. completes the explanation and application work with direct evidence
2. explains the structure or reasoning used in the node
3. applies the node in a bounded changed context
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
