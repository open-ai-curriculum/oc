# C6 Node Package

## Node

`C6` - Advanced civic argument, policy design, and participation

## Purpose

This package defines the capstone civics node for the social studies domain.

It operationalizes the learner's ability to build advanced civic arguments, design bounded policy responses, and match participation strategies to real institutional settings.
