# E4C Failure Taxonomy

### E4C-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of markets labor and public resource decisions but does not yet build a stable structure for the decomposition node

### E4C-FM-02 Organization Drift

- description: learner begins the explanation and application work but loses organization or classification discipline during the task

### E4C-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of markets labor and public resource decisions but struggles to apply the work in a bounded new context

### E4C-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
