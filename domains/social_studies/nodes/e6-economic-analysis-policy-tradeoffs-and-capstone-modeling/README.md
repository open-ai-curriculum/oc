# E6 Node Package

## Node

`E6` - Economic analysis, policy tradeoffs, and capstone modeling

## Purpose

This package defines the capstone economics node for the social studies domain.

It operationalizes the learner's ability to analyze economic scenarios, model tradeoffs, and support bounded public or market recommendations.
