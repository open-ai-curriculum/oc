# E6 Node Specification

## Node Identity

- id: `E6`
- name: `economic_analysis_policy_tradeoffs_and_capstone_modeling`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `economic-analysis-capstone-node-package`
- evidence class: `Synthetic`

## Purpose

The learner performs economic analysis, models tradeoffs in public or market decisions, and uses bounded economic reasoning to support capstone recommendations.

## Dependencies

- `E5` macroeconomic patterns, global trade, and finance basics
- `C4` constitutional principles and public policy
- `Q6` synthesis, argument, and public communication

## Exit Conditions

The learner is secure on `E6` only when all of the following are true:

1. builds a bounded economic model or explanation for a policy or market problem
2. compares tradeoffs using explicit assumptions and constraints
3. explains how public-policy conditions interact with the economic recommendation
4. succeeds in changed economic-analysis or policy-tradeoff contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`
