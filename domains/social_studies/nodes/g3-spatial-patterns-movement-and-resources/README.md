# G3 Node Package

## Node

`G3` - Spatial patterns, movement, and resources

## Purpose

This package defines the next geography node for the social studies domain.

It operationalizes the learner's ability to identify spatial patterns, explain basic movement of people or goods, and reason about resource distribution.
