# H6 Node Package

## Node

`H6` - Historiography, public history, and capstone argument

## Purpose

This package defines the capstone history node for the social studies domain.

It operationalizes the learner's ability to compare historical interpretations, reason about public-history framing, and produce a capstone historical argument.
