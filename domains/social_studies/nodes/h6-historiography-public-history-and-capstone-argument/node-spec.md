# H6 Node Specification

## Node Identity

- id: `H6`
- name: `historiography_public_history_and_capstone_argument`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `historiography-public-history-node-package`
- evidence class: `Synthetic`

## Purpose

The learner evaluates competing historical interpretations, connects scholarship to public history, and constructs a capstone historical argument for a defined audience.

## Dependencies

- `H5` thematic historical analysis and periodization
- `Q6` synthesis, argument, and public communication

## Exit Conditions

The learner is secure on `H6` only when all of the following are true:

1. compares at least two historical interpretations or public-history framings of the same problem
2. defends a capstone historical argument with explicit use of evidence and interpretation
3. explains how audience, medium, or public purpose shapes the historical presentation
4. succeeds with changed interpretation sets or public-history contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`
