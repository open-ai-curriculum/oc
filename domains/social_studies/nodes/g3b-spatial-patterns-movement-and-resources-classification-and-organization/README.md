# G3B Node Package

## Node

`G3B` - Spatial Patterns Movement And Resources Classification And Organization

## Purpose

This package defines a decomposition node extending `G3` for the social studies domain.

It operationalizes the learner's ability to organize compare and classify elements of spatial patterns movement and resources.
