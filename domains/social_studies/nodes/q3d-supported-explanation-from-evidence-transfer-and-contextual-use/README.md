# Q3D Node Package

## Node

`Q3D` - Supported Explanation From Evidence Transfer And Contextual Use

## Purpose

This package defines a decomposition node extending `Q3` for the social studies domain.

It operationalizes the learner's ability to transfer and adapt supported explanation from evidence.
