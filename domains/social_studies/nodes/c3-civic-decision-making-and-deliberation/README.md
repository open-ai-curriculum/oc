# C3 Node Package

## Node

`C3` - Civic decision-making and deliberation

## Purpose

This package defines the next civics node for the social studies domain.

It operationalizes the learner's ability to compare options in a civic problem, weigh reasons, and participate in basic deliberation with evidence and fairness constraints.
