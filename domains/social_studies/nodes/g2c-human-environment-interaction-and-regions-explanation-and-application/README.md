# G2C Node Package

## Node

`G2C` - Human Environment Interaction And Regions Explanation And Application

## Purpose

This package defines a decomposition node extending `G2` for the social studies domain.

It operationalizes the learner's ability to explain and apply human environment interaction and regions.
