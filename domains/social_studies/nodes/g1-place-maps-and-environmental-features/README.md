# G1 Node Package

## Node

`G1` - Place, maps, and environmental features

## Purpose

This package defines the entry geography node for the social studies domain.

It operationalizes the learner's ability to identify places, use simple map-like representations, and notice environmental features before later regional, spatial, and movement reasoning begins.
