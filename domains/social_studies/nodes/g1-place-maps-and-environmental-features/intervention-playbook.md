# G1 Intervention Playbook

## Purpose

Map `G1` failure modes to bounded intervention responses.

## Intervention Rules

### For G1-FM-01 Representation-Place Disconnect

Primary interventions:

- connect the map or model directly to the real place
- trace one feature from representation to reality and back
- use simple labels and legend-like supports

### For G1-FM-02 Feature Identification Drift

Primary interventions:

- highlight one feature category at a time
- compare multiple place scenes with the same feature
- sort features by land water built space or pathway

### For G1-FM-03 Scale Or Orientation Fragility

Primary interventions:

- rotate or resize one representation at a time
- use matching tasks between two views of the same place
- verbalize same place different view

### For G1-FM-04 Access Risk In Spatial Display

Primary interventions:

- simplify the spatial field
- use tactile or high-contrast maps where appropriate
- check whether access adjustments materially change performance
