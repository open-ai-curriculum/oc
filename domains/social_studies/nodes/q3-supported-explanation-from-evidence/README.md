# Q3 Node Package

## Node

`Q3` - Supported explanation from evidence

## Purpose

This package defines the first explanatory inquiry node for the social studies domain.

It operationalizes the learner's ability to make a claim or explanation that is connected to observable evidence rather than unsupported opinion or recall.
