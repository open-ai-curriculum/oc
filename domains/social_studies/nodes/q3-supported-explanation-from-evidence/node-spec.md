# Q3 Node Specification

## Node Identity

- id: `Q3`
- name: `supported_explanation_from_evidence`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `social-studies-explanation-node-package`
- evidence class: `Synthetic`

## Purpose

The learner gives a basic explanation or claim and supports it with relevant evidence from a source, representation, or scenario.

## Dependencies

- `Q2` compare, classify, and sequence evidence

## Exit Conditions

The learner is secure on `Q3` only when all of the following are true:

1. states a usable explanation or claim
2. selects or cites relevant supporting evidence
3. connects the evidence to the explanation with usable clarity
4. succeeds across changed source types or contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
