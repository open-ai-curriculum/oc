# Q3 Verification Model

## Gate Set

### G1. Direct Performance Gate

- purpose: produce a supported explanation from evidence
- pass rule: at least 85% accuracy across node-aligned tasks

### G2. Meaning Gate

- purpose: explain how the evidence supports the claim
- pass rule: usable connection in at least 2 of 3 opportunities

### G3. Transfer Gate

- purpose: apply the same evidence-use logic to a changed context
- pass rule: one successful transfer task

### G4. Retention Gate

- purpose: confirm persistence after spacing
- pass rule: at least 80% delayed recheck accuracy after about 14 days
