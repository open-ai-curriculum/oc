# Q6 Node Package

## Node

`Q6` - Synthesis, argument, and public communication

## Purpose

This package defines the next inquiry and discourse node for the social studies domain.

It operationalizes the learner's ability to synthesize evidence across sources, construct bounded arguments, and communicate conclusions for an identified audience and purpose.
