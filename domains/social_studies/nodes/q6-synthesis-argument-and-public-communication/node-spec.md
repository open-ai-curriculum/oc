# Q6 Node Specification

## Node Identity

- id: `Q6`
- name: `synthesis_argument_and_public_communication`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `social-studies-synthesis-and-argument-node-package`
- evidence class: `Synthetic`

## Purpose

The learner synthesizes evidence across sources, builds a bounded argument or explanation, and communicates the result for an identified audience and purpose.

## Dependencies

- `Q5` evaluation of sources, models, and claims

## Exit Conditions

The learner is secure on `Q6` only when all of the following are true:

1. synthesizes multiple relevant pieces of evidence into one coherent claim or explanation
2. selects communication moves that fit the audience and task purpose
3. explains how the chosen evidence supports the conclusion
4. succeeds across changed audience or communication contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
