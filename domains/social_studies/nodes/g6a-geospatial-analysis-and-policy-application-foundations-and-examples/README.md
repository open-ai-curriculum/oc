# G6A Node Package

## Node

`G6A` - Geospatial Analysis And Policy Application Foundations And Examples

## Purpose

This package defines a decomposition node extending `G6` for the social studies domain.

It operationalizes the learner's ability to build foundational recognition and guided examples within geospatial analysis and policy application.
