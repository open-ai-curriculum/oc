# H5 Intervention Playbook

## Purpose

Map `H5` failure modes to bounded intervention responses.

## Intervention Rules

### For H5-FM-01 Theme Tracking Fragmentation

Primary interventions:

- use one running theme tracker across eras before period labels are assigned
- require each example to be tagged as continuity change or both
- model how multiple events fit under one historical theme

### For H5-FM-02 Period Boundary Arbitrary Or Unjustified

Primary interventions:

- compare two possible period boundaries and defend one with evidence
- require a because statement for each proposed turning point
- sort date changes from meaningful structural changes

### For H5-FM-03 Continuity-Change Imbalance

Primary interventions:

- require one sentence for continuity and one for change in the same response
- use t-charts that connect persistence and rupture to the same theme
- compare overchange and no-change examples before independent work

### For H5-FM-04 Multi-Era Source Load Or Vocabulary Risk

Primary interventions:

- reduce the source set and preteach key terms
- sequence eras one at a time before asking for full synthesis
- verify whether vocabulary or source-load supports materially change performance
