# H3D Node Package

## Node

`H3D` - Causation Context And Change Over Time Transfer And Contextual Use

## Purpose

This package defines a decomposition node extending `H3` for the social studies domain.

It operationalizes the learner's ability to transfer and adapt causation context and change over time.
