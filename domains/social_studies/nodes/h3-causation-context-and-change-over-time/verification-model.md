# H3 Verification Model

## Gate Set

### G1. Direct Performance Gate

- purpose: identify causation context and change over time
- pass rule: at least 85% accuracy across node-aligned tasks

### G2. Meaning Gate

- purpose: explain how context supports the causal or change-over-time claim
- pass rule: usable explanation in at least 2 of 3 opportunities

### G3. Transfer Gate

- purpose: apply the same historical reasoning in a changed context
- pass rule: one successful transfer task

### G4. Retention Gate

- purpose: confirm persistence after spacing
- pass rule: at least 80% delayed recheck accuracy after about 14 days
