# H3 Intervention Playbook

## Purpose

Map `H3` failure modes to bounded intervention responses.

## Intervention Rules

### For H3-FM-01 Single-Cause Collapse

Primary interventions:

- identify multiple contributing factors in a constrained source set
- sort causes by direct and indirect influence
- compare one-factor and multi-factor explanations

### For H3-FM-02 Context Dropout

Primary interventions:

- build a short context frame before explaining the event
- highlight source details that establish setting and conditions
- ask what was happening around this event that matters

### For H3-FM-03 Presentist Interpretation

Primary interventions:

- contrast what we think now with what people knew then
- separate judgment from contextual explanation
- use source-based perspective reminders

### For H3-FM-04 Language Or Source-Load Risk

Primary interventions:

- simplify the source set and preteach key terms
- allow annotation with oral explanation
- verify whether comprehension supports materially change performance
