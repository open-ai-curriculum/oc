# E3 Node Package

## Node

`E3` - Incentives, trade, and simple systems

## Purpose

This package defines the next economics node for the social studies domain.

It operationalizes the learner's ability to explain how incentives affect choices, how trade changes available options, and how simple systems connect participants.
