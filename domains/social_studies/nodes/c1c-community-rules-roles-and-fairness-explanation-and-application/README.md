# C1C Node Package

## Node

`C1C` - Community Rules Roles And Fairness Explanation And Application

## Purpose

This package defines a decomposition node extending `C1` for the social studies domain.

It operationalizes the learner's ability to explain and apply community rules roles and fairness.
