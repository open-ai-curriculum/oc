# C4 Node Package

## Node

`C4` - Constitutional principles and public policy

## Purpose

This package defines the next civics node for the social studies domain.

It operationalizes the learner's ability to connect constitutional or foundational civic principles to public-policy questions and explain basic policy tradeoffs.
