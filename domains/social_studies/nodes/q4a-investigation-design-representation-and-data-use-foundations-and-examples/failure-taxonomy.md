# Q4A Failure Taxonomy

### Q4A-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of investigation design representation and data use but does not yet build a stable structure for the decomposition node

### Q4A-FM-02 Organization Drift

- description: learner begins the foundations and examples work but loses organization or classification discipline during the task

### Q4A-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of investigation design representation and data use but struggles to apply the work in a bounded new context

### Q4A-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
