# H6C Node Package

## Node

`H6C` - Historiography Public History And Capstone Argument Explanation And Application

## Purpose

This package defines a decomposition node extending `H6` for the social studies domain.

It operationalizes the learner's ability to explain and apply historiography public history and capstone argument.
