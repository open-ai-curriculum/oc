# H6C Failure Taxonomy

### H6C-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of historiography public history and capstone argument but does not yet build a stable structure for the decomposition node

### H6C-FM-02 Organization Drift

- description: learner begins the explanation and application work but loses organization or classification discipline during the task

### H6C-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of historiography public history and capstone argument but struggles to apply the work in a bounded new context

### H6C-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
