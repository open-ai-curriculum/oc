# H1C Node Package

## Node

`H1C` - Chronology Identity And Community Memory Explanation And Application

## Purpose

This package defines a decomposition node extending `H1` for the social studies domain.

It operationalizes the learner's ability to explain and apply chronology identity and community memory.
