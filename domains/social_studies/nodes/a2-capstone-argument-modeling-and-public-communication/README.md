# A2 Node Package

## Node

`A2` - Capstone argument, modeling, and public communication

## Purpose

This package defines the final cross-domain capstone node for the social studies domain.

It operationalizes the learner's ability to integrate strand capstones into a final public-facing argument or model with explicit audience and medium choices.
