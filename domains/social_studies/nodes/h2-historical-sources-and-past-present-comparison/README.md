# H2 Node Package

## Node

`H2` - Historical sources and past-present comparison

## Purpose

This package defines the second history node for the social studies domain.

It operationalizes the learner's ability to inspect basic historical sources and compare features of the past and present using evidence rather than vague impression alone.
