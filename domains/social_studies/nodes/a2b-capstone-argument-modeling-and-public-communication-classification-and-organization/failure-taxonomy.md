# A2B Failure Taxonomy

### A2B-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of capstone argument modeling and public communication but does not yet build a stable structure for the decomposition node

### A2B-FM-02 Organization Drift

- description: learner begins the classification and organization work but loses organization or classification discipline during the task

### A2B-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of capstone argument modeling and public communication but struggles to apply the work in a bounded new context

### A2B-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
