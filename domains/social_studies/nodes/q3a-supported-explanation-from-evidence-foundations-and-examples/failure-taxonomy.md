# Q3A Failure Taxonomy

### Q3A-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of supported explanation from evidence but does not yet build a stable structure for the decomposition node

### Q3A-FM-02 Organization Drift

- description: learner begins the foundations and examples work but loses organization or classification discipline during the task

### Q3A-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of supported explanation from evidence but struggles to apply the work in a bounded new context

### Q3A-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
