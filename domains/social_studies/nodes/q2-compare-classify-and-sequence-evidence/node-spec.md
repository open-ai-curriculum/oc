# Q2 Node Specification

## Node Identity

- id: `Q2`
- name: `compare_classify_and_sequence_evidence`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `inquiry-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner compares observable evidence, groups items or ideas by usable attributes, and orders events or evidence in sequence well enough to support later explanation and causation work.

## Dependencies

- `Q1` observe, describe, and question

## Exit Conditions

The learner is secure on `Q2` only when all of the following are true:

1. identifies a meaningful similarity or difference
2. groups evidence using a usable rule
3. sequences evidence or events in a defensible order
4. succeeds across changed source types or contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- compare two sources, objects, events, or scenes
- explain why items belong in the same or different groups
- order events, steps, or evidence with usable logic

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
