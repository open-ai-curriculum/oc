# Q2 Intervention Playbook

## Purpose

Map `Q2` failure modes to bounded intervention responses.

## Intervention Rules

### For Q2-FM-01 Surface-Only Comparison

Primary interventions:

- model compare stems such as same because and different because
- limit the feature set to one comparison focus at a time
- revisit paired examples with teacher think-alouds

Expected effect:

- learner identifies a meaningful comparison dimension

Reverification target:

- one direct comparison item

### For Q2-FM-02 Unstable Classification Rule

Primary interventions:

- provide candidate rules and ask the learner to test them
- sort with visible labels and then fade supports
- ask the learner to justify why one item does not belong

Expected effect:

- learner maintains a consistent grouping rule

Reverification target:

- one classify item

### For Q2-FM-03 Sequence Logic Drift

Primary interventions:

- use simple before-after-next language with visual supports
- reconstruct the sequence from concrete evidence
- compare correct and incorrect orderings and explain why

Expected effect:

- learner sequences with a clearer temporal or logical basis

Reverification target:

- one sequence item

### For Q2-FM-04 Explanation Or Language Compression Risk

Primary interventions:

- allow pointing plus sentence frames
- reduce expressive load while preserving the same reasoning demand
- verify the rule through sorting behavior before concluding the concept is absent

Expected effect:

- assessment captures comparison or classification reasoning more accurately

Reverification target:

- one re-administered meaning item with adjusted supports
