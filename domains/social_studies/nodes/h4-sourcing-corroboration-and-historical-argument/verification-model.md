# H4 Verification Model

## Gate Set

### G1. Direct Performance Gate

- purpose: source evidence corroborate accounts and build a historical argument
- pass rule: at least 85% accuracy across node-aligned tasks

### G2. Meaning Gate

- purpose: explain how sourcing and corroboration affect the argument
- pass rule: usable explanation in at least 2 of 3 opportunities

### G3. Transfer Gate

- purpose: apply the same historical-argument logic to a changed source set
- pass rule: one successful transfer task

### G4. Retention Gate

- purpose: confirm persistence after spacing
- pass rule: at least 80% delayed recheck accuracy after about 14 days
