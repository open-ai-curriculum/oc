# Q6D Failure Taxonomy

### Q6D-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of synthesis argument and public communication but does not yet build a stable structure for the decomposition node

### Q6D-FM-02 Organization Drift

- description: learner begins the transfer and contextual use work but loses organization or classification discipline during the task

### Q6D-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of synthesis argument and public communication but struggles to apply the work in a bounded new context

### Q6D-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
