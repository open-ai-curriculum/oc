# Q6D Node Package

## Node

`Q6D` - Synthesis Argument And Public Communication Transfer And Contextual Use

## Purpose

This package defines a decomposition node extending `Q6` for the social studies domain.

It operationalizes the learner's ability to transfer and adapt synthesis argument and public communication.
