# Q1 Node Specification

## Node Identity

- id: `Q1`
- name: `observe_describe_and_question`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `inquiry-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices relevant features in a phenomenon, image, map, artifact, or social situation, describes what is present with usable precision, and asks questions that can guide later inquiry.

## Dependencies

None

## Exit Conditions

The learner is secure on `Q1` only when all of the following are true:

1. notices relevant visible, audible, or otherwise accessible features
2. describes what is present without immediate fabrication or drift
3. asks at least one question that is connected to the observed situation
4. succeeds across changed sources or contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- identify salient details in a source or situation
- distinguish observation from unsupported inference at an entry level
- describe what is present using accessible language or supported response modes
- ask a question that could guide next investigation

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
