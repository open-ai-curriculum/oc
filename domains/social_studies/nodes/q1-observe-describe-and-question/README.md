# Q1 Node Package

## Node

`Q1` - Observe, describe, and question

## Purpose

This package defines the entry-point inquiry node for the social studies domain.

It operationalizes the learner's ability to notice relevant features, describe what is present, and ask usable questions before later comparison, explanation, investigation, or disciplinary analysis begins.
