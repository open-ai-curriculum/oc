# G5C Failure Taxonomy

### G5C-FM-01 Surface Recognition Without Structure

- description: learner notices pieces of global interdependence migration and sustainability but does not yet build a stable structure for the decomposition node

### G5C-FM-02 Organization Drift

- description: learner begins the explanation and application work but loses organization or classification discipline during the task

### G5C-FM-03 Application Breakdown In Context

- description: learner can perform guided parts of global interdependence migration and sustainability but struggles to apply the work in a bounded new context

### G5C-FM-04 Vocabulary Or Task-Load Risk

- description: learner performance may be limited by vocabulary task structure or representation load rather than by the target reasoning alone
