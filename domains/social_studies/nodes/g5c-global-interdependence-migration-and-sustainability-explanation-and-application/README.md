# G5C Node Package

## Node

`G5C` - Global Interdependence Migration And Sustainability Explanation And Application

## Purpose

This package defines a decomposition node extending `G5` for the social studies domain.

It operationalizes the learner's ability to explain and apply global interdependence migration and sustainability.
