# E5 Node Package

## Node

`E5` - Macroeconomic patterns, global trade, and finance basics

## Purpose

This package defines the next economics node for the social studies domain.

It operationalizes the learner's ability to explain labor roles, market relationships, and basic public resource decisions with attention to tradeoffs and incentives.
