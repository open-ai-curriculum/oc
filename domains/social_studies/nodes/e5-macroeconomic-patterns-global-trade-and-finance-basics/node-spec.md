# E5 Node Specification

## Node Identity

- id: `E5`
- name: `macroeconomic_patterns_global_trade_and_finance_basics`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `economics-macroeconomics-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets macroeconomic patterns, explains global trade relationships, and uses finance basics to reason about bounded economic scenarios.

## Dependencies

- `E4` markets, labor, and public resource decisions
- `G4` geographic reasoning with maps, data, and scale
- `Q5` evaluation of sources, models, and claims

## Exit Conditions

The learner is secure on `E5` only when all of the following are true:

1. interprets a macroeconomic pattern or indicator in a bounded context
2. explains reciprocal effects or constraints in a trade relationship
3. uses finance basics to support an economic judgment rather than just naming details
4. succeeds in changed macroeconomic or trade contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`
