# Q4 Node Specification

## Node Identity

- id: `Q4`
- name: `investigation_design_representation_and_data_use`
- domain: `social_studies`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `social-studies-investigation-node-package`
- evidence class: `Synthetic`

## Purpose

The learner frames a basic investigation question, selects or interprets an appropriate representation, and uses simple data or organized evidence to answer that question.

## Dependencies

- `Q3` supported explanation from evidence

## Exit Conditions

The learner is secure on `Q4` only when all of the following are true:

1. identifies or frames a usable investigation question
2. selects or interprets a relevant representation or evidence display
3. uses the representation or data to answer the question
4. succeeds across changed evidence or representation types
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
