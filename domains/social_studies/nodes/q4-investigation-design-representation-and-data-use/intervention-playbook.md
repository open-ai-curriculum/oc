# Q4 Intervention Playbook

## Purpose

Map `Q4` failure modes to bounded intervention responses.

## Intervention Rules

### For Q4-FM-01 Question-Representation Mismatch

Primary interventions:

- match several questions to several evidence displays
- identify what kind of information each representation is best for
- contrast a relevant and irrelevant display for the same question

### For Q4-FM-02 Data Readout Without Use

Primary interventions:

- ask what the detail means for the question after each observation
- use sentence frames linking data to conclusion
- reduce the task to one data point and one inference at a time

### For Q4-FM-03 Investigation Scope Drift

Primary interventions:

- narrow the question to what the evidence can actually answer
- separate answerable and unanswerable questions with the same dataset
- constrain conclusions to the available evidence window

### For Q4-FM-04 Representation Or Language Access Risk

Primary interventions:

- simplify the display and preteach key terms
- allow annotation plus oral explanation
- verify whether access supports materially change performance
