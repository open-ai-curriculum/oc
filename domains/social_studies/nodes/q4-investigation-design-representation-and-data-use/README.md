# Q4 Node Package

## Node

`Q4` - Investigation design, representation, and data use

## Purpose

This package defines the next inquiry node for the social studies domain.

It operationalizes the learner's ability to frame a basic investigation, choose or interpret a representation, and use simple data or evidence displays to answer a question.
