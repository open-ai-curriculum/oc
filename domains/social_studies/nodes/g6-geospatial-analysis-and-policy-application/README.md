# G6 Node Package

## Node

`G6` - Geospatial analysis and policy application

## Purpose

This package defines the capstone geography node for the social studies domain.

It operationalizes the learner's ability to perform geospatial analysis and apply geographic evidence to bounded planning or policy problems.
