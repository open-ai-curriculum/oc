# A1 Node Package

## Node

`A1` - Interdisciplinary research and evidence synthesis

## Purpose

This package defines the first advanced interdisciplinary synthesis node for the social studies domain.

It operationalizes the learner's ability to assemble a bounded interdisciplinary research problem, synthesize evidence across history civics geography and economics, and produce a defensible cross-domain account.
