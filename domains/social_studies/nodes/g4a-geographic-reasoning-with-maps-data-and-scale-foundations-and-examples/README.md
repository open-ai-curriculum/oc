# G4A Node Package

## Node

`G4A` - Geographic Reasoning With Maps Data And Scale Foundations And Examples

## Purpose

This package defines a decomposition node extending `G4` for the social studies domain.

It operationalizes the learner's ability to build foundational recognition and guided examples within geographic reasoning with maps data and scale.
