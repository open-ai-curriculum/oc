# C3D Node Package

## Node

`C3D` - Civic Decision Making And Deliberation Transfer And Contextual Use

## Purpose

This package defines a decomposition node extending `C3` for the social studies domain.

It operationalizes the learner's ability to transfer and adapt civic decision making and deliberation.
