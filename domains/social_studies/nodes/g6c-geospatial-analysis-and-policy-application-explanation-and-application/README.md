# G6C Node Package

## Node

`G6C` - Geospatial Analysis And Policy Application Explanation And Application

## Purpose

This package defines a decomposition node extending `G6` for the social studies domain.

It operationalizes the learner's ability to explain and apply geospatial analysis and policy application.
