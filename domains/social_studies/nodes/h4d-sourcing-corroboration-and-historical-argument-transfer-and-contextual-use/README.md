# H4D Node Package

## Node

`H4D` - Sourcing Corroboration And Historical Argument Transfer And Contextual Use

## Purpose

This package defines a decomposition node extending `H4` for the social studies domain.

It operationalizes the learner's ability to transfer and adapt sourcing corroboration and historical argument.
