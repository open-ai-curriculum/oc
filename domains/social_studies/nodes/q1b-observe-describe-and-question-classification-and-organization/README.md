# Q1B Node Package

## Node

`Q1B` - Observe Describe And Question Classification And Organization

## Purpose

This package defines a decomposition node extending `Q1` for the social studies domain.

It operationalizes the learner's ability to organize compare and classify elements of observe describe and question.
