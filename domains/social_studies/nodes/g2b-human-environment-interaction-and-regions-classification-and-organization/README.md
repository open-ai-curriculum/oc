# G2B Node Package

## Node

`G2B` - Human Environment Interaction And Regions Classification And Organization

## Purpose

This package defines a decomposition node extending `G2` for the social studies domain.

It operationalizes the learner's ability to organize compare and classify elements of human environment interaction and regions.
