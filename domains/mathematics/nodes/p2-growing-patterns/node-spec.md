# P2 Node Specification

## Node Identity

- id: `P2`
- name: `growing_patterns`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices how a pattern changes by a consistent growth rule and extends it accurately.

## Dependencies

- `P1`

## Exit Conditions

The learner is secure on `P2` only when all of the following are true:

1. identifies how a growing pattern changes from one step to the next
2. extends a simple growing pattern accurately
3. explains the growth rule in concrete or verbal terms
4. succeeds across objects, drawings, and simple number-linked forms
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- say that a pattern adds one more, two more, or another fixed increment each step
- extend a staircase, tower, or number pattern by the same growth rule
- distinguish a growing pattern from a repeating pattern
- explain what changes and what stays consistent in the growth

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
