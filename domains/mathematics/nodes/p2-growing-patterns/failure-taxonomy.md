# P2 Failure Taxonomy

### P2-FM-01 Growth-Rule Misidentification

- description: learner does not identify the correct amount of change between steps

### P2-FM-02 Extension Drift

- description: learner initially identifies the rule but extends the pattern inaccurately

### P2-FM-03 Repeating-vs-Growing Confusion

- description: learner treats growing patterns as if they simply repeat

### P2-FM-04 Change-Language Weakness

- description: learner cannot explain how the pattern changes each step
