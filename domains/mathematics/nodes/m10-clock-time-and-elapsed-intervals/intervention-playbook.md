# M10 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `clock-hand role rehearsal`: Use explicit contrasting tasks that isolate hour and minute information.
- `timeline jumps`: Represent elapsed time as jumps on an interval line rather than as static labels.
- `schedule reconstruction tasks`: Rebuild daily sequences from scrambled time and duration cards.
