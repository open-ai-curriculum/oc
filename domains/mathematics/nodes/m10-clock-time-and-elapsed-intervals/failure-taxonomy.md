# M10 Failure Taxonomy

## Classified Failure Modes

- `hour_minute_role_confusion`: Misreads the roles of hour and minute indicators or digit positions.
- `clock_label_interval_blend`: Treats a time label as if it already expresses elapsed duration.
- `count_on_time_breakdown`: Can read a time but cannot track forward or backward intervals.
- `schedule_order_drift`: Loses event sequence when comparing multiple start times and durations.
