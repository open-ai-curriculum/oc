# Failure Taxonomy

- `additive_comparison_substitution`: Treats a ratio as only a difference between quantities rather than a multiplicative relationship.
- `part_part_part_whole_collapse`: Confuses part-to-part ratios with part-to-whole descriptions.
- `representation_disconnect`: Cannot coordinate word, table, diagram, and symbolic ratio forms as the same relationship.
- `order_reversal`: Reverses ratio order and changes the relationship being described.
