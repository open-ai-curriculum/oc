# Q1 Node Specification

## Node Identity

- id: `Q1`
- name: `ratio_as_multiplicative_comparison`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets a ratio as a multiplicative comparison between quantities and distinguishes ratio structure from additive comparison.

## Dependencies

- `F6`
- `T4`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets ratio language as a multiplicative relationship between two quantities
2. represents simple ratios in words, diagrams, tables, and symbolic ratio forms
3. distinguishes multiplicative comparison from additive difference in bounded cases
4. succeeds across part-to-part and part-to-whole ratio contexts without collapsing them into the same claim
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- explain statements such as 2 to 3 or 3 for every 5 as coordinated multiplicative relationships
- match verbal, pictorial, tabular, and symbolic ratio representations in bounded cases
- reject additive interpretations when a task is ratio-based
- compare simple ratio situations using multiplicative rather than difference-based reasoning


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
