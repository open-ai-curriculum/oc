# Intervention Playbook

## `additive_comparison_substitution`

- compare additive and multiplicative comparison task pairs
- teacher think-aloud on for every language
- same numbers different structure sorting tasks

## `part_part_part_whole_collapse`

- use color-coded part-to-part versus part-to-whole diagrams
- match ratio statements to correct visual models
- explicit contrast routines for whole inclusion versus comparison only

## `representation_disconnect`

- translate one ratio across diagram, table, words, and symbolic form
- ratio card sorts across representation types
- teacher modeling of invariant relationship across formats

## `order_reversal`

- ordered-pair ratio reading rehearsal
- swap-order non-example tasks
- label first quantity and second quantity before writing the ratio
