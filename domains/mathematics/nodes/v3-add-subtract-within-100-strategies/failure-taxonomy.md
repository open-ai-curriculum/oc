# V3 Failure Taxonomy

### V3-FM-01 Strategy Selection Weakness

- description: learner cannot choose a viable place-value strategy

### V3-FM-02 Tens-Ones Decomposition Error

- description: learner breaks or recombines tens and ones incorrectly

### V3-FM-03 Operation-Meaning Drift

- description: learner loses whether the task is adding or subtracting while using a strategy

### V3-FM-04 Explanation Weakness

- description: learner can produce an answer without explaining the place-value strategy
