# V3 Node Specification

## Node Identity

- id: `V3`
- name: `add_subtract_within_100_strategies`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses place-value-based strategies to add and subtract within 100 with understanding.

## Dependencies

- `V1`
- `N4`

## Exit Conditions

The learner is secure on `V3` only when all of the following are true:

1. solves within-100 addition and subtraction problems using place-value-aware strategies
2. explains a strategy such as decomposing, counting on by tens/ones, or compensation
3. connects the strategy to tens-and-ones structure
4. succeeds across symbolic, story, and model forms
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- add and subtract within 100 using tens-and-ones strategies
- explain how decomposing or composing tens helps
- connect mental, drawn, and equation strategies
- maintain accuracy in mixed addition/subtraction within 100

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
