# X3 Node Specification

## Node Identity

- id: `X3`
- name: `compare_and_order_decimals`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner compares and orders decimals using place value, benchmarks, and magnitude reasoning rather than whole-number heuristics.

## Dependencies

- `X2`
- `L3`

## Exit Conditions

The learner is secure on `X3` only when all of the following are true:

1. compares and orders decimals reliably
2. explains comparisons through place value and magnitude, not superficial digit reading
3. uses benchmark reasoning in direct and transfer tasks
4. succeeds across notation, charts, and bounded number-line contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- compares decimals correctly using aligned place-value reasoning
- orders short decimal sets by magnitude
- uses benchmarks such as zero, one half, and one in bounded cases
- justifies comparisons with place or magnitude language

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
