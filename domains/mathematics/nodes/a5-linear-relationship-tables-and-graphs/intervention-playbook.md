# Intervention Playbook

## `table_graph_disconnect`

- translate one relationship through rule, table, ordered pairs, and graph in a fixed sequence
- connect each table row explicitly to one plotted point
- use readback routines from graph to table and back again

## `rate_structure_shallow_reading`

- annotate constant change or constant factor between successive rows
- compare two relationships with different rates using matched tables and graphs
- teacher think-alouds naming what stays invariant across the relationship

## `proportional_linear_misclassification`

- sort examples into proportional, other linear, and non-linear/non-matching cases
- contrast through-origin and non-through-origin graphs in bounded settings
- link proportional tables to multiplicative invariance before comparing with general linear change

## `axis_variable_role_confusion`

- name the x-quantity and y-quantity before constructing the table or graph
- use context labels on axes for every task
- reverse tasks that ask which variable changed and where that appears in the graph
