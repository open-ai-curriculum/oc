# Failure Taxonomy

- `table_graph_disconnect`: Cannot preserve the same relationship when moving between a rule, table, ordered pairs, and graph.
- `rate_structure_shallow_reading`: Notices point placement or row changes without understanding the constant rate structure underneath.
- `proportional_linear_misclassification`: Confuses proportional relationships, general linear relationships, and unrelated additive patterns.
- `axis_variable_role_confusion`: Loses which quantity belongs on which axis or what each coordinate component represents.
