# A5 Node Specification

## Node Identity

- id: `A5`
- name: `linear_relationship_tables_and_graphs`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets linear relationships through tables, ordered pairs, and graphs, coordinating constant rate structure with coordinate-plane representation rather than treating graphs as disconnected pictures.

## Dependencies

- `A1`
- `A4`
- `Q3`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets a linear relationship as a consistent rate of change across a bounded table, graph, or rule
2. generates and coordinates ordered pairs from simple linear rules or proportional relationships
3. plots and reads linear patterns on a coordinate plane with coherent axis meaning
4. succeeds across tables, verbal rules, graphs, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify constant change or constant multiplicative rate in bounded linear or proportional tables
- generate ordered pairs from simple rules and plot them accurately
- explain how a table, a set of points, and a verbal rule can represent the same relationship
- interpret what slope-like change means in bounded contexts without collapsing all patterns into additive sameness


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
