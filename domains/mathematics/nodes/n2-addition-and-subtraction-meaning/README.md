# N2 Node Package

## Node

`N2` - Addition and subtraction meaning

## Purpose

This package defines the learner's ability to interpret addition and subtraction as meaningful actions on quantities rather than as disconnected procedures.

It exists to operationalize joining, separating, adding to, taking from, and comparing situations that later support counting strategies, fluency, and equation meaning.
