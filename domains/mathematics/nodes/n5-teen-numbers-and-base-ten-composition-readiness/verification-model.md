# N5 Verification Model

## Verification Intent

`N5` verifies whether the learner treats teen numbers as structured quantities built from one ten and some ones.

## Gate Set

### G1 Direct Performance

- target: accurate teen-number composition and decomposition
- threshold: `>= 0.90`

### G2 Meaning

- target: explanation of the one-ten-plus-ones structure
- threshold: successful explanation in at least `2 of 3` prompts

### G3 Transfer

- target: success across changed representations
- threshold: at least `1` successful transfer task

### G4 Retention

- target: durable teen-number structure after spacing
- threshold: `>= 0.85` on delayed recheck after about `14` days

## Verification Notes

- correctness without bundled-unit reasoning is insufficient
- tasks should distinguish `14` as `10 + 4` from superficial digit naming
- at least one item should test moving from number word to grouped representation
