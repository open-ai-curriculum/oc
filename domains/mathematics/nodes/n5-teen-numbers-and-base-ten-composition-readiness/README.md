# N5 Node Package

## Node

`N5` - Teen numbers and base-ten composition readiness

## Purpose

This package defines the learner's ability to interpret teen numbers as a ten and some more rather than as unrelated count words.

It exists to operationalize the conceptual bridge from small-number composition into early base-ten structure.
