# N5 Learner State Model

Tracked state for `N5` includes:

- `status`
- `direct_accuracy`
- `meaning_status`
- `transfer_status`
- `retention_status`
- `dominant_failure_modes`
- `recommended_interventions`

The learner is not `secure` until teen-number structure is accurate, explainable, transferable, and stable after delay.
