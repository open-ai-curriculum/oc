# N5 Failure Taxonomy

### N5-FM-01 Unbundled Teen Interpretation

- description: learner treats a teen number as separate digits or count words rather than as a ten and ones

### N5-FM-02 Number Word-Symbol Drift

- description: learner cannot coordinate teen number words, numerals, and grouped representations

### N5-FM-03 Ten-Unit Instability

- description: learner does not preserve the ten as a single bundled unit during representation or explanation

### N5-FM-04 Comparison Weakness

- description: learner can build one teen number but cannot compare teen numbers through ten-plus-ones logic
