# D3 Node Specification

## Node Identity

- id: `D3`
- name: `read_simple_picture_graphs_and_tallies`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner reads simple data displays such as picture graphs and tally charts to answer basic questions.

## Dependencies

- `D2`

## Exit Conditions

The learner is secure on `D3` only when all of the following are true:

1. reads counts correctly from simple picture graphs or tallies
2. answers basic compare questions using the display
3. explains how the display represents category totals
4. succeeds across picture graphs, tally marks, and small category tables
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- read how many belong in a category from a simple display
- identify the most, least, equal, or difference in basic terms
- connect a display row or column to the underlying category
- avoid treating the display as decorative rather than quantitative

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
