# D3 Failure Taxonomy

### D3-FM-01 Display Count Reading Error

- description: learner misreads how many are shown in a category display

### D3-FM-02 Label-to-Quantity Disconnect

- description: learner does not connect the displayed marks or pictures to the correct category

### D3-FM-03 Representation Meaning Weakness

- description: learner uses the display superficially without understanding how it encodes totals

### D3-FM-04 Question-to-Display Mismatch

- description: learner cannot map a comparison or count question onto the information shown in the display
