# D3 Intervention Playbook

### D3-FM-01 Display Count Reading Error

- count each display row or column carefully with pointing or marking support
- separate one display type at a time before mixing pictures and tallies

### D3-FM-02 Label-to-Quantity Disconnect

- trace from the label to the displayed marks before answering
- cover unrelated categories while reading one category

### D3-FM-03 Representation Meaning Weakness

- build a display from a known data set before reading it
- ask what each mark or picture stands for in the display

### D3-FM-04 Question-to-Display Mismatch

- highlight the question word such as most, least, or how many before reading
- rehearse mapping common question types to display-reading actions
