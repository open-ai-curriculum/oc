# P3 Intervention Playbook

### P3-FM-01 Repeating-Missing Inference Failure

- rebuild the full repeating unit before asking for the missing element
- cover and reveal one position at a time within a stable repeating cycle

### P3-FM-02 Growing-Missing Inference Failure

- compare the change before and after the missing step
- use growth tables or step drawings to make the increment visible

### P3-FM-03 Local-Cue Guessing

- require the learner to check the inferred answer against the full pattern
- contrast a locally plausible wrong answer with the rule-consistent answer

### P3-FM-04 Justification Weakness

- use sentence frames such as "it must be __ because the rule is __"
- require explanation before confirmation of correctness
