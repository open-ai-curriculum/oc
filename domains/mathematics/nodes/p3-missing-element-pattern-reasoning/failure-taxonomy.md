# P3 Failure Taxonomy

### P3-FM-01 Repeating-Missing Inference Failure

- description: learner cannot infer the missing item in a repeating pattern

### P3-FM-02 Growing-Missing Inference Failure

- description: learner cannot infer the missing step in a growing pattern

### P3-FM-03 Local-Cue Guessing

- description: learner relies on nearby visible items instead of the full rule

### P3-FM-04 Justification Weakness

- description: learner may guess correctly but cannot explain why the answer fits the pattern
