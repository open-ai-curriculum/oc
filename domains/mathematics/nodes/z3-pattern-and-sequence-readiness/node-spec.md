# Z3 Node Specification

## Node Identity

- id: `Z3`
- name: `pattern_and_sequence_readiness`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices what repeats in a pattern and what comes next in a simple ordered sequence.

## Dependencies

- `Z2`

## Exit Conditions

The learner is secure on `Z3` only when all of the following are true:

1. identifies repeating patterns
2. continues a simple repeating or ordered sequence
3. explains what part repeats or what rule determines the next element
4. succeeds across objects, pictures, and simple symbol-like placeholders
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- identify the repeating core of a simple pattern
- continue a sequence correctly
- explain what comes next and why
- support later counting, patterning, and structure noticing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
