# Z3 Teacher Observability View

Teachers should see:

- current `Z3` status
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner identifies the repeating core and can extend the sequence
- recommended intervention

Class-level view should highlight:

- who guesses the next element without using structure
- who can identify a pattern but not continue it
- who loses pattern reasoning across different materials
