# Z3 Failure Taxonomy

### Z3-FM-01 Repeating-Core Instability

- description: learner does not preserve the repeating unit of the pattern

### Z3-FM-02 Next-Element Guessing

- description: learner guesses the next element without grounding it in the visible structure

### Z3-FM-03 Sequence Extension Drift

- description: learner identifies the pattern but cannot continue it reliably

### Z3-FM-04 Context Boundary

- description: learner reasons about patterns in one medium but not across other materials or sensory formats
