# Z3 Intervention Playbook

## Purpose

Map `Z3` failure modes to bounded intervention responses.

## Intervention Rules

### For Z3-FM-01 Repeating-Core Instability

Primary interventions:

- isolate the smallest repeating chunk
- mark the repeated core visually
- compare complete and broken patterns side by side

Expected effect:

- learner stabilizes the repeating unit

Reverification target:

- one direct pattern item

### For Z3-FM-02 Next-Element Guessing

Primary interventions:

- require the learner to say the pattern before naming the next element
- cover the last item and ask what repeats
- use slow step-by-step continuation routines

Expected effect:

- learner bases the next element on visible structure instead of guessing

Reverification target:

- one direct item and one explanation item

### For Z3-FM-03 Sequence Extension Drift

Primary interventions:

- continue only one step at a time
- use manipulatives to build the next position
- restart from the repeating core rather than the whole line

Expected effect:

- learner extends the pattern reliably

Reverification target:

- one direct continuation item

### For Z3-FM-04 Context Boundary

Primary interventions:

- move between object, picture, and sound patterns with the same language
- ask what repeats in each medium
- use paired examples across formats

Expected effect:

- learner transfers pattern and sequence reasoning across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
