# F6 Node Specification

## Node Identity

- id: `F6`
- name: `fraction_magnitude_beyond_one_whole`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner understands improper fractions and mixed numbers as legitimate magnitudes greater than one whole and can move among those representations coherently.

## Dependencies

- `F3`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets improper fractions as quantities that may exceed one whole
2. connects mixed numbers and improper fractions in bounded equivalent cases
3. compares magnitudes greater than one whole using benchmark and composition reasoning
4. succeeds across visual models, symbolic forms, and early number-line situations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- explain why 5/4 represents one whole and one fourth more
- match bounded mixed numbers and improper fractions that describe the same quantity
- compare simple magnitudes greater than one whole without reverting to whole-number-only logic
- use whole-and-part composition language for fraction quantities beyond one


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
