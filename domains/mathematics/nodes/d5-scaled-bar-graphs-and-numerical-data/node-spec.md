# D5 Node Specification

## Node Identity

- id: `D5`
- name: `scaled_bar_graphs_and_numerical_data`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner reads, represents, and compares numerical data in scaled bar graphs using additive reasoning and scale awareness.

## Dependencies

- `D4`
- `V2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. reads category values from a scaled bar graph using the stated scale correctly
2. represents a small numerical data set in a scaled bar graph with labels and consistent scale
3. answers compare and total questions using the data display rather than guesswork
4. succeeds across read, build, and interpret tasks with different scales
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- match bar lengths to numerical values using the graph scale
- construct a scaled bar graph from a provided table or tally set
- solve additive comparison questions based on displayed data
- explain why an inaccurate scale or label choice distorts the graph


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
