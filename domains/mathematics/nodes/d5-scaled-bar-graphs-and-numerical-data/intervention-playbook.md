# Intervention Playbook

## `scale_ignorance`

- explicit scale-reading routines before any question answering
- same graph with different scales comparison set
- teacher modeling of count-by-scale interpretation

## `label_axis_confusion`

- guided graph annotation and question unpacking
- category-to-bar matching exercises
- axis label check protocol before solving

## `comparison_computation_error`

- separate value extraction from computation steps
- bar-graph compare/total sentence frames
- targeted additive reasoning rehearsal using graph values

## `representation_distortion`

- graph construction with pre-marked intervals
- scale consistency checks during drawing
- non-example review of distorted bar graphs
