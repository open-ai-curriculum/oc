# Failure Taxonomy

- `scale_ignorance`: Reads the visual bar length without using the numerical scale.
- `label_axis_confusion`: Misreads categories, axes, or graph labels and therefore answers the wrong question.
- `comparison_computation_error`: Reads the values correctly but fails on total or difference calculations.
- `representation_distortion`: Builds a graph with inconsistent intervals, bars, or labels that distort the data.
