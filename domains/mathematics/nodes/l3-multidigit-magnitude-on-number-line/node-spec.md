# L3 Node Specification

## Node Identity

- id: `L3`
- name: `multidigit_magnitude_on_number_line`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner places, compares, and estimates multidigit whole numbers on bounded number lines using place-value magnitude rather than only digit reading.

## Dependencies

- `V2`
- `L2`

## Exit Conditions

The learner is secure on `L3` only when all of the following are true:

1. locates multidigit numbers plausibly on bounded number lines
2. uses benchmark reasoning to compare and order multidigit magnitudes
3. connects place-value structure to spacing and estimation on a line
4. succeeds across tens, hundreds, and bounded interval tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- places multidigit numbers within bounded intervals on a line
- uses benchmark anchors such as tens or hundreds to estimate position
- compares multidigit magnitudes through relative location on a line
- explains why closer position reflects smaller difference

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
