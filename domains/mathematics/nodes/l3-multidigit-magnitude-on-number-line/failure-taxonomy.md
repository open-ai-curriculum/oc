# L3 Failure Taxonomy

## Classified Failure Modes

- `digit_reading_without_scale`: Reads the numeral but cannot place it relative to interval benchmarks.
- `benchmark_anchor_loss`: Fails to use available endpoints or midpoint anchors for estimation.
- `proportional_spacing_breakdown`: Places numbers in order but with implausible spacing relative to size differences.
- `boundary_interval_confusion`: Misreads the interval represented by the line endpoints.
