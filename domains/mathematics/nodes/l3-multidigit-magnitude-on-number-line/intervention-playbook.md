# L3 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `anchor-first placement`: Require endpoint and midpoint identification before placing additional numbers.
- `compare-estimate-check cycles`: Estimate, justify, then verify placement using benchmark partitions.
- `same-line spacing critiques`: Compare plausible and implausible placements on one bounded line.
