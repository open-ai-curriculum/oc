# A4 Node Specification

## Node Identity

- id: `A4`
- name: `coordinate_plane_and_ordered_pairs`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets and uses the coordinate plane through ordered pairs, axes, origin, direction, and signed location structure rather than treating points as unlabeled plotting procedures.

## Dependencies

- `S1`
- `G2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets ordered pairs as coordinated horizontal and vertical locations on a plane
2. plots and identifies points accurately using axes, origin, and direction conventions
3. explains quadrant and axis meaning in bounded signed-coordinate cases
4. succeeds across plotted diagrams, verbal directions, tables, and simple contextual maps
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify x- and y-axis roles and use the origin as a reference point
- plot ordered pairs and read plotted coordinates accurately
- explain how sign and direction determine location in the four-quadrant plane
- connect tabular or directional descriptions to plotted points in bounded cases


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
