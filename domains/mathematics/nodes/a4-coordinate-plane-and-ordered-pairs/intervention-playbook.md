# Intervention Playbook

## `axis_order_reversal`

- horizontal-then-vertical tracing routines from the origin
- color-code x-axis and y-axis movement separately
- plot-and-readback tasks emphasizing ordered-pair sequence

## `origin_reference_instability`

- repeated origin reset routines before plotting each new point
- use blank-grid overlays that require naming the origin and axes first
- compare point locations relative to the same starting reference

## `quadrant_sign_confusion`

- signed-location card sorts by quadrant and axis
- number-line-to-coordinate-plane bridging tasks
- teacher think-alouds linking sign to direction on each axis

## `representation_mapping_breakdown`

- match tables, verbal directions, and plotted points to the same location
- simple map-grid translation tasks
- teacher compare-and-connect routines across plotted and described coordinates
