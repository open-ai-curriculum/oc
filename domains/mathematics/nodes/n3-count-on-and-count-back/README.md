# N3 Node Package

## Node

`N3` - Count on and count back

## Purpose

This package defines the learner's ability to use the counting sequence strategically to add to or take from a known quantity without rebuilding the whole from zero.

It exists to operationalize a bridge from operation meaning into efficient counting-based strategies that later support fluency and number-line reasoning.
