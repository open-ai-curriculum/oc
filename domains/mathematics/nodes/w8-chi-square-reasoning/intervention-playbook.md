# Intervention Playbook

## `sample_space_model_fragmentation`

- build complete sample spaces with lists, tables, or trees before computing any probabilities
- compare the same event represented with two different correct models to reinforce structure
- require learners to name the set of outcomes being counted before writing a fraction or percent

## `independence_conditioning_opacity`

- compare matched problems where independence holds and where one event changes the sample space for the other
- use teacher think-alouds naming whether the second event “knows” the result of the first
- require learners to restate the condition in words before computing a conditional probability

## `overlap_double_counting`

- use Venn-diagram or table routines that mark shared outcomes explicitly
- compare addition problems with disjoint events and overlapping events side by side
- require learners to explain whether any outcomes were counted twice before finalizing the answer

## `short_run_variation_overreaction`

- simulate or compare repeated short runs and longer runs of the same random process
- discuss why experimental results vary and why large-trial behavior is more stable
- require learners to judge whether a deviation is surprising rather than assuming every mismatch is an error
