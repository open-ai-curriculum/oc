# AP1 Node Specification

## Node Identity

- id: `AP1`
- name: `ap_statistics_extension`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner integrates descriptive statistics, study design, probability, sampling distributions, and the major AP Statistics inference strands into a coherent AP Statistics-level reasoning system rather than compressing the course into one undifferentiated statistics capstone.

## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. integrates descriptive statistics, study design, probability, and multiple inferential strands to justify bounded AP Statistics-style claims
2. selects and justifies the correct inferential family for proportions, means, categorical counts, or regression slope rather than treating all inference as one routine
3. interprets chance variation, sampling distributions, interval meaning, and significance evidence as connected parts of one inferential system
4. critiques statistical arguments for design weakness, unsupported causality, inferential mismatch, or misread evidence strength
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.79`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- integrate distribution, association, design, probability, and the correct inferential routine in one bounded AP Statistics-style argument
- justify whether a scenario calls for proportion, mean, chi-square, or regression-slope inference
- explain how chance variation and study structure interact when interpreting evidence strength
- critique AP-style statistical claims for inferential mismatch, bias, causal overreach, or misused probability


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
