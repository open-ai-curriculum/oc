# D4 Intervention Playbook

### D4-FM-01 Data-to-Display Mapping Error

- map one category at a time from source data to display
- check totals against the source set after each category is recorded

### D4-FM-02 Labeling Weakness

- provide category-label frames before marks are added
- require reading the labels aloud before completing the display

### D4-FM-03 Display Construction Instability

- use pre-drawn templates before open construction
- build one row or column at a time in a fixed order

### D4-FM-04 Representation Explanation Weakness

- ask the learner to read their own display back to another person
- use sentence frames such as "this row shows..."
