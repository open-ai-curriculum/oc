# D4 Failure Taxonomy

### D4-FM-01 Data-to-Display Mapping Error

- description: learner does not place the correct totals or marks into the representation

### D4-FM-02 Labeling Weakness

- description: learner omits or misuses category labels, making the display ambiguous

### D4-FM-03 Display Construction Instability

- description: learner cannot maintain an organized structure while building the display

### D4-FM-04 Representation Explanation Weakness

- description: learner creates a display but cannot explain how it communicates the data
