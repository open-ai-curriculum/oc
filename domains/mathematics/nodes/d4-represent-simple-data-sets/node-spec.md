# D4 Node Specification

## Node Identity

- id: `D4`
- name: `represent_simple_data_sets`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner organizes a simple data set into a structured representation that can be read by others.

## Dependencies

- `D3`

## Exit Conditions

The learner is secure on `D4` only when all of the following are true:

1. records a small data set in a simple structured display
2. labels categories and places counts or marks correctly
3. explains how the representation shows the data set
4. succeeds across sort charts, tally charts, and simple picture-graph-like forms
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- create a simple chart from observed or provided data
- represent each category total correctly
- label the display so another reader can use it
- check whether the display matches the source data

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
