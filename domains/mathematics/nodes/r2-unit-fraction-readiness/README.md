# R2 Node Package

## Node

`R2` - Unit fraction readiness

## Purpose

This package defines the second direct readiness node for fraction structure.

It exists to operationalize the learner's ability to interpret one equal part out of a partitioned whole as a named unit that can later scale into non-unit fractions.
