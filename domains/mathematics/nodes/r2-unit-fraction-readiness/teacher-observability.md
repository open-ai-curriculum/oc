# R2 Teacher Observability View

Teachers should see:

- current `R2` status
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner treats unit-part size correctly
- recommended intervention

Class-level view should highlight:

- who still counts pieces without unit meaning
- who reverses unit-part size when partition count changes
- who can handle fair-sharing language but not formal unit reasoning
