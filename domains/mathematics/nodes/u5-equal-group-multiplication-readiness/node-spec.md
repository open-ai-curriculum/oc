# U5 Node Specification

## Node Identity

- id: `U5`
- name: `equal_group_multiplication_readiness`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner recognizes situations composed of equal groups and can represent the total as repeated same-size units.

## Dependencies

- `U2`

## Exit Conditions

The learner is secure on `U5` only when all of the following are true:

1. identifies when a situation consists of equal groups rather than unrelated items
2. represents a total as repeated groups of the same size
3. explains why the groups must remain equal for the structure to hold
4. succeeds across objects, drawings, stories, and simple symbolic forms
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- show `3 groups of 4` with objects or drawings
- distinguish equal-group situations from additive collections with unlike parts
- explain why `4 + 4 + 4` matches `3 groups of 4`
- compare two group structures by both number of groups and size of groups

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
