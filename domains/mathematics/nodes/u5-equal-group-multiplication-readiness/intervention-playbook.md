# U5 Intervention Playbook

### U5-FM-01 Equal-Group Misidentification

- sort examples into equal-group and not-equal-group sets with explanation
- rebuild incorrect groupings so all groups match in size

### U5-FM-02 Repeated-Unit Weakness

- use physically separated groups with one repeated unit size
- rehearse one group first, then replicate it without changing the size

### U5-FM-03 Group-Count Only Bias

- compare examples with the same number of groups but different group sizes
- require naming both number of groups and size of each group

### U5-FM-04 Expression Mapping Drift

- map grouped drawings to repeated-addition expressions in both directions
- use sentence frames such as "__ groups of __ means __ + __ + __"
