# U5 Failure Taxonomy

### U5-FM-01 Equal-Group Misidentification

- description: learner treats unequal sets as equal groups or fails to recognize valid equal groups

### U5-FM-02 Repeated-Unit Weakness

- description: learner does not preserve the repeated same-size unit across the full structure

### U5-FM-03 Group-Count Only Bias

- description: learner attends only to the number of groups and ignores group size

### U5-FM-04 Expression Mapping Drift

- description: learner cannot connect equal-group structure to repeated-addition expression forms
