# G3 Intervention Playbook

### G3-FM-01 Composition Weakness

- start with exact outline matching before open composition tasks
- build from two-piece compositions before increasing complexity

### G3-FM-02 Decomposition Weakness

- draw cut lines on wholes and ask the learner to name the parts
- compare multiple valid decompositions of the same whole

### G3-FM-03 Rotation-and-Movement Confusion

- rotate one piece while keeping the shape name constant
- use transparent overlays or tracing to show sameness under movement

### G3-FM-04 Part-Whole Explanation Weakness

- ask the learner to narrate how each piece contributes to the whole
- use sentence frames such as "these two parts make one __"
