# G3 Node Specification

## Node Identity

- id: `G3`
- name: `compose_and_decompose_shapes`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner composes larger shapes from smaller shapes and decomposes given shapes into smaller component shapes.

## Dependencies

- `G2`

## Exit Conditions

The learner is secure on `G3` only when all of the following are true:

1. combines smaller shapes to create a target larger shape
2. decomposes a given shape into smaller recognizable parts
3. explains how the parts fit together to preserve the whole
4. succeeds across physical manipulatives, drawings, and outline-based tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- build a rectangle from two squares or a larger triangle from smaller triangles
- identify more than one way to decompose a shape when appropriate
- explain how rotating or moving a part changes position but not shape identity
- connect part-whole shape structure to later partition and symmetry reasoning

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
