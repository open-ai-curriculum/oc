# M4 Failure Taxonomy

### M4-FM-01 Misalignment Error

- description: learner compares lengths without aligning objects or endpoints correctly

### M4-FM-02 Ordering Instability

- description: learner can compare one pair but loses coherence when ordering three or more quantities

### M4-FM-03 Attribute Drift

- description: learner shifts attention to a different attribute instead of the target measurable attribute

### M4-FM-04 Comparative Language Weakness

- description: learner cannot explain or consistently use the comparative language needed for the judgment
