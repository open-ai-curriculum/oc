# M4 Intervention Playbook

### M4-FM-01 Misalignment Error

- use explicit endpoint-alignment routines with physical guides
- contrast aligned and misaligned examples and ask which comparison is fair

### M4-FM-02 Ordering Instability

- build ordering with two objects first, then insert a third object into the sequence
- use ordering mats labeled shortest to longest

### M4-FM-03 Attribute Drift

- sort tasks by the target attribute before comparing
- name the attribute aloud before each comparison

### M4-FM-04 Comparative Language Weakness

- rehearse sentence frames such as "___ is longer than ___"
- pair gesture, pointing, and verbal comparison in one routine
