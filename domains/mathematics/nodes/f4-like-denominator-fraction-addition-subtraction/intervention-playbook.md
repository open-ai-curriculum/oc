# F4 Intervention Playbook

### For F4-FM-01 Adds Denominators

- return to same-size partition models
- explicitly contrast denominator as unit size versus numerator as count

### For F4-FM-02 Improper Fraction Confusion

- use fraction strips and mixed-number connection work
- represent totals greater than one visually

### For F4-FM-03 Procedure Without Quantity

- require diagram or verbal explanation alongside the computation
- compare symbolic result to a model
