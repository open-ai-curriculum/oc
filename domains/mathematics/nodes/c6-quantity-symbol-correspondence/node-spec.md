# C6 Node Specification

## Node Identity

- id: `C6`
- name: `quantity_symbol_correspondence`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner links written numerals to actual quantities so a number symbol is understood as representing a countable amount.

## Dependencies

- `C4`
- `C5`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. matches bounded quantities to the correct numerals and numerals to the correct quantities
2. uses counting or subitizing evidence to justify numeral-choice decisions
3. detects when a numeral label does not match the shown set
4. succeeds across objects, pictures, cards, and simple story contexts
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- choose the numeral that names a shown quantity
- build or select a quantity that matches a shown numeral
- reject mismatched numeral labels on bounded sets
- move between spoken number, written numeral, and counted set coherently


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
