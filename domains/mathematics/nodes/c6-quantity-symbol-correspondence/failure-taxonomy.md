# Failure Taxonomy

- `count_symbol_disconnect`: Can count a set but does not connect the final count to the matching numeral.
- `symbol_quantity_guessing`: Guesses numeral matches from visual familiarity instead of quantity evidence.
- `final_count_loss`: Counts correctly but forgets or misstates the quantity the numeral should represent.
- `label_acceptance_error`: Accepts incorrect numeral labels without checking the set size.
