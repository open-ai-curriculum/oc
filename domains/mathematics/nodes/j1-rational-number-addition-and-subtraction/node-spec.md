# J1 Node Specification

## Node Identity

- id: `J1`
- name: `rational_number_addition_and_subtraction`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner adds and subtracts rational numbers across fraction, decimal, and signed forms using magnitude, common-unit, and signed-quantity structure rather than disconnected procedures.

## Dependencies

- `F5`
- `X4`
- `S2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. adds and subtracts rational numbers accurately across bounded fraction, decimal, and signed-number forms
2. preserves magnitude and sign meaning while operating on positive and negative rational values
3. explains how common units, equivalent forms, or signed magnitude structure justify the operation
4. succeeds across symbolic, number-line, verbal, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- add and subtract fractions, decimals, and signed rational quantities using defensible structure
- convert or align representations when needed without losing quantity meaning
- explain why the computed result is reasonable using magnitude, benchmark, or direction language
- connect symbolic work to number-line movement and bounded contextual situations


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
