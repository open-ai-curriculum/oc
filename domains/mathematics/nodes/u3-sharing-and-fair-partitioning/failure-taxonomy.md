# U3 Failure Taxonomy

### U3-FM-01 Fairness Quantity Drift

- description: learner does not preserve equal amount across recipients

### U3-FM-02 Count-Only Fairness

- description: learner assumes the same number of pieces guarantees fairness even when amounts differ

### U3-FM-03 Distribution Instability

- description: learner understands fairness verbally but cannot distribute shares reliably

### U3-FM-04 Whole-And-Set Disconnect

- description: learner reasons correctly in set sharing or whole sharing, but not both
