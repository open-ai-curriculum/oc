# U3 Intervention Playbook

## Purpose

Map `U3` failure modes to bounded intervention responses.

## Intervention Rules

### For U3-FM-01 Fairness Quantity Drift

Primary interventions:

- use one-to-one distribution routines
- compare each recipient's share side by side
- ask whether anyone got more or less and how to tell

Expected effect:

- learner preserves equal amount across recipients

Reverification target:

- one direct fair-share item and one explanation item

### For U3-FM-02 Count-Only Fairness

Primary interventions:

- contrast same-count but unequal-size shares
- use food or strip models where piece size is visible
- separate "same number" from "same amount" in teacher language

Expected effect:

- learner stops using piece count as the only fairness criterion

Reverification target:

- one error-diagnosis item and one direct item

### For U3-FM-03 Distribution Instability

Primary interventions:

- distribute concrete objects one at a time
- use turn-taking sharing routines
- simplify to two recipients before increasing complexity

Expected effect:

- learner can execute fair sharing, not just describe it

Reverification target:

- one production task

### For U3-FM-04 Whole-And-Set Disconnect

Primary interventions:

- map fair-set sharing to fair-whole partitioning explicitly
- use parallel examples with objects and partitioned wholes
- ask what stays the same about fairness in both cases

Expected effect:

- learner transfers the fairness principle across sharing types

Reverification target:

- one changed-context transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
