# U3 Node Package

## Node

`U3` - Sharing and fair partitioning

## Purpose

This package defines the fairness-based sharing foundation that later equal partitioning depends on.

It exists to operationalize the learner's ability to reason that a whole or set can be shared so each recipient gets the same amount.
