# Intervention Playbook

## `representation_translation_fragmentation`

- translate one linear function through equation, table, graph, and verbal rule in a fixed sequence
- anchor every translation with a shared pair set and the same rate-of-change statement
- compare correct and drifted translations side by side

## `parameter_meaning_collapse`

- name slope and starting value explicitly in each representation before converting forms
- use teacher think-alouds that tie coefficient changes to graph steepness and table differences
- connect contextual quantities to equation parameters through repeated annotation routines

## `graph_equation_alignment_breakdown`

- pair each graph with a small value table and equation before asking for matches
- require learners to justify matches using rate of change and starting value, not visual similarity alone
- compare near-miss graphs or equations that differ by one structural feature

## `linearity_overgeneralization`

- compare linear and non-linear tables, graphs, and contexts using explicit constant-difference checks
- use bounded counterexamples where one feature looks linear but the overall structure is not
- require learners to state why a representation is linear, not only label it
