# Failure Taxonomy

- `representation_translation_fragmentation`: Can work in one representation but fails to preserve the same function while translating to another.
- `parameter_meaning_collapse`: Manipulates slope or intercept-like quantities without understanding what they mean in context or across forms.
- `graph_equation_alignment_breakdown`: Matches a graph and equation by superficial cues rather than by shared rate and starting value.
- `linearity_overgeneralization`: Treats any straight-looking pattern or repeated change claim as linear without checking function structure.
