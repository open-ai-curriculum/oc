# Verification Model

This node is verified through four gate types.

## Gates

- `G1 direct performance`: learner meets at least 0.87 accuracy on direct tasks
- `G2 meaning`: learner explains linear-function structure coherently across representations
- `G3 transfer`: learner succeeds in at least one less-familiar representation-translation context
- `G4 retention`: learner passes a delayed recheck after 21 days

## Item Families

- direct linear-function representation tasks
- explanation and justification
- error diagnosis
- transfer in a varied linear-function context
