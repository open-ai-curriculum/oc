# AP8 Node Specification

## Node Identity

- id: `AP8`
- name: `differential_equations_foundations`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets simple differential equations as models of changing quantities and connects slope fields, separable structure, and accumulation reasoning into a coherent introductory differential-equations foundation.

## Dependencies

- `AP4`
- `AP6`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets simple differential equations as statements about rate relationships rather than as isolated notational forms
2. connects slope fields, solution behavior, and initial conditions in bounded graphical and contextual settings
3. solves or interprets bounded separable differential-equation situations where AP-level methods are appropriate
4. explains how accumulation and rate reasoning work together when a differential-equation model is used
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.8`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- interpret simple differential equations and slope fields in bounded symbolic, graphical, and contextual settings
- connect an initial condition to the behavior of a specific solution
- use bounded separable-model reasoning where AP-level methods apply
- justify differential-equation conclusions with correct rate and accumulation meaning


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
