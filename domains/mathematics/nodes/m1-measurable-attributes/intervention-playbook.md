# M1 Intervention Playbook

## Purpose

Map `M1` failure modes to bounded intervention responses.

## Intervention Rules

### For M1-FM-01 Attribute Identification Drift

Primary interventions:

- ask "what are we comparing?" before asking for the result
- sort tasks by length, area, and amount
- use teacher sentence frames naming the attribute explicitly

Expected effect:

- learner identifies the relevant measurable attribute first

Reverification target:

- one direct attribute item and one explanation item

### For M1-FM-02 Comparison Without Attribute Anchor

Primary interventions:

- require comparison statements to include the attribute word
- compare the same objects on different attributes
- use visuals that isolate one attribute at a time

Expected effect:

- learner anchors comparison language to a measurable attribute

Reverification target:

- one direct comparison item

### For M1-FM-03 Single-Cue Overgeneralization

Primary interventions:

- contrast tall-versus-full or long-versus-covered-space cases
- ask which attribute matters in each case
- use pairs where one visible cue is misleading

Expected effect:

- learner stops applying one visual cue to every comparison

Reverification target:

- one error-diagnosis item and one transfer item

### For M1-FM-04 Context Boundary

Primary interventions:

- move from length to area to amount using the same comparison language
- ask what stays the same when the attribute changes
- use parallel examples across contexts

Expected effect:

- learner transfers measurable-attribute reasoning across contexts

Reverification target:

- one changed-context transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
