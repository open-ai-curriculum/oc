# M1 Failure Taxonomy

### M1-FM-01 Attribute Identification Drift

- description: learner does not identify which attribute is being compared

### M1-FM-02 Comparison Without Attribute Anchor

- description: learner gives an answer such as bigger or smaller without grounding it in a measurable attribute

### M1-FM-03 Single-Cue Overgeneralization

- description: learner uses one visible cue such as height for every comparison even when another attribute is relevant

### M1-FM-04 Context Boundary

- description: learner can compare one attribute such as length but not transfer the idea to area or amount
