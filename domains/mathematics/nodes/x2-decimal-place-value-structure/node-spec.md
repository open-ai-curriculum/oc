# X2 Node Specification

## Node Identity

- id: `X2`
- name: `decimal_place_value_structure`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets decimal notation as an extension of place-value structure beyond ones into tenths and hundredths.

## Dependencies

- `X1`
- `V2`

## Exit Conditions

The learner is secure on `X2` only when all of the following are true:

1. interprets decimal notation through place-value relationships
2. represents decimals in structured ones-tenths-hundredths forms
3. explains tenfold unit relationships across place positions
4. succeeds across charts, notation, expanded forms, and model-based tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- reads and represents decimals using ones, tenths, and bounded hundredths structure
- explains how adjacent places represent tenfold relationships across the decimal point
- translates among notation, place-value charts, and composed quantities
- compares decimal digits by place value rather than superficial digit size

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
