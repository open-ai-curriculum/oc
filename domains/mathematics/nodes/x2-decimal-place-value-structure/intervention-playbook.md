# X2 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `place-value chart decomposition`: Decompose decimals into ones, tenths, and hundredths using explicit chart structure.
- `tenfold unit ladders`: Trace unit-size shifts across adjacent places on both sides of the decimal point.
- `false-comparison critiques`: Diagnose and repair decimal comparisons driven by superficial digit size.
