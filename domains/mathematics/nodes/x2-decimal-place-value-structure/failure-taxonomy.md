# X2 Failure Taxonomy

## Classified Failure Modes

- `whole_number_place_transfer_error`: Extends whole-number comparison rules to decimals without respecting decimal places.
- `tenths_hundredths_blur`: Confuses tenths and hundredths as if they were interchangeable labels.
- `expanded_form_instability`: Cannot decompose a decimal into place-value components coherently.
- `cross_point_unit_break`: Does not preserve tenfold relationships when crossing the decimal point.
