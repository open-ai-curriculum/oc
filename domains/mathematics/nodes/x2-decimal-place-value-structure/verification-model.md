# X2 Verification Model

## Verification Gates

- `G1` direct performance gate
- `G2` meaning gate
- `G3` transfer gate
- `G4` retention gate

## Evidence Targets

- reads and represents decimals using ones, tenths, and bounded hundredths structure
- explains how adjacent places represent tenfold relationships across the decimal point
- translates among notation, place-value charts, and composed quantities
- compares decimal digits by place value rather than superficial digit size

## Minimum Verification Rule

A learner does not clear `X2` until direct accuracy, explanation, transfer, and delayed retention evidence all meet threshold.
