# E2 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `step-annotation routines`: Annotate each transformation with the operation and reason before proceeding.
- `midstream-sign audits`: Pause after each line to verify sign preservation and equivalence.
- `worked-error reconstruction`: Repair flawed multi-step solutions by identifying the first invalid transformation.
