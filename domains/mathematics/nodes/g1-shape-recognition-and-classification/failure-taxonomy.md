# G1 Failure Taxonomy

### G1-FM-01 Shape Identity Weakness

- description: learner cannot reliably identify common shapes

### G1-FM-02 Attribute Classification Weakness

- description: learner cannot sort shapes by visible geometric attributes

### G1-FM-03 Orientation Dependence

- description: learner treats rotated shapes as different shape classes

### G1-FM-04 Non-Geometric Cue Bias

- description: learner classifies shapes by color, size, or context rather than geometric structure
