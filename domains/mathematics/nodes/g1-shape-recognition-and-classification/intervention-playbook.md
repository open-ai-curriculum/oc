# G1 Intervention Playbook

### G1-FM-01 Shape Identity Weakness

- rehearse shape naming with repeated exposure across multiple examples
- pair touch-tracing with naming of sides and corners where applicable

### G1-FM-02 Attribute Classification Weakness

- sort by one explicit attribute at a time
- compare shapes that differ on only one feature

### G1-FM-03 Orientation Dependence

- rotate the same shape through multiple positions and keep the name constant
- contrast orientation changes with actual shape changes

### G1-FM-04 Non-Geometric Cue Bias

- use same-color mixed-shape sets and different-color same-shape sets
- ask "what about the shape stays the same?" before classification
