# Failure Taxonomy

- `variable_role_swap`: Reverses or confuses which quantity is independent and which is dependent.
- `paired_value_fragmentation`: Treats paired values as isolated data points rather than as coordinated values of one relationship.
- `representation_role_disconnect`: Can identify variable roles in one representation but loses them in another.
- `context_dependency_breakdown`: Can talk about the relationship symbolically or tabularly but loses it in contextual situations.
