# K1 Node Specification

## Node Identity

- id: `K1`
- name: `variable_relationships_and_independent_dependent_quantities`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets relationships between variables by distinguishing which quantity changes independently and which quantity depends on it, rather than treating paired values as unrelated numbers.

## Dependencies

- `A1`
- `A5`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. identifies and explains variable relationships in bounded tables, rules, graphs, and contexts
2. distinguishes independent and dependent quantities coherently
3. explains how changing one quantity affects another in a structured relationship
4. succeeds across symbolic, table, verbal, graph, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify which quantity is treated as the input-like or driving quantity in a bounded relationship
- identify which quantity changes in response and how it is recorded
- explain paired values as related measures of one relationship rather than as separate facts
- connect verbal relationship statements to tables, points, and simple symbolic rules


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
