# Intervention Playbook

## `variable_role_swap`

- label the driving quantity and responding quantity explicitly before solving
- use input-output tables with highlighted variable roles
- compare correct and reversed role assignments in matched situations

## `paired_value_fragmentation`

- use one-relationship card sorts where multiple ordered pairs must be recognized as belonging together
- rehearse "same relationship, different input" language
- teacher think-alouds tying each pair back to one consistent relationship

## `representation_role_disconnect`

- translate one relationship across table, graph, rule, and verbal form while preserving variable labels
- require learners to restate which variable is changing and which variable depends on it after each representation shift
- compare correct and incorrect cross-representation mappings

## `context_dependency_breakdown`

- use contexts such as cost, distance, and time with explicit "what changes first?" routines
- pair each table or graph with a plain-language explanation prompt
- teacher compare-and-connect routines across symbolic, tabular, graph, and contextual relationship situations
