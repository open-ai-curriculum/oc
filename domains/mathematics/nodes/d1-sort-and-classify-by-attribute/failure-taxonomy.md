# D1 Failure Taxonomy

### D1-FM-01 Sorting Execution Weakness

- description: learner cannot place items into correct groups consistently

### D1-FM-02 Attribute Switching Weakness

- description: learner cannot re-sort the same set by a different attribute

### D1-FM-03 Mixed-Rule Confusion

- description: learner blends multiple grouping rules without keeping the classification coherent

### D1-FM-04 Rule Explanation Weakness

- description: learner can sort items but cannot explain the category rule
