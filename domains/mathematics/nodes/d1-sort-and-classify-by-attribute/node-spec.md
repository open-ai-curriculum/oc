# D1 Node Specification

## Node Identity

- id: `D1`
- name: `sort_and_classify_by_attribute`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner sorts and classifies a set of items by a selected attribute and explains the classification rule.

## Dependencies

- `Z2`

## Exit Conditions

The learner is secure on `D1` only when all of the following are true:

1. sorts a mixed set into groups based on a named attribute
2. classifies items by more than one possible attribute when prompted
3. explains what makes each group belong together
4. succeeds across concrete objects, pictures, and simple category-label contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- sort objects by color, shape, size, or type
- re-sort the same set by a different attribute
- explain the rule for each category
- recognize when an item does not fit the stated classification rule

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
