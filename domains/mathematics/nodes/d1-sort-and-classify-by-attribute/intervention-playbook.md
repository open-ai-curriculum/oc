# D1 Intervention Playbook

### D1-FM-01 Sorting Execution Weakness

- sort with explicit category labels or mats
- compare one item at a time to a model example in each group

### D1-FM-02 Attribute Switching Weakness

- keep the set constant and name the new attribute before resorting
- contrast first-sort and second-sort rules explicitly

### D1-FM-03 Mixed-Rule Confusion

- use only one target attribute at a time
- ask the learner to state the rule before placing each item

### D1-FM-04 Rule Explanation Weakness

- use sentence frames such as "these belong together because..."
- require one spoken justification for each group after sorting
