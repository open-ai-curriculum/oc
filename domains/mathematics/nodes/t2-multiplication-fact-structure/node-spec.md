# T2 Node Specification

## Node Identity

- id: `T2`
- name: `multiplication_fact_structure`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets multiplication facts as structured equal-group or array relationships rather than isolated memorized strings.

## Dependencies

- `T1`
- `N4`

## Exit Conditions

The learner is secure on `T2` only when all of the following are true:

1. interprets and solves multiplication facts through structural reasoning
2. derives unknown facts from known facts using equal-group or array patterns
3. explains the meaning of factors and product in context
4. succeeds across verbal, visual, symbolic, and comparison tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- interprets a multiplication expression as groups of equal size
- uses known structures to derive related multiplication facts
- explains commutative pairings through array rotation or grouping logic
- connects multiplication facts to repeated addition where appropriate

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
