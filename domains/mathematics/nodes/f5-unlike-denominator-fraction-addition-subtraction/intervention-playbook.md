# F5 Intervention Playbook

## Purpose

Map `F5` failure modes to bounded intervention responses.

## Intervention Rules

### For F5-FM-01 Incorrect Common Denominator

Primary interventions:

- visual fraction-strip alignment with shared partitioning
- rewrite-first-then-compute routine
- scaffolded denominator-matching sequence

Expected effect:

- learner stabilizes common-unit selection before computation

Reverification target:

- direct execution plus one explanation item

### For F5-FM-02 Direct Addition Error

Primary interventions:

- return to same-whole partition model
- explicit contrast: unit size versus number of parts
- teacher-guided compare-then-compute discussion

Expected effect:

- learner stops combining denominators as if they were totals

Reverification target:

- one visual task, one symbolic task, one explanation task

### For F5-FM-03 Equivalence Breakdown

Primary interventions:

- revisit `F2` with scaling tables and area models
- match original and rewritten fractions before solving
- use "same value, smaller pieces" explanation routine

Expected effect:

- learner preserves value while changing unit size

Reverification target:

- equivalent-fraction rewrite plus one full `F5` item

### For F5-FM-04 Mixed Number Loss

Primary interventions:

- decompose mixed numbers into whole-plus-fraction explicitly
- use bounded regrouping routines with teacher modeling
- temporarily narrow item complexity before returning to full form

Expected effect:

- learner preserves whole-number structure during rewriting and computation

Reverification target:

- bounded mixed-number execution set

### For F5-FM-05 Magnitude Blindness

Primary interventions:

- benchmark estimation before solving
- answer-plausibility check after solving
- compare result to `1/2`, `1`, and the addends or minuend

Expected effect:

- learner checks whether the result is structurally sensible

Reverification target:

- transfer task with explicit estimate-and-check component

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
