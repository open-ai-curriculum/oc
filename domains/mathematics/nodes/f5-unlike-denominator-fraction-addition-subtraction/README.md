# F5 Node Package

## Node

`F5` - Unlike-denominator fraction addition and subtraction

## Purpose

This package is the first full operational node package in the repository's spec-driven learning-system model.

It exists to prove that a capability node can be specified tightly enough to drive:

- instruction
- assessment
- failure classification
- intervention
- learner-state tracking
- teacher-facing visibility
- agent behavior
