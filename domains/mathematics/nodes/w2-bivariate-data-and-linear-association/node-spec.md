# W2 Node Specification

## Node Identity

- id: `W2`
- name: `bivariate_data_and_linear_association`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets bivariate data through association, trend, clustering, and deviation from fit, and uses scatter-plot and linear-association reasoning rather than treating data pairs as isolated points.

## Dependencies

- `W1`
- `A5`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets scatter plots using direction, form, strength, clusters, and outliers in bounded bivariate data sets
2. distinguishes association from causation and explains what a trend can and cannot justify
3. uses informal line-of-best-fit reasoning to describe and estimate linear association in bounded cases
4. compares observed data points to a linear trend and interprets residual-like deviation in introductory form
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.84`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- describe direction, form, strength, and unusual features of bounded scatter plots
- draw or evaluate an informal line of best fit for roughly linear data
- estimate or compare predicted versus observed values in bounded contexts
- justify what association claims are supported and what causal claims are not supported by the data alone


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
