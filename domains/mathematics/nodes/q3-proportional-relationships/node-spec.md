# Q3 Node Specification

## Node Identity

- id: `Q3`
- name: `proportional_relationships`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner recognizes proportional relationships as multiplicative structures with invariant scaling and uses that structure across tables, equations, and contextual situations.

## Dependencies

- `Q2`
- `T4`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. recognizes when two quantities are in a proportional relationship and when they are not
2. extends proportional relationships consistently through tables, equivalent-ratio forms, and bounded contexts
3. explains invariant multiplicative structure rather than relying on repeated additive change
4. succeeds across ratio tables, missing-value tasks, and simple constant-of-proportionality contexts
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify whether paired quantities scale proportionally in bounded cases
- complete missing values in proportional tables using multiplicative logic
- justify why equal additive differences do not by themselves establish proportionality
- coordinate one-unit reasoning with equivalent-ratio growth across multiple representations


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
