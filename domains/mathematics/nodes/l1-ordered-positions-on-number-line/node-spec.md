# L1 Node Specification

## Node Identity

- id: `L1`
- name: `ordered_positions_on_number_line`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets a number line as an ordered set of positions with consistent direction and spacing, not merely a decorated counting strip.

## Dependencies

- `C4`
- `N3`

## Exit Conditions

The learner is secure on `L1` only when all of the following are true:

1. interprets the number line as ordered position rather than object count alone
2. locates and names bounded whole-number positions correctly
3. uses direction and spacing to compare numbers
4. succeeds across blank lines, labeled lines, and jump representations
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- places bounded whole numbers in correct order on a line
- uses left-right direction consistently to interpret less and greater
- explains that steps on the line represent equal intervals
- matches spoken, symbolic, and visual number-line positions

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
