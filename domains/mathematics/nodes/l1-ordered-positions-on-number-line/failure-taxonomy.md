# L1 Failure Taxonomy

## Classified Failure Modes

- `label_only_tracking`: Treats labels as isolated markers without preserving interval structure.
- `direction_reversal`: Reverses the left-right order of greater and lesser quantities.
- `unequal_spacing_acceptance`: Accepts inconsistent spacing as if the line still preserved magnitude.
- `count_strip_dependency`: Can use fully labeled count strips but breaks on more abstract number-line layouts.
