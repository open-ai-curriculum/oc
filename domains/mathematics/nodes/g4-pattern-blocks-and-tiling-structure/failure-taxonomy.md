# G4 Failure Taxonomy

### G4-FM-01 Shape-Fit Weakness

- description: learner cannot determine which shapes fit a region or repeated structure

### G4-FM-02 Gap-Overlap Blindness

- description: learner does not recognize that gaps or overlaps invalidate a tiling

### G4-FM-03 Single-Arrangement Lock

- description: learner can imitate one arrangement but cannot adapt to a changed region or shape set

### G4-FM-04 Tiling Explanation Weakness

- description: learner can place shapes but cannot explain why the tiling works or fails
