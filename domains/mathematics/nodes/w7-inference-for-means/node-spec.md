# W7 Node Specification

## Node Identity

- id: `W7`
- name: `inference_for_means`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses confidence intervals and significance tests for means to reason about average outcomes under uncertainty, connecting distributions, sampling variability, and study design rather than treating inference as a button-press routine.

## Dependencies

- `W5`
- `W3`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. constructs and interprets bounded confidence intervals for means in one-sample or comparative settings with correct claim scope
2. sets up and interprets bounded significance tests for means using null, alternative, statistic, and evidence logic coherently
3. explains how sampling variability, distribution shape conditions, and sample size affect the credibility of mean-based inference
4. connects interval and test results for means while preserving the study-design limits on the conclusion
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.83`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- build and interpret bounded confidence intervals for means with correct population meaning
- conduct bounded one- or two-sample mean inference and explain the resulting evidence claim
- explain how sample size and distribution conditions influence mean-based inference credibility
- critique mean-based inference that overstates what the design or assumptions support


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
