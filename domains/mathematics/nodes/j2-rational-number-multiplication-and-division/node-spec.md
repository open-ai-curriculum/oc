# J2 Node Specification

## Node Identity

- id: `J2`
- name: `rational_number_multiplication_and_division`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner multiplies and divides rational numbers across fraction, decimal, rate, scale, and signed forms using unitizing, scaling, and inverse structure rather than memorized detached rules.

## Dependencies

- `J1`
- `Q2`
- `T4`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. multiplies and divides rational numbers accurately across bounded fraction, decimal, signed, and rate-based forms
2. preserves scale, unit, and sign meaning while operating on rational quantities
3. explains how unit-rate, scaling, inverse, or fraction-of-a-fraction structure justifies the operation
4. succeeds across symbolic, table, number-line, verbal, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- multiply and divide rational quantities using scaling, unit-rate, or inverse structure
- interpret products and quotients involving fractions, decimals, and signed values coherently
- explain why the result is reasonable using unit, scale, benchmark, or sign language
- connect symbolic work to rate, scaling, grouping, and bounded contextual situations


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
