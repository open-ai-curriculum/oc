# M7 Failure Taxonomy

### M7-FM-01 Unit Identification Weakness

- description: learner cannot reliably identify the early money or value units in use

### M7-FM-02 Value Comparison Weakness

- description: learner cannot compare simple value amounts correctly

### M7-FM-03 Count-of-Objects Bias

- description: learner treats the number of coins or markers as more important than their value

### M7-FM-04 Exchange-Meaning Weakness

- description: learner cannot explain enough/not-enough or simple exchange judgments
