# H3 Node Specification

## Node Identity

- id: `H3`
- name: `proof_structure_and_geometric_argument`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner constructs and critiques bounded geometric arguments using definitions, givens, properties, and theorem-based reasoning, and understands proof as a structured justification rather than a memorized format.

## Dependencies

- `H2`
- `A2`
- `P4`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. identifies claims, givens, assumptions, and needed subclaims in bounded geometric proof situations
2. constructs multi-step geometric arguments using definitions, congruence/similarity results, angle relationships, and algebraic properties
3. justifies each step with an appropriate reason rather than unsupported visual assertion
4. critiques incomplete or invalid arguments and repairs bounded proof gaps
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.8`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify the structure of a bounded proof situation and the intermediate claims needed
- supply valid reasons for bounded geometric statements and theorem uses
- complete or critique short geometric arguments with coherent claim-evidence flow
- connect algebraic manipulation and geometric properties inside a single bounded proof


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
