# K3 Node Specification

## Node Identity

- id: `K3`
- name: `slope_rate_of_change_and_linear_growth`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets slope or rate of change as the consistent change in one quantity relative to another in linear situations, rather than as a memorized formula detached from growth structure.

## Dependencies

- `K2`
- `Q3`
- `S1`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. identifies and computes rate of change in bounded linear situations accurately
2. explains slope as consistent change in output per unit change in input
3. distinguishes positive, negative, zero, and steeper or less-steep linear growth in bounded cases
4. succeeds across symbolic, table, verbal, graph, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- compute bounded rates of change from tables, graphs, or contextual data
- explain how a constant rate corresponds to linear growth
- compare graphs or tables by direction and steepness using rate language
- connect proportional growth and signed change to slope meaning in bounded cases


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
