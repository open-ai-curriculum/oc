# Intervention Playbook

## `change_ratio_breakdown`

- annotate change in output and change in input explicitly before computing the rate
- use table-difference routines before slope notation compression
- compare correct and incorrect change-pair selections in matched tables

## `steepness_without_rate_meaning`

- compare multiple linear graphs with explicit rate values attached
- link verbal steepness descriptions to quantitative rate comparisons
- teacher think-alouds naming slope as change per one-unit input increase

## `sign_direction_confusion`

- sort graphs and contexts into increasing, decreasing, and constant categories
- compare signed rates with visual graph direction and contextual meaning
- use number-line and signed-change bridges before full graph interpretation

## `representation_growth_disconnect`

- translate one linear-growth situation across table, graph, verbal statement, and symbolic summary
- pair each graph with a table-of-changes routine
- teacher compare-and-connect routines across symbolic, tabular, graph, and contextual rate situations
