# Intervention Playbook

## `family_synthesis_fragmentation`

- compare matched modeling situations where different function families fit for different structural reasons
- require learners to state the dominant feature of the situation before selecting or composing a model
- use side-by-side analyses of two plausible models and justify which one actually matches the behavior

## `approaching_behavior_opacity`

- use near-value tables and graphs to study what a function is approaching from both sides before any symbolic simplification
- compare continuous and discontinuous examples with similar surface appearance
- require learners to explain what is happening “near” a point before stating a conclusion

## `composition_representation_breakdown`

- trace outer and inner functions separately before composing them
- compare symbolic transformations with their graphical effects in matched examples
- require learners to identify what each transformation or composition step does to the input or output

## `cross_family_feature_blindness`

- compare functions with different formulas but the same dominant feature, and functions with similar formulas but different dominant features
- use feature-ranking routines: growth, oscillation, asymptote, intercept, local change
- require learners to name which feature matters most for the question before computing
