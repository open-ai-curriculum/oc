# B5 Node Specification

## Node Identity

- id: `B5`
- name: `precalculus_function_synthesis_and_limits_foundations`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner synthesizes multiple function families, reasons about transformations and composition across representations, and develops introductory limit intuition so advanced-function work becomes a coherent precalculus system rather than a set of separate chapters.

## Dependencies

- `B3`
- `B4`
- `K2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. compares and composes multiple function families to model or interpret bounded situations using the most revealing representation
2. reasons about transformations, inverses, composition, and parameter effects across polynomial, exponential, logarithmic, rational, and trigonometric functions
3. interprets rate, growth, periodicity, and asymptotic behavior together when comparing advanced-function models
4. develops introductory limit-based reasoning about approaching behavior, continuity, and local change in bounded precalculus settings
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.8`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- choose and justify an appropriate advanced-function family or composition for a bounded modeling or interpretation task
- compare transformed, composed, or inverse functions across algebraic and graphical representations
- explain approaching behavior and continuity in bounded near-limit situations without relying on formal calculus procedures
- connect cross-family features such as growth, asymptotes, oscillation, and local behavior in one coherent analysis


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
