# F1 Intervention Playbook

### For F1-FM-01 Whole Number Separation

- use equal-part models and unit-fraction language
- rehearse "parts of size" sentence frames

### For F1-FM-02 Denominator Size Inversion

- compare same-whole partitions with different denominator values
- emphasize smaller parts as denominator increases

### For F1-FM-03 Symbolic Visual Disconnect

- match and sort symbols with diagrams
- require bidirectional mapping from symbol to picture and picture to symbol
