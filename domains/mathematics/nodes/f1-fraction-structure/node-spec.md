# F1 Node Specification

## Node Identity

- id: `F1`
- name: `fraction_structure`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `downstream-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets fractions as quantities made of equal parts and connects symbolic notation to visual models.

## Dependencies

- `R1`
- `R2`
- `R3`

## Exit Conditions

The learner is secure on `F1` only when all of the following are true:

1. identifies numerator and denominator correctly
2. explains the denominator as the unit partition and the numerator as the count of parts
3. maps diagrams to symbols and symbols to diagrams
4. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- interpret `3/4` as three parts of size one fourth
- identify a unit fraction
- compare symbolic and visual representations
- reason about equal partitioning of a whole
- preserve the distinction between selected-part count and whole-part count

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
