# F1 Failure Taxonomy

### F1-FM-01 Whole Number Separation

- description: learner treats numerator and denominator as unrelated whole numbers

### F1-FM-02 Denominator Size Inversion

- description: learner assumes a larger denominator means a larger quantity

### F1-FM-03 Symbolic Visual Disconnect

- description: learner cannot connect fraction notation to equal-part models
