# X5 Failure Taxonomy

## Classified Failure Modes

- `tick_label_overreading`: Reads decimal labels without preserving interval structure between them.
- `partition_density_confusion`: Cannot coordinate tenths and hundredths partitions on the same bounded line.
- `benchmark_anchor_loss`: Fails to use endpoints or intermediate benchmark anchors to place decimals plausibly.
- `notation_to_location_disconnect`: Can compare decimal notation but cannot map it to a line location.
