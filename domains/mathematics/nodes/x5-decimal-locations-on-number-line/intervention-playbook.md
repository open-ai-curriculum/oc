# X5 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `tenths-then-hundredths partitioning`: Partition lines coarsely into tenths before refining to hundredths.
- `anchor-first placement`: Place 0, 0.5, and 1 or other bounded anchors before adding more decimals.
- `same-decimal multiple-lines comparison`: Place the same decimal on different bounded intervals to preserve relative meaning.
