# X5 Verification Model

## Verification Gates

- `G1` direct performance gate
- `G2` meaning gate
- `G3` transfer gate
- `G4` retention gate

## Evidence Targets

- places decimals on bounded number lines using interval reasoning
- uses tenths and hundredths partitions coherently
- compares decimal magnitudes by relative position on a line
- explains why decimal spacing depends on interval size and benchmark anchors

## Minimum Verification Rule

A learner does not clear `X5` until direct accuracy, explanation, transfer, and delayed retention evidence all meet threshold.
