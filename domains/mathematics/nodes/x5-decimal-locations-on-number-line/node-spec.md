# X5 Node Specification

## Node Identity

- id: `X5`
- name: `decimal_locations_on_number_line`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner places decimals on the number line using place-value intervals and benchmark locations rather than only reading notation.

## Dependencies

- `X3`
- `L4`

## Exit Conditions

The learner is secure on `X5` only when all of the following are true:

1. locates decimals accurately on bounded number lines
2. uses decimal partitions and benchmarks to justify placements
3. connects decimal notation to linear magnitude structure
4. succeeds across tenths, hundredths, benchmark, and transfer tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- places decimals on bounded number lines using interval reasoning
- uses tenths and hundredths partitions coherently
- compares decimal magnitudes by relative position on a line
- explains why decimal spacing depends on interval size and benchmark anchors

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
