# R3 Node Package

## Node

`R3` - Symbol preparation

## Purpose

This package defines the final direct readiness node before fraction structure.

It exists to operationalize the learner's ability to map spoken and visual part-whole reasoning onto early symbolic fraction notation without yet requiring the full `F1` interpretation load.
