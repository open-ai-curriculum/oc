# R3 Intervention Playbook

## Purpose

Map `R3` failure modes to bounded intervention responses.

## Intervention Rules

### For R3-FM-01 Symbol Position Swap

Primary interventions:

- use two-column mapping routines for selected parts versus whole parts
- annotate visuals before writing symbols
- require verbal rehearsal of each quantity before placement

Expected effect:

- learner preserves the distinction between selected-part count and whole-part count

Reverification target:

- one direct mapping item and one error-diagnosis item

### For R3-FM-02 Role Collapse

Primary interventions:

- explicitly label the top and bottom positions with their meaning roles
- sort examples where the two counts are intentionally swapped
- use sentence frames that pair each count with its job

Expected effect:

- learner understands that notation positions are semantically different

Reverification target:

- one explanation item and one direct mapping item

### For R3-FM-03 Oral Symbol Disconnect

Primary interventions:

- map oral phrases to cards before writing symbols
- use teacher read-aloud and learner write-back routines
- reduce symbolic load, then rebuild from spoken part-whole language

Expected effect:

- learner bridges oral understanding into written symbolic form

Reverification target:

- one oral-to-symbol task and one transfer task

### For R3-FM-04 Symbol Without Quantity

Primary interventions:

- require visual justification for each written symbol
- connect each symbol to a concrete or diagrammed whole
- compare two symbols and ask which picture each would match

Expected effect:

- learner anchors notation in quantity meaning rather than pattern memory

Reverification target:

- one visual-symbol match and one explanation item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
