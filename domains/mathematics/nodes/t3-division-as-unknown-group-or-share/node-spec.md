# T3 Node Specification

## Node Identity

- id: `T3`
- name: `division_as_unknown_group_or_share`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets division as finding an unknown number of groups or an unknown share size within equal-group structure.

## Dependencies

- `U6`
- `T1`

## Exit Conditions

The learner is secure on `T3` only when all of the following are true:

1. interprets division as an unknown within equal-group structure
2. distinguishes equal-share from how-many-groups division contexts
3. connects division outcomes to inverse multiplication checks
4. succeeds across stories, drawings, tables, and symbolic equations
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- solves equal-share and equal-group division situations
- explains whether the unknown is the number of groups or the size of each group
- matches division situations to arrays, repeated subtraction, or inverse multiplication relationships
- checks whether a proposed quotient is structurally reasonable

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
