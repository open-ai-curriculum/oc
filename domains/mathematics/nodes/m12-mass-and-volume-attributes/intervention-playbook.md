# M12 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `attribute-sorting routines`: Sort situations by which measurement attribute is actually being considered.
- `same-size-different-mass contrasts`: Use examples where visual size does not determine mass.
- `container-comparison tasks`: Compare differently shaped containers holding the same or different amounts.
