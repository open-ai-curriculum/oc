# V1 Node Specification

## Node Identity

- id: `V1`
- name: `two_digit_number_structure`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner understands two-digit numbers as structured tens-and-ones quantities rather than as isolated digit pairs.

## Dependencies

- `N6`

## Exit Conditions

The learner is secure on `V1` only when all of the following are true:

1. represents two-digit numbers as tens and ones reliably
2. translates between spoken numbers, numerals, and grouped models
3. explains the role of tens and ones in the number structure
4. succeeds across concrete, pictorial, and symbolic contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- show a two-digit number as tens and ones in multiple forms
- connect numeral, spoken name, and grouped representation
- explain why the tens place changes the value of a digit
- treat two-digit quantities as structured amounts rather than separate digits

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
