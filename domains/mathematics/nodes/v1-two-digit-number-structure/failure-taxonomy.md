# V1 Failure Taxonomy

### V1-FM-01 Tens-Ones Structure Weakness

- description: learner does not reliably represent two-digit numbers as tens and ones

### V1-FM-02 Form-Translation Drift

- description: learner cannot coordinate spoken, symbolic, and grouped forms

### V1-FM-03 Digit-Only Interpretation

- description: learner reads digits without place-based quantity meaning

### V1-FM-04 Two-Digit Quantity Instability

- description: learner treats a two-digit number as two unrelated one-digit values
