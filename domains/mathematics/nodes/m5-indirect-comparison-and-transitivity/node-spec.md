# M5 Node Specification

## Node Identity

- id: `M5`
- name: `indirect_comparison_and_transitivity`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses indirect comparison and transitive reasoning to decide relationships when direct comparison is not available.

## Dependencies

- `M4`

## Exit Conditions

The learner is secure on `M5` only when all of the following are true:

1. infers comparison relations from shared reference objects or prior comparisons
2. applies transitivity correctly across at least three linked quantities
3. explains why the conclusion follows from the chain of comparisons
4. succeeds across concrete, drawn, and language-based comparison chains
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- infer that object A is longer than object C after comparing A to B and B to C
- use a shared unit or reference object to compare indirectly
- preserve order when objects are not all present together
- explain why the result follows from earlier comparisons

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
