# Intervention Playbook

## `pair_equation_mismatch`

- generate several ordered pairs from one equation and test them explicitly
- use table-building routines from the equation before graphing
- compare satisfying and non-satisfying pairs side by side

## `symbol_relation_disconnect`

- rewrite equation language as "for each pair that works" before symbolic manipulation
- connect verbal rules to equation forms through matched examples
- teacher think-alouds naming the equation as a relationship, not just a formula string

## `substitution_pair_breakdown`

- use ordered-pair substitution routines with variable positions highlighted
- compare correct and reversed substitutions in matched examples
- require learners to state which coordinate value goes with which variable before evaluating

## `representation_relation_disconnect`

- translate one linear equation across symbolic form, table, graph, and verbal rule while preserving the same pair structure
- pair each symbolic equation with a small satisfaction table and plotted points
- teacher compare-and-connect routines across symbolic, tabular, graph, and contextual linear-equation situations
