# K4 Node Specification

## Node Identity

- id: `K4`
- name: `linear_equations_in_two_variables`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets linear equations in two variables as rules relating pairs of values in a plane or table, rather than as isolated symbolic strings to manipulate without relational meaning.

## Dependencies

- `K2`
- `K3`
- `A2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets and generates bounded linear equations in two variables accurately
2. explains how an equation determines related pairs of values
3. connects linear equations to tables, points, and growth rules coherently
4. succeeds across symbolic, table, verbal, graph, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- generate ordered pairs that satisfy a bounded linear equation in two variables
- determine whether a given pair satisfies a linear equation
- explain how the equation, table, and graph describe the same linear relationship
- connect slope and intercept-style structure to the behavior of the related pairs in bounded cases


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
