# N1 Node Package

## Node

`N1` - Compose and decompose small numbers

## Purpose

This package defines the whole-number part-whole foundation that later fraction reasoning depends on.

It exists to operationalize the learner's ability to see a quantity as made of smaller quantities and to move flexibly between whole and part views.
