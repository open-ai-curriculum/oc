# N1 Intervention Playbook

## Purpose

Map `N1` failure modes to bounded intervention responses.

## Intervention Rules

### For N1-FM-01 Whole-Part Instability

Primary interventions:

- use part-part-whole mats with concrete objects
- require the learner to restate the whole before naming parts
- use cover-and-reveal routines to connect missing parts back to the whole

Expected effect:

- learner preserves the total while reasoning about parts

Reverification target:

- one direct composition item and one explanation item

### For N1-FM-02 Single-Decomposition Lock

Primary interventions:

- generate multiple decompositions of the same number with counters
- sort examples by "same whole, different parts"
- use teacher prompts that ask for another valid way

Expected effect:

- learner understands that a whole can be composed in more than one way

Reverification target:

- one multi-solution item

### For N1-FM-03 Counting Drift

Primary interventions:

- slow the task with one-to-one counting checks
- anchor each part with concrete objects before combining
- revisit cardinality language from `C3`

Expected effect:

- learner tracks quantity accurately through composition and decomposition

Reverification target:

- one concrete direct item and one error-diagnosis item

### For N1-FM-04 Rearrangement Misunderstanding

Primary interventions:

- physically rearrange the same objects without adding or removing any
- ask whether the total changed and why
- use fixed-total comparisons before symbolic notation

Expected effect:

- learner sees regrouping as reorganization rather than quantity change

Reverification target:

- one transfer item with explanation

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
