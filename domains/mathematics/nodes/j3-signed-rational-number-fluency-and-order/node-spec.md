# J3 Node Specification

## Node Identity

- id: `J3`
- name: `signed_rational_number_fluency_and_order`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner compares, orders, and operates fluently with signed rational numbers across fraction, decimal, and number-line forms using magnitude and direction structure rather than isolated sign habits.

## Dependencies

- `J1`
- `J2`
- `S1`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. compares, orders, and locates signed rational numbers accurately across bounded fraction, decimal, and mixed-form representations
2. coordinates sign and magnitude meaning while operating fluently with signed rational values
3. explains why a signed rational result is reasonable using reference points, direction, and benchmark magnitude
4. succeeds across symbolic, number-line, table, verbal, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- compare and order signed fractions and decimals using reference-point reasoning
- locate signed rational values on a number line and interpret distance from zero
- operate fluently on signed rational numbers while preserving sign and magnitude meaning
- explain why a result makes sense using direction, benchmark, or absolute-size language


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
