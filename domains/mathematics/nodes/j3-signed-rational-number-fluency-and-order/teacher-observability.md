# Teacher Observability

## Look-fors

- Orders signed rational values inconsistently because zero or benchmark values are not functioning as stable references.
- Talks about a value being "bigger" without distinguishing greater value from greater distance from zero.
- Breaks when fractions and decimals are mixed in the same signed ordering or comparison task.
- Computes some signed rational operations but cannot interpret the result on a number line or in a bounded context.
