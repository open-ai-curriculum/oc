# M11 Verification Model

## Verification Gates

- `G1` direct performance gate
- `G2` meaning gate
- `G3` transfer gate
- `G4` retention gate

## Evidence Targets

- builds the same total value in more than one way with common coins or units
- compares two amounts using composed value rather than raw coin count
- determines whether an amount is enough for a bounded price
- explains simple exchanges that preserve or increase/decrease total value

## Minimum Verification Rule

A learner does not clear `M11` until direct accuracy, explanation, transfer, and delayed retention evidence all meet threshold.
