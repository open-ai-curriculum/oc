# P1 Intervention Playbook

### P1-FM-01 Repeating-Unit Misidentification

- mark one full cycle visually before asking for extension
- compare candidate repeating units and ask which one reproduces the whole pattern

### P1-FM-02 Extension Error

- stop after one full identified cycle and restart the pattern from there
- practice extending only one or two additional steps at a time

### P1-FM-03 Surface-Feature Lock

- use brackets or color blocks to group full cycles
- contrast same-start patterns with different repeating units

### P1-FM-04 Rule Language Weakness

- use sentence frames such as "it repeats __, then __"
- require oral explanation before extension
