# P1 Failure Taxonomy

### P1-FM-01 Repeating-Unit Misidentification

- description: learner does not correctly identify the full repeating unit

### P1-FM-02 Extension Error

- description: learner identifies part of the pattern but extends it incorrectly

### P1-FM-03 Surface-Feature Lock

- description: learner focuses on isolated adjacent items instead of the full cycle

### P1-FM-04 Rule Language Weakness

- description: learner can continue the pattern but cannot explain what repeats or why
