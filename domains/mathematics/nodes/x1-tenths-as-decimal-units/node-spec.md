# X1 Node Specification

## Node Identity

- id: `X1`
- name: `tenths_as_decimal_units`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets tenths as structured decimal units connected to equal partitions of a whole and to base-ten notation.

## Dependencies

- `V1`
- `F1`

## Exit Conditions

The learner is secure on `X1` only when all of the following are true:

1. interprets tenths as decimal units of a whole
2. connects tenths notation to equal-part and place-value structure
3. distinguishes tenths from whole-number digit reading
4. succeeds across partition models, decimal notation, and familiar context tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- identifies tenths in partitioned wholes and base-ten style representations
- connects one tenth to decimal notation in bounded contexts
- explains that tenths name equal parts of one whole rather than separate counting objects
- matches visual, verbal, symbolic, and money-like representations of tenths

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
