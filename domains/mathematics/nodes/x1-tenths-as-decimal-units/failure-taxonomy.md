# X1 Failure Taxonomy

## Classified Failure Modes

- `tenths_as_two_whole_digits`: Reads a decimal with tenths as if it were two separate whole-number digits.
- `partition_unit_disconnect`: Cannot connect tenths notation to equal partitions of one whole.
- `decimal_point_ignored`: Treats the decimal point as a decorative mark rather than a unit boundary.
- `whole_size_drift`: Changes the reference whole without recognizing the effect on tenths meaning.
