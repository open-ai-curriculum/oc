# X1 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `whole-and-tenths matching`: Match partitioned wholes, decimal numerals, and named tenths statements.
- `decimal-point meaning rehearsal`: Use contrast cases to show how the decimal point changes unit interpretation.
- `bounded whole consistency tasks`: Hold the whole constant while varying the number of tenths represented.
