# U2 Failure Taxonomy

### U2-FM-01 Unit Identity Drift

- description: learner does not preserve what the repeated unit is across the pattern or grouping

### U2-FM-02 Unequal-Group Acceptance

- description: learner accepts nonmatching groups as part of the same repeated-unit structure

### U2-FM-03 Group-As-Collection Only

- description: learner sees separate collections but not a repeatable same-sized unit

### U2-FM-04 Context Boundary

- description: learner reasons correctly with objects but not with tiled or measurement-adjacent contexts
