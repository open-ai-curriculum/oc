# U2 Node Specification

## Node Identity

- id: `U2`
- name: `repeated_unit_reasoning`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner understands that the same group or unit can be repeated and treated as a countable structure rather than as unrelated individual pieces.

## Dependencies

- `U1`

## Exit Conditions

The learner is secure on `U2` only when all of the following are true:

1. identifies repeated equal groups or units
2. treats the same group or unit as a repeatable countable chunk
3. explains what stays the same across the repeated groups or units
4. succeeds across object-grouping and simple measurement-style contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- recognize that several groups are made of the same-sized unit
- count or describe repeated groups as structured units
- explain why changing one group size would break the repeated-unit structure
- prepare for fair sharing and partitioning by stabilizing the idea of equal repeatable units

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
