# U2 Node Package

## Node

`U2` - Repeated unit reasoning

## Purpose

This package defines the bridge from counting individual objects to counting repeated groups or repeated equal units.

It exists to operationalize the learner's ability to treat the same-sized group or unit as a repeatable countable entity.
