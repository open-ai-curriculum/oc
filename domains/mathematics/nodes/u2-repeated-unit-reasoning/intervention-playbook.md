# U2 Intervention Playbook

## Purpose

Map `U2` failure modes to bounded intervention responses.

## Intervention Rules

### For U2-FM-01 Unit Identity Drift

Primary interventions:

- label the repeated unit before counting or describing the pattern
- circle or mark each equal group
- compare "same unit repeated" with "different groups mixed together"

Expected effect:

- learner preserves the identity of the repeated unit

Reverification target:

- one direct repeated-unit item

### For U2-FM-02 Unequal-Group Acceptance

Primary interventions:

- sort equal and unequal group sets
- compare one mismatching group against stable repeated groups
- ask what must stay the same for the pattern to hold

Expected effect:

- learner rejects nonmatching groups from repeated-unit structures

Reverification target:

- one error-diagnosis item and one direct item

### For U2-FM-03 Group-As-Collection Only

Primary interventions:

- count by groups with teacher modeling
- use repeated language such as "same group again"
- connect equal groups to tiled strips or repeated tiles

Expected effect:

- learner treats the group or tile as a repeatable unit

Reverification target:

- one explanation item and one transfer item

### For U2-FM-04 Context Boundary

Primary interventions:

- move between objects, strips, and tiled spaces using the same repeated-unit language
- ask what repeats in each context
- use parallel examples across contexts

Expected effect:

- learner transfers repeated-unit reasoning beyond one material format

Reverification target:

- one changed-context transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
