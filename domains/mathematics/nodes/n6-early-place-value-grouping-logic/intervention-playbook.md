# N6 Intervention Playbook

### N6-FM-01 Digit-Only Interpretation

- require bundling tasks before reading numerals
- connect each written numeral to tens-and-ones statements aloud

### N6-FM-02 Tens-Comparison Failure

- compare numbers with clear tens differences first, then move to closer cases
- use "compare tens first, then ones" talk moves consistently

### N6-FM-03 Representation Drift

- cycle among blocks, drawings, and numerals in one routine
- require matching tasks before independent production

### N6-FM-04 Place-Role Instability

- contrast reversed-digit pairs such as 23 and 32
- use sentence frames such as "in 32, the 3 means 3 tens"
