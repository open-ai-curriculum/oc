# N6 Learner State Model

Tracked state for `N6` includes:

- `status`
- `direct_accuracy`
- `meaning_status`
- `transfer_status`
- `retention_status`
- `dominant_failure_modes`
- `recommended_interventions`

The learner is not `secure` until grouped place-value reasoning is accurate, explainable, transferable, and stable after delay.
