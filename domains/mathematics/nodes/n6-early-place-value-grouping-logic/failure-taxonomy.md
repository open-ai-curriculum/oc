# N6 Failure Taxonomy

### N6-FM-01 Digit-Only Interpretation

- description: learner reads digits but does not connect them to grouped tens-and-ones value

### N6-FM-02 Tens-Comparison Failure

- description: learner compares two-digit numbers using ones first or ignores the tens structure

### N6-FM-03 Representation Drift

- description: learner cannot keep concrete, pictorial, and symbolic place-value forms aligned

### N6-FM-04 Place-Role Instability

- description: learner does not understand that the same digit can represent different values in different places
