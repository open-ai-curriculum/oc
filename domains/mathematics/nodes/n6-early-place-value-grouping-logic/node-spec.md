# N6 Node Specification

## Node Identity

- id: `N6`
- name: `early_place_value_grouping_logic`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner understands that numbers can be represented with grouped tens and ones and that the value of a digit depends on its place within that grouped structure.

## Dependencies

- `N5`

## Exit Conditions

The learner is secure on `N6` only when all of the following are true:

1. represents two-digit numbers with tens and ones groupings
2. compares and orders two-digit numbers by reasoning about tens first, then ones
3. explains why the same digit can have different value roles in different places
4. succeeds across concrete grouping, drawings, spoken explanation, and symbolic forms
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- show `23` as `2 tens and 3 ones`
- explain why `32` is different from `23`
- compare `28` and `31` by reasoning about tens and ones
- connect bundled groups to two-digit numerals without relying on rote digit naming only

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
