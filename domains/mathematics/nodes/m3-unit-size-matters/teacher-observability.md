# M3 Teacher Observability View

Teachers should see:

- current `M3` status
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner is using raw count instead of unit-size meaning
- recommended intervention

Class-level view should highlight:

- who thinks more units always means more quantity
- who reverses partition-count logic
- who can reason in one measurement context but not another
