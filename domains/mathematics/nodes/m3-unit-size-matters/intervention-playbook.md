# M3 Intervention Playbook

## Purpose

Map `M3` failure modes to bounded intervention responses.

## Intervention Rules

### For M3-FM-01 Unit Count Without Unit Meaning

Primary interventions:

- compare the same quantity measured with visibly different units
- label both the count and the unit name in every response
- ask "how big is each unit?" before accepting the count

Expected effect:

- learner connects the count to the size of the unit

Reverification target:

- one direct comparison item and one explanation item

### For M3-FM-02 More-Number-Means-More-Quantity

Primary interventions:

- use same-quantity examples with contrasting unit sizes
- explicitly compare "more units because they are smaller"
- sort examples where larger counts mean smaller units

Expected effect:

- learner stops using the raw count as a proxy for magnitude

Reverification target:

- one error-diagnosis item and one direct item

### For M3-FM-03 Partition Count Reversal

Primary interventions:

- compare one whole partitioned into different counts
- use overlays or stacked models to compare part sizes
- connect partition count to smaller part size in the same whole

Expected effect:

- learner understands that more equal parts of the same whole produce smaller units

Reverification target:

- one partition-based transfer item

### For M3-FM-04 Context-Bound Reasoning

Primary interventions:

- move the same principle across length, area, and sharing contexts
- ask for the invariant idea before changing the surface form
- use parallel examples with different materials

Expected effect:

- learner transfers unit-size reasoning beyond one familiar setting

Reverification target:

- one changed-context transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
