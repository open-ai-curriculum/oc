# M3 Node Package

## Node

`M3` - Unit size matters

## Purpose

This package defines the measurement foundation that later fraction and denominator reasoning depend on.

It exists to operationalize the learner's ability to understand that measurement results depend on the size of the unit, not just the count of units.
