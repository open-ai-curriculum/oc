# M3 Failure Taxonomy

### M3-FM-01 Unit Count Without Unit Meaning

- description: learner tracks the number of units but not what the unit size means

### M3-FM-02 More-Number-Means-More-Quantity

- description: learner assumes a larger count always represents a larger quantity

### M3-FM-03 Partition Count Reversal

- description: learner assumes more equal parts create larger parts instead of smaller ones

### M3-FM-04 Context-Bound Reasoning

- description: learner can reason about one measurement setting but cannot transfer the idea to another
