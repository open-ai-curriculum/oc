# R1 Failure Taxonomy

### R1-FM-01 Equality Blindness

- description: learner accepts unequal partitions as valid equal parts of a whole

### R1-FM-02 Whole Reference Drift

- description: learner loses track of which whole a part belongs to

### R1-FM-03 Part As Object

- description: learner treats a part as an isolated object rather than as a relational quantity

### R1-FM-04 Unit Size Blindness

- description: learner assumes same-looking pieces have equal value even when wholes differ
