# R1 Node Package

## Node

`R1` - Part-whole reasoning with equal parts

## Purpose

This package defines the first direct readiness node for fraction structure.

It exists to operationalize the learner's ability to see a whole as composed of equal parts and to distinguish valid equal partitioning from arbitrary breaking-apart.
