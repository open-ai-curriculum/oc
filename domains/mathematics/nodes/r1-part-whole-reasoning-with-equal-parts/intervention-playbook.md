# R1 Intervention Playbook

## Purpose

Map `R1` failure modes to bounded intervention responses.

## Intervention Rules

### For R1-FM-01 Equality Blindness

Primary interventions:

- sort examples into equal and unequal partitions
- fold, cut, or overlay shapes to test equality physically or visually
- use teacher language that contrasts "same number of pieces" with "same-sized pieces"

Expected effect:

- learner stops confusing partition count with equal partition quality

Reverification target:

- one visual identification task and one explanation task

### For R1-FM-02 Whole Reference Drift

Primary interventions:

- explicitly label each whole before comparing parts
- use matching routines where learners connect a part to its original whole
- contrast same-sized pieces from different wholes

Expected effect:

- learner maintains the reference whole while reasoning about a part

Reverification target:

- direct recognition task with mixed wholes

### For R1-FM-03 Part As Object

Primary interventions:

- ask "part of what whole?" before accepting an answer
- use part-whole diagrams with missing-whole and missing-part prompts
- revisit composition and decomposition language from `N1`

Expected effect:

- learner reasons relationally rather than treating the part as a standalone count

Reverification target:

- explanation task plus one transfer item

### For R1-FM-04 Unit Size Blindness

Primary interventions:

- compare equal-shaped pieces from different-size wholes
- use measurement overlays or tracing to show changing whole size
- require estimate language such as "larger piece because the whole is larger"

Expected effect:

- learner links part value to whole size and unit size

Reverification target:

- one error-diagnosis item and one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
