# M8 Failure Taxonomy

## Classified Failure Modes

- `endpoint_misalignment`: Starts measuring from the tool edge or the wrong mark and produces shifted measurements.
- `hashmark_counting_confusion`: Counts visible marks rather than unit intervals.
- `unit_label_loss`: Records a number without preserving the length unit.
- `tool_transfer_breakdown`: Can measure with one tool type but not another equivalent standard unit tool.
