# M8 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `zero-mark alignment rehearsal`: Use transparent examples that contrast tool edge and zero-mark starts.
- `interval-versus-mark sort`: Have learners classify which count represents spaces between marks versus marks themselves.
- `same-object multi-tool checks`: Measure the same object with different standard tools and reconcile the readings.
