# M8 Node Specification

## Node Identity

- id: `M8`
- name: `standard_length_units_and_tool_use`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner measures length with standard units and uses measurement tools in a way that preserves unit iteration and endpoint alignment.

## Dependencies

- `M5`
- `V1`

## Exit Conditions

The learner is secure on `M8` only when all of the following are true:

1. uses standard length units with correct starting point and endpoint logic
2. reads and records measured length in bounded contexts
3. explains common tool-use errors such as starting at the wrong mark
4. succeeds across rulers, measuring strips, and classroom object tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- measures objects using bounded standard length tools
- aligns endpoints correctly when using rulers or unit-marked tools
- explains why the measurement count depends on the chosen unit marks rather than the physical tool length alone
- compares measured lengths using recorded unit counts

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
