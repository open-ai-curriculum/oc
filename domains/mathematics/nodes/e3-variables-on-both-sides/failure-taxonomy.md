# E3 Failure Taxonomy

## Classified Failure Modes

- `variable_transfer_inconsistency`: Moves variable terms inconsistently or loses term structure across sides.
- `identity_contradiction_confusion`: Confuses no-solution and infinitely-many-solution outcomes.
- `structural_misclassification`: Treats equation class as an arithmetic mistake instead of a valid structural result.
- `solution_class_check_gap`: Does not verify whether the final classification fits the original equation.
