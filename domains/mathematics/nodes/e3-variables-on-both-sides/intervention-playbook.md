# E3 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `equation-class sort`: Sort completed equations into one-solution, contradiction, and identity families with justification.
- `term-collection scaffolds`: Use explicit marking for variable terms and constants before combining across sides.
- `classification-check routines`: Require each final answer to include both a solution statement and an equation-class statement.
