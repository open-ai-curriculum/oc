# Y2 Failure Taxonomy

## Classified Failure Modes

- `operation_estimate_direction_error`: Produces estimates with the wrong directional relationship to the exact result.
- `benchmark_selection_instability`: Chooses inconsistent or unhelpful benchmarks for an operation.
- `plausibility_check_absent`: Computes exactly but does not use estimation to judge reasonableness.
- `cross_operation_strategy_blur`: Uses the same estimation move even when operation structure differs.
