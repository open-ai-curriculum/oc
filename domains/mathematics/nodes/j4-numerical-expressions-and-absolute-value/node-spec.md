# J4 Node Specification

## Node Identity

- id: `J4`
- name: `numerical_expressions_and_absolute_value`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner evaluates numerical expressions with rational numbers and interprets absolute value as distance from zero or magnitude, rather than as a procedural symbol with no semantic role.

## Dependencies

- `J3`
- `A1`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. evaluates bounded numerical expressions with rational numbers accurately and coherently
2. interprets absolute value as magnitude or distance from zero rather than as a sign-removal shortcut
3. explains how grouping, order structure, and absolute-value meaning determine the result
4. succeeds across symbolic, number-line, verbal, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- evaluate bounded rational-number expressions while preserving operation and grouping structure
- interpret absolute value on a number line as distance from zero
- distinguish value, opposite, and absolute value in bounded cases
- explain why an expression result is reasonable using magnitude, grouping, and sign structure


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
