# Intervention Playbook

## `structure_tracking_breakdown`

- use expression-markup routines that annotate grouping and operation sequence before evaluation
- rewrite one bounded step at a time while preserving the original structure visibly
- compare correct and incorrect simplifications to surface where structure was lost

## `absolute_value_as_sign_removal`

- use number-line distance tasks with pairs such as `|-3|` and `|3|`
- contrast value, opposite, and absolute value explicitly
- teacher think-alouds linking absolute value to distance from zero, not positivity

## `grouping_order_blindness`

- rehearse parentheses-sensitive evaluation with bounded rational expressions
- use card sorts that separate expression value from expression structure
- require learners to state which operation or grouping is being resolved and why

## `context_expression_transfer_breakdown`

- translate among symbolic expressions, temperature/distance contexts, and number-line magnitude statements
- pair each evaluated expression with a reasonableness or distance interpretation question
- teacher compare-and-connect routines across symbolic, visual, and contextual absolute-value and expression situations
