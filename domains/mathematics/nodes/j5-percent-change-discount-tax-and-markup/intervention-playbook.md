# Intervention Playbook

## `base_value_tracking_breakdown`

- label original amount, percent amount, and result explicitly before calculation
- use percent-bar or table models that keep the base quantity visible
- compare multiple problems where the same percent is applied to different bases

## `additive_percent_misread`

- contrast additive difference problems with percent-change problems directly
- use hundred-model and rate-table routines before symbolic shortcuts
- teacher think-alouds naming the percent as a fraction of the original amount

## `result_direction_confusion`

- sort contexts into increase versus decrease before computing
- annotate whether the percent amount should be added to or removed from the base
- compare matched tax and discount cases with the same numbers

## `context_reasonableness_breakdown`

- use price, measurement, and quantity contexts with explicit estimate-before-compute routines
- pair each answer with a question about whether it should be larger or smaller than the original amount
- teacher compare-and-connect routines across symbolic, tabular, and contextual percent-change situations
