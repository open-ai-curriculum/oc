# Verification Model

This node is verified through four gate types.

## Gates

- `G1 direct performance`: learner meets at least 0.88 accuracy on direct tasks
- `G2 meaning`: learner explains what the variable and expression represent
- `G3 transfer`: learner succeeds in at least one less-familiar expression context
- `G4 retention`: learner passes a delayed recheck after 21 days

## Item Families

- direct evaluation or classification
- explanation and justification
- error diagnosis
- transfer in a varied expression context
