# Failure Taxonomy

- `expression_equation_collapse`: Treats an expression as if it must be solved like an equation.
- `substitution_placeholding_error`: Substitutes values inconsistently or in the wrong structural position.
- `variable_as_label_only`: Treats the variable only as a label and not as a quantity that can vary or be assigned a value.
- `operation_order_disruption`: Loses expression structure during evaluation and computes in an incoherent order.
