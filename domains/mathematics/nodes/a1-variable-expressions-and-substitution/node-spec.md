# A1 Node Specification

## Node Identity

- id: `A1`
- name: `variable_expressions_and_substitution`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets variables as quantities that can vary or stand for numbers in expressions and evaluates expressions through substitution with structural meaning.

## Dependencies

- `S2`
- `E1`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets variables in expressions as standing for quantities rather than as mere letters to manipulate
2. evaluates bounded expressions correctly by substitution
3. distinguishes expressions from equations and explains the difference in meaning
4. succeeds across symbolic, verbal, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- explain what a variable represents in a bounded expression context
- substitute values into expressions and compute accurately
- sort expressions and equations by structure and purpose
- connect a verbal rule or quantity description to an algebraic expression in bounded cases


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
