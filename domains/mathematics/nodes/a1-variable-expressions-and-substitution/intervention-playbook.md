# Intervention Playbook

## `expression_equation_collapse`

- sort expressions versus equations with explanation prompts
- teacher modeling of names versus statements
- non-example review where equals sign changes task type

## `substitution_placeholding_error`

- substitute-and-circle routines for each variable occurrence
- color-code value insertion before computation
- one-expression multiple-value rehearsal

## `variable_as_label_only`

- context-to-expression matching tasks
- vary-the-value tasks showing output changes
- teacher think-aloud on variable as quantity placeholder

## `operation_order_disruption`

- structure-preserving rewrite steps
- evaluate with visible grouping and order checks
- bounded order-of-operations rehearsal in substitution contexts
