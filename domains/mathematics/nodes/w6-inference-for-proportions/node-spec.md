# W6 Node Specification

## Node Identity

- id: `W6`
- name: `inference_for_proportions`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses confidence intervals and significance tests for proportions to evaluate population claims in bounded settings, connecting sample outcomes, chance variation, and study validity into one inferential argument.

## Dependencies

- `W5`
- `W3`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. constructs and interprets bounded confidence intervals for one- and two-proportion settings with correct population meaning
2. sets up and interprets bounded significance tests for proportions using null, alternative, statistic, and evidence logic coherently
3. distinguishes statistical significance from practical importance and explains how study design affects interpretive legitimacy
4. connects interval and test results for proportions without treating them as unrelated procedures
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.83`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- build and interpret bounded confidence intervals for proportions with correct claim scope
- conduct bounded one- or two-proportion significance reasoning and explain the resulting evidence claim
- distinguish statistical significance from practical importance in context
- critique proportion-based inference that ignores weak design or invalid assumptions


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
