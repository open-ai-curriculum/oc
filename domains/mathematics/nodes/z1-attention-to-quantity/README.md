# Z1 Node Package

## Node

`Z1` - Attention to quantity

## Purpose

This package defines the earliest zero-state quantitative readiness in the domain.

It exists to operationalize the learner's ability to notice sets, collections, and meaningful changes in amount before formal counting begins.
