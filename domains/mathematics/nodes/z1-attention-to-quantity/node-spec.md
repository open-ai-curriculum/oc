# Z1 Node Specification

## Node Identity

- id: `Z1`
- name: `attention_to_quantity`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices that sets and collections differ in quantity and that quantity can increase, decrease, appear, or disappear.

## Dependencies

None

## Exit Conditions

The learner is secure on `Z1` only when all of the following are true:

1. notices the presence of one versus many in practical contexts
2. notices when a set has changed in amount
3. distinguishes sameness versus difference in broad quantity terms
4. succeeds across concrete, visual, and simple everyday contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- notice when one object becomes many or many becomes one
- respond to simple quantity contrasts
- recognize that a set changed when items were added or removed
- establish pre-counting awareness of quantity as a meaningful feature

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
