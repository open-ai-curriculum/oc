# Z1 Intervention Playbook

## Purpose

Map `Z1` failure modes to bounded intervention responses.

## Intervention Rules

### For Z1-FM-01 One-Many Instability

Primary interventions:

- contrast one-object and many-object trays
- use matching routines for one versus many
- repeat broad quantity language in concrete settings

Expected effect:

- learner stabilizes basic broad quantity contrasts

Reverification target:

- one direct quantity-attention item

### For Z1-FM-02 Quantity-Change Blindness

Primary interventions:

- use hide-add-reveal and hide-remove-reveal routines
- ask what changed after each action
- slow the sequence so the learner attends to the change event

Expected effect:

- learner notices that quantity changed when items were added or removed

Reverification target:

- one change-detection item

### For Z1-FM-03 Quantity-State Confusion

Primary interventions:

- sort examples into same and different quantity states
- pair simple more versus less examples side by side
- use teacher language naming the state explicitly

Expected effect:

- learner organizes broad quantity states more reliably

Reverification target:

- one direct item and one error-diagnosis item

### For Z1-FM-04 Context Boundary

Primary interventions:

- move from objects to pictures to familiar everyday situations
- ask the same quantity question in multiple contexts
- use parallel examples with different materials

Expected effect:

- learner transfers quantity attention beyond one setting

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
