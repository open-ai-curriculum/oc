# K6 Node Specification

## Node Identity

- id: `K6`
- name: `systems_of_linear_equations_and_inequalities`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets systems of linear equations and inequalities as interacting constraints, solves them in bounded cases, and explains solution meaning rather than treating solving methods as isolated symbol moves.

## Dependencies

- `K4`
- `K5`
- `A3`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. solves bounded systems of linear equations using graph, table, or symbolic elimination/substitution routes as appropriate
2. explains a system solution as a value pair satisfying both constraints simultaneously
3. interprets no-solution and infinitely-many-solution cases structurally in bounded linear settings
4. represents and interprets bounded linear inequalities and simple systems of inequalities in coordinate form
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- solve or verify bounded systems of linear equations in multiple representations
- explain why a solution pair must satisfy both equations or inequalities simultaneously
- classify bounded system outcomes as one solution, no solution, or infinitely many solutions
- interpret and graph bounded linear inequalities and overlap regions in simple cases


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
