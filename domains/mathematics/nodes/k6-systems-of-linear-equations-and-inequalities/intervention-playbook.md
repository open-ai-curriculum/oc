# Intervention Playbook

## `constraint_intersection_misread`

- test candidate pairs against both equations before solving symbolically
- graph or table each constraint separately and identify only the overlap as the solution set
- compare pairs that satisfy one equation with pairs that satisfy both

## `simultaneous_satisfaction_gap`

- require learners to explain each solve step as preserving both constraints
- use matching tasks that pair symbolic work with graph intersections and table overlap
- teacher think-alouds naming the solution as shared truth, not just computed output

## `elimination_substitution_drift`

- annotate elimination or substitution steps with preserved-equation checks
- compare correct and incorrect solve paths where one algebraic move breaks equivalence
- use bounded systems where graph verification can confirm symbolic work

## `inequality_region_misclassification`

- use boundary-line and test-point routines before shading
- compare correct and reversed inequality regions in matched examples
- require verbal explanation of why the overlap region satisfies all listed inequalities
