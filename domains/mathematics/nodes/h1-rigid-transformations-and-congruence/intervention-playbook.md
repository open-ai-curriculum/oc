# Intervention Playbook

## `motion_rule_execution_breakdown`

- use stepwise grid-based transformation routines with explicit start and image coordinates
- mark centers, directions, and turn amounts before executing each transformation
- compare correct and drifted image placements side by side

## `preservation_meaning_gap`

- measure and compare lengths and angles before and after rigid motions
- use teacher think-alouds naming which properties are preserved and which orientation features may change
- compare rigid and non-rigid actions explicitly to separate congruence-preserving from non-preserving moves

## `correspondence_label_drift`

- label original and image vertices in matched order before reasoning
- use tracing or overlay routines to maintain point correspondence
- require verbal readback of which original point maps to which image point

## `visual_congruence_overreach`

- require a transformation path or symmetry argument before accepting a congruence claim
- compare visually similar non-congruent figures with actual congruent pairs
- use overlay or coordinate verification to test appearance-based claims
