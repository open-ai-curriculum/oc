# Failure Taxonomy

- `correspondence_mapping_breakdown`: Fails to pair corresponding quantities correctly in a proportional application.
- `additive_application_drift`: Treats a proportional application as if equal additive change were sufficient.
- `scale_factor_instability`: Applies inconsistent or shifting scale factors across the same situation.
- `contextual_scale_reasonableness_breakdown`: Can compute symbolically but cannot judge whether a scaled result is sensible in context.
