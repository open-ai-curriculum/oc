# Intervention Playbook

## `correspondence_mapping_breakdown`

- label corresponding quantities explicitly before solving
- use double-number lines, ratio tables, or matched diagrams to preserve pairings
- compare correct and incorrect pairings in similar contexts

## `additive_application_drift`

- contrast additive and proportional application cases with the same surface numbers
- require scale-factor or unit-rate justification before solving
- teacher think-alouds naming the invariant ratio across corresponding values

## `scale_factor_instability`

- use single-scale-factor rehearsal across all pairs in a task
- annotate the factor connecting original and scaled quantities before computing
- compare correct and incorrect table extensions to surface drifting factors

## `contextual_scale_reasonableness_breakdown`

- use map, recipe, similar-figure, and measurement contexts with explicit estimate-before-compute routines
- pair each answer with a question about whether the scaled result should be larger, smaller, or equivalent
- teacher compare-and-connect routines across symbolic, tabular, diagrammatic, and contextual scale situations
