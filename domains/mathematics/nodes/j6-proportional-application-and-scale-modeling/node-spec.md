# J6 Node Specification

## Node Identity

- id: `J6`
- name: `proportional_application_and_scale_modeling`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner applies proportional reasoning to scale models, similar figures, maps, recipes, measurement conversions, and other multiplicative situations using invariant ratio structure rather than additive shortcuts.

## Dependencies

- `Q3`
- `J5`
- `M14`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. solves bounded proportional application and scale situations accurately
2. distinguishes proportional structure from additive patterning in applied contexts
3. explains how invariant ratio or scale factor determines unknown values
4. succeeds across table, diagram, verbal, measurement, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- solve for unknowns in scale drawings, map scales, recipes, similar-shape, and conversion-style proportional contexts
- represent proportional applications using tables, unit-rate reasoning, or scale-factor structure
- explain why the same multiplicative relationship must hold across corresponding quantities
- judge whether a proportional result is reasonable relative to the original quantities and scale


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
