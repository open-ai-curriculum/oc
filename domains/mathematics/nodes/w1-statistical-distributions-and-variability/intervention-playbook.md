# Intervention Playbook

## `summary_selection_fragmentation`

- classify distributions before choosing mean, median, or spread summaries
- compare matched data sets where only skew or outliers change and the best summary changes with them
- require learners to name what question the summary is answering before computing

## `distribution_feature_opacity`

- annotate plots with center, spread, clusters, gaps, and outliers before computing summaries
- use teacher think-alouds that connect numerical summaries back to visible distribution features
- compare distributions with similar centers but different spread or shape

## `outlier_summary_blindness`

- compare the same data set before and after one or two extreme values are added
- calculate mean and median side by side and discuss which moved more and why
- use removal-and-recompute routines to expose outlier sensitivity

## `graph_claim_disconnect`

- require every comparison claim to cite one graphical feature and one numerical summary when available
- compare unsupported verbal claims with claims actually backed by the displayed distribution
- use “what does the graph show” versus “what are we inferring” routines during analysis
