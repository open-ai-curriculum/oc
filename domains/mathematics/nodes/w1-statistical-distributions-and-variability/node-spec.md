# W1 Node Specification

## Node Identity

- id: `W1`
- name: `statistical_distributions_and_variability`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets univariate data distributions through center, spread, shape, and variability, and compares data sets using structured statistical reasoning rather than isolated summary numbers.

## Dependencies

- `D6`
- `Y4`
- `V2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets distributions using measures of center, spread, clusters, gaps, and outliers in bounded data sets
2. selects and explains appropriate summaries such as mean, median, interquartile range, and mean absolute deviation in bounded cases
3. compares two or more distributions using coherent claims about typical value and variability
4. reasons from graphical and numerical summaries without treating any single statistic as the whole story
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- describe the shape and variability of bounded data distributions from plots or tables
- compute and interpret mean, median, range, interquartile range, and mean absolute deviation in bounded cases
- compare distributions using both numerical summaries and distribution features
- justify why a chosen summary is informative for the distribution at hand


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
