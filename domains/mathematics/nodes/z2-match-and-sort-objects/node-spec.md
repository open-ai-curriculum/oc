# Z2 Node Specification

## Node Identity

- id: `Z2`
- name: `match_and_sort_objects`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner matches objects or sets and sorts them by shared attributes such as color, shape, size, or type.

## Dependencies

- `Z1`

## Exit Conditions

The learner is secure on `Z2` only when all of the following are true:

1. matches like objects or sets reliably
2. sorts a mixed set by a named or visible attribute
3. explains what the grouped objects share
4. succeeds across concrete objects, pictures, and simple category prompts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- match like with like
- place objects into attribute-based groups
- explain what makes a group belong together
- support later counting, patterning, and data reasoning by stabilizing attribute grouping

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
