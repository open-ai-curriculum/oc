# Z2 Failure Taxonomy

### Z2-FM-01 Sameness Rule Instability

- description: learner does not preserve a stable matching or sorting rule

### Z2-FM-02 Attribute Drift

- description: learner changes the grouping attribute mid-task or uses the wrong attribute

### Z2-FM-03 Odd-Item Inclusion

- description: learner includes nonmatching items in a group

### Z2-FM-04 Context Boundary

- description: learner matches or sorts in one material format but not another
