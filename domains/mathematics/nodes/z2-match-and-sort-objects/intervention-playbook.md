# Z2 Intervention Playbook

## Purpose

Map `Z2` failure modes to bounded intervention responses.

## Intervention Rules

### For Z2-FM-01 Sameness Rule Instability

Primary interventions:

- name the rule before sorting
- compare two possible sorting rules explicitly
- keep the task to one attribute at a time

Expected effect:

- learner stabilizes one matching or sorting rule

Reverification target:

- one direct sorting item

### For Z2-FM-02 Attribute Drift

Primary interventions:

- highlight the target attribute visually
- sort the same set twice under two different named rules
- ask what stays the same inside each group

Expected effect:

- learner tracks the intended attribute more reliably

Reverification target:

- one direct item and one explanation item

### For Z2-FM-03 Odd-Item Inclusion

Primary interventions:

- ask which item does not belong and why
- compare correct and incorrect groups side by side
- remove one mismatched item at a time

Expected effect:

- learner excludes nonmatching items from a group

Reverification target:

- one error-diagnosis item

### For Z2-FM-04 Context Boundary

Primary interventions:

- move from concrete objects to picture cards using the same rule language
- use parallel sorting tasks across materials
- ask the learner to restate the rule in each context

Expected effect:

- learner transfers matching and sorting across formats

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
