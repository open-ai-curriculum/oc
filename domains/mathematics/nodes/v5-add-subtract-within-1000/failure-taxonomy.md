# V5 Failure Taxonomy

### V5-FM-01 Hundreds-Tens-Ones Coordination Weakness

- description: learner cannot coordinate all three place values

### V5-FM-02 Cross-Place Regrouping Error

- description: learner breaks regrouping across places

### V5-FM-03 Written-Method Fragility

- description: learner loses structure when working in written formats

### V5-FM-04 Explanation-and-Transfer Weakness

- description: learner can compute but cannot explain or transfer the method
