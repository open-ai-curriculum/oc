## Purpose

The learner is secure on `V5` only when all of the following are true:

1. adds and subtracts within 1000 accurately with place-value reasoning
2. uses hundreds/tens/ones structure explicitly in solution methods
3. explains regrouping or decomposition across multiple places
4. succeeds across symbolic, model, and story contexts
5. passes a delayed recheck after spacing

- explanation required: `true`
- transfer required: `true`

- solve addition and subtraction within 1000 using place-value structure
