# V2 Node Specification

## Node Identity

- id: `V2`
- name: `compare_and_order_multidigit_numbers`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner compares and orders multidigit whole numbers using place-value reasoning rather than superficial digit reading.

## Dependencies

- `V1`

## Exit Conditions

The learner is secure on `V2` only when all of the following are true:

1. compares multidigit numbers correctly
2. orders sets of multidigit numbers by value
3. explains comparisons using place-value structure
4. succeeds across numerals, grouped models, and verbal forms
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- compare two multidigit numbers by highest-value place first
- order a short list of numbers correctly
- justify a comparison using place value language
- detect when a larger ones digit does not make the whole number larger

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
