# V2 Failure Taxonomy

### V2-FM-01 Place-First Comparison Weakness

- description: learner does not compare from the highest-value place

### V2-FM-02 Digit-Magnitude Confusion

- description: learner treats a larger later digit as making the whole number larger

### V2-FM-03 Ordering Drift

- description: learner can compare a pair but loses coherence when ordering several numbers

### V2-FM-04 Comparison Justification Weakness

- description: learner states the answer without place-value justification
