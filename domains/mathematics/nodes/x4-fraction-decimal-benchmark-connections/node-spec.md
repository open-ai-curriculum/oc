# X4 Node Specification

## Node Identity

- id: `X4`
- name: `fraction_decimal_benchmark_connections`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner connects common fraction benchmarks and decimal benchmarks as coordinated representations of the same magnitude.

## Dependencies

- `F3`
- `X3`

## Exit Conditions

The learner is secure on `X4` only when all of the following are true:

1. connects common fraction and decimal benchmarks coherently
2. uses benchmark equivalence to support magnitude reasoning across representations
3. distinguishes exact benchmark matches from approximate reasonableness judgments
4. succeeds across symbolic, visual, and number-line transfer tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- matches common benchmark fractions to familiar decimal values in bounded cases
- uses shared benchmarks to compare fraction and decimal magnitudes
- explains why two representations can name the same quantity
- moves between fraction and decimal benchmark language on number lines or models

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
