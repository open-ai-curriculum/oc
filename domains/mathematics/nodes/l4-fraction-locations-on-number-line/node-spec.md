# L4 Node Specification

## Node Identity

- id: `L4`
- name: `fraction_locations_on_number_line`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets fractions as numbers that occupy locations on the number line and uses partitioned intervals to place and compare them.

## Dependencies

- `F3`
- `F6`
- `L3`

## Exit Conditions

The learner is secure on `L4` only when all of the following are true:

1. places fractions on a number line using equal-part partition logic
2. uses number-line location to compare and reason about fraction magnitude
3. connects part-whole fraction structure to linear magnitude representation
4. succeeds across unit fractions, simple improper fractions, mixed-number benchmarks, and comparison tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- places unit fractions and simple non-unit fractions on bounded lines
- partitions a whole interval into equal fractional intervals
- compares fractions using relative position and benchmark locations
- explains why equal partitions are required for valid fraction placement

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
