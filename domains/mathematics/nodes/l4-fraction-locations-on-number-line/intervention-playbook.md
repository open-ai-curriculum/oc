# L4 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `equal-part interval folding`: Partition whole intervals into equal subintervals before placing labels.
- `benchmark anchor placements`: Place 0, 1/2, and 1 first, then locate nearby fractions.
- `same-fraction different-lines comparison`: Compare how the same fraction appears on different bounded lines while preserving its relative location.
