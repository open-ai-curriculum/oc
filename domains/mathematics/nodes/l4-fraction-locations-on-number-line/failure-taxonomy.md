# L4 Failure Taxonomy

## Classified Failure Modes

- `whole_number_tick_bias`: Places fractions by counting labels as if they were whole-number ticks only.
- `unequal_partition_acceptance`: Accepts unequal subintervals as valid fraction partitions.
- `numerator_denominator_position_blend`: Confuses the count of parts with the size of each interval on the line.
- `benchmark_fraction_gap`: Cannot use 0, 1/2, or 1 as anchors for comparison and placement.
