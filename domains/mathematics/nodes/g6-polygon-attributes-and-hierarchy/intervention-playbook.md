# Intervention Playbook

## `nondefining_attribute_bias`

- attribute contrast sorts with rotated and resized examples
- teacher think-aloud separating defining and non-defining features
- same-shape different-orientation comparison set

## `attribute_omission`

- guided attribute checklists for side and angle features
- polygon description routines before naming
- match shape cards to explicit attribute statements

## `hierarchy_reversal`

- subset and Venn-style polygon family sorts
- square-rectangle-rhombus comparison discussions
- class membership explanation stems

## `vocabulary_label_confusion`

- targeted geometry vocabulary rehearsal
- label-to-example and non-example matching
- oral naming with immediate corrective feedback
