# N4 Learner State Model

Tracked state for `N4` includes:

- `status`
- `direct_accuracy`
- `meaning_status`
- `transfer_status`
- `retention_status`
- `dominant_failure_modes`
- `recommended_interventions`

The learner is not `secure` until mixed facts are accurate, strategy use is efficient and explainable, and delayed rechecks remain stable.
