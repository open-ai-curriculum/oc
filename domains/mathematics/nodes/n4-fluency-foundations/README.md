# N4 Node Package

## Node

`N4` - Fluency foundations

## Purpose

This package defines the learner's ability to solve small addition and subtraction combinations with efficient, stable strategies rather than slow reconstruction.

It exists to operationalize the transition from operation meaning and counting strategies into early fluency that supports later place-value and fraction work.
