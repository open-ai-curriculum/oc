# N4 Intervention Playbook

### N4-FM-01 Fact Instability

- narrow the range and rebuild stable fact families with visual supports
- use repeated mixed retrieval with immediate feedback on only a small set at a time

### N4-FM-02 Recount Dependence

- contrast recounting with count-on or known-fact strategies and measure the work saved
- require explicit strategy selection before solving

### N4-FM-03 Mixed-Set Collapse

- interleave a small number of addition and subtraction items deliberately
- add sorting or strategy labeling before solving mixed sets

### N4-FM-04 Strategy Inarticulation

- use sentence frames such as "I knew __ so I used __"
- have the learner compare two valid strategies on the same problem
