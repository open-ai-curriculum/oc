# Y3 Failure Taxonomy

## Classified Failure Modes

- `unit_scale_mismatch`: Selects a unit or benchmark that is grossly mismatched to the object scale.
- `measurement_plausibility_blindness`: Accepts implausible measurement values without challenge.
- `benchmark_object_gap`: Cannot connect familiar objects to benchmark measurement quantities.
- `estimate_exact_role_blur`: Confuses approximate measurement judgment with exact tool reading.
