# S1 Failure Taxonomy

## Classified Failure Modes

- `negative_as_decoration`: Treats the negative sign as a surface marker rather than as part of value meaning.
- `ordering_reversal`: Reverses order when comparing negative numbers or crossing zero.
- `subtraction_change_confusion`: Cannot distinguish static signed value from signed change or movement.
- `zero_reference_instability`: Does not use zero consistently as the anchor for comparison.
