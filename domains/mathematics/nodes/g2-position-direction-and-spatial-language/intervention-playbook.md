# G2 Intervention Playbook

### G2-FM-01 Position-Term Confusion

- rehearse paired opposite terms with direct physical examples
- contrast examples and non-examples for each target term

### G2-FM-02 Left-Right Instability

- anchor left/right to the learner's own body first, then transfer to object scenes
- use repeated gesture-plus-language pairings

### G2-FM-03 Multi-Step Direction Drift

- reduce to one-step directions and rebuild to two-step sequences
- have the learner restate each step before acting

### G2-FM-04 Representation Context Dependence

- cycle from physical movement to object placement to picture interpretation
- keep the same term set constant while changing the representation
