# G2 Node Specification

## Node Identity

- id: `G2`
- name: `position_direction_and_spatial_language`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses and interprets spatial language such as above, below, beside, in front of, behind, left, right, and turn.

## Dependencies

- `G1`

## Exit Conditions

The learner is secure on `G2` only when all of the following are true:

1. interprets spatial and directional language correctly in action and description tasks
2. describes relative positions using appropriate vocabulary
3. follows and produces simple directional sequences accurately
4. succeeds across body movement, object placement, and drawn-scene contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- place an object above, below, beside, or between other objects
- follow a one-step or two-step directional instruction
- describe where an object is relative to another object
- distinguish left/right and front/behind in bounded classroom contexts

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
