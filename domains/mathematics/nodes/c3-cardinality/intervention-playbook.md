# C3 Intervention Playbook

## Purpose

Map `C3` failure modes to bounded intervention responses.

## Intervention Rules

### For C3-FM-01 Count-To-Answer Disconnect

Primary interventions:

- use count-and-freeze routines where the last word is repeated as the answer
- point to the completed set while restating the total
- ask "how many altogether?" immediately after the count

Expected effect:

- learner links the count sequence to the set total

Reverification target:

- one direct counting item and one explanation item

### For C3-FM-02 Endless Recounting

Primary interventions:

- rehearse stopping after the final count word
- use teacher echo of the last count as the answer
- build confidence with small fixed sets

Expected effect:

- learner treats the final count word as the stable answer

Reverification target:

- one direct counting item without recount

### For C3-FM-03 Rearrangement Quantity Drift

Primary interventions:

- rearrange the same objects repeatedly without adding or removing any
- ask whether anything was added or taken away
- compare before-and-after layouts with fixed totals

Expected effect:

- learner understands that arrangement does not change quantity

Reverification target:

- one rearrangement item and one error-diagnosis item

### For C3-FM-04 Context Boundary

Primary interventions:

- shift from direct objects to pictures and stories gradually
- anchor each context back to a concrete set model
- ask the learner to identify which objects belong to the counted set

Expected effect:

- learner transfers cardinality beyond direct object counting

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
