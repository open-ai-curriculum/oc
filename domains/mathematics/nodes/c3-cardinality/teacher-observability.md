# C3 Teacher Observability View

Teachers should see:

- current `C3` status
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner uses the last count word as the set total
- recommended intervention

Class-level view should highlight:

- who can count but not state the quantity
- who loses quantity when the same set is rearranged
- who shows cardinality only in direct counting contexts
