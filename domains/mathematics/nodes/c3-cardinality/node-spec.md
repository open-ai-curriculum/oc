# C3 Node Specification

## Node Identity

- id: `C3`
- name: `cardinality`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner understands that the final count word represents how many objects are in the set.

## Dependencies

- `C2`

## Exit Conditions

The learner is secure on `C3` only when all of the following are true:

1. counts a set and states the total correctly
2. answers "how many?" with the final count word rather than recounting indefinitely
3. preserves quantity understanding when the same set is rearranged
4. succeeds across objects, pictures, and short story contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- count a set and name its quantity
- tell how many without restarting the full count after a correct count
- recognize that rearranging the same set does not change how many
- connect counted totals to simple comparison or number-building tasks

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
