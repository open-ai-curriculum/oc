# C3 Failure Taxonomy

### C3-FM-01 Count-To-Answer Disconnect

- description: learner can count but does not use the final count word as the total

### C3-FM-02 Endless Recounting

- description: learner does not treat the completed count as authoritative and restarts instead

### C3-FM-03 Rearrangement Quantity Drift

- description: learner changes the quantity judgment when the same set is rearranged

### C3-FM-04 Context Boundary

- description: learner shows cardinality in direct counting but loses it in picture or story contexts
