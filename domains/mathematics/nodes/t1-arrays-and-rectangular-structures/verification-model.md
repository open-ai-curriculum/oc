# T1 Verification Model

## Verification Gates

- `G1` direct performance gate
- `G2` meaning gate
- `G3` transfer gate
- `G4` retention gate

## Evidence Targets

- builds equal-group situations as rows and columns
- explains how rows and columns structure the total
- matches repeated addition, array drawings, and rectangular layouts
- distinguishes true arrays from irregular collections

## Minimum Verification Rule

A learner does not clear `T1` until direct accuracy, explanation, transfer, and delayed retention evidence all meet threshold.
