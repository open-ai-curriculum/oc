# T1 Node Specification

## Node Identity

- id: `T1`
- name: `arrays_and_rectangular_structures`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner organizes equal groups into rows and columns and uses rectangular structure to interpret total quantity.

## Dependencies

- `U5`
- `G4`

## Exit Conditions

The learner is secure on `T1` only when all of the following are true:

1. constructs and interprets equal-group arrays using rows and columns
2. connects an array representation to the total quantity without recounting every object when appropriate
3. explains how row count and column count determine the structure of the whole
4. succeeds across objects, drawings, grid layouts, and story contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- builds equal-group situations as rows and columns
- explains how rows and columns structure the total
- matches repeated addition, array drawings, and rectangular layouts
- distinguishes true arrays from irregular collections

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
