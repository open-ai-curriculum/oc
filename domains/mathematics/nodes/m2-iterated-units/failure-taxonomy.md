# M2 Failure Taxonomy

### M2-FM-01 Unit Inconsistency

- description: learner changes unit size or meaning across the same measurement

### M2-FM-02 Gap-Or-Overlap Measurement

- description: learner leaves spaces or overlaps between repeated units

### M2-FM-03 Surface-Only Measurement

- description: learner can place units but does not connect them to the quantity being measured

### M2-FM-04 Context-Bound Iteration

- description: learner can repeat units in one context but not transfer the idea to another
