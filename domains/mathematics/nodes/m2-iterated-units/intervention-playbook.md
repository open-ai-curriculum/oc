# M2 Intervention Playbook

## Purpose

Map `M2` failure modes to bounded intervention responses.

## Intervention Rules

### For M2-FM-01 Unit Inconsistency

Primary interventions:

- require the learner to name the unit before measuring
- use only one visibly consistent unit set per task
- compare correct and incorrect examples of mixed-unit measurement

Expected effect:

- learner maintains one consistent unit through the measurement

Reverification target:

- one direct measurement item and one explanation item

### For M2-FM-02 Gap-Or-Overlap Measurement

Primary interventions:

- snap or place units edge to edge with teacher modeling
- highlight gaps and overlaps visually
- use tracing or alignment guides

Expected effect:

- learner measures with continuous repeated units

Reverification target:

- one corrected measurement task

### For M2-FM-03 Surface-Only Measurement

Primary interventions:

- ask what the units are covering or spanning
- connect each placed unit to the quantity measured
- compare measurement with and without complete coverage

Expected effect:

- learner understands that repeated units represent the measured quantity

Reverification target:

- one transfer item with explanation

### For M2-FM-04 Context-Bound Iteration

Primary interventions:

- move from length to area or object comparison using the same logic
- ask what remains the same when the material changes
- use paired examples across contexts

Expected effect:

- learner transfers iterated-unit reasoning across contexts

Reverification target:

- one changed-context transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
