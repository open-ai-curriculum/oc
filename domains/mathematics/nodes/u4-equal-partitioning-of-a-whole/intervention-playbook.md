# U4 Intervention Playbook

## Purpose

Map `U4` failure modes to bounded intervention responses.

## Intervention Rules

### For U4-FM-01 Equality Blindness

Primary interventions:

- sort candidate partitions into equal and unequal examples
- use fold or overlay checks for partition equality
- ask whether each share is the same size, not just the same count

Expected effect:

- learner checks equality rather than relying on piece count

Reverification target:

- one identification item and one explanation item

### For U4-FM-02 Count-Only Partition Logic

Primary interventions:

- contrast requested piece count with actual equal share size
- compare two-part and three-part cuts that vary in equality
- use teacher language that separates "how many pieces" from "how big each piece is"

Expected effect:

- learner distinguishes partition count from equal-part structure

Reverification target:

- one error-diagnosis item and one direct item

### For U4-FM-03 Production Instability

Primary interventions:

- use guided drawing or folding routines
- partition strips and rectangles before moving to complex shapes
- mark benchmark points before drawing partition lines

Expected effect:

- learner can produce equal partitions, not just recognize them

Reverification target:

- one production task

### For U4-FM-04 Fairness Without Geometry

Primary interventions:

- map fair-share stories onto drawn wholes
- explicitly link fairness language to equal size and equal measure
- compare unfair and fair shares visually and verbally

Expected effect:

- learner connects social fairness language to mathematical partition structure

Reverification target:

- one transfer item with explanation

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
