# U4 Node Package

## Node

`U4` - Equal partitioning of a whole

## Purpose

This package defines the direct partitioning foundation that fraction structure depends on.

It exists to operationalize the learner's ability to partition one whole into equal parts and to distinguish fair equal partitioning from arbitrary cutting.
