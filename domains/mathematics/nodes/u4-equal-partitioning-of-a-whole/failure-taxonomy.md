# U4 Failure Taxonomy

### U4-FM-01 Equality Blindness

- description: learner accepts partitions based on piece count without checking equal size

### U4-FM-02 Count-Only Partition Logic

- description: learner assumes any split into the requested number of pieces satisfies equal partitioning

### U4-FM-03 Production Instability

- description: learner understands equal sharing verbally but cannot produce equal partitions reliably

### U4-FM-04 Fairness Without Geometry

- description: learner uses fairness language but cannot connect it to equal size or equal measure
