# C1 Failure Taxonomy

### C1-FM-01 Sequence Skip-Or-Substitution

- description: learner skips count words or substitutes the wrong next word

### C1-FM-02 Order Instability

- description: learner does not preserve stable count-word order

### C1-FM-03 Restart Dependence

- description: learner can only count from the beginning and cannot continue from a bounded entry point

### C1-FM-04 Sequence Without Meaning

- description: learner recites partially by memory pattern but cannot explain why order matters
