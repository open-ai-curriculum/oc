# C1 Intervention Playbook

## Purpose

Map `C1` failure modes to bounded intervention responses.

## Intervention Rules

### For C1-FM-01 Sequence Skip-Or-Substitution

Primary interventions:

- chant and point with visual number supports
- isolate short bounded sequence windows
- use call-and-response next-number routines

Expected effect:

- learner stabilizes the next-word progression

Reverification target:

- one direct sequence item

### For C1-FM-02 Order Instability

Primary interventions:

- compare correct and scrambled sequences
- use ordered number-path supports
- rehearse sequence with teacher pacing

Expected effect:

- learner preserves stable count order

Reverification target:

- one direct item and one error-diagnosis item

### For C1-FM-03 Restart Dependence

Primary interventions:

- practice starting from bounded middle points
- use start-here markers on number paths
- prompt "what comes after?" rather than restarting from one

Expected effect:

- learner can enter the sequence from bounded starting points

Reverification target:

- one transfer item

### For C1-FM-04 Sequence Without Meaning

Primary interventions:

- ask why order matters after recitation
- compare what happens when a number is skipped
- connect stable order to later counting objects

Expected effect:

- learner links memorized sequence to stable structure

Reverification target:

- one explanation item and one error-diagnosis item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
