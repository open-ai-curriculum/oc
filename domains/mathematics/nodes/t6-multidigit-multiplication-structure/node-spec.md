# T6 Node Specification

## Node Identity

- id: `T6`
- name: `multidigit_multiplication_structure`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner solves multidigit multiplication problems through place-value decomposition, partial products, and area structure rather than opaque procedures.

## Dependencies

- `T2`
- `V4`

## Exit Conditions

The learner is secure on `T6` only when all of the following are true:

1. solves bounded multidigit multiplication using defensible structural methods
2. explains partial products through place-value decomposition
3. connects area models, expanded forms, and written multiplication strategies
4. succeeds across symbolic, model-based, and error-diagnosis tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- decomposes multidigit factors into place-value components
- computes partial products and recombines them coherently
- connects area or rectangular models to symbolic multiplication work
- explains why the multiplication method preserves total value and structure

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
