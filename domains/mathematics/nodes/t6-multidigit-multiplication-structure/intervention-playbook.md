# T6 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `expanded-form multiplication routines`: Decompose each factor and compute explicit partial products before compression.
- `area-to-symbol translation`: Translate repeatedly between rectangular models and symbolic multiplication.
- `place-value labeling of subproducts`: Label each partial product by its unit value before recombining.
