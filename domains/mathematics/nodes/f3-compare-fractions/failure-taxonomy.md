# F3 Failure Taxonomy

### F3-FM-01 Whole Number Bias

- description: learner compares numerators or denominators as whole numbers without quantity reasoning

### F3-FM-02 Denominator Magnitude Error

- description: learner assumes larger denominator means larger fraction without considering unit size

### F3-FM-03 Strategy Fragility

- description: learner can compare only in one narrow pattern and breaks when the form changes
