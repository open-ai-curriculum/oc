# F3 Verification Model

### G1. Direct Performance Gate

- pass rule: at least 90% accuracy across ordering and comparison items

### G2. Meaning Gate

- pass rule: correct reasoning using benchmarks, common units, or unit-size logic

### G3. Transfer Gate

- pass rule: one successful contextual or visual comparison task

### G4. Retention Gate

- pass rule: at least 85% delayed recheck accuracy after about 14 days
