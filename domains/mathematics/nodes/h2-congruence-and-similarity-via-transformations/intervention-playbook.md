# Intervention Playbook

## `correspondence_ratio_drift`

- label corresponding vertices and sides in matched order before comparing measurements
- use side-angle correspondence tables before setting up ratios or transformation claims
- compare correct and misaligned correspondence maps side by side

## `scale_factor_meaning_gap`

- use dilation routines that show center, scale factor, and resulting length changes explicitly
- compare several figures with the same angle structure but different scale factors
- teacher think-alouds naming what changes and what is preserved under dilation

## `congruence_similarity_conflation`

- compare congruent and merely similar figure pairs in matched examples
- require learners to name whether a rigid motion alone is sufficient or whether dilation is required
- verify claims by checking both angle structure and side-length relationship

## `visual_match_overreliance`

- require an explicit transformation or proportional argument before accepting a claim
- compare visually convincing distractors with structurally valid congruent and similar pairs
- use overlay, tracing, and measurement checks to challenge appearance-only judgments
