# D2 Failure Taxonomy

### D2-FM-01 Category Count Error

- description: learner counts one or more categories inaccurately

### D2-FM-02 Comparison Language Error

- description: learner misuses more, fewer, most, least, or equal when comparing categories

### D2-FM-03 Equal-Category Blindness

- description: learner fails to recognize when two categories have the same total

### D2-FM-04 Visual-Bias Override

- description: learner uses size or appearance rather than counted totals to make the comparison
