# D2 Intervention Playbook

### D2-FM-01 Category Count Error

- count one category at a time with physical separation
- label each category total before comparing

### D2-FM-02 Comparison Language Error

- rehearse comparison vocabulary with matched examples
- require the learner to state both category totals before naming the relation

### D2-FM-03 Equal-Category Blindness

- use paired categories with equal totals explicitly
- compare side-by-side counts and ask whether the totals match

### D2-FM-04 Visual-Bias Override

- contrast large-looking items with smaller but more numerous items
- enforce count-first comparison routines
