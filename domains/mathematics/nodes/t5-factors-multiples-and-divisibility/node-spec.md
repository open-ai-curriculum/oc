# T5 Node Specification

## Node Identity

- id: `T5`
- name: `factors_multiples_and_divisibility`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets factors, multiples, and divisibility as structural properties of multiplication and division relationships rather than isolated vocabulary facts.

## Dependencies

- `T2`
- `V2`

## Exit Conditions

The learner is secure on `T5` only when all of the following are true:

1. interprets factors, multiples, and divisibility structurally
2. identifies and justifies divisibility in bounded whole-number cases
3. uses factor-multiple relationships to support later multiplicative reasoning
4. succeeds across arrays, lists, symbolic prompts, and classification tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identifies factors and multiples through structured equal-group or array reasoning
- explains divisibility as a no-remainder grouping relationship
- uses factor and multiple structure to compare or classify whole numbers
- connects divisibility observations to multiplication families rather than memorized labels alone

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
