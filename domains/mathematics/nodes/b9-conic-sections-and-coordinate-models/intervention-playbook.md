# Intervention Playbook

## `formula_structure_fragmentation`

- classify bounded tasks by structure before selecting circumference, area, arc-length, sector, midpoint, or distance routines
- compare matched problems where only one structural cue changes and the correct formula changes with it
- require learners to state what geometric quantity is being measured before computing

## `arc_angle_meaning_gap`

- use circle diagrams that pair central angles, inscribed angles, intercepted arcs, and chords in matched examples
- teacher think-alouds naming which measure controls which other measure
- compare correct and incorrect arc-angle interpretations side by side

## `coordinate_measure_substitution_error`

- annotate x- and y-coordinate substitutions line by line for midpoint and distance routines
- use plotted points so each numerical step can be checked against the diagram
- compare correct and swapped-coordinate work in matched examples

## `measurement_context_disconnect`

- estimate whether the answer should be larger or smaller than a benchmark before calculation
- compare computed measures against diagram constraints such as radius, diameter, or segment size
- require a post-computation reasonableness statement for each nontrivial circle or coordinate-geometry problem
