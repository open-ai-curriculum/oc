# C5 Node Specification

## Node Identity

- id: `C5`
- name: `numeral_recognition_and_writing_readiness`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner recognizes, names, and begins forming written numerals as stable mathematical symbols rather than decorative marks.

## Dependencies

- `C3`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. recognizes bounded early numerals reliably in varied formats
2. matches spoken number names to written numerals
3. forms bounded early numerals legibly enough to preserve identity
4. succeeds across cards, charts, tracing, and independent writing tasks
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- identify named numerals in mixed sets
- say the number name for shown numerals and select a numeral for a spoken number
- write bounded early numerals with stable orientation and distinguishing features
- distinguish numerals from letters, shapes, or quantity pictures


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
