# Intervention Playbook

## `symbol_name_instability`

- daily numeral naming routines with mixed cards
- spoken-name to symbol matching sets
- teacher think-aloud linking number words to written numerals

## `orientation_reversal`

- guided numeral formation practice with directional cues
- trace-copy-write sequence for unstable numerals
- contrast pairs for commonly reversed numerals

## `symbol_set_confusion`

- numeral versus non-numeral sorting tasks
- highlight the defining features of target numerals
- use sparse uncluttered symbol sets before dense displays

## `rote_sequence_without_symbol_access`

- interrupt count recitation with point-to-symbol prompts
- count-word to numeral jump games
- small-set numeral hunts tied to oral counting
