# C2 Failure Taxonomy

### C2-FM-01 Skip-Or-Double-Count

- description: learner skips objects or counts some objects more than once

### C2-FM-02 Count-Tracking Instability

- description: learner loses place while coordinating count words and objects

### C2-FM-03 Layout Sensitivity

- description: learner counts correctly in one layout but not in another

### C2-FM-04 Procedure Without Meaning

- description: learner imitates counting motions but cannot explain the one-count-per-object rule
