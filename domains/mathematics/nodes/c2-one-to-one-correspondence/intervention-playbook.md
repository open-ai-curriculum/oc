# C2 Intervention Playbook

## Purpose

Map `C2` failure modes to bounded intervention responses.

## Intervention Rules

### For C2-FM-01 Skip-Or-Double-Count

Primary interventions:

- use touch-and-move counting routines
- separate counted and uncounted objects physically
- slow the count while matching each word to one object

Expected effect:

- learner stabilizes one count word per object

Reverification target:

- one direct counting control item

### For C2-FM-02 Count-Tracking Instability

Primary interventions:

- count short fixed sets first
- use teacher-modeled pointing or tapping
- add visual markers to support place-keeping

Expected effect:

- learner maintains counting control across the set

Reverification target:

- one direct item and one explanation item

### For C2-FM-03 Layout Sensitivity

Primary interventions:

- practice with lined-up, circular, and scattered sets
- ask what stays the same even when the layout changes
- use the same quantity in multiple layouts

Expected effect:

- learner transfers one-to-one control across arrangements

Reverification target:

- one changed-layout transfer item

### For C2-FM-04 Procedure Without Meaning

Primary interventions:

- ask the learner to explain why each object gets one count
- compare correct and incorrect counting demonstrations
- use sentence frames such as "each object gets one number"

Expected effect:

- learner connects counting procedure to the one-to-one rule

Reverification target:

- one explanation item and one error-diagnosis item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
