# C2 Node Specification

## Node Identity

- id: `C2`
- name: `one_to_one_correspondence`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner coordinates counting words with individual objects so that each object is counted once and only once.

## Dependencies

- `C1`

## Exit Conditions

The learner is secure on `C2` only when all of the following are true:

1. assigns one count word to each object
2. avoids skipping or double-counting objects
3. maintains one-to-one control when objects are rearranged or spaced differently
4. succeeds across concrete objects, pictures, and moved sets
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- touch or track each object once while counting
- explain that every object must get one count word
- preserve counting control with scattered or lined-up sets
- support later cardinality because the count process is stable

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
