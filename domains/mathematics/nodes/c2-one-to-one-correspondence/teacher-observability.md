# C2 Teacher Observability View

Teachers should see:

- current `C2` status
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner assigns one count word to one object across layouts
- recommended intervention

Class-level view should highlight:

- who skips or double-counts
- who loses control when layout changes
- who imitates counting procedure without understanding the one-to-one rule
