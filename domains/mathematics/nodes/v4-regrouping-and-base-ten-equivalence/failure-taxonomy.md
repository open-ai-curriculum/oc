# V4 Failure Taxonomy

### V4-FM-01 Regrouping Equivalence Weakness

- description: learner does not understand that one ten equals ten ones

### V4-FM-02 Borrow-Carry Procedure Without Meaning

- description: learner imitates steps without preserving quantity logic

### V4-FM-03 Rename-Then-Lose-Value Error

- description: learner renames units but loses or duplicates value

### V4-FM-04 Model-to-Algorithm Disconnect

- description: learner cannot connect physical/drawn regrouping to written work
