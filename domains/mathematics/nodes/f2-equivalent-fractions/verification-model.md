# F2 Verification Model

### G1. Direct Performance Gate

- pass rule: at least 90% accuracy across equivalent-fraction generation and recognition items

### G2. Meaning Gate

- pass rule: correct explanation of why multiplying or dividing numerator and denominator by the same nonzero number preserves value

### G3. Transfer Gate

- pass rule: one successful alternate-form representation or context task

### G4. Retention Gate

- pass rule: at least 85% delayed recheck accuracy after about 14 days
