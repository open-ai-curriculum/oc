# F2 Failure Taxonomy

### F2-FM-01 Additive Distortion

- description: learner adds to numerator and denominator instead of scaling

### F2-FM-02 Pattern Without Meaning

- description: learner recognizes surface patterns but cannot justify equivalence

### F2-FM-03 Unjustified Equivalence

- description: learner labels fractions as equal without preserving value correctly
