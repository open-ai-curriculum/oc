# P4 Failure Taxonomy

### P4-FM-01 Rule Statement Weakness

- description: learner cannot state the governing rule explicitly

### P4-FM-02 Cross-Representation Structure Weakness

- description: learner cannot recognize the same structure across different representations

### P4-FM-03 Item-Naming Instead-of-Rule

- description: learner names surface elements without describing the structural relation

### P4-FM-04 Prediction-From-Rule Weakness

- description: learner states a partial rule but cannot use it to predict or justify
