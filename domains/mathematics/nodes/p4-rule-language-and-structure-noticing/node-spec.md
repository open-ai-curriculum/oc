# P4 Node Specification

## Node Identity

- id: `P4`
- name: `rule_language_and_structure_noticing`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices and states the rule or structure governing a pattern, sequence, or organized relation in explicit language.

## Dependencies

- `P3`

## Exit Conditions

The learner is secure on `P4` only when all of the following are true:

1. states a rule for a repeating or growing pattern in explicit language
2. identifies structure similarities across different representations of the same rule
3. explains how the rule predicts new or missing cases
4. succeeds across object, visual, number, and verbal pattern forms
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- say a rule such as "it adds one each time" or "it repeats red, blue, yellow"
- notice the same structure in different materials or symbols
- explain how the rule helps determine the next or missing element
- distinguish rule language from simple item naming

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
