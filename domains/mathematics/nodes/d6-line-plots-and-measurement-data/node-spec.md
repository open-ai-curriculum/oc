# D6 Node Specification

## Node Identity

- id: `D6`
- name: `line_plots_and_measurement_data`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner represents and interprets measurement data on line plots, including repeated values and fractional-unit contexts.

## Dependencies

- `D5`
- `M8`
- `F2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. plots measurement values on a line plot using a consistent scale and correct repeated marks
2. reads and compares frequency at marked values including clustered or repeated measurements
3. answers questions about totals, differences, and simple distribution features from a line plot
4. succeeds across whole-number and fraction-marked measurement contexts
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- place X marks correctly above measurement values on a line plot
- interpret repeated measures and identify most/least frequent values
- use the line plot to answer additive and comparison questions
- connect fraction-marked measurement positions to plotted data accurately


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
