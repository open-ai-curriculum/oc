# Intervention Playbook

## `mark_position_error`

- guided plotting on pre-labeled number lines
- value-to-tick matching before plotting
- fraction tick rehearsal with measurement examples

## `frequency_tracking_loss`

- stacked-mark counting routines
- sort-table-to-line-plot transfer tasks
- teacher modeling of one-mark-per-measure discipline

## `measurement_scale_disconnect`

- measurement collection before plotting
- fraction-unit line labeling practice
- tool-use to record and then plot the same data

## `distribution_question_confusion`

- question stems for total, difference, and frequency analysis
- highlight-and-count annotation protocol
- compare two line plots with guided discussion
