# T4 Node Specification

## Node Identity

- id: `T4`
- name: `multiplicative_comparison_and_scaling`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner reasons about one quantity as a scaled version of another and interprets multiplicative comparisons as coordinated changes in size.

## Dependencies

- `T2`
- `T3`

## Exit Conditions

The learner is secure on `T4` only when all of the following are true:

1. models one quantity as a multiplicative comparison of another
2. selects multiplication or division appropriately in scale contexts
3. rejects additive interpretations when a comparison is multiplicative
4. succeeds across lengths, sets, money, and comparison stories
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- interprets statements such as twice as many or three times as long
- distinguishes multiplicative comparison from additive difference
- solves scale-up and scale-down situations using known factor structure
- explains why repeated equal-size change represents multiplicative growth

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
