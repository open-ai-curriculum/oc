# T4 Agent Behavior

## Agent Responsibilities

- classify evidence against `T4` rather than awarding completion from exposure
- identify dominant failure modes using the taxonomy
- recommend only bounded interventions from the intervention map
- preserve dependency integrity and block premature progression

## Agent Must Not

- mark the learner secure without retention evidence
- collapse additive and multiplicative reasoning into one generic error class
- override human review requirements for learner-impacting deployment
