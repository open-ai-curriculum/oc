# M13 Node Specification

## Node Identity

- id: `M13`
- name: `standard_mass_volume_units_and_comparison`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses bounded standard units for mass and volume and compares measured quantities with explicit unit meaning.

## Dependencies

- `M12`
- `V1`

## Exit Conditions

The learner is secure on `M13` only when all of the following are true:

1. uses standard mass and volume units meaningfully in bounded tasks
2. compares measured values while preserving unit interpretation
3. distinguishes numerical value from unit label in mass/volume contexts
4. succeeds across symbolic measures, object contexts, and plausibility tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- reads or uses bounded standard units for mass and volume in familiar contexts
- compares measured mass or volume quantities with attention to units
- explains why the same numerical value can differ in meaning if the unit changes
- matches measurement statements to plausible objects or containers

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
