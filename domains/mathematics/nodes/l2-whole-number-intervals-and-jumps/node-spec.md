# L2 Node Specification

## Node Identity

- id: `L2`
- name: `whole_number_intervals_and_jumps`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses the number line to represent whole-number change, distance, and count-on/count-back movement as measurable intervals.

## Dependencies

- `L1`
- `N4`

## Exit Conditions

The learner is secure on `L2` only when all of the following are true:

1. uses interval jumps to model whole-number change accurately
2. determines simple distances on a line without collapsing them into endpoint labels
3. connects count-on/count-back reasoning to linear magnitude
4. succeeds across open number lines, hop diagrams, and change stories
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- represents addition and subtraction moves as jumps on a line
- counts intervals rather than only reading landing labels
- explains distance between two positions as a measurable span
- matches story changes to number-line movement

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
