# A2 Node Specification

## Node Identity

- id: `A2`
- name: `distributive_reasoning_and_combining_like_terms`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses distributive structure to rewrite expressions and combines like terms while preserving quantitative meaning rather than performing symbol moves by imitation.

## Dependencies

- `A1`
- `T6`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. recognizes and explains distributive structure in bounded algebraic expressions
2. rewrites expressions using distribution or factoring while preserving equivalence in bounded cases
3. combines like terms correctly and explains why unlike terms cannot be combined
4. succeeds across symbolic, area-structure, and verbalized expression contexts
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify common factors or distributed groups in an expression
- expand or factor simple expressions using distributive structure
- combine like terms while preserving term meaning and coefficient structure
- connect area or grouped-quantity structure to distributive rewriting


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
