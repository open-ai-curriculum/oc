# C4 Intervention Playbook

## Purpose

Map `C4` failure modes to bounded intervention responses.

## Intervention Rules

### For C4-FM-01 Comparison Without Stable Count Basis

Primary interventions:

- count both sets before comparing
- match objects one to one across sets
- restate the quantity of each set before choosing more or less

Expected effect:

- learner bases comparison on stable quantities

Reverification target:

- one direct comparison item and one explanation item

### For C4-FM-02 More-Less Reversal

Primary interventions:

- use side-by-side labeled sets
- rehearse more versus less with visual supports
- ask which set would have extra objects if matched

Expected effect:

- learner stabilizes the directional meaning of more and less

Reverification target:

- one direct comparison item

### For C4-FM-03 Spread-Out-Means-More

Primary interventions:

- rearrange the same set while preserving count
- compare compact and spread-out layouts of equal sets
- ask whether anything was added or removed

Expected effect:

- learner separates spatial spread from quantity

Reverification target:

- one error-diagnosis item and one rearranged-set item

### For C4-FM-04 Context Boundary

Primary interventions:

- move from concrete sets to pictures to short stories gradually
- anchor each context back to counted sets
- ask the learner to name both quantities before comparing

Expected effect:

- learner transfers more-less-same reasoning across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
