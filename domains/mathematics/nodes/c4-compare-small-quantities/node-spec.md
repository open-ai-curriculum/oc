# C4 Node Specification

## Node Identity

- id: `C4`
- name: `compare_small_quantities`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner determines whether one small quantity is more, less, or the same as another and can justify the comparison.

## Dependencies

- `C3`

## Exit Conditions

The learner is secure on `C4` only when all of the following are true:

1. compares bounded concrete sets correctly
2. identifies more, less, and same reliably
3. explains the basis for the comparison
4. succeeds across concrete, pictured, and simple story contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- say which set has more, less, or the same
- justify the comparison with counting or matching
- preserve comparison meaning when sets are rearranged
- support later measurable-attribute reasoning by stabilizing quantitative comparison

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
