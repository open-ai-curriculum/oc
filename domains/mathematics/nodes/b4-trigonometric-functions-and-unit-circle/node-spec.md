# B4 Node Specification

## Node Identity

- id: `B4`
- name: `trigonometric_functions_and_unit_circle`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets trigonometric functions through unit-circle structure, periodic behavior, and coordinate relationships, and connects angle measure, graph behavior, and function values rather than treating trigonometry as disconnected triangle formulas.

## Dependencies

- `H4`
- `A4`
- `K1`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. explains sine, cosine, and tangent values on the unit circle and connects them to angle and coordinate structure
2. interprets and graphs periodic trigonometric functions in bounded cases using amplitude, period, phase, and symmetry ideas
3. connects right-triangle ratios to unit-circle definitions and extends trigonometric reasoning beyond acute angles
4. solves and interprets bounded trigonometric-function questions using structural periodic reasoning rather than only memorized identities
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.81`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify and explain unit-circle coordinates and corresponding trigonometric values in bounded angle sets
- graph and interpret bounded sine, cosine, and tangent behavior using structural features
- connect periodic graph behavior to angle movement around the unit circle
- solve bounded trigonometric-function questions while preserving periodic and coordinate meaning


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
