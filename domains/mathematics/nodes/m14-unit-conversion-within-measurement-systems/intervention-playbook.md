# M14 Intervention Playbook

## Intervention Principles

- intervention must match the dominant failure mode
- corrective work must preserve the structural meaning of the node
- progression resumes only after local re-verification

## Bounded Interventions

- `larger-versus-smaller unit ladders`: Trace how the count changes when the unit size changes for the same quantity.
- `same-quantity relabeling tasks`: Rewrite one measured quantity in multiple units while preserving meaning.
- `estimate-before-convert routines`: Estimate the direction and rough size of the conversion before computing.
