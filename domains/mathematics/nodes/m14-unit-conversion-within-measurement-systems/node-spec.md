# M14 Node Specification

## Node Identity

- id: `M14`
- name: `unit_conversion_within_measurement_systems`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner converts between related measurement units within the same system by reasoning about unit size and multiplicative structure rather than by isolated formula use.

## Dependencies

- `M8`
- `M13`
- `T4`
- `V2`

## Exit Conditions

The learner is secure on `M14` only when all of the following are true:

1. converts within measurement systems using unit-size reasoning
2. explains the multiplicative relationship between larger and smaller units
3. judges whether a conversion result is plausible before accepting it
4. succeeds across length, mass, and volume bounded conversion tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- converts between related measurement units within one system in bounded cases
- explains why smaller units produce larger numerical counts for the same quantity
- compares equivalent measurements expressed in different units
- checks whether a converted value is reasonable for the original quantity

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
