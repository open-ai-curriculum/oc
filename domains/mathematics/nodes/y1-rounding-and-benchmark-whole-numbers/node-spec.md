# Y1 Node Specification

## Node Identity

- id: `Y1`
- name: `rounding_and_benchmark_whole_numbers`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses benchmark numbers and rounding to represent whole-number magnitude approximately rather than only exactly.

## Dependencies

- `V2`

## Exit Conditions

The learner is secure on `Y1` only when all of the following are true:

1. rounds whole numbers to stated benchmark places accurately
2. uses benchmark numbers to support approximate reasoning
3. explains the relationship between an exact value and its benchmark representation
4. succeeds across notation, number-line, and context tasks
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- rounds bounded whole numbers to stated place-value benchmarks
- uses benchmark values to describe approximate magnitude
- explains why a rounded value is reasonable for the original quantity
- connects benchmark choice to problem context and scale

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
