# Y1 Verification Model

## Verification Gates

- `G1` direct performance gate
- `G2` meaning gate
- `G3` transfer gate
- `G4` retention gate

## Evidence Targets

- rounds bounded whole numbers to stated place-value benchmarks
- uses benchmark values to describe approximate magnitude
- explains why a rounded value is reasonable for the original quantity
- connects benchmark choice to problem context and scale

## Minimum Verification Rule

A learner does not clear `Y1` until direct accuracy, explanation, transfer, and delayed retention evidence all meet threshold.
