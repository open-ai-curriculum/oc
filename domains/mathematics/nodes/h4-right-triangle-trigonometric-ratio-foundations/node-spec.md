# H4 Node Specification

## Node Identity

- id: `H4`
- name: `right_triangle_trigonometric_ratio_foundations`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets sine, cosine, and tangent as stable ratios in similar right triangles, connects those ratios to side-angle structure, and uses them to solve bounded geometric problems without reducing the topic to button-pushing.

## Dependencies

- `H2`
- `G7`
- `Q3`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. explains why sine, cosine, and tangent remain stable for a fixed acute angle across similar right triangles
2. identifies opposite, adjacent, and hypotenuse correctly relative to a chosen acute angle
3. computes and uses trigonometric ratios to find unknown sides or angles in bounded right-triangle settings
4. interprets inverse trigonometric use and reasonableness of results in introductory geometric or contextual problems
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.83`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- explain ratio stability using similar right triangles
- identify correct side-angle relationships for sine, cosine, and tangent
- solve bounded right-triangle side and angle problems using trigonometric ratios
- verify that chosen ratios and resulting measures are reasonable for the given diagram or context


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
