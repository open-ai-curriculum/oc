# Intervention Playbook

## `ratio_selection_mismatch`

- use angle-side identification routines before selecting a trigonometric ratio
- compare matched right-triangle problems where only the known/unknown pair changes and the correct ratio changes with it
- require learners to state the ratio relationship in words before computing

## `similarity_ratio_opacity`

- build matched similar right triangles and compare side ratios for the same acute angle
- use teacher think-alouds connecting trig ratios to similarity rather than calculator procedures
- compare fixed-angle triangles of different sizes to show ratio stability explicitly

## `side_reference_drift`

- relabel opposite and adjacent after changing the reference angle in the same triangle
- use color or marker routines that tie each side name to one chosen angle
- compare correct and swapped side labels in matched examples

## `inverse_reasonableness_gap`

- estimate whether the answer should be an acute or obtuse angle and whether the side should be shorter or longer before calculation
- compare computed results against diagram constraints and benchmark triangles
- require a post-computation reasonableness statement for each nontrivial trig problem
