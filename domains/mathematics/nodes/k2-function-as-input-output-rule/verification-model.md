# Verification Model

This node is verified through four gate types.

## Gates

- `G1 direct performance`: learner meets at least 0.88 accuracy on direct tasks
- `G2 meaning`: learner explains input-output and function-rule structure coherently
- `G3 transfer`: learner succeeds in at least one less-familiar function context
- `G4 retention`: learner passes a delayed recheck after 21 days

## Item Families

- direct function-identification or input-output tasks
- explanation and justification
- error diagnosis
- transfer in a varied function context
