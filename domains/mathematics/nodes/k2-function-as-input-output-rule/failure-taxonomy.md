# Failure Taxonomy

- `input_output_role_blur`: Confuses which quantity is treated as input and which as output in a function situation.
- `single_output_condition_miss`: Fails to use the condition that each input must have one output.
- `representation_function_disconnect`: Can recognize a function in one representation but not in another.
- `context_rule_transfer_breakdown`: Can identify symbolic or tabular functions but loses the rule structure in contextual situations.
