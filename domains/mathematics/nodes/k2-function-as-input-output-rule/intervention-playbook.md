# Intervention Playbook

## `input_output_role_blur`

- label inputs and outputs explicitly before interpreting the relation
- use mapping diagrams and input-output tables with role highlighting
- compare correct and reversed input-output assignments in matched situations

## `single_output_condition_miss`

- sort examples into functions and non-functions with explicit one-input-to-one-output justification
- use mapping diagrams where one input maps to multiple outputs as contrast cases
- teacher think-alouds naming why repeated outputs are allowed but repeated inputs with different outputs break function status

## `representation_function_disconnect`

- translate one function across verbal rule, table, graph, and symbolic form while preserving the same input-output meaning
- require learners to restate the rule after each representation shift
- compare correct and incorrect cross-representation function matches

## `context_rule_transfer_breakdown`

- use cost, distance, and measurement contexts with explicit "for each input, what output results?" routines
- pair each contextual relationship with a table or mapping diagram
- teacher compare-and-connect routines across symbolic, tabular, graph, and contextual function situations
