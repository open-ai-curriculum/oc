# Failure Taxonomy

- `angle_coordinate_period_fragmentation`: Uses angle, coordinate, and graph information separately without preserving their trigonometric relationship.
- `unit_circle_period_opacity`: Memorizes values or graph shapes without understanding how periodic behavior arises from unit-circle motion.
- `quadrant_sign_misread`: Assigns incorrect signs or values because quadrant structure is not tracked correctly.
- `triangle_unit_circle_disconnect`: Treats right-triangle ratios and unit-circle function definitions as separate systems instead of one coherent trigonometric structure.
