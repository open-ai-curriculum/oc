# Intervention Playbook

## `angle_coordinate_period_fragmentation`

- connect angle position, unit-circle coordinates, and graph points in one matched routine
- compare the same angle shown on the unit circle and on a sine/cosine graph side by side
- require learners to narrate how one angle movement changes the corresponding function value

## `unit_circle_period_opacity`

- use full-rotation tracing routines to show how repeated angle movement creates periodic graphs
- compare one-cycle and multi-cycle trigonometric graphs with their unit-circle source
- use teacher think-alouds naming why the graph repeats and where symmetry appears

## `quadrant_sign_misread`

- mark quadrants and expected sign patterns before computing values
- compare matched angles with the same reference angle in different quadrants
- require learners to predict sign and rough magnitude before using exact values

## `triangle_unit_circle_disconnect`

- pair right-triangle ratio reasoning with unit-circle definitions for the same trig function
- compare acute-angle triangle values with corresponding unit-circle values to show extension beyond acute cases
- require learners to explain how triangle-based and unit-circle-based reasoning agree in overlapping cases
