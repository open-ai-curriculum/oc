# B6 Node Specification

## Node Identity

- id: `B6`
- name: `trigonometric_identities_equations_and_analytic_trig`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner extends unit-circle trigonometry into analytic trigonometric reasoning, using identities, equivalent forms, transformed graphs, and equation solving to treat trigonometry as a structured function system rather than a table of formulas.

## Dependencies

- `B4`
- `A2`
- `K2`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. rewrites bounded trigonometric expressions using fundamental identities and explains why the rewritten forms preserve meaning
2. solves bounded trigonometric equations over stated intervals using periodic structure, inverse reasoning, and unit-circle logic
3. interprets transformed trigonometric graphs using amplitude, period, phase shift, symmetry, and equivalent analytic forms
4. connects identities, graphs, and equation-solving moves so analytic trigonometric reasoning remains representation-consistent
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.81`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- rewrite and justify bounded trigonometric expressions using core identities and equivalent forms
- solve bounded trigonometric equations and explain the role of periodicity in the solution set
- interpret transformed sine, cosine, and tangent graphs using analytic and graphical features together
- connect unit-circle values, identities, graph structure, and equation solving in one coherent analysis


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
