# Intervention Playbook

## `limit_process_static_view`

- compare tables and graphs where nearby behavior matters more than function value at a point
- use one-sided and two-sided approach routines before any symbolic simplification
- require learners to describe what values are doing near the target before naming the limit

## `derivative_meaning_fragmentation`

- compare slope, velocity, and local linear approximation interpretations of the same derivative statement
- use secant-to-tangent routines that force the learner to describe what changes as intervals shrink
- require learners to state what the derivative means in context before computing or simplifying

## `symbolic_graphical_motion_disconnect`

- compare derivative sign charts, graphs, and motion descriptions for the same situation and identify where they agree or conflict
- use matched examples where a symbolic error creates a graph or motion claim that is obviously inconsistent
- require learners to reconcile algebraic work with a graphical or contextual interpretation before closure

## `optimization_local_global_confusion`

- compare bounded optimization tasks where derivative-zero points are relevant but not sufficient without interval or context review
- use routines that separate local evidence, endpoint evidence, and contextual constraints
- require learners to justify whether the claim is about instantaneous change, local extremum, or whole-interval behavior
