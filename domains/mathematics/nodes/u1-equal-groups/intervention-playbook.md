# U1 Intervention Playbook

## Purpose

Map `U1` failure modes to bounded intervention responses.

## Intervention Rules

### For U1-FM-01 Same-Group-Count Confusion

Primary interventions:

- contrast "same number of groups" with "same number in each group"
- compare matched and mismatched group sets
- use teacher language naming group size explicitly

Expected effect:

- learner bases equality on group size, not group count

Reverification target:

- one direct equal-group item and one explanation item

### For U1-FM-02 Group-Size Drift

Primary interventions:

- count each group after making it
- line up groups side by side for comparison
- adjust one mismatched group to match the others

Expected effect:

- learner preserves the same size across groups

Reverification target:

- one production task

### For U1-FM-03 Production Instability

Primary interventions:

- use small sets first
- distribute one object per group in rounds
- mark group spaces before placing objects

Expected effect:

- learner can construct equal groups, not only recognize them

Reverification target:

- one direct production item

### For U1-FM-04 Context Boundary

Primary interventions:

- move from objects to pictures to short stories using the same group language
- ask what makes the groups equal in each case
- use paired object and story examples

Expected effect:

- learner transfers equal-group reasoning across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to dependency review and inspect upstream node stability rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when attention, language, confidence, or context suggests a different explanation of the learner state.
