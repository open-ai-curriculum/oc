# U1 Failure Taxonomy

### U1-FM-01 Same-Group-Count Confusion

- description: learner treats the number of groups as the equality condition rather than the size of each group

### U1-FM-02 Group-Size Drift

- description: learner does not preserve the same number in each group

### U1-FM-03 Production Instability

- description: learner can identify equal groups but cannot reliably make them

### U1-FM-04 Context Boundary

- description: learner reasons about equal groups in direct object tasks but not in story or picture contexts
