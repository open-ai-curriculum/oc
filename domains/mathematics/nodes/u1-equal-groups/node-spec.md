# U1 Node Specification

## Node Identity

- id: `U1`
- name: `equal_groups`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner understands that equal groups have the same number of objects and can distinguish those groups from unequal groupings.

## Dependencies

- `C3`

## Exit Conditions

The learner is secure on `U1` only when all of the following are true:

1. identifies equal and unequal groups
2. makes equal groups from a set or critiques existing groupings
3. explains that the groups are equal because each has the same number
4. succeeds across concrete objects, drawings, and simple story contexts
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- tell whether groups match in size
- make equal groups from a set of objects
- explain why one mismatched group breaks equal-group structure
- support later repeated-unit reasoning by stabilizing same-sized grouping

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
