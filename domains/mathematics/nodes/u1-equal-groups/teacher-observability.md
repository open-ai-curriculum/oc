# U1 Teacher Observability View

Teachers should see:

- current `U1` status
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner is anchoring equality in group size rather than number of groups
- recommended intervention

Class-level view should highlight:

- who confuses number of groups with equal group size
- who can recognize equal groups but not build them
- who loses equal-group reasoning in story contexts
