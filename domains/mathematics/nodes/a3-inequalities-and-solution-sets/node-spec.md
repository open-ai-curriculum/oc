# A3 Node Specification

## Node Identity

- id: `A3`
- name: `inequalities_and_solution_sets`
- domain: `mathematics`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner interprets inequalities as sets of values satisfying a condition and represents those sets through number-line, symbolic, and contextual reasoning rather than treating inequality symbols as one-answer equation markers.

## Dependencies

- `E1`
- `S1`
- `L4`


## Exit Conditions

The learner is secure on this node only when all of the following are true:

1. interprets an inequality as describing a range or set of values rather than a single answer
2. solves bounded one-step inequalities while preserving comparison meaning
3. represents and interprets solution sets on a number line with correct directional and endpoint meaning
4. succeeds across symbolic, number-line, verbal, and simple contextual representations
5. passes a delayed recheck after spacing


## Mastery Criteria

- direct accuracy threshold: `0.88`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- solve bounded one-step inequalities and explain the comparison relationship that remains true
- represent solution sets with correct open or closed endpoint conventions in bounded cases
- explain why many values can satisfy an inequality and test candidate values appropriately
- connect symbolic inequalities to directional number-line and contextual threshold representations


## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
