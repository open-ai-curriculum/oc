# Failure Taxonomy

- `equation_inequality_collapse`: Treats an inequality as if it should yield one equation-like answer rather than a set of satisfying values.
- `solution_set_as_single_value`: Identifies a boundary value but not the full range of values that satisfy the inequality.
- `endpoint_direction_confusion`: Misuses open or closed endpoints or shades the wrong direction on the number line.
- `context_threshold_breakdown`: Can solve symbolically but loses meaning in verbal or threshold-based contexts.
