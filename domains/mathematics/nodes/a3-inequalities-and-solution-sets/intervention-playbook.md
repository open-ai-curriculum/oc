# Intervention Playbook

## `equation_inequality_collapse`

- compare matched equations and inequalities to surface one-solution versus many-solution structure
- test multiple candidate values after solving, not only the boundary value
- use balance-to-comparison bridge tasks that preserve meaning language

## `solution_set_as_single_value`

- generate true and false value sets for one inequality before graphing
- sort candidate values into satisfies or does-not-satisfy groups
- teacher rehearsal of range language such as greater than, less than, at least, and at most

## `endpoint_direction_confusion`

- match verbal phrases to open or closed endpoint conventions
- number-line graphing routines with explicit endpoint justification
- compare nearly identical inequalities that differ only by strict versus inclusive language

## `context_threshold_breakdown`

- translate contextual limits such as temperature, budget, or score thresholds into symbolic inequalities
- map symbolic inequalities back into decision or threshold statements
- teacher compare-and-connect routines across symbolic and contextual inequality forms
