# Mathematics Artifact Attachments

This directory contains machine-readable attachment registries that bind curriculum, assessment, intervention, and support artifacts to graph-native mathematics entities.

## Purpose

These registries operationalize the [graph artifact attachment contract](../graph-artifact-attachment-contract.md) by giving the mathematics graph a concrete attachment layer that can later drive curriculum compilation and adaptive navigation.

## Files

- `registry.schema.json`: canonical JSON Schema for mathematics artifact attachment registries
- `fraction-spine-attachment-registry.yaml`: initial draft registry for the `F1` through `F5` fraction spine
- `assessment-surface-attachment-registry.yaml`: draft registry for ACT and IB assessment-surface artifacts attached to maintained overlays and overlay units

## Claim Boundary

The registry entries in this directory are draft attachment records. They define governed attachment intent and graph linkage. They do not by themselves prove that the referenced curriculum assets are authored, reviewed, or classroom-ready.
