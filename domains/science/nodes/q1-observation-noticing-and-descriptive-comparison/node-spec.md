# Q1 Node Specification

## Node Identity

- id: `Q1`
- name: `observation_noticing_and_descriptive_comparison`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices observable features of objects, organisms, materials, and events, then describes similarities, differences, and visible changes using direct observation rather than unsupported guessing.

## Dependencies

None

## Exit Conditions

The learner is secure on `Q1` only when all of the following are true:

1. notices relevant visible properties in common science contexts
2. compares two observations accurately using descriptive language
3. identifies visible change over time or after an event
4. succeeds across classroom objects, images, and familiar real-world settings
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- name what is observed before making a claim
- compare two objects or events using observable features
- note that something changed and describe the visible difference
- separate direct observation from unsupported inference at a basic level

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
