# Q1 Intervention Playbook

## Purpose

Map `Q1` failure modes to bounded intervention responses.

## Intervention Rules

### For Q1-FM-01 Feature Blindness

Primary interventions:

- slow the observation task and direct attention to one feature at a time
- contrast two items with one highly visible difference
- use teacher think-alouds naming what is directly seen

Expected effect:

- learner notices relevant observable features more reliably

Reverification target:

- one direct observation item

### For Q1-FM-02 Inference-Observation Confusion

Primary interventions:

- separate "what we see" from "what we think" in paired examples
- ask whether the property is visible, testable, or only guessed
- model sentence frames beginning with "I notice"

Expected effect:

- learner distinguishes direct observation from unsupported inference

Reverification target:

- one explanation item and one error-diagnosis item

### For Q1-FM-03 Comparison Language Instability

Primary interventions:

- rehearse same, different, changed, and not changed with concrete examples
- use side-by-side comparison cards
- prompt the learner to name one same feature and one different feature

Expected effect:

- learner describes comparisons with more stable language

Reverification target:

- one direct comparison item

### For Q1-FM-04 Context Boundary

Primary interventions:

- repeat the same observation routine across objects, photos, and outdoor examples
- preserve the question while changing the materials
- move from familiar to slightly less familiar settings

Expected effect:

- learner transfers observation routines across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for language load, attentional demands, or task-design mismatch rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when language access, sensory access, attention, or context suggests a different explanation of the learner state.
