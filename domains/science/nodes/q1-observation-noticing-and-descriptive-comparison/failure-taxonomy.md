# Q1 Failure Taxonomy

### Q1-FM-01 Feature Blindness

- description: learner misses an observable feature that is necessary for comparison

### Q1-FM-02 Inference-Observation Confusion

- description: learner reports a guess or hidden property as if it were directly observed

### Q1-FM-03 Comparison Language Instability

- description: learner notices features but cannot state sameness, difference, or change reliably

### Q1-FM-04 Context Boundary

- description: learner can observe in one familiar setting but not in a changed material or environmental context
