# L2 Intervention Playbook

## Purpose

Map `L2` failure modes to bounded intervention responses.

## Intervention Rules

### For L2-FM-01 Life-Cycle Ordering Breakdown

Primary interventions:

- sequence life-cycle images from early to later stages
- compare what changes from one stage to the next
- use arrows and stage labels on a simple life-cycle model

Expected effect:

- learner orders and interprets life-cycle stages more reliably

Reverification target:

- one life-cycle item

### For L2-FM-02 Trait-Sharing Confusion

Primary interventions:

- compare parent-offspring pairs and name shared visible traits
- contrast shared traits with differences caused by age or context
- use "same as parent / different from parent" sorting routines

Expected effect:

- learner identifies shared traits more reliably

Reverification target:

- one trait-sharing item

### For L2-FM-03 Ecosystem-Relationship Disconnect

Primary interventions:

- pair organisms with the support they give or receive in a simple ecosystem
- ask how this living thing helps or depends on the other
- use arrows or simple system maps for local ecosystem examples

Expected effect:

- learner explains ecosystem relationships more reliably

Reverification target:

- one ecosystem-relationship item

### For L2-FM-04 Context Boundary

Primary interventions:

- repeat the same reasoning routine across plants, animals, and pond or garden systems
- preserve the life-systems structure while changing the example
- fade scaffolds after one successful familiar example

Expected effect:

- learner transfers life-science system reasoning across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for background-knowledge mismatch, language load, or prompt-design mismatch rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when local ecological context, language access, or cultural familiarity with organisms suggests a different explanation of the learner state.
