# L2 Failure Taxonomy

### L2-FM-01 Life-Cycle Ordering Breakdown

- description: learner cannot reliably order or interpret life-cycle stages

### L2-FM-02 Trait-Sharing Confusion

- description: learner cannot distinguish shared inherited traits from unrelated similarities

### L2-FM-03 Ecosystem-Relationship Disconnect

- description: learner cannot explain a basic organism-environment or organism-organism relationship

### L2-FM-04 Context Boundary

- description: learner can reason in one familiar organism context but not in a changed one
