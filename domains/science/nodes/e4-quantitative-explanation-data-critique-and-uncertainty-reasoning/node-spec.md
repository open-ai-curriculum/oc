# E4 Node Specification

## Node Identity

- id: `E4`
- name: `quantitative_explanation_data_critique_and_uncertainty_reasoning`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner builds and critiques scientific explanations using quantitative evidence while calibrating claims to uncertainty, method quality, and model fit.

## Dependencies

- `E3` model-based argument, revision, and source comparison
- `I4` multivariable investigation, sampling, and error awareness

## Exit Conditions

The learner is secure on `E4` only when all of the following are true:

1. integrates quantitative evidence into explanations without treating numbers as self-interpreting
2. critiques explanations using uncertainty and method-quality reasoning
3. calibrates claims so they are neither overstated nor dismissed without cause
4. succeeds across more than one disciplinary explanation context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `35`

## Observable Evidence

The learner can:

- explain what a quantitative pattern does and does not support
- critique claims using sample quality, measurement limits, and uncertainty
- revise explanations when stronger evidence or better models appear
- preserve evidence-claim alignment across multiple science domains

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
