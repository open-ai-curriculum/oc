# E4 Intervention Playbook

- For `E4-FM-01`, use claim-from-graph routines where numbers must be translated into bounded verbal claims.
- For `E4-FM-02`, use evidence-weight comparison tasks that rank studies or datasets by methodological strength.
- For `E4-FM-03`, use uncertainty-calibration prompts that require both a supported claim and an explicit limitation.
- For `E4-FM-04`, use explanation-revision cycles where critique must change the stated conclusion.
