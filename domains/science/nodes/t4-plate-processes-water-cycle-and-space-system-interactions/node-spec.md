# T4 Node Specification

## Node Identity

- id: `T4`
- name: `plate_processes_water_cycle_and_space_system_interactions`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains large-scale Earth and space processes using bounded system models, including plate-driven change, water-cycle movement, and space-system interactions.

## Dependencies

- `T3` climate, resources, Earth-Sun-Moon systems, and map scale
- `M2` mechanism, cause-effect, and process models
- `E3` model-based argument, revision, and source comparison

## Exit Conditions

The learner is secure on `T4` only when all of the following are true:

1. explains at least one surface or geological change using a plate or Earth-process model
2. traces water movement through more than one part of the water cycle
3. uses system interaction language for a bounded space or Earth-space case
4. succeeds across more than one geoscience context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- connect geological features or events to underlying Earth processes in bounded models
- explain how water moves among reservoirs without treating the cycle as a single loop picture only
- interpret interactions among Earth systems or space systems with controlled cause-effect language
- compare geoscience explanations against models and available evidence

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
