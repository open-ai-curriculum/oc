# C4 Failure Taxonomy

- `C4-FM-01` rate-equilibrium confusion: learner confuses reaction speed with equilibrium position or completion.
- `C4-FM-02` static-equilibrium belief: learner treats equilibrium as no activity rather than dynamic balance.
- `C4-FM-03` acid-base label recall only: learner uses acid or base labels without transfer-model explanation.
- `C4-FM-04` condition-effect overgeneralization: learner states condition changes without bounded system reasoning.
