# C4 Node Specification

## Node Identity

- id: `C4`
- name: `kinetics_equilibrium_and_acid_base_systems`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains reversible chemical systems through rate, equilibrium, and proton-transfer reasoning without treating dynamic systems as static endpoints only.

## Dependencies

- `C3`
- `M4`

## Exit Conditions

The learner is secure on `C4` only when all of the following are true:

1. explains reaction rate changes using collision or system-condition reasoning
2. explains equilibrium as dynamic balance rather than reaction completion only
3. uses acid-base models to explain proton transfer and system change in bounded cases
4. succeeds across more than one advanced chemistry context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- distinguish rate from extent or completion of reaction
- explain dynamic equilibrium using forward and reverse process reasoning
- connect acid-base behavior to transfer models and system conditions
- avoid overgeneralized catalyst or pH claims

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
