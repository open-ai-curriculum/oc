# M2 Node Specification

## Node Identity

- id: `M2`
- name: `mechanism_cause_effect_and_process_models`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains a science phenomenon by tracing a simple process or mechanism, linking causes to effects in an ordered way rather than only naming the end result.

## Dependencies

- `M1` part-whole structure, function, and simple system models
- `E1` observation-based explanation and evidence talk

## Exit Conditions

The learner is secure on `M2` only when all of the following are true:

1. identifies a plausible process or mechanism underlying an observed outcome
2. links causes and effects in a coherent sequence
3. uses a model to represent the process rather than only the result
4. succeeds across more than one science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- explain what happens in a process step by step
- connect an observed cause to an observed effect
- use a simple mechanism model to support the explanation
- avoid replacing mechanism with labels alone

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
