# M2 Failure Taxonomy

### M2-FM-01 Process-Step Collapse

- description: learner cannot order or sustain the steps of the process coherently

### M2-FM-02 Cause-Effect Reversal Or Blur

- description: learner confuses the cause and the effect or leaves the link unclear

### M2-FM-03 Mechanism Omission

- description: learner names the result but does not explain how the process works

### M2-FM-04 Context Boundary

- description: learner can explain mechanism in one familiar context but not in a changed one
