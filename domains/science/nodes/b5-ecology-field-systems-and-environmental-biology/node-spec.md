# B5 Node Specification

## Node Identity

- id: `B5`
- name: `ecology_field_systems_and_environmental_biology`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains ecological and field-based biological systems using interacting populations, abiotic constraints, measurement evidence, and environmental change models without collapsing outcomes to one variable.

## Dependencies

- `L4`
- `L6`
- `T5`
- `I4`

## Exit Conditions

The learner is secure on `B5` only when all of the following are true:

1. explains ecological change using interacting biotic and abiotic factors
2. uses field evidence and sampling logic in environmental-biology claims
3. connects ecosystem outcomes to environmental-system change without overclaiming certainty
4. succeeds across more than one field-biology or ecology context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- integrate population, resource, and abiotic-condition reasoning
- use field methods and sampling limits when evaluating ecosystem claims
- connect local biological observations to larger environmental-system models appropriately
- avoid single-cause explanations for ecological change

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
