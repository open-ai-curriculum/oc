# M1 Item Bank Schema

## Required Item Families

### Family A. Parts Models And Function

- identify parts within a whole
- use a simple model or diagram
- match structure to function

### Family B. Explanation

- explain what the model represents and how the structure supports the function

### Family C. Error Diagnosis

- identify when a model leaves out a critical part or mismaps the system

### Family D. Transfer

- reason across organism, object, and Earth-system examples
