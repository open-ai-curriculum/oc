# M1 Intervention Playbook

## Purpose

Map `M1` failure modes to bounded intervention responses.

## Intervention Rules

### For M1-FM-01 Part-Whole Collapse

Primary interventions:

- highlight one important part at a time within the whole
- label parts on a simple diagram
- contrast important and non-critical details for the same model

Expected effect:

- learner identifies meaningful system parts more reliably

Reverification target:

- one parts item

### For M1-FM-02 Model-Referent Confusion

Primary interventions:

- point from the model to the real object or picture repeatedly
- use sentence frames such as "this part of the model stands for..."
- compare a model with the thing it represents

Expected effect:

- learner maps the model to the referent more accurately

Reverification target:

- one model explanation item

### For M1-FM-03 Structure-Function Disconnect

Primary interventions:

- pair each visible structure with one concrete function
- ask what this part helps the system do
- compare two structures with different jobs

Expected effect:

- learner connects structure and function more reliably

Reverification target:

- one structure-function item

### For M1-FM-04 Context Boundary

Primary interventions:

- repeat the same modeling routine across organisms, objects, and Earth/space examples
- preserve the part-function structure while changing the domain
- fade scaffolds after a successful familiar example

Expected effect:

- learner transfers simple modeling across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for spatial-representation load, language load, or prompt-design mismatch rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when drawing demands, spatial access, or response-mode constraints suggest a different explanation of the learner state.
