# M1 Node Specification

## Node Identity

- id: `M1`
- name: `part_whole_structure_function_and_simple_system_models`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-bridge-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses a simple model, drawing, diagram, or representation to show parts of a system and explain how a structure or part supports a function in a familiar science context.

## Dependencies

- `Q3` phenomenon framing and shared sensemaking routines

## Exit Conditions

The learner is secure on `M1` only when all of the following are true:

1. identifies meaningful parts within a whole system
2. uses a simple model or diagram that maps to the real system
3. explains how a structure or part supports a function
4. succeeds across more than one science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- use a drawing or diagram to represent a simple system
- identify parts of a plant, animal, object, or Earth system component
- explain a basic structure-function relationship
- avoid treating the model as the thing itself without mapping correspondences

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
