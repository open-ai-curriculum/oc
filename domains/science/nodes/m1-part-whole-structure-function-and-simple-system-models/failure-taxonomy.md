# M1 Failure Taxonomy

### M1-FM-01 Part-Whole Collapse

- description: learner cannot reliably distinguish important parts of the system from the whole

### M1-FM-02 Model-Referent Confusion

- description: learner cannot explain what the model stands for or treats the model as the thing itself

### M1-FM-03 Structure-Function Disconnect

- description: learner notices a part but cannot connect it to what it does

### M1-FM-04 Context Boundary

- description: learner can use a simple model in one familiar context but not in a changed one
