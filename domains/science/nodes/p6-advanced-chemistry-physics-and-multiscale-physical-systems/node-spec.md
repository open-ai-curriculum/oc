# P6 Node Specification

## Node Identity

- id: `P6`
- name: `advanced_chemistry_physics_and_multiscale_physical_systems`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner integrates chemistry and physics explanations across scales using quantitative evidence, coordinated models, and disciplined critique.

## Dependencies

- `P5` fields, atomic interactions, and quantitative conservation models
- `I5` quantitative investigation design, uncertainty, and method critique
- `E5` cross-domain synthesis using multiple models and evidence streams

## Exit Conditions

The learner is secure on `P6` only when all of the following are true:

1. coordinates microscopic and macroscopic physical explanations in bounded systems
2. integrates chemistry and physics models without false equivalence or vocabulary stacking
3. critiques advanced physical explanations using quantitative evidence, uncertainty, and model limits
4. succeeds across more than one advanced physical-systems context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- connect atomic-scale interaction models to observable large-scale physical behavior
- coordinate multiple physical models while stating what each contributes
- critique multiscale physical explanations for evidence mismatch or model overreach
- synthesize advanced chemistry and physics evidence streams in a bounded account

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
