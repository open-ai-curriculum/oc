# P6 Learner State Model

The learner-state for `P6` tracks:

- dependency security on `P5`, `I5`, and `E5`
- direct accuracy on multiscale physical explanation, model coordination, and critique tasks
- explanation quality for advanced chemistry and physics synthesis
- transfer performance across unfamiliar advanced physical contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
