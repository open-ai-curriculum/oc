# A1 Learner State Model

The learner-state for `A1` tracks:

- dependency security on `L6`, `P6`, `T6`, and `E5`
- direct accuracy on interdisciplinary phenomenon explanation, model coordination, and critique tasks
- explanation quality for coordinated multi-domain synthesis
- transfer performance across unfamiliar integrated phenomena
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
