# A1 Failure Taxonomy

- `A1-FM-01` partial-domain explanation: learner explains an integrated phenomenon using only one or two necessary domain strands.
- `A1-FM-02` model-role collapse: learner coordinates multiple models without stating what each contributes or limits.
- `A1-FM-03` pseudo-interdisciplinary evidence gap: learner uses multiple domain labels without matching evidence across strands.
- `A1-FM-04` phenomenon-scale overreach: learner makes interdisciplinary claims broader than the coordinated evidence supports.
