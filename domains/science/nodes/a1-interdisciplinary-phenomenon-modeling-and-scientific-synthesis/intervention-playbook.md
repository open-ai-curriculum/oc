# A1 Intervention Playbook

- For `A1-FM-01`, use phenomenon-to-domain mapping where learners must identify every required strand before explaining.
- For `A1-FM-02`, use model-role matrices with one row per domain model and explicit contribution and limit fields.
- For `A1-FM-03`, use evidence-coverage audits that reveal which strands are evidenced and which are only named.
- For `A1-FM-04`, use claim-scope reviews that compare interdisciplinary breadth with actual coordinated support.
