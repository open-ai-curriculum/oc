# D1 Node Package

## Node

`D1` - Problem definition, criteria, constraints, and solution sketching

## Purpose

This package defines the learner's early engineering-design readiness in the domain.

It operationalizes the learner's ability to define a problem, identify criteria and constraints, and sketch a simple solution that remains tied to the problem conditions.
