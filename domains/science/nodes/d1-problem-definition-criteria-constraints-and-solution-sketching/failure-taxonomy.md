# D1 Failure Taxonomy

### D1-FM-01 Problem Framing Blur

- description: learner cannot state the problem clearly enough to guide a design

### D1-FM-02 Criteria-Constraint Confusion

- description: learner confuses success conditions with limits or materials constraints

### D1-FM-03 Solution-Fit Breakdown

- description: learner proposes a solution that does not actually fit the stated criteria or constraints

### D1-FM-04 Context Boundary

- description: learner can reason through one familiar design problem but not a changed one
