# F4 Node Specification

## Node Identity

- id: `F4`
- name: `electrostatics_circuits_and_magnetic_fields`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains electric and magnetic phenomena using field, circuit, and interaction models without collapsing electricity into current-only or formula-only reasoning.

## Dependencies

- `P5`
- `M5`
- `I4`

## Exit Conditions

The learner is secure on `F4` only when all of the following are true:

1. explains electric interactions and fields using bounded system models
2. explains circuits through potential, current, resistance, and system relations
3. connects magnetic behavior to moving charge or field interaction models in bounded cases
4. succeeds across more than one advanced electricity and magnetism context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- distinguish charge, field, potential, current, and resistance roles appropriately
- reason about circuits as systems rather than isolated component facts
- connect magnetic effects to interaction models instead of memorized direction rules only
- avoid overextending one electrical relation to all contexts

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
