# B1 Node Specification

## Node Identity

- id: `B1`
- name: `biomolecules_cellular_structure_and_metabolism`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains cellular function using biomolecule structure, membrane transport, organelle roles, and metabolic energy-transformation models without collapsing cell behavior into memorized part lists.

## Dependencies

- `L3`
- `L5`
- `P5`
- `M4`

## Exit Conditions

The learner is secure on `B1` only when all of the following are true:

1. explains how biomolecule structure constrains function in bounded biological cases
2. connects organelles, membranes, and transport processes into coherent cellular-system explanations
3. uses metabolism models to explain energy capture, release, or transfer in cells
4. succeeds across more than one advanced cell-biology context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- distinguish structural descriptions from functional cellular explanations
- explain membrane transport and organelle roles using mechanism language
- connect ATP, respiration, or photosynthetic energy transformations to cell activity
- coordinate molecular and cellular evidence without scale collapse

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
