# G5 Intervention Playbook

- For `G5-FM-01`, use tradeoff matrices that require multiple sustainability criteria to be compared.
- For `G5-FM-02`, use impact-claim audits with explicit evidence and uncertainty checks.
- For `G5-FM-03`, use human-environment system maps connecting causes, constraints, and outcomes.
- For `G5-FM-04`, use source-evaluation routines for environmental data and argument quality.
