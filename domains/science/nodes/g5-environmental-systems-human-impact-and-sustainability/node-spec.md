# G5 Node Specification

## Node Identity

- id: `G5`
- name: `environmental_systems_human_impact_and_sustainability`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner evaluates environmental systems and human impacts using evidence, tradeoff, and sustainability reasoning without collapsing environmental decisions into one-variable advocacy claims.

## Dependencies

- `T5`
- `L4`
- `E4`
- `D4`

## Exit Conditions

The learner is secure on `G5` only when all of the following are true:

1. explains environmental-system change using interacting biophysical and human factors
2. evaluates impact claims with explicit evidence and uncertainty posture
3. uses sustainability reasoning to compare tradeoffs in bounded environmental cases
4. succeeds across more than one advanced environmental-science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- connect biological and Earth-system evidence in environmental explanations
- compare tradeoffs and consequences rather than endorsing one metric only
- use uncertainty and source-quality language in impact claims
- avoid substituting position statements for evidence-based system reasoning

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
