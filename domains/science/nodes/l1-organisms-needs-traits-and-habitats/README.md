# L1 Node Package

## Node

`L1` - Organisms, needs, traits, and habitats

## Purpose

This package defines the earliest life-science readiness in the domain.

It operationalizes the learner's ability to recognize that living things have needs, observable traits, and habitat relationships that can be described and supported with direct evidence.
