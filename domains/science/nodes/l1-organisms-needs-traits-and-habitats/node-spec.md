# L1 Node Specification

## Node Identity

- id: `L1`
- name: `organisms_needs_traits_and_habitats`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner identifies basic organism needs, notices observable traits, and connects living things to habitats that help meet those needs.

## Dependencies

- `Q1` observation, noticing, and descriptive comparison
- `E1` observation-based explanation and evidence talk

## Exit Conditions

The learner is secure on `L1` only when all of the following are true:

1. identifies basic needs such as water, food, air, light, or shelter where appropriate
2. describes observable traits of a living thing using direct evidence
3. links an organism to a habitat feature that helps meet a need
4. succeeds across more than one organism context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- describe what a plant or animal needs to live
- point to visible traits such as color, body parts, or covering
- explain how a habitat supports a living thing using observed features
- avoid relying on fantasy or unsupported personification as evidence

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
