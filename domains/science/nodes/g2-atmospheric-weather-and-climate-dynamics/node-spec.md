# G2 Node Specification

## Node Identity

- id: `G2`
- name: `atmospheric_weather_and_climate_dynamics`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains atmospheric circulation, weather systems, and climate dynamics using Earth-system evidence without collapsing climate reasoning into local weather anecdotes.

## Dependencies

- `T5`
- `M4`
- `I4`

## Exit Conditions

The learner is secure on `G2` only when all of the following are true:

1. explains weather and climate using atmospheric-system models
2. distinguishes event-level weather claims from longer-pattern climate reasoning
3. uses atmospheric evidence and model limits in bounded climate explanations
4. succeeds across more than one advanced atmosphere or climate context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- connect pressure, circulation, energy, and moisture ideas in atmospheric explanations
- separate local event descriptions from broader climate pattern claims
- use evidence and uncertainty language appropriately in climate reasoning
- avoid overgeneralizing from one region or one event

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
