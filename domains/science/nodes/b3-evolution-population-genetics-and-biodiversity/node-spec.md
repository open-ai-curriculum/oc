# B3 Node Specification

## Node Identity

- id: `B3`
- name: `evolution_population_genetics_and_biodiversity`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains evolutionary change and biodiversity using variation, inheritance, selection, drift, and population-level evidence without reverting to need-based organism stories.

## Dependencies

- `B2`
- `L5`
- `E4`

## Exit Conditions

The learner is secure on `B3` only when all of the following are true:

1. explains evolutionary change at the population level using bounded mechanisms
2. uses variation and inheritance evidence to explain biodiversity patterns
3. distinguishes selection, drift, and related evolutionary processes in bounded cases
4. succeeds across more than one biodiversity or population-genetics context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- separate organism survival stories from population-level change explanations
- use variation, allele-frequency, or comparative-evidence language appropriately
- explain biodiversity as an outcome of multiple evolutionary processes and ecological context
- critique overconfident or teleological evolutionary claims

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
