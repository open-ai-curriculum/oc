# B3 Intervention Playbook

- For `B3-FM-01`, use individual-versus-population explanation sorts to remove need-based reasoning.
- For `B3-FM-02`, use mechanism comparison routines separating selection, drift, migration, and mutation roles.
- For `B3-FM-03`, use biodiversity case analyses that require process explanations rather than description only.
- For `B3-FM-04`, use population-evidence checks that tie claims to observable frequency or comparative patterns.
