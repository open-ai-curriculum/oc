# R4 Failure Taxonomy

- `R4-FM-01` simulation-as-reality: learner treats computational output as direct reality rather than model-dependent representation.
- `R4-FM-02` assumption-opacity: learner uses a computational model without identifying key assumptions or limitations.
- `R4-FM-03` output-interpretation drift: learner reads graphs or tables without connecting them to scientific quantities and claims.
- `R4-FM-04` tool-completion substitution: learner treats successful tool execution as evidence of strong scientific reasoning.
