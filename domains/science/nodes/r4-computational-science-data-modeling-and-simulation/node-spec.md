# R4 Node Specification

## Node Identity

- id: `R4`
- name: `computational_science_data_modeling_and_simulation`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses computational tools and simulation to analyze scientific systems and data without treating model output as reality or as a substitute for scientific interpretation.

## Dependencies

- `A2`
- `M5`
- `I5`

## Exit Conditions

The learner is secure on `R4` only when all of the following are true:

1. uses computational models and simulations with explicit assumptions and system meaning
2. connects data workflows, model outputs, and scientific claims in bounded cases
3. evaluates when simulation helps and when it can mislead if its assumptions are ignored
4. succeeds across more than one computational-science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- state model assumptions and representational choices explicitly
- connect computational output back to scientific quantities and evidence
- treat simulation as a bounded explanatory tool rather than self-validating truth
- avoid overreading graphs, dashboards, or outputs without system interpretation

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
