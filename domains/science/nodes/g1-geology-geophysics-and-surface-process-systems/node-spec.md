# G1 Node Specification

## Node Identity

- id: `G1`
- name: `geology_geophysics_and_surface_process_systems`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains rocks, Earth materials, tectonics, and surface processes through geologic-system models without reducing Earth change to isolated landform descriptions.

## Dependencies

- `T4`
- `M3`
- `I3`

## Exit Conditions

The learner is secure on `G1` only when all of the following are true:

1. explains geologic and surface-process change using system and process models
2. connects rock, tectonic, and Earth-material evidence to bounded geologic claims
3. distinguishes short-term surface change from longer geologic-process reasoning
4. succeeds across more than one advanced Earth-science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- connect landforms to causal Earth-process explanations
- use rock and material evidence with explicit process interpretation
- distinguish geologic timescale claims from short-term observation only
- avoid one-process explanations when interacting Earth systems matter

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
