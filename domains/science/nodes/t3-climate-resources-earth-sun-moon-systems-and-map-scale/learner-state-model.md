# T3 Learner State Model

The learner-state for `T3` tracks:

- dependency security on `T2`, `M3`, and `I3`
- direct accuracy on climate, Earth-Sun-Moon, resource, and map-scale tasks
- explanation quality for Earth-space and representation reasoning
- transfer performance in unfamiliar Earth-system contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
