# T3 Node Specification

## Node Identity

- id: `T3`
- name: `climate_resources_earth_sun_moon_systems_and_map_scale`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains recurring Earth-space patterns, uses map scale and system models responsibly, and reasons about climate and resource use without collapsing long-term and short-term phenomena.

## Dependencies

- `T2` Earth surface change, solar/lunar patterns, and water movement
- `M3` scale, proportion, and system interaction models
- `I3` repeated trials, data reliability, and pattern summary

## Exit Conditions

The learner is secure on `T3` only when all of the following are true:

1. distinguishes weather-scale events from climate-scale pattern reasoning in bounded cases
2. uses Earth-Sun-Moon system relationships to explain at least one recurring pattern
3. interprets simple map-scale or resource-use representations without scale collapse
4. succeeds across more than one Earth-space context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- explain why long-term patterns cannot be inferred from one short event alone
- connect Earth-Sun-Moon relationships to observed cycles in bounded models
- use map scale or resource models without treating representations as literal size or time matches
- compare claims about climate or resources against the provided data representation

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
