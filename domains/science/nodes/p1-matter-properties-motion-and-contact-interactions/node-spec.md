# P1 Node Specification

## Node Identity

- id: `P1`
- name: `matter_properties_motion_and_contact_interactions`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner identifies observable material properties, notices simple motion changes, and explains what happens when one object pushes or pulls another through contact.

## Dependencies

- `Q1` observation, noticing, and descriptive comparison
- `E1` observation-based explanation and evidence talk

## Exit Conditions

The learner is secure on `P1` only when all of the following are true:

1. describes observable material properties such as hard, soft, smooth, rough, or flexible
2. identifies visible changes in motion such as starting, stopping, speeding up, or changing direction
3. connects a push or pull to a directly observed motion result
4. succeeds across more than one material or motion context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- sort objects by observable properties
- describe what happened when an object was pushed or pulled
- identify motion outcomes from direct evidence
- avoid treating hidden causes as known without observation

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
