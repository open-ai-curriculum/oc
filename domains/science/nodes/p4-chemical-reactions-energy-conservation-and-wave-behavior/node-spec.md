# P4 Node Specification

## Node Identity

- id: `P4`
- name: `chemical_reactions_energy_conservation_and_wave_behavior`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains chemical reactions, applies conservation reasoning in bounded systems, and interprets wave behavior without reducing all change to surface appearance.

## Dependencies

- `P3` particle models, waves, and force interactions
- `I3` repeated trials, data reliability, and pattern summary
- `M3` scale, proportion, and system interaction models

## Exit Conditions

The learner is secure on `P4` only when all of the following are true:

1. identifies bounded evidence that a chemical reaction has occurred
2. uses conservation reasoning when interpreting physical systems and changes
3. explains at least one wave behavior using a model rather than vocabulary alone
4. succeeds across more than one physical context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- distinguish reaction evidence from simple state or size changes in bounded examples
- reason that total matter or energy accounting must remain controlled within the modeled system
- explain reflection, absorption, transmission, or other wave behavior in a constrained case
- compare competing physical explanations against data or model fit

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
