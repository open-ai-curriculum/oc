# T2 Node Package

## Node

`T2` - Earth surface change, solar/lunar patterns, and water movement

## Purpose

This package defines the learner's ability to reason about visible Earth-surface changes, recurring sun-moon patterns, and simple movement of water across familiar Earth systems.
