# I1 Failure Taxonomy

### I1-FM-01 Fairness Breakdown

- description: learner changes too many conditions or cannot identify what should stay the same

### I1-FM-02 Tool-Purpose Mismatch

- description: learner selects or uses a tool that does not match the intended observation or measurement

### I1-FM-03 Recording Instability

- description: learner does not record outcomes consistently enough to support comparison

### I1-FM-04 Context Boundary

- description: learner can follow the routine in one familiar investigation but not in a changed context
