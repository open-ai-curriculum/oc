# I1 Node Specification

## Node Identity

- id: `I1`
- name: `fair_tests_tool_use_and_measurement_routines`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner follows a simple fair-test routine, uses a basic tool for the intended measurement or observation purpose, and records outcomes in a stable enough way to support comparison.

## Dependencies

- `Q2` question formation, prediction language, and pattern noticing

## Exit Conditions

The learner is secure on `I1` only when all of the following are true:

1. participates in a bounded test without changing multiple conditions at once
2. chooses or uses a basic tool appropriately
3. records or reports outcomes consistently enough for comparison
4. succeeds in more than one material or phenomenon context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- identify what is being tested
- keep the routine stable across repeated tries
- use common classroom science tools for the intended purpose
- compare observed outcomes after recording them

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
