# I1 Teacher Observability View

Teachers should see:

- current `I1` status
- dependency posture for `Q2`
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner keeps a fair test stable and uses tools appropriately
- recommended intervention

Class-level view should highlight:

- who changes too many conditions at once
- who selects the wrong tool for the intended measurement
- who can investigate only in one familiar context
