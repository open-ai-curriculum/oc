# I3 Node Specification

## Node Identity

- id: `I3`
- name: `repeated_trials_data_reliability_and_pattern_summary`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses repeated trials to judge whether results are reliable enough to summarize and compare, and then states a bounded pattern based on the data.

## Dependencies

- `I2` variable awareness, data recording, and basic comparison

## Exit Conditions

The learner is secure on `I3` only when all of the following are true:

1. recognizes why repeating trials can strengthen a result
2. distinguishes more reliable from less reliable data patterns
3. summarizes a bounded pattern across multiple trials
4. succeeds across more than one investigation context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- identify when one trial is not enough
- compare repeated results for consistency
- summarize a trend or pattern without overstating certainty
- use the record to judge whether a result is trustworthy enough to discuss

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
