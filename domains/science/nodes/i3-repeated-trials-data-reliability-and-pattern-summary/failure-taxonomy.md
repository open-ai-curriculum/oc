# I3 Failure Taxonomy

### I3-FM-01 Repetition-Value Blindness

- description: learner does not recognize why repeated trials strengthen or qualify a result

### I3-FM-02 Reliability Misjudgment

- description: learner misjudges whether a set of results is consistent enough to support a summary

### I3-FM-03 Pattern-Overstatement

- description: learner states a stronger pattern conclusion than the data supports

### I3-FM-04 Context Boundary

- description: learner uses reliability reasoning in one familiar investigation but not in a changed one
