# M5 Intervention Playbook

- For `M5-FM-01`, use model-versus-reality routines that explicitly separate representation from literal claim.
- For `M5-FM-02`, use evidence-to-mechanism mapping tasks where every mechanism claim must cite an observed pattern.
- For `M5-FM-03`, use analogy-boundary charts listing transferable and non-transferable features.
- For `M5-FM-04`, use competing-model comparison protocols with explicit evidence criteria.
