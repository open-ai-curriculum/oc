# I4 Failure Taxonomy

- `I4-FM-01` variable-control collapse: learner identifies variables but cannot reason about which must be controlled or separated.
- `I4-FM-02` convenience-sample overreach: learner treats easy sampling as equivalent to representative sampling.
- `I4-FM-03` undifferentiated error talk: learner uses error language without distinguishing source or consequence.
- `I4-FM-04` design-confidence mismatch: learner draws strong conclusions from weak or underpowered methods.
