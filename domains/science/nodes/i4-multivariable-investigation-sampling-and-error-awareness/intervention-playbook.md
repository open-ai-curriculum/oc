# I4 Intervention Playbook

- For `I4-FM-01`, use variable-prioritization routines that require learners to justify which variables must remain fixed and why.
- For `I4-FM-02`, use sampling-plan comparison tasks with explicit attention to representativeness and missing cases.
- For `I4-FM-03`, use error-source sorting where learners classify uncertainty statements by likely origin and impact.
- For `I4-FM-04`, use evidence-strength checks that compare conclusions against actual method quality before claims are accepted.
