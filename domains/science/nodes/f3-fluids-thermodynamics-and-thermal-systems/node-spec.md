# F3 Node Specification

## Node Identity

- id: `F3`
- name: `fluids_thermodynamics_and_thermal_systems`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains pressure, flow, heat, and thermal-system behavior using physical models without reducing thermal systems to one-variable temperature talk.

## Dependencies

- `F1`
- `P6`
- `M4`

## Exit Conditions

The learner is secure on `F3` only when all of the following are true:

1. explains fluid and thermal behavior using pressure, energy, and system-condition reasoning
2. distinguishes heat, temperature, and energy-change claims in bounded cases
3. connects macroscopic thermal behavior to system models without overextending simplifications
4. succeeds across more than one fluids or thermodynamics context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- use pressure and flow concepts with bounded system interpretation
- distinguish temperature descriptions from energy-transfer explanations
- reason about thermal systems using state change, conservation, or process models
- avoid one-variable explanations for multi-condition systems

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
