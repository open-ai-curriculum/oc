# C1 Node Specification

## Node Identity

- id: `C1`
- name: `atomic_structure_periodicity_and_quantum_models`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains matter using atomic structure, periodicity, and quantized-energy models without reducing atomic behavior to memorized table patterns only.

## Dependencies

- `P5`
- `M5`

## Exit Conditions

The learner is secure on `C1` only when all of the following are true:

1. explains atomic structure using bounded particle and energy-level models
2. uses periodicity to predict properties with model-based justification
3. connects quantum-model ideas to observable chemical behavior in limited cases
4. succeeds across more than one advanced chemistry context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- distinguish model purpose from literal picturing of atomic structure
- use periodic trends with mechanistic justification rather than recall only
- relate electron arrangement and energy levels to observable property differences
- avoid overclaiming from simplified atomic representations

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
