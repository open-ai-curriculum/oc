# T6 Node Specification

## Node Identity

- id: `T6`
- name: `earth_space_evolution_planetary_systems_and_global_change_synthesis`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner synthesizes Earth-space evolution, planetary-system reasoning, and global change evidence without flattening scale, uncertainty, or model differences.

## Dependencies

- `T5` geologic time, climate systems, and human impact models
- `E4` quantitative explanation, data critique, and uncertainty reasoning
- `I5` quantitative investigation design, uncertainty, and method critique

## Exit Conditions

The learner is secure on `T6` only when all of the following are true:

1. coordinates Earth and space evidence streams in bounded planetary explanations
2. explains global change using multi-system evidence without collapsing all processes into one narrative
3. critiques planetary or Earth-space claims for uncertainty, timescale, and model mismatch
4. succeeds across more than one advanced Earth-space synthesis context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- integrate geological, climatic, and space-system evidence in one bounded explanation
- compare planetary-system models while preserving what each explains
- critique Earth-space claims for evidence gaps or timescale mismatch
- synthesize global change reasoning with explicit model and uncertainty control

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
