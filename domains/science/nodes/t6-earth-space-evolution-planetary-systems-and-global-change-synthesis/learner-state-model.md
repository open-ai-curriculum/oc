# T6 Learner State Model

The learner-state for `T6` tracks:

- dependency security on `T5`, `E4`, and `I5`
- direct accuracy on Earth-space synthesis, global-change, and critique tasks
- explanation quality for advanced Earth-space integration
- transfer performance across unfamiliar Earth-space contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
