# T6 Intervention Playbook

- For `T6-FM-01`, use evidence-stream planners that force explicit roles for geological, climatic, and space evidence.
- For `T6-FM-02`, use global-change system maps with uncertainty annotations before explanation.
- For `T6-FM-03`, use critique stems requiring the exact timescale, evidence, or model-role flaw to be named.
- For `T6-FM-04`, use synthesis-boundary reviews that compare claim scope with integrated evidence scope.
