# R3 Intervention Playbook

- For `R3-FM-01`, use instrumentation maps linking physical quantity, sensor, circuit, and output.
- For `R3-FM-02`, use calibration and noise audits before data interpretation.
- For `R3-FM-03`, use embedded-measurement tasks that force setup reasoning, not output trust alone.
- For `R3-FM-04`, use validation routines comparing instrument readings with independent checks.
