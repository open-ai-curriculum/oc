# L4 Node Specification

## Node Identity

- id: `L4`
- name: `populations_ecosystems_and_matter_energy_in_living_systems`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains how populations interact within ecosystems and how matter and energy move through living systems without collapsing all ecological relationships into a single chain.

## Dependencies

- `L2` life cycles, inheritance, and ecosystem relationships
- `M3` scale, proportion, and system interaction models
- `E3` model-based argument, revision, and source comparison

## Exit Conditions

The learner is secure on `L4` only when all of the following are true:

1. distinguishes organism, population, community, and ecosystem levels in bounded cases
2. explains at least one matter or energy pathway through living systems
3. uses interaction language that includes more than a single predator-prey relation
4. succeeds across more than one ecosystem context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- describe how populations depend on resources and on interactions with other populations
- trace matter movement and energy transfer through a bounded ecosystem model
- compare ecosystem changes without overstating certainty beyond the data or model
- use system language that preserves scale and interaction structure

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
