# G4 Node Specification

## Node Identity

- id: `G4`
- name: `astronomy_astrophysics_and_cosmic_evolution`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains planetary, stellar, and cosmological phenomena using physical and observational models without treating astronomy as a disconnected fact catalog.

## Dependencies

- `T6`
- `M5`
- `E4`

## Exit Conditions

The learner is secure on `G4` only when all of the following are true:

1. explains astronomy and astrophysics phenomena using bounded physical models
2. connects observations to planetary, stellar, or cosmic-evolution claims appropriately
3. distinguishes model-based inference from direct observation in astronomy contexts
4. succeeds across more than one advanced astronomy context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- use light, motion, scale, and interaction evidence in astronomy explanations
- explain how observations support inference in space science
- coordinate model choice with the astronomical phenomenon being explained
- avoid overclaiming certainty from indirect evidence

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
