# C2 Node Specification

## Node Identity

- id: `C2`
- name: `bonding_structure_and_intermolecular_forces`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner predicts material behavior from bonding, structure, polarity, and intermolecular-force models without collapsing all interactions into one generic bond idea.

## Dependencies

- `C1`
- `P5`

## Exit Conditions

The learner is secure on `C2` only when all of the following are true:

1. distinguishes major bonding and interaction models in bounded cases
2. uses structure and polarity to predict material properties
3. connects intermolecular forces to observable outcomes such as phase or solubility behavior
4. succeeds across more than one advanced chemistry context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- distinguish intramolecular and intermolecular reasoning appropriately
- connect shape, polarity, or network structure to material behavior
- justify property predictions with interaction models
- avoid treating all attractions as equivalent

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
