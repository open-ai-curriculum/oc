# C2 Failure Taxonomy

- `C2-FM-01` bond-type collapse: learner treats ionic, covalent, metallic, and intermolecular interactions as the same relation.
- `C2-FM-02` shape-polarity disconnect: learner cannot connect geometry and polarity to behavior.
- `C2-FM-03` property-without-model support: learner predicts properties without interaction-based justification.
- `C2-FM-04` force-strength overclaim: learner overstates one interaction type without bounded comparison.
