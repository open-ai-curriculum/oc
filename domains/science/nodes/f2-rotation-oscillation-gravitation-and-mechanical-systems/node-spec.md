# F2 Node Specification

## Node Identity

- id: `F2`
- name: `rotation_oscillation_gravitation_and_mechanical_systems`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner extends mechanics into rotational, oscillatory, and gravitational systems without treating these topics as disconnected special-case formulas.

## Dependencies

- `F1`
- `P5`

## Exit Conditions

The learner is secure on `F2` only when all of the following are true:

1. explains rotational and oscillatory behavior using coherent mechanical-system reasoning
2. connects gravitation to orbital or field-based mechanics in bounded cases
3. distinguishes linear and rotational analogies without collapsing them into identical relations
4. succeeds across more than one advanced mechanics context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- use rotational variables and system constraints meaningfully
- explain oscillation with restoring-effect reasoning instead of pattern description only
- connect orbital motion to interaction models and system conditions
- avoid overextending linear analogies without checking model fit

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
