# Q3 Intervention Playbook

## Purpose

Map `Q3` failure modes to bounded intervention responses.

## Intervention Rules

### For Q3-FM-01 Phenomenon Blur

Primary interventions:

- restate the phenomenon with a visible anchor image or event
- use sentence frames such as "We are trying to figure out..."
- contrast the focal question with near-but-different questions

Expected effect:

- learner identifies the focal phenomenon more reliably

Reverification target:

- one phenomenon-framing item

### For Q3-FM-02 Topic Drift

Primary interventions:

- pause before discussion contributions and restate the class question
- sort on-topic and off-topic statements
- use a visible prompt card naming the science focus

Expected effect:

- learner keeps contributions tied to the shared frame

Reverification target:

- one on-topic contribution item

### For Q3-FM-03 Relevance Misjudgment

Primary interventions:

- compare relevant and irrelevant observations for the same question
- ask which detail helps explain the phenomenon
- highlight evidence that directly bears on the focal question

Expected effect:

- learner judges relevance more reliably

Reverification target:

- one relevance item

### For Q3-FM-04 Context Boundary

Primary interventions:

- repeat the same framing routine across life, physical, and Earth/space examples
- preserve the sensemaking structure while changing the phenomenon
- fade scaffolds after one successful familiar example

Expected effect:

- learner transfers phenomenon framing across science contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for language load, discourse-support needs, or prompt-design mismatch rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when language access, confidence, or classroom discussion norms suggest a different explanation of the learner state.
