# Q3 Node Specification

## Node Identity

- id: `Q3`
- name: `phenomenon_framing_and_shared_sensemaking_routines`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-bridge-node-package`
- evidence class: `Synthetic`

## Purpose

The learner identifies the focal phenomenon in a science discussion, states what is being figured out, and contributes observations or ideas that remain tied to the shared investigation.

## Dependencies

- `Q2` question formation, prediction language, and pattern noticing

## Exit Conditions

The learner is secure on `Q3` only when all of the following are true:

1. identifies the phenomenon or event under discussion
2. states a bounded "what we are trying to figure out" frame
3. contributes observations or ideas tied to that frame
4. succeeds across more than one science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- stay on the focal question during discussion
- distinguish the target phenomenon from related background topics
- contribute evidence-linked ideas to a shared sensemaking routine
- use the class frame to organize what is being investigated

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
