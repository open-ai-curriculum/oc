# E3 Teacher Observability View

Teachers should see:

- current `E3` status
- dependency posture for `E2` and `M2`
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner revises arguments using stronger models or sources
- recommended intervention

Class-level view should highlight:

- who resists revision when evidence changes
- who cannot coordinate models and sources
- who chooses weaker sources over stronger evidence
