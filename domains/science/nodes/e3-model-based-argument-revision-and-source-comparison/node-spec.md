# E3 Node Specification

## Node Identity

- id: `E3`
- name: `model_based_argument_revision_and_source_comparison`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses models and evidence to argue for an explanation, revises that explanation when new evidence appears, and compares sources for relevance, fit, or explanatory value.

## Dependencies

- `E2` claim-evidence-reasoning and competing explanations
- `M2` mechanism, cause-effect, and process models

## Exit Conditions

The learner is secure on `E3` only when all of the following are true:

1. uses a model as part of an evidence-based argument
2. revises a claim or explanation when a stronger source or new evidence appears
3. compares sources for relevance or support quality
4. succeeds across more than one science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- use a model and evidence together in an argument
- explain why a revision is warranted
- compare stronger and weaker sources or representations
- avoid sticking to an earlier claim when the evidence clearly shifts

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
