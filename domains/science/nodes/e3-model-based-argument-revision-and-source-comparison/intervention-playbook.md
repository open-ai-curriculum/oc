# E3 Intervention Playbook

## Purpose

Map `E3` failure modes to bounded intervention responses.

## Intervention Rules

### For E3-FM-01 Revision Resistance

Primary interventions:

- compare an original claim with new evidence that changes the best explanation
- ask what must change in the argument and why
- model short revision statements such as "I used to think..., now I think..."

Expected effect:

- learner revises explanations more appropriately when evidence changes

Reverification target:

- one revision item

### For E3-FM-02 Model-Source Coordination Breakdown

Primary interventions:

- pair each claim with both a model and a data source
- ask what each source contributes to the argument
- compare arguments that use one source versus coordinated sources

Expected effect:

- learner coordinates model and evidence sources more reliably

Reverification target:

- one model-source item

### For E3-FM-03 Source-Quality Misjudgment

Primary interventions:

- sort stronger and weaker sources for the same question
- highlight relevance, directness, and explanatory fit
- compare evidence-based and unsupported source choices

Expected effect:

- learner judges source quality more accurately

Reverification target:

- one source-comparison item

### For E3-FM-04 Context Boundary

Primary interventions:

- repeat revision routines across life, physical, and Earth/space argument tasks
- preserve the revision structure while changing the science domain
- fade scaffolds after one successful familiar example

Expected effect:

- learner transfers model-based revision across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for literacy load, source-display design, or prompt-design mismatch rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when a nonstandard revision remains evidence-defensible, or when discourse demands and reading load suggest a different explanation of the learner state.
