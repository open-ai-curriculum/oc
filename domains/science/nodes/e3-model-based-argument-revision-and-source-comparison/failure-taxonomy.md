# E3 Failure Taxonomy

### E3-FM-01 Revision Resistance

- description: learner does not revise the explanation when stronger evidence or models appear

### E3-FM-02 Model-Source Coordination Breakdown

- description: learner cannot coordinate model information with evidence or source quality

### E3-FM-03 Source-Quality Misjudgment

- description: learner chooses a weaker or irrelevant source over a stronger one

### E3-FM-04 Context Boundary

- description: learner can revise arguments in one familiar context but not in a changed one
