# E1 Teacher Observability View

Teachers should see:

- current `E1` status
- dependency posture for `Q2`
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner supports claims with direct evidence
- recommended intervention

Class-level view should highlight:

- who makes unsupported assertions
- who can name observations but not use them as evidence
- who can explain only in one familiar context
