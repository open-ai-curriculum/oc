# E1 Node Specification

## Node Identity

- id: `E1`
- name: `observation_based_explanation_and_evidence_talk`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner states a simple science explanation or claim about an observed event and supports it with direct observation, recorded results, or another immediately available evidence source.

## Dependencies

- `Q2` question formation, prediction language, and pattern noticing

## Exit Conditions

The learner is secure on `E1` only when all of the following are true:

1. states what happened or what claim is being made
2. identifies a relevant observation as evidence
3. links the claim to the evidence with simple reasoning language
4. succeeds across more than one phenomenon context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- tell what was observed and what it shows
- point to or name evidence for a simple claim
- avoid relying only on preference or unsupported assertion
- explain simple classroom science outcomes using direct evidence talk

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
