# E1 Failure Taxonomy

### E1-FM-01 Claim-Evidence Disconnect

- description: learner states a claim but cannot connect it to a relevant observation

### E1-FM-02 Unsupported Assertion

- description: learner gives an explanation based on preference guesswork or authority rather than evidence

### E1-FM-03 Evidence Selection Error

- description: learner chooses an irrelevant or weaker observation when stronger evidence is available

### E1-FM-04 Context Boundary

- description: learner can use evidence talk in one familiar context but not in a changed phenomenon context
