# Q2 Intervention Playbook

## Purpose

Map `Q2` failure modes to bounded intervention responses.

## Intervention Rules

### For Q2-FM-01 Uncheckable Questioning

Primary interventions:

- sort example questions into checkable and not-checkable
- ask what evidence could answer the question
- rewrite one broad question into a bounded observational question

Expected effect:

- learner forms questions that fit observation or simple testing

Reverification target:

- one question-formation item

### For Q2-FM-02 Prediction Without Basis

Primary interventions:

- require the learner to point to the observed clue before predicting
- compare two predictions and ask which one matches the setup
- use sentence frames such as "I think this because I notice"

Expected effect:

- learner ties predictions to visible evidence or prior repeated pattern

Reverification target:

- one prediction item

### For Q2-FM-03 Pattern Blindness

Primary interventions:

- highlight repeated events on a chart or sequence strip
- ask what happened again and again
- use before-next comparisons to surface recurrence

Expected effect:

- learner notices and states recurring patterns more reliably

Reverification target:

- one pattern item

### For Q2-FM-04 Topic Drift

Primary interventions:

- restate the focal phenomenon before asking for a question
- narrow the prompt to one object event or variable
- use a visible anchor such as "our question is about this"

Expected effect:

- learner keeps questioning tied to the target phenomenon

Reverification target:

- one question item in a changed context

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for language load, attention to task boundaries, or unclear prompt design rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when language access, confidence, or context suggests a different explanation of the learner state.
