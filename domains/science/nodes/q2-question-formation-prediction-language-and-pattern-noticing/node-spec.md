# Q2 Node Specification

## Node Identity

- id: `Q2`
- name: `question_formation_prediction_language_and_pattern_noticing`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner turns direct observation into a bounded science question, a simple prediction, or a statement about a recurring pattern without drifting into unrelated or unobservable claims.

## Dependencies

- `Q1` observation, noticing, and descriptive comparison

## Exit Conditions

The learner is secure on `Q2` only when all of the following are true:

1. asks a question that fits the observed phenomenon
2. states a simple prediction connected to the observation
3. notices and names a recurring visible pattern
4. succeeds across more than one phenomenon context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- ask what will happen next in a bounded observable situation
- use prediction language tied to an observed setup
- describe a pattern such as repeated change or repeated sequence
- keep a question grounded in what can be observed or checked

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
