# F5 Node Specification

## Node Identity

- id: `F5`
- name: `waves_optics_quantum_and_modern_physics`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains optical, wave, and modern-physics phenomena using multiscale models without treating wave behavior and quantum ideas as disconnected curiosity topics.

## Dependencies

- `F4`
- `P6`
- `M5`

## Exit Conditions

The learner is secure on `F5` only when all of the following are true:

1. explains wave and optical behavior using coherent propagation and interaction models
2. connects light behavior to both wave and bounded modern-physics perspectives where appropriate
3. uses model choice carefully in quantum or modern-physics contexts
4. succeeds across more than one advanced wave or modern-physics context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- distinguish what wave models explain well in the bounded case
- connect optical behavior to interaction, path, or representation choices
- treat modern-physics models as explanatory tools with limits rather than literal pictures
- avoid mixing incompatible model claims without justification

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
