# F5 Failure Taxonomy

- `F5-FM-01` wave-feature listing: learner names wave features without using a propagation or interaction model.
- `F5-FM-02` optics-ray-only collapse: learner uses one optics representation without checking whether it fits the case.
- `F5-FM-03` quantum-literalism: learner treats modern-physics models as literal pictures rather than bounded explanations.
- `F5-FM-04` model-mixing without-boundary: learner blends wave, ray, or quantum ideas without justification.
