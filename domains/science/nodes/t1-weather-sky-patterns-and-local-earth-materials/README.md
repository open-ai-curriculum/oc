# T1 Node Package

## Node

`T1` - Weather, sky patterns, and local Earth materials

## Purpose

This package defines the earliest Earth and space science readiness in the domain.

It operationalizes the learner's ability to notice recurring weather and sky patterns and describe observable properties of local Earth materials using direct evidence.
