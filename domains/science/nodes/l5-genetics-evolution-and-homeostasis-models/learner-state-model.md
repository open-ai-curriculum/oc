# L5 Learner State Model

The learner-state for `L5` tracks:

- dependency security on `L4`, `M4`, and `E3`
- direct accuracy on genetics, evolution, and homeostasis tasks
- explanation quality for advanced life-science modeling
- transfer performance across unfamiliar biological contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
