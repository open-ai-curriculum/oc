# L5 Intervention Playbook

- For `L5-FM-01`, use inheritance-versus-acquisition comparison routines that force source-of-trait explanations.
- For `L5-FM-02`, use individual-versus-population sorts that separate organism stories from population change models.
- For `L5-FM-03`, use regulation graphs showing fluctuation within ranges rather than fixed points.
- For `L5-FM-04`, use gene-to-trait modeling tasks that add expression, environment, and system context explicitly.
