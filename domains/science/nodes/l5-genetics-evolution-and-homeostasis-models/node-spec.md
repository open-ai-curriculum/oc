# L5 Node Specification

## Node Identity

- id: `L5`
- name: `genetics_evolution_and_homeostasis_models`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains inheritance, population change, and homeostatic regulation using bounded biological models without collapsing these processes into simple trait stories or purposeful adaptation claims.

## Dependencies

- `L4` populations, ecosystems, and matter-energy in living systems
- `M4` conservation, cycle, and energy flow models
- `E3` model-based argument, revision, and source comparison

## Exit Conditions

The learner is secure on `L5` only when all of the following are true:

1. explains inheritance using genes or information flow models in bounded cases
2. explains population-level evolutionary change without attributing change to need or intention
3. explains homeostasis as dynamic regulation rather than static balance only
4. succeeds across more than one advanced life-science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- distinguish inheritance patterns from environmental influence claims
- explain selection and population change without teleological drift
- describe feedback and regulation in maintaining biological stability
- compare competing biological explanations against evidence and model fit

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
