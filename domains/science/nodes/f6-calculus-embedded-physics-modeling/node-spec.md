The learner is secure on `F6` only when all of the following are true:

1. uses calculus-supported reasoning to model change and accumulation in bounded physics cases
2. connects mathematical operations back to physical quantities, assumptions, and system meaning
3. uses advanced modeling to extend mechanics or electricity-magnetism explanations without overclaiming scope
4. succeeds across more than one calculus-embedded physics context
5. passes a delayed recheck after spacing
