# B4 Node Specification

## Node Identity

- id: `B4`
- name: `anatomy_physiology_and_system_regulation`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains coordinated body-system function and regulation using feedback, transport, and interaction models without reducing physiology to isolated organ facts.

## Dependencies

- `B1`
- `L5`
- `M4`

## Exit Conditions

The learner is secure on `B4` only when all of the following are true:

1. explains body-system coordination using transport, signaling, and regulation models
2. uses feedback language to explain stability and response in physiological systems
3. distinguishes structure naming from functional system explanation
4. succeeds across more than one anatomy or physiology context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- coordinate multiple body systems in a single functional explanation
- explain physiological regulation as dynamic feedback rather than static balance only
- use anatomy details only when they support mechanism explanations
- identify where organ-level and system-level evidence must both matter

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
