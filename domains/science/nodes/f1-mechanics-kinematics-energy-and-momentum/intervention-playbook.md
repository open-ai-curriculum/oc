# F1 Intervention Playbook

- For `F1-FM-01`, use representation-matching routines that require one mechanics situation to be explained across words, graphs, and equations.
- For `F1-FM-02`, use model-selection tasks comparing force, energy, and momentum explanations for the same event.
- For `F1-FM-03`, use collision and motion analyses that bound when momentum reasoning is justified.
- For `F1-FM-04`, use graph-to-measurement checks tying symbolic work back to observed motion.
