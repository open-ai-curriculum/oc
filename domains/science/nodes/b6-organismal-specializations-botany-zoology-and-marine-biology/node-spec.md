# B6 Node Specification

## Node Identity

- id: `B6`
- name: `organismal_specializations_botany_zoology_and_marine_biology`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner extends advanced biological reasoning into plant, animal, and marine organism systems using habitat, adaptation, physiology, and environmental interaction models without fragmenting organism study from broader system logic.

## Dependencies

- `B4`
- `B5`
- `T6`

## Exit Conditions

The learner is secure on `B6` only when all of the following are true:

1. explains organismal specialization using physiology, environment, and adaptation together
2. connects plant, animal, or marine cases to broader biological and Earth-system constraints
3. uses comparative organism evidence without reducing explanation to trait cataloging
4. succeeds across more than one organism-specialization context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- compare organism groups using functional and environmental reasoning
- connect marine or terrestrial organism behavior to habitat-system conditions
- explain specialization through coordinated structure, physiology, and ecological fit
- avoid trait-listing without system explanation

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
