# E2 Failure Taxonomy

### E2-FM-01 Claim-Evidence Mismatch

- description: learner states a claim but selects evidence that does not support it well

### E2-FM-02 Reasoning Gap

- description: learner names the claim and evidence but cannot explain why the evidence supports the claim

### E2-FM-03 Competing-Explanation Misjudgment

- description: learner cannot distinguish a stronger explanation from a weaker one

### E2-FM-04 Context Boundary

- description: learner can use CER in one familiar context but not in a changed science context
