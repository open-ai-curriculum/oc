# E2 Node Specification

## Node Identity

- id: `E2`
- name: `claim_evidence_reasoning_and_competing_explanations`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-bridge-node-package`
- evidence class: `Synthetic`

## Purpose

The learner states a science claim, selects relevant evidence, gives a basic reason connecting the evidence to the claim, and compares competing explanations for strength of support.

## Dependencies

- `E1` observation-based explanation and evidence talk
- `I2` variable awareness, data recording, and basic comparison

## Exit Conditions

The learner is secure on `E2` only when all of the following are true:

1. states a bounded science claim about an investigated phenomenon
2. selects relevant evidence from observations or recorded data
3. explains why the evidence supports the claim
4. distinguishes a stronger explanation from a weaker one
5. succeeds across more than one context and passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- state a claim tied to the investigation question
- select evidence from the record rather than from preference
- give a basic because-statement linking evidence to claim
- compare competing explanations using evidence support

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
