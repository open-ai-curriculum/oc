# E2 Intervention Playbook

## Purpose

Map `E2` failure modes to bounded intervention responses.

## Intervention Rules

### For E2-FM-01 Claim-Evidence Mismatch

Primary interventions:

- match claims to the most relevant evidence from a small set
- highlight which row or observation belongs to the focal question
- compare supporting and non-supporting evidence for the same claim

Expected effect:

- learner aligns claims with better evidence

Reverification target:

- one claim-evidence item

### For E2-FM-02 Reasoning Gap

Primary interventions:

- use sentence frames such as "This evidence supports the claim because..."
- model short because-statements from a visible table
- ask what the evidence shows before stating the reason

Expected effect:

- learner links evidence to the claim with basic reasoning

Reverification target:

- one reasoning item

### For E2-FM-03 Competing-Explanation Misjudgment

Primary interventions:

- compare two explanations and ask which one uses the actual data
- sort stronger and weaker explanations
- highlight what each explanation does or does not account for

Expected effect:

- learner judges explanation strength more accurately

Reverification target:

- one competing-explanations item

### For E2-FM-04 Context Boundary

Primary interventions:

- repeat the CER routine across plant, motion, and weather examples
- preserve the CER structure while changing the domain
- fade scaffolds after a successful familiar example

Expected effect:

- learner transfers CER reasoning across science contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for language load, evidence-display design, or prompt-design mismatch rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when discourse demands, reading load, or response-mode constraints suggest a different explanation of the learner state.
