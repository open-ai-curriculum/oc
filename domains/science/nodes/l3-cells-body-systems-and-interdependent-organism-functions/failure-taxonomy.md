# L3 Failure Taxonomy

- `L3-FM-01` cell idea collapse: learner treats cells as optional details instead of basic living units.
- `L3-FM-02` part naming without function: learner names organs or structures but does not explain what they do.
- `L3-FM-03` system isolation error: learner treats body systems or organism functions as unrelated processes.
- `L3-FM-04` scale confusion: learner mixes cell, organ, organism, and system levels without control.
