# L3 Node Specification

## Node Identity

- id: `L3`
- name: `cells_body_systems_and_interdependent_organism_functions`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains that living things are made of cells, that body structures form interacting systems, and that coordinated organism functions support survival and reproduction.

## Dependencies

- `L2` life cycles, inheritance, and ecosystem relationships
- `M2` mechanism, cause-effect, and process models

## Exit Conditions

The learner is secure on `L3` only when all of the following are true:

1. identifies cells as the basic unit of living things in bounded instructional contexts
2. explains how structures within a body system contribute to a larger function
3. connects multiple organism functions rather than treating systems as isolated parts
4. succeeds across more than one organism or body-system context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- distinguish cell-level, organ-level, and system-level descriptions
- explain how structures such as lungs, leaves, roots, or circulatory components support function
- describe how systems interact to move matter or energy needed by the organism
- use mechanism language to explain organism function rather than naming parts only

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
