# L3 Learner State Model

The learner-state for `L3` tracks:

- dependency security on `L2` and `M2`
- direct accuracy on cell, structure-function, and system-interaction tasks
- explanation quality for coordinated organism function
- transfer performance in unfamiliar organism contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
