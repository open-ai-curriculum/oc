# D2 Failure Taxonomy

### D2-FM-01 Test-Result Misread

- description: learner cannot accurately interpret what the prototype test showed

### D2-FM-02 Redesign Disconnect

- description: learner proposes a redesign that does not respond to the test evidence

### D2-FM-03 Tradeoff Blindness

- description: learner improves one design feature without noticing a related drawback or constraint conflict

### D2-FM-04 Context Boundary

- description: learner can redesign in one familiar context but not in a changed one
