# M3 Failure Taxonomy

### M3-FM-01 Scale Blindness

- description: learner does not recognize that the same system may need different interpretations at different scales

### M3-FM-02 Relative-Change Instability

- description: learner struggles to reason about qualitative proportional or relative changes

### M3-FM-03 Interaction Omission

- description: learner explains one part of the system while ignoring important interacting parts

### M3-FM-04 Context Boundary

- description: learner can reason about scale or interactions in one familiar context but not in a changed one
