# M3 Node Specification

## Node Identity

- id: `M3`
- name: `scale_proportion_and_system_interaction_models`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains how scale matters in a system, reasons about proportional changes qualitatively, and tracks how one part of a system affects another.

## Dependencies

- `M2` mechanism, cause-effect, and process models
- `I2` variable awareness, data recording, and basic comparison

## Exit Conditions

The learner is secure on `M3` only when all of the following are true:

1. recognizes that the same system may look or behave differently at different scales
2. describes a simple proportional or relative change in a model or system
3. explains how interacting parts influence one another
4. succeeds across more than one science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- compare small and large scales of the same phenomenon or system
- describe relative increases, decreases, or interactions qualitatively
- use a system model to explain how one change affects another part
- avoid treating isolated parts as independent when the model shows interaction

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
