# E5 Intervention Playbook

- For `E5-FM-01`, use evidence-source mapping where each major claim must link to more than one supporting stream when appropriate.
- For `E5-FM-02`, use model-role tables that force learners to state contribution and limit for each model.
- For `E5-FM-03`, use pseudo-integration critiques that distinguish real cross-domain links from vocabulary accumulation.
- For `E5-FM-04`, use synthesis-boundary checks that compare claim scope against coordinated evidence.
