# E5 Learner State Model

The learner-state for `E5` tracks:

- dependency security on `E4` and `M5`
- direct accuracy on synthesis, multi-model, and integrated-evidence tasks
- explanation quality for interdisciplinary evidence coordination
- transfer performance across unfamiliar synthesis contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
