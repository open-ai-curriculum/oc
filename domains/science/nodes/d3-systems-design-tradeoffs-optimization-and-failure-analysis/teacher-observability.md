# D3 Teacher Observability View

Teachers should see:

- current `D3` status
- dependency posture for `D2`, `E3`, and `M3`
- direct accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner can optimize using system-level tradeoff reasoning
- recommended intervention

Class-level view should highlight:

- who narrows failure to one part instead of the system
- who misjudges tradeoffs
- who proposes optimizations that ignore system evidence
