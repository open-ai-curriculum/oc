# D3 Failure Taxonomy

### D3-FM-01 System-Failure Narrowing

- description: learner focuses on one part and misses the system-level source of failure

### D3-FM-02 Tradeoff Misjudgment

- description: learner cannot accurately weigh what improves against what gets worse

### D3-FM-03 Optimization Disconnect

- description: learner proposes an optimization that does not fit the broader system evidence

### D3-FM-04 Context Boundary

- description: learner can optimize one familiar system but not a changed engineering context
