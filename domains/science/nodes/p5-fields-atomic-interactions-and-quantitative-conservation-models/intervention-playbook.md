# P5 Intervention Playbook

- For `P5-FM-01`, use contact-versus-field comparison tasks that require evidence-backed model choice.
- For `P5-FM-02`, use model-versus-picture routines that separate representational aids from literal atomic claims.
- For `P5-FM-03`, use closed-system accounting checks before quantitative conclusions are accepted.
- For `P5-FM-04`, use quantity-to-meaning prompts that require each calculation to be interpreted in model terms.
