# P5 Node Specification

## Node Identity

- id: `P5`
- name: `fields_atomic_interactions_and_quantitative_conservation_models`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains advanced physical systems using field models, atomic interaction models, and quantitative conservation reasoning without confusing representational tools with literal pictures.

## Dependencies

- `P4` chemical reactions, energy conservation, and wave behavior
- `M5` field, force, and invisible mechanism models
- `E4` quantitative explanation, data critique, and uncertainty reasoning

## Exit Conditions

The learner is secure on `P5` only when all of the following are true:

1. uses field and interaction models to explain physical behavior in bounded cases
2. applies quantitative conservation reasoning across advanced physical systems
3. explains atomic-scale interactions without collapsing to naive visible-particle stories
4. succeeds across more than one advanced physical-science context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `42`

## Observable Evidence

The learner can:

- justify the role of field models in explaining interaction at a distance or distributed effects
- use quantitative conservation logic to test physical explanations
- describe atomic interactions as modeled processes rather than as visible mechanical miniatures only
- compare competing physical explanations against evidence and model adequacy

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
