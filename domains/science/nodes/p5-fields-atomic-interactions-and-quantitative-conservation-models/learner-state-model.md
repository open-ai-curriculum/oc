# P5 Learner State Model

The learner-state for `P5` tracks:

- dependency security on `P4`, `M5`, and `E4`
- direct accuracy on field, atomic-interaction, and quantitative-conservation tasks
- explanation quality for advanced physical-science models
- transfer performance across unfamiliar physical-science contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
