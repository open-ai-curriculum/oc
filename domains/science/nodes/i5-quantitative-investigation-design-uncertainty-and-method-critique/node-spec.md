# I5 Node Specification

## Node Identity

- id: `I5`
- name: `quantitative_investigation_design_uncertainty_and_method_critique`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-secondary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner designs and critiques quantitative investigations using explicit uncertainty reasoning, methodological justification, and disciplined comparison of competing approaches.

## Dependencies

- `I4` multivariable investigation, sampling, and error awareness

## Exit Conditions

The learner is secure on `I5` only when all of the following are true:

1. justifies quantitative design choices using model fit and measurement needs
2. interprets uncertainty as part of inference rather than as a reason to dismiss all evidence
3. critiques competing methods using validity, precision, scale, and feasibility
4. succeeds across more than one advanced investigation context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `35`

## Observable Evidence

The learner can:

- compare methods with explicit reference to what each can and cannot support
- reason about uncertainty bands, range, or variation in a bounded way
- justify instrumentation or measurement choices for a scientific purpose
- revise an investigation plan when critique reveals methodological mismatch

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
