# I2 Failure Taxonomy

### I2-FM-01 Variable Confusion

- description: learner cannot reliably identify what changed and what stayed the same

### I2-FM-02 Recording Breakdown

- description: learner records data inconsistently enough to weaken comparison

### I2-FM-03 Comparison Misread

- description: learner cannot accurately compare recorded outcomes

### I2-FM-04 Context Boundary

- description: learner can use variable and data logic in one familiar investigation but not in a changed context
