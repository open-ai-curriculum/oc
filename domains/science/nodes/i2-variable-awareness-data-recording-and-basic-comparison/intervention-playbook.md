# I2 Intervention Playbook

## Purpose

Map `I2` failure modes to bounded intervention responses.

## Intervention Rules

### For I2-FM-01 Variable Confusion

Primary interventions:

- use "what changed / what stayed the same" sorting routines
- point to one focal changed factor before reading the data
- compare two investigations with different changed factors

Expected effect:

- learner identifies the focal variable more reliably

Reverification target:

- one variable item

### For I2-FM-02 Recording Breakdown

Primary interventions:

- model one-row-per-trial recording
- use a simplified table with fewer fields
- compare clean and messy records before drawing conclusions

Expected effect:

- learner records outcomes more consistently

Reverification target:

- one data-recording item

### For I2-FM-03 Comparison Misread

Primary interventions:

- compare two entries side by side
- highlight which quantity or outcome should be compared
- use sentence frames such as "This one is greater because..."

Expected effect:

- learner compares investigation outcomes more accurately

Reverification target:

- one comparison item

### For I2-FM-04 Context Boundary

Primary interventions:

- repeat the same variable/data routine across plant, motion, and materials tests
- preserve the comparison structure while changing the phenomenon
- fade scaffolds after success in one familiar task

Expected effect:

- learner transfers variable-aware data comparison across contexts

Reverification target:

- one transfer item

## Escalation Rule

If the same dominant failure mode remains `blocking` after two bounded intervention cycles, escalate to review for executive-load demands, language load, or prompt-design mismatch rather than intensifying the same local practice blindly.

## Human Review Rule

Interventions may be suggested automatically, but the teacher or human reviewer must remain able to override the recommendation when motor access, table-reading skills, or response-mode constraints suggest a different explanation of the learner state.
