# I2 Node Specification

## Node Identity

- id: `I2`
- name: `variable_awareness_data_recording_and_basic_comparison`
- domain: `science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `science-bridge-node-package`
- evidence class: `Synthetic`

## Purpose

The learner identifies what changed in an investigation, records simple observations consistently, and compares results to support a basic science conclusion.

## Dependencies

- `I1` fair tests, tool use, and measurement routines
- `Q3` phenomenon framing and shared sensemaking routines

## Exit Conditions

The learner is secure on `I2` only when all of the following are true:

1. identifies the focal changed factor or condition in a simple investigation
2. records observations in a stable simple format
3. compares at least two outcomes accurately
4. succeeds across more than one investigation context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- state what changed and what stayed the same
- record data in rows, pictures, marks, or tables consistently enough for comparison
- compare outcomes using direct evidence
- use the investigation record to support a basic claim

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
