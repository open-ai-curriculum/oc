# R2 Goal Setting, Progress Tracking, And Simple Reflection

## Purpose

This node defines the next governed self-regulation surface in the `learning_to_learn` domain.

It exists so later persistence and metacognitive evaluation work does not assume learners can already set a bounded goal, track progress, and reflect on what helped or did not help.
