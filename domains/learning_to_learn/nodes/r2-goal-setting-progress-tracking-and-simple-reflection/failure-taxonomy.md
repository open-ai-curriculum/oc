# R2 Failure Taxonomy

### R2-FM-01 Unusable Goal

- description: learner sets a goal that is too vague, too broad, or not connected to the task block

### R2-FM-02 No Progress Monitoring

- description: learner sets a goal but does not track whether progress or accuracy is moving toward it

### R2-FM-03 Reflection Without Instructional Value

- description: learner reflects in a way that does not identify what helped, what did not help, or what should change next
