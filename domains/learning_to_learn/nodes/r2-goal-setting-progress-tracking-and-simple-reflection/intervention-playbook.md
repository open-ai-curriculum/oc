# R2 Intervention Playbook

## Purpose

Map `R2` failure modes to bounded intervention responses.

## Intervention Rules

### For R2-FM-01 Unusable Goal

Primary interventions:

- reduce the goal to one bounded task block or accuracy target
- model a usable goal beside an unusable one
- tie the goal directly to the next work cycle

Expected effect:

- learner sets a goal that is concrete enough to guide work

Reverification target:

- one short goal-setting routine

### For R2-FM-02 No Progress Monitoring

Primary interventions:

- add one visible progress checkpoint
- use a simple count, checklist, or accuracy marker
- require a mid-task progress readout

Expected effect:

- learner tracks progress or accuracy during the task

Reverification target:

- one goal-plus-progress cycle

### For R2-FM-03 Reflection Without Instructional Value

Primary interventions:

- narrow reflection to what helped, what blocked, and what to try next
- compare informative and noninformative reflections
- tie reflection to the immediately completed task block

Expected effect:

- learner produces a reflection that changes the next learning move

Reverification target:

- one task-complete-then-reflect cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language load, self-report demands, and task framing rather than only extending reflection prompts.
