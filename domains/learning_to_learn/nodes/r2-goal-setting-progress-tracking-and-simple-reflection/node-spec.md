# R2 Node Specification

## Node Identity

- id: `R2`
- name: `goal_setting_progress_tracking_and_simple_reflection`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner sets a bounded goal, tracks progress or accuracy against it, and reflects on what helped or did not help.

## Dependencies

- `R1` self-monitoring, help-seeking, and feedback acceptance
- `S1` productive practice routine selection

## Exit Conditions

The learner is secure on `R2` only when all of the following are true:

1. sets a bounded and usable goal for the task or work block
2. tracks progress or accuracy against that goal during or after work
3. identifies at least one support or strategy that helped or did not help
4. succeeds across more than one task type
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- state a short task goal
- record or report progress against it
- name one factor that supported or limited success

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
