# M2 Failure Taxonomy

### M2-FM-01 Immediate Reread Dependence

- description: learner returns to the answer immediately instead of attempting retrieval first

### M2-FM-02 Cue Misuse

- description: learner has a cue available but does not use it to retrieve the target information productively

### M2-FM-03 No Spacing Routine

- description: learner treats review as one massed block rather than distributed retrieval over time
