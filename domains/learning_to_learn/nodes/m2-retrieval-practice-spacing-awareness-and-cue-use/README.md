# M2 Retrieval Practice, Spacing Awareness, And Cue Use

## Purpose

This node defines the next governed memory surface in the `learning_to_learn` domain.

It exists so later durable retention work does not assume learners can already attempt recall before rereading and use spacing and cues to strengthen memory over time.
