# M2 Intervention Playbook

## Purpose

Map `M2` failure modes to bounded intervention responses.

## Intervention Rules

### For M2-FM-01 Immediate Reread Dependence

Primary interventions:

- require one retrieval attempt before reopening the answer
- time-box the retrieval attempt so failure is safe
- model the difference between checking and copying

Expected effect:

- learner attempts retrieval before rereading

Reverification target:

- one retrieve-then-check cycle

### For M2-FM-02 Cue Misuse

Primary interventions:

- reduce cues to one high-value prompt at a time
- model how a cue should trigger recall
- remove extra supports after one successful retrieval cycle

Expected effect:

- learner uses the cue to retrieve rather than only to reread

Reverification target:

- one cue-supported retrieval routine

### For M2-FM-03 No Spacing Routine

Primary interventions:

- schedule one short later review instead of one longer immediate block
- mark the next retrieval point visibly
- compare massed review with spaced retrieval results

Expected effect:

- learner participates in distributed retrieval over time

Reverification target:

- one spaced review cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of memory load, schedule feasibility, and support design rather than only increasing repetition.
