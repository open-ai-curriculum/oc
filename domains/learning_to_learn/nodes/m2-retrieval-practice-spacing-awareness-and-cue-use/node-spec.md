# M2 Node Specification

## Node Identity

- id: `M2`
- name: `retrieval_practice_spacing_awareness_and_cue_use`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner attempts to recall information before rereading and uses spaced review and cues to strengthen memory over time.

## Dependencies

- `M1` encoding through chunking, representation, and rehearsal
- `S1` productive practice routine selection

## Exit Conditions

The learner is secure on `M2` only when all of the following are true:

1. attempts recall before immediately returning to the answer
2. uses at least one cue or retrieval prompt productively
3. revisits information across time rather than only massing one practice block
4. succeeds across more than one content type
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- try to retrieve before rereading
- use a cue card prompt or retrieval prompt
- participate in spaced review rather than one-time exposure

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
