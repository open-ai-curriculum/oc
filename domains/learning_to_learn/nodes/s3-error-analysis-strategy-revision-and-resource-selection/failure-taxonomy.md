# S3 Failure Taxonomy

### S3-FM-01 Error Without Diagnosis

- description: learner recognizes that an attempt failed but cannot identify why

### S3-FM-02 Same-Strategy Repetition

- description: learner repeats the same ineffective strategy after failure

### S3-FM-03 Resource Mismatch

- description: learner chooses a resource that does not match the diagnosed breakdown
