# S3 Teacher Observability View

Teachers should see:

- current `S3` status
- dependency posture for `S2`, `R2`, and `C3`
- direct strategy-revision stability
- transfer and retention posture
- dominant failure mode
- recommended intervention

Class-level view should highlight:

- who cannot diagnose why an attempt failed
- who repeats the same ineffective strategy
- who selects the wrong resource for the diagnosed problem
