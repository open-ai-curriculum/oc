# A1 Teacher Observability View

Teachers should see:

- current `A1` status
- direct task-entry stability
- transfer and retention posture
- dominant failure mode
- whether the learner orients to the correct task target
- recommended intervention

Class-level view should highlight:

- who does not enter tasks in a usable time window
- who cannot sustain orientation through the first routine
- who starts but does not complete the initial bounded task segment
