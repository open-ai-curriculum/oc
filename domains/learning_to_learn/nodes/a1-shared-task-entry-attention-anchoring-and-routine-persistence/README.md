# A1 Shared Task Entry, Attention Anchoring, And Routine Persistence

## Purpose

This node defines the earliest governed task-entry readiness surface in the `learning_to_learn` domain.

It exists so later learning-process work does not assume learners can already enter a task, orient to the target, and remain engaged long enough for learning to begin.
