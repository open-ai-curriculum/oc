# A1 Intervention Playbook

## Purpose

Map `A1` failure modes to bounded intervention responses.

## Intervention Rules

### For A1-FM-01 Task Entry Refusal Or Delay

Primary interventions:

- use a short predictable entry routine with one clear start signal
- reduce nonessential choices at the start of the task
- begin with a brief high-success first action

Expected effect:

- learner joins the task in a usable time window

Reverification target:

- one direct task-entry routine

### For A1-FM-02 Attention Anchor Instability

Primary interventions:

- make the target explicit through pointing, highlighting, or workspace simplification
- present one focal stimulus at a time
- restate the task target before the first action

Expected effect:

- learner orients to the relevant target long enough to begin

Reverification target:

- one orientation-sensitive routine

### For A1-FM-03 Routine Persistence Breakdown

Primary interventions:

- shorten the first work segment so completion is reachable
- use a visible first-then frame
- reinforce completion of the initial routine before extending duration

Expected effect:

- learner remains engaged through the first bounded task segment

Reverification target:

- one full start-to-first-completion routine

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of attention, regulation, language access, and context factors rather than repeating the same entry routine blindly.
