# A1 Failure Taxonomy

### A1-FM-01 Task Entry Refusal Or Delay

- description: learner does not join the task in a usable time window without repeated restart attempts

### A1-FM-02 Attention Anchor Instability

- description: learner enters the routine but does not orient to the relevant target long enough to begin meaningfully

### A1-FM-03 Routine Persistence Breakdown

- description: learner starts but drops out before completing the first bounded task segment
