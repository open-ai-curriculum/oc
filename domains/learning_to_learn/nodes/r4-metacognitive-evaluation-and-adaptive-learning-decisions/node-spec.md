# R4 Node Specification

## Node Identity

- id: `R4`
- name: `metacognitive_evaluation_and_adaptive_learning_decisions`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner judges whether current learning is actually working and makes adaptive decisions about strategy, pacing, help-seeking, and review rather than relying on surface comfort or habit alone.

## Dependencies

- `R3` feedback integration, frustration regulation, and persistence decisions
- `S3` error analysis, strategy revision, and resource selection
- `M3` durable retention planning and mixed retrieval

## Exit Conditions

The learner is secure on `R4` only when all of the following are true:

1. evaluates learning quality using evidence rather than feeling alone
2. detects when a strategy, schedule, or support choice is no longer sufficient
3. makes an adaptive decision and explains why it is warranted
4. uses the routine across more than one context
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- explain whether a study approach is working and why
- decide when to switch strategies, seek help, or change pacing
- avoid overconfidence based only on fluency or familiarity

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
