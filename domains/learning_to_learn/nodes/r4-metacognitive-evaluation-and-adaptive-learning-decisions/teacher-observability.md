# R4 Teacher Observability View

Teachers should see:

- current `R4` status
- dependency posture for `R3`, `S3`, and `M3`
- whether self-evaluation is grounded in evidence
- whether the learner makes an adaptive decision when the current approach fails
- dominant failure mode
- recommended intervention

Class-level view should highlight:

- who mistakes fluency for mastery
- who notices failure but does not adapt
- who changes plans too often without evidence
