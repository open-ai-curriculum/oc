# M4 Teacher Observability View

Teachers should see:

- current `M4` status
- dependency posture for `M3`, `S3`, and `T1`
- whether knowledge is consolidated into connected structures
- whether long-horizon retrieval planning is actually used
- dominant failure mode
- recommended intervention

Class-level view should highlight:

- who studies only for the next checkpoint
- who cannot synthesize related ideas
- who collapses on longer-delay retrieval
