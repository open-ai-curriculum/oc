# M4 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `M4`.

## Guidance

- prompt learners to connect ideas across lessons and units, not just rehearse isolated pieces
- require retrieval before re-study when diagnosing retention
- surface a visible long-horizon review plan
- separate synthesis failure from memory-access failure before choosing support
- escalate to human review when course pacing makes long-horizon retention structurally unrealistic
