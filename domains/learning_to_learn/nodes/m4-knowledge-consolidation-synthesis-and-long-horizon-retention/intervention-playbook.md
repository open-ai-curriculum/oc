# M4 Intervention Playbook

## Purpose

Map `M4` failure modes to bounded intervention responses.

## Intervention Rules

### For M4-FM-01 Fragmented Retention Plan

Primary interventions:

- build a visible long-horizon retrieval calendar
- group related ideas into one consolidation routine
- replace massed review with scheduled mixed retrieval

Expected effect:

- learner uses a retention plan that extends beyond the next assignment

Reverification target:

- one multi-week retrieval cycle

### For M4-FM-02 Unsynthesized Knowledge

Primary interventions:

- require linked summaries or concept maps across related topics
- prompt explanation of how ideas connect before additional practice
- compare two related units or problem types explicitly

Expected effect:

- learner organizes knowledge into connected structures

Reverification target:

- one synthesis and retrieval cycle

### For M4-FM-03 Long-Delay Retrieval Collapse

Primary interventions:

- widen spacing gradually rather than jumping directly to long delays
- cue retrieval before re-study
- diagnose whether failure comes from weak consolidation or weak access cues

Expected effect:

- learner retrieves after longer delays without full reteach

Reverification target:

- one delayed retrieval cycle beyond the short-term window

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of workload assumptions, course pacing, and cross-course interference rather than only increasing retrieval volume.
