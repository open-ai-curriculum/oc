# M4 Node Specification

## Node Identity

- id: `M4`
- name: `knowledge_consolidation_synthesis_and_long_horizon_retention`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner consolidates learning across lessons and units, builds connected summaries, and manages long-horizon retrieval so important knowledge remains usable beyond short recheck windows.

## Dependencies

- `M3` durable retention planning and mixed retrieval
- `S3` error analysis, strategy revision, and resource selection
- `T1` near transfer across tasks, contexts, and representations

## Exit Conditions

The learner is secure on `M4` only when all of the following are true:

1. organizes knowledge into connected structures rather than isolated fragments
2. uses a deliberate long-horizon retention plan
3. synthesizes ideas across lessons, units, or related domains
4. retrieves and applies consolidated knowledge after delay
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `35`

## Observable Evidence

The learner can:

- create linked summaries or retrieval plans across multiple topics
- explain how related ideas fit together
- retain usable knowledge across longer instructional gaps

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
