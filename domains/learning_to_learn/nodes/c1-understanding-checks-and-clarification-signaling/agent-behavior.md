# C1 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `C1`.

## Guidance

- treat clarification as a normal learning move rather than an exception
- prompt for the specific unclear part, not only a generic help request
- keep the reattempt close to the clarification so usability can be observed
- do not infer understanding from compliance or silence alone
- escalate to human review when confusion signaling remains inaccessible after repeated bounded support
