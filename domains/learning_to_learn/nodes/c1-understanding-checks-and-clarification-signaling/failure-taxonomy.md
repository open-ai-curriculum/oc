# C1 Failure Taxonomy

### C1-FM-01 Silent Confusion

- description: learner does not signal misunderstanding even when performance shows the task demand was not understood

### C1-FM-02 Global Help Request Without Specific Need

- description: learner signals being stuck but cannot indicate what needs to be repeated, clarified, or shown

### C1-FM-03 Clarification Not Used

- description: learner receives clarification but does not use it to reattempt the task meaningfully
