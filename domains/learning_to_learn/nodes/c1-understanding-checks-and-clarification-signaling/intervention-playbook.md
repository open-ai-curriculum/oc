# C1 Intervention Playbook

## Purpose

Map `C1` failure modes to bounded intervention responses.

## Intervention Rules

### For C1-FM-01 Silent Confusion

Primary interventions:

- model simple confusion signals such as repeat please or show me
- pause before error escalation and invite a check for understanding
- normalize clarification as part of learning rather than as failure

Expected effect:

- learner signals misunderstanding before extended unproductive work

Reverification target:

- one supported instruction-following task

### For C1-FM-02 Global Help Request Without Specific Need

Primary interventions:

- offer sentence frames that name likely clarification needs
- prompt the learner to point to the unclear step word or example
- separate repeat-the-whole-task from repeat-one-part requests

Expected effect:

- learner can identify a usable clarification target

Reverification target:

- one targeted clarification exchange

### For C1-FM-03 Clarification Not Used

Primary interventions:

- provide clarification immediately before a reattempt
- require one next action after the clarification
- keep the reattempt short enough to show whether the new understanding is usable

Expected effect:

- learner uses the clarification to restart the task meaningfully

Reverification target:

- one clarify-then-reattempt cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language load, communication access, and task-design factors rather than repeating the same clarification prompt blindly.
