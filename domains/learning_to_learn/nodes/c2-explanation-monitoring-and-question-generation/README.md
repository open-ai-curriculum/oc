# C2 Explanation Monitoring And Question Generation

## Purpose

This node defines the next governed comprehension-monitoring surface in the `learning_to_learn` domain.

It exists so later diagnosis and transfer work does not assume learners can already notice when an explanation does not make sense and ask a targeted question about the missing part.
