# C2 Failure Taxonomy

### C2-FM-01 Passive Listening Without Check

- description: learner hears an explanation but does not monitor whether it makes sense

### C2-FM-02 Vague Questioning

- description: learner asks for help but cannot target the missing step, meaning, or example

### C2-FM-03 Answer Not Applied

- description: learner receives the answer to a targeted question but does not use it to continue
