# C2 Intervention Playbook

## Purpose

Map `C2` failure modes to bounded intervention responses.

## Intervention Rules

### For C2-FM-01 Passive Listening Without Check

Primary interventions:

- insert stop-and-check points during explanations
- ask the learner to name the last part that still made sense
- normalize pausing an explanation to monitor understanding

Expected effect:

- learner actively checks explanations for meaning

Reverification target:

- one pause-and-check explanation routine

### For C2-FM-02 Vague Questioning

Primary interventions:

- provide stems for step, meaning, and example questions
- ask the learner to point to the unclear part before asking
- limit the question to one missing piece at a time

Expected effect:

- learner asks a specific question tied to the actual gap

Reverification target:

- one targeted question exchange

### For C2-FM-03 Answer Not Applied

Primary interventions:

- require one immediate next action after the answer is given
- shorten the distance between answer and reattempt
- compare before-and-after attempts to show what changed

Expected effect:

- learner uses the answer to continue the task

Reverification target:

- one answer-then-continue cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language complexity, explanation format, and communication access rather than repeating the same questioning frame.
