# S2 Failure Taxonomy

### S2-FM-01 One-Strategy Default

- description: learner uses the same strategy regardless of task demand

### S2-FM-02 Worked Example Copying

- description: learner copies the worked example surface without understanding how to adapt it

### S2-FM-03 Notes Without Reuse Value

- description: learner records notes that do not preserve the information needed for later performance
