# S2 Node Specification

## Node Identity

- id: `S2`
- name: `strategy_matching_worked_example_use_and_note_capture`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner chooses among more than one strategy based on task type and uses worked examples or notes to support later performance.

## Dependencies

- `S1` productive practice routine selection
- `C2` explanation monitoring and question generation

## Exit Conditions

The learner is secure on `S2` only when all of the following are true:

1. chooses a strategy that fits the task demand from more than one option
2. uses a worked example to guide, not replace, problem solving
3. records notes or organizational cues that support later re-use
4. succeeds across more than one task type
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- distinguish between at least two strategy options
- use a worked example as a model
- capture notes that support later recall or execution

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
