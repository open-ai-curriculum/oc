# S2 Intervention Playbook

## Purpose

Map `S2` failure modes to bounded intervention responses.

## Intervention Rules

### For S2-FM-01 One-Strategy Default

Primary interventions:

- contrast two strategy choices on the same task family
- label when each strategy fits best
- reduce choices to two high-value options first

Expected effect:

- learner chooses between strategies based on task features

Reverification target:

- one strategy-match routine

### For S2-FM-02 Worked Example Copying

Primary interventions:

- hide one step of the worked example and require adaptation
- compare the model problem with a near-transfer problem
- prompt the learner to explain what must change

Expected effect:

- learner uses the worked example as a model rather than a script to copy

Reverification target:

- one model-to-new-problem adaptation cycle

### For S2-FM-03 Notes Without Reuse Value

Primary interventions:

- narrow notes to one reusable structure or cue
- review whether the note supports later action
- compare a reusable note with a nonreusable note

Expected effect:

- learner captures notes that support later retrieval or execution

Reverification target:

- one note-capture-then-reuse cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of task complexity, notation demands, and support format rather than only adding more examples.
