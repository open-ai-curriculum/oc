# M1 Node Specification

## Node Identity

- id: `M1`
- name: `encoding_through_chunking_representation_and_rehearsal`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses simple repetition, grouping, representation, gesture, or verbal rehearsal to hold and encode new information instead of relying on passive exposure alone.

## Dependencies

- `A1` shared task entry, attention anchoring, and routine persistence
- `C1` understanding checks and clarification signaling

## Exit Conditions

The learner is secure on `M1` only when all of the following are true:

1. uses a simple encoding routine when new information is introduced
2. groups or represents information in a way that supports later recall
3. rehearses key information long enough to support immediate reuse
4. succeeds across more than one content type or representation format
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- repeat or rehearse new information with purpose
- use grouping, drawing, gesture, or verbal structure to hold information
- recall the encoded information shortly after initial presentation

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
