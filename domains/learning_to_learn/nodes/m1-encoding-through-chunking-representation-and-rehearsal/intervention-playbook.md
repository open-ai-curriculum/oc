# M1 Intervention Playbook

## Purpose

Map `M1` failure modes to bounded intervention responses.

## Intervention Rules

### For M1-FM-01 Passive Exposure Reliance

Primary interventions:

- require an active repeat say show or mark move immediately after presentation
- model one encoding routine before expecting independent use
- keep the amount of new information small enough for one deliberate encoding cycle

Expected effect:

- learner performs an active encoding move instead of only observing

Reverification target:

- one encode-then-use routine

### For M1-FM-02 Fragile Holding Of Information

Primary interventions:

- shorten the amount of information held at one time
- use visible supports during the first rehearsal cycle
- repeat the encode-and-recall cycle before adding new information

Expected effect:

- learner holds the information long enough for immediate reuse

Reverification target:

- one short-delay recall attempt

### For M1-FM-03 Mismatched Representation Or Chunking

Primary interventions:

- choose a representation that matches the structure of the information
- model smaller or more stable chunks
- compare a successful encoding example with an unsuccessful one

Expected effect:

- learner uses a representation or grouping that preserves the intended structure

Reverification target:

- one represent-and-recall routine

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of working-memory load, language load, and accessibility factors rather than only repeating the same rehearsal demand.
