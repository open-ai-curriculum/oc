# R1 Node Specification

## Node Identity

- id: `R1`
- name: `self_monitoring_help_seeking_and_feedback_acceptance`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices when work is stuck, requests help in a usable way, and attempts revision after feedback instead of treating correction as the end of the task.

## Dependencies

- `A1` shared task entry, attention anchoring, and routine persistence

## Exit Conditions

The learner is secure on `R1` only when all of the following are true:

1. notices and signals when progress has stalled
2. requests help in a way that provides a usable next step
3. receives feedback without abandoning the task
4. attempts a revision after feedback or support
5. remains stable across more than one task type and on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- indicate being stuck before extended unproductive behavior
- ask for help with a usable signal or request
- make a revision attempt after feedback

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
