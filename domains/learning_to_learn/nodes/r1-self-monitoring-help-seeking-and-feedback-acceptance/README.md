# R1 Self-Monitoring, Help-Seeking, And Feedback Acceptance

## Purpose

This node defines the earliest governed self-regulation surface in the `learning_to_learn` domain.

It exists so later reflection and adaptive decision work does not assume learners can already notice being stuck, ask for support in a usable way, and remain available for revision after feedback.
