# R1 Failure Taxonomy

### R1-FM-01 Stuck State Not Signaled

- description: learner remains unproductively stuck without signaling the need for help

### R1-FM-02 Help Request Not Usable

- description: learner asks for help in a way that does not identify a usable next step or target

### R1-FM-03 Feedback Rejection Or Withdrawal

- description: learner disengages, resists, or does not attempt revision after receiving feedback
