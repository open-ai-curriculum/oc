# T1 Failure Taxonomy

### T1-FM-01 Context Lock

- description: learner can use the strategy only in the original training context

### T1-FM-02 Representation Shift Breakdown

- description: learner loses the strategy when the task is presented in a new representation or format

### T1-FM-03 Transfer Without Monitoring

- description: learner attempts transfer but does not check whether the strategy still fits the new context
