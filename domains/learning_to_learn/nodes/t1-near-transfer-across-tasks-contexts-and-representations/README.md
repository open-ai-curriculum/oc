# T1 Near Transfer Across Tasks, Contexts, And Representations

## Purpose

This node defines the first governed transfer surface in the `learning_to_learn` domain.

It exists so far-transfer and independent-learning claims are not made before learners can apply a stable learning strategy in a new but related task, context, or representation.
