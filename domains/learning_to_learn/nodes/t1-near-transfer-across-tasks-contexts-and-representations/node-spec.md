# T1 Node Specification

## Node Identity

- id: `T1`
- name: `near_transfer_across_tasks_contexts_and_representations`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner applies a known learning strategy in a new but related task, context, or representation rather than only in the original training format.

## Dependencies

- `C2` explanation monitoring and question generation
- `M2` retrieval practice, spacing awareness, and cue use
- `S2` strategy matching, worked-example use, and note capture
- `R2` goal setting, progress tracking, and simple reflection

## Exit Conditions

The learner is secure on `T1` only when all of the following are true:

1. identifies a familiar learning move that still applies in the new context
2. adapts the move to a changed task, representation, or subject surface
3. monitors whether the transfer move is still working
4. succeeds across more than one near-transfer context
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- bring a strategy from one task into a related new task
- adapt the strategy to a changed representation or context
- notice whether the transfer is still productive

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
