# T2 Failure Taxonomy

### T2-FM-01 Context-Bound Strategy Use

- description: learner can transfer only when the new context still closely resembles the original one

### T2-FM-02 Weak Abstraction

- description: learner cannot explain the underlying learning principle well enough to reuse it in a novel context

### T2-FM-03 Independent Orchestration Breakdown

- description: learner identifies a plausible transfer move but cannot coordinate planning, retention, revision, and self-monitoring long enough to use it independently
