# T2 Intervention Playbook

## Purpose

Map `T2` failure modes to bounded intervention responses.

## Intervention Rules

### For T2-FM-01 Context-Bound Strategy Use

Primary interventions:

- compare the familiar and novel contexts explicitly
- name the abstract feature that survives the surface change
- practice transfer in one moderately changed context before a fully novel one

Expected effect:

- learner sees what actually transfers beyond surface similarity

Reverification target:

- one bounded far-transfer cycle

### For T2-FM-02 Weak Abstraction

Primary interventions:

- require an explicit generalization statement before transfer
- compare multiple examples to derive the shared learning principle
- use reflection prompts that target principle rather than procedure wording

Expected effect:

- learner can state the abstraction that drives transfer

Reverification target:

- one abstraction-and-application cycle

### For T2-FM-03 Independent Orchestration Breakdown

Primary interventions:

- reduce the novelty level while preserving the need for orchestration
- externalize the independent learning plan across planning retention and revision steps
- verify each component of the orchestration separately before recombining

Expected effect:

- learner coordinates the full independent routine under novel conditions

Reverification target:

- one complete novel-context orchestration cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of novelty load, domain-language demands, and whether the target transfer claim is too broad for the current evidence.
