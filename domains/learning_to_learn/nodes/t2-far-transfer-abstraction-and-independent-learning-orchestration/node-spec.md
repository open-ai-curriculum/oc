# T2 Node Specification

## Node Identity

- id: `T2`
- name: `far_transfer_abstraction_and_independent_learning_orchestration`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner abstracts effective learning routines from prior experience and orchestrates independent learning across substantially changed tasks, disciplines, or representations without depending on one familiar script.

## Dependencies

- `T1` near transfer across tasks, contexts, and representations
- `A3` independent task structuring and attention management
- `M4` knowledge consolidation, synthesis, and long-horizon retention
- `S4` multi-strategy planning, source evaluation, and self-directed revision
- `R4` metacognitive evaluation and adaptive learning decisions

## Exit Conditions

The learner is secure on `T2` only when all of the following are true:

1. identifies abstract learning moves that survive major context changes
2. builds an independent learning plan for a novel context
3. selects and revises strategies, sources, pacing, and review without waiting for direct scripting
4. succeeds across more than one far-transfer context
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `35`

## Observable Evidence

The learner can:

- explain what general learning principle transfers to a new domain
- design an independent learning approach for a novel task
- sustain the approach long enough to produce usable evidence of learning

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
