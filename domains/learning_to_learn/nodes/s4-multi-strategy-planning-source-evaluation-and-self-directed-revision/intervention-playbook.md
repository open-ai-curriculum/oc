# S4 Intervention Playbook

## Purpose

Map `S4` failure modes to bounded intervention responses.

## Intervention Rules

### For S4-FM-01 Single-Strategy Fixation

Primary interventions:

- require two plausible strategy options before work proceeds
- compare expected strengths and limits of each option
- treat switching as evidence-based revision, not failure

Expected effect:

- learner considers more than one viable approach

Reverification target:

- one strategy-comparison cycle

### For S4-FM-02 Poor Source Judgment

Primary interventions:

- use a short source-fit checklist tied to the task
- compare one stronger and one weaker source explicitly
- require justification for source selection before use

Expected effect:

- learner selects sources based on usefulness and credibility

Reverification target:

- one source-evaluation cycle

### For S4-FM-03 Passive Revision Dependence

Primary interventions:

- require one independent revision move before external help is given
- make the evidence for revision visible
- reinforce self-correction when it is justified, not only when it is perfect

Expected effect:

- learner revises from evidence without waiting passively

Reverification target:

- one independent revision cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of task ambiguity, source accessibility, and expectations for independent judgment rather than only adding more options.
