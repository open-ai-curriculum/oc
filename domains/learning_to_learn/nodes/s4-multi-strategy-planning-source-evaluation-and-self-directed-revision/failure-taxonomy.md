# S4 Failure Taxonomy

### S4-FM-01 Single-Strategy Fixation

- description: learner commits to one approach and does not compare alternatives even when the first strategy is weak

### S4-FM-02 Poor Source Judgment

- description: learner selects sources by convenience or familiarity rather than credibility, task fit, or evidence value

### S4-FM-03 Passive Revision Dependence

- description: learner waits for external correction instead of revising independently from available evidence
