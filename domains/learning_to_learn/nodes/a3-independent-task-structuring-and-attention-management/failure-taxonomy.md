# A3 Failure Taxonomy

### A3-FM-01 No Usable Task Structure

- description: learner starts complex work without converting it into a workable sequence or time-bounded plan

### A3-FM-02 Attention Load Collapse

- description: learner's plan exists but attention drifts, overload, or pacing collapse prevents sustained execution

### A3-FM-03 Plan Abandonment Under Friction

- description: learner encounters difficulty, feedback, or delay and abandons the plan rather than adapting it
