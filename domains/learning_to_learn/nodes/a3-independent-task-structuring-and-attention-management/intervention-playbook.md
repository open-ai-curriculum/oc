# A3 Intervention Playbook

## Purpose

Map `A3` failure modes to bounded intervention responses.

## Intervention Rules

### For A3-FM-01 No Usable Task Structure

Primary interventions:

- model a short planning template with start, checkpoint, and finish conditions
- require a visible sequence before work begins
- reduce task scope until the planning routine is actually used

Expected effect:

- learner begins with a workable execution plan

Reverification target:

- one independently planned work block

### For A3-FM-02 Attention Load Collapse

Primary interventions:

- shorten the time horizon of the plan
- add one self-timed checkpoint with a reset decision
- separate cognitive overload from simple distraction before reteaching

Expected effect:

- learner sustains attention through the next planned checkpoint

Reverification target:

- one full work cycle with checkpoint recovery

### For A3-FM-03 Plan Abandonment Under Friction

Primary interventions:

- rehearse one adaptation rule for when the original plan stops working
- require revision of the plan instead of restarting from zero
- reinforce completion through adaptation rather than perfection

Expected effect:

- learner adjusts the plan and finishes instead of abandoning the task

Reverification target:

- one friction-and-adaptation cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of executive-load assumptions, access supports, and task-design fit rather than only increasing independence demands.
