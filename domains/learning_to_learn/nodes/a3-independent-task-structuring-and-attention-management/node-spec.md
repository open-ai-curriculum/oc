# A3 Node Specification

## Node Identity

- id: `A3`
- name: `independent_task_structuring_and_attention_management`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner independently scopes a task, sequences the work, allocates attention, and completes a bounded academic block without relying on constant external re-entry support.

## Dependencies

- `A2` attention planning, distraction recovery, and task completion
- `R3` feedback integration, frustration regulation, and persistence decisions

## Exit Conditions

The learner is secure on `A3` only when all of the following are true:

1. breaks a task into an executable sequence before starting
2. sets an attention and pacing plan that can survive common distractions
3. monitors progress and adjusts the plan without abandoning the task
4. completes the work across more than one context
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- create a workable task plan without teacher decomposition
- sustain attention across a longer work interval
- revise pacing or sequencing when the original plan starts to fail

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
