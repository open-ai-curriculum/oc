# A3 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `A3`.

## Guidance

- require a visible work sequence before independent execution begins
- treat pacing and attention management as part of the plan, not as afterthoughts
- prompt adaptation of the plan before allowing abandonment and restart
- distinguish overload from distraction before choosing support
- escalate to human review when independent work repeatedly collapses despite bounded planning support
