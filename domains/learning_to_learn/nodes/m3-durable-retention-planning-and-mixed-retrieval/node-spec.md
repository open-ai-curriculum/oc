# M3 Node Specification

## Node Identity

- id: `M3`
- name: `durable_retention_planning_and_mixed_retrieval`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner plans review across time, mixes related content sets, and uses retrieval under reduced support to strengthen durable retention.

## Dependencies

- `M2` retrieval practice, spacing awareness, and cue use
- `R2` goal setting, progress tracking, and simple reflection

## Exit Conditions

The learner is secure on `M3` only when all of the following are true:

1. plans more than one retrieval point across time
2. uses mixed retrieval rather than only blocked same-item review
3. sustains retrieval with reduced cue support
4. succeeds across more than one content set
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- schedule later review deliberately
- mix related but distinct items or concepts
- retrieve with less support over time

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
