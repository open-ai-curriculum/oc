# M3 Durable Retention Planning And Mixed Retrieval

## Purpose

This node defines the middle-wave retention-planning surface in the `learning_to_learn` domain.

It exists so later long-horizon consolidation work does not assume learners can already plan review across time, mix related content, and sustain retrieval with reduced support.
