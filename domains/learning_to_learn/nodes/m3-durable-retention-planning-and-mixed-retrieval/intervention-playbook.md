# M3 Intervention Playbook

## Purpose

Map `M3` failure modes to bounded intervention responses.

## Intervention Rules

### For M3-FM-01 No Retention Plan

Primary interventions:

- schedule two concrete retrieval points instead of one
- attach review planning to a visible calendar or checkpoint system
- keep the interval plan small enough to execute reliably

Expected effect:

- learner plans distributed review intentionally

Reverification target:

- one multi-point retrieval plan

### For M3-FM-02 Blocked-Only Review Dependence

Primary interventions:

- mix two related item types first before expanding
- compare blocked and mixed review results explicitly
- keep the mixed set small enough for success

Expected effect:

- learner tolerates and uses mixed retrieval productively

Reverification target:

- one bounded mixed-retrieval set

### For M3-FM-03 Support Collapse

Primary interventions:

- fade one support at a time
- keep retrieval prompts shorter while preserving task structure
- compare supported and less-supported retrieval performance

Expected effect:

- learner retrieves successfully with reduced support

Reverification target:

- one faded-support retrieval cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of memory load, schedule conditions, and support design rather than only extending review frequency.
