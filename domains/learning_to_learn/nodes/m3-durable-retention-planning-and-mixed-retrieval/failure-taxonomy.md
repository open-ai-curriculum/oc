# M3 Failure Taxonomy

### M3-FM-01 No Retention Plan

- description: learner does not plan more than one retrieval point across time

### M3-FM-02 Blocked-Only Review Dependence

- description: learner reviews only one kind of item at a time and cannot tolerate mixed retrieval

### M3-FM-03 Support Collapse

- description: learner can retrieve with heavy supports but not when those supports are reduced
