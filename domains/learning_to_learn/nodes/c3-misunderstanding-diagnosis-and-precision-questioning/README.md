# C3 Misunderstanding Diagnosis And Precision Questioning

## Purpose

This node defines the middle-wave comprehension-diagnosis surface in the `learning_to_learn` domain.

It exists so later transfer and adaptive revision work does not assume learners can already identify where understanding broke down and ask a precise question about that specific breakdown.
