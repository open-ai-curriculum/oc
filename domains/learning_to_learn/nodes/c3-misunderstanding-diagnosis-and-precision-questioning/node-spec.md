# C3 Node Specification

## Node Identity

- id: `C3`
- name: `misunderstanding_diagnosis_and_precision_questioning`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner identifies where comprehension broke down and asks a precise question about the missing step, meaning, rule, or representation.

## Dependencies

- `C2` explanation monitoring and question generation
- `R2` goal setting, progress tracking, and simple reflection

## Exit Conditions

The learner is secure on `C3` only when all of the following are true:

1. identifies the specific location of misunderstanding
2. distinguishes between a knowledge gap, procedure gap, and representation gap
3. asks a precise question tied to that diagnosis
4. succeeds across more than one task or domain context
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- locate the step or idea that stopped making sense
- classify what kind of misunderstanding occurred
- ask a question that yields a usable next step

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
