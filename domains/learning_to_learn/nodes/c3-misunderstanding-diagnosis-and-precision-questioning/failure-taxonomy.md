# C3 Failure Taxonomy

### C3-FM-01 Global Confusion Label

- description: learner reports not understanding but cannot locate where the breakdown occurred

### C3-FM-02 Misclassified Breakdown

- description: learner identifies a problem but misclassifies whether it is about meaning, procedure, or representation

### C3-FM-03 Precision Gap In Questioning

- description: learner locates the breakdown but still asks a question too broad to produce a usable next step
