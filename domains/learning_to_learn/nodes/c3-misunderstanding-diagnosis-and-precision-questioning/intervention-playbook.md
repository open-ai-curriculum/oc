# C3 Intervention Playbook

## Purpose

Map `C3` failure modes to bounded intervention responses.

## Intervention Rules

### For C3-FM-01 Global Confusion Label

Primary interventions:

- require the learner to point to the first place the breakdown appears
- compare the last clear step with the first unclear step
- use one bounded work sample for diagnosis instead of the whole task

Expected effect:

- learner locates the actual breakdown point

Reverification target:

- one locate-the-breakdown routine

### For C3-FM-02 Misclassified Breakdown

Primary interventions:

- contrast meaning, procedure, and representation errors with short examples
- ask the learner to sort the current problem into one of those buckets
- keep the classification choice explicit before questioning

Expected effect:

- learner classifies the misunderstanding more accurately

Reverification target:

- one classify-the-breakdown routine

### For C3-FM-03 Precision Gap In Questioning

Primary interventions:

- narrow the question to one missing piece
- provide precision stems for rule, step, or representation questions
- require a question that can be answered in one bounded move

Expected effect:

- learner asks a question that yields a usable next step

Reverification target:

- one precision-question exchange

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language load, task complexity, and explanation design rather than only extending diagnosis prompts.
