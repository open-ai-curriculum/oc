# S1 Intervention Playbook

## Purpose

Map `S1` failure modes to bounded intervention responses.

## Intervention Rules

### For S1-FM-01 Guess-And-Move-On Pattern

Primary interventions:

- require a visible check step after each attempt
- provide a model answer cue or success criterion for comparison
- slow the pace so checking happens before the next attempt

Expected effect:

- learner checks an attempt before moving on

Reverification target:

- one short practice cycle with explicit check step

### For S1-FM-02 Error Ends The Routine

Primary interventions:

- make one correction attempt the expected next move after an error
- use brief error recovery routines with immediate restart
- reinforce revised attempts, not only first-try success

Expected effect:

- learner continues into a correction step after an initial error

Reverification target:

- one error-then-correct cycle

### For S1-FM-03 Ineffective Routine Match

Primary interventions:

- narrow the routine options to one or two task-matched choices
- model why one routine fits the task better than another
- pair the routine name with the task type repeatedly

Expected effect:

- learner chooses a practice routine that fits the task demand

Reverification target:

- one routine-selection task followed by practice

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of task difficulty, motivation context, and accessibility supports rather than only increasing repetition.
