# S1 Productive Practice Routine Selection

## Purpose

This node defines the earliest governed practice-quality surface in the `learning_to_learn` domain.

It exists so later strategy and revision work does not assume learners can already use a bounded productive-practice routine instead of repeating errors or abandoning the task immediately.
