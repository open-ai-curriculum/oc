# S1 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `S1`.

## Guidance

- require a bounded routine rather than open-ended repetition
- make the check step visible and unavoidable
- normalize correction as part of practice, not as punishment
- keep the routine matched to the task demand
- escalate to human review when productive practice remains inaccessible despite reduced task complexity and explicit support
