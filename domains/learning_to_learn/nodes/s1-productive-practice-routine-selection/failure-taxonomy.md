# S1 Failure Taxonomy

### S1-FM-01 Guess-And-Move-On Pattern

- description: learner produces attempts without checking them against a cue, example, or expected result

### S1-FM-02 Error Ends The Routine

- description: learner stops or disengages after an initial error instead of making a correction attempt

### S1-FM-03 Ineffective Routine Match

- description: learner uses a routine that does not fit the task demand well enough to support progress
