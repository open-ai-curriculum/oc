# R3 Feedback Integration, Frustration Regulation, And Persistence Decisions

## Purpose

This node defines the middle-wave self-regulation surface in the `learning_to_learn` domain.

It exists so later adaptive decision-making work does not assume learners can already incorporate critique, regulate frustration, and decide whether to persist, pause, or seek another support move.
