# R3 Failure Taxonomy

### R3-FM-01 Feedback Avoidance

- description: learner disengages or deflects before feedback can be used

### R3-FM-02 Frustration Escalation

- description: learner becomes dysregulated enough that revision cannot proceed productively

### R3-FM-03 Poor Persistence Decision

- description: learner chooses to stop, continue, or seek help in a way that does not fit the actual task state
