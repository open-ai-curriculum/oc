# R3 Intervention Playbook

## Purpose

Map `R3` failure modes to bounded intervention responses.

## Intervention Rules

### For R3-FM-01 Feedback Avoidance

Primary interventions:

- reduce feedback to one bounded change target
- deliver critique with an immediate revision action
- separate evaluation of the work from evaluation of the learner

Expected effect:

- learner remains available to use feedback

Reverification target:

- one feedback-then-revise cycle

### For R3-FM-02 Frustration Escalation

Primary interventions:

- use one preselected regulation routine before revision
- shorten the revision demand
- re-enter revision only after a visible reset step

Expected effect:

- learner regulates enough to continue with revision or a supported pause

Reverification target:

- one regulate-then-revise cycle

### For R3-FM-03 Poor Persistence Decision

Primary interventions:

- define stop continue and seek-help thresholds in advance
- compare productive and unproductive persistence choices
- ask for the next best move based on current evidence

Expected effect:

- learner chooses a better next action when effort becomes difficult

Reverification target:

- one challenge-point decision cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of affect load, trust conditions, and support design rather than only increasing performance pressure.
