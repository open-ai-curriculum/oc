# A2 Node Specification

## Node Identity

- id: `A2`
- name: `attention_planning_distraction_recovery_and_task_completion`
- domain: `learning_to_learn`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `cross-domain-learning-support-node-package`
- evidence class: `Synthetic`

## Purpose

The learner anticipates simple distractions, uses a reset routine, and completes bounded work without constant adult-led re-entry.

## Dependencies

- `A1` shared task entry, attention anchoring, and routine persistence
- `R1` self-monitoring, help-seeking, and feedback acceptance

## Exit Conditions

The learner is secure on `A2` only when all of the following are true:

1. identifies at least one likely distraction or off-task pattern before or during work
2. uses a reset routine to return to the task after drift
3. completes a bounded task segment with limited re-entry support
4. succeeds across more than one task format
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.80`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- name or recognize common distraction points
- return to the task using a known reset routine
- persist through a bounded work block

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
