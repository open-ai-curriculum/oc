# A2 Failure Taxonomy

### A2-FM-01 No Distraction Plan

- description: learner enters work without anticipating common distraction points or support routines

### A2-FM-02 Drift Without Recovery

- description: learner goes off task and does not use a known reset routine to return

### A2-FM-03 Incomplete Work Cycle

- description: learner can re-enter briefly but does not sustain effort through the bounded task segment
