# A2 Intervention Playbook

## Purpose

Map `A2` failure modes to bounded intervention responses.

## Intervention Rules

### For A2-FM-01 No Distraction Plan

Primary interventions:

- identify one predictable distraction point before work begins
- choose one concrete reset routine in advance
- externalize the plan in a visible cue

Expected effect:

- learner starts work with a usable plan for distraction and recovery

Reverification target:

- one planned work block

### For A2-FM-02 Drift Without Recovery

Primary interventions:

- rehearse one reset routine outside the full task first
- prompt recovery at the earliest sign of drift
- reduce the distance between drift and reset action

Expected effect:

- learner returns to the task after brief drift

Reverification target:

- one drift-and-reset cycle

### For A2-FM-03 Incomplete Work Cycle

Primary interventions:

- shorten the bounded work segment
- add one mid-task checkpoint
- reinforce completion of the full cycle rather than only starting

Expected effect:

- learner completes the planned work block

Reverification target:

- one full planned work cycle

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of attention load, environmental supports, and task-length assumptions rather than only increasing prompts.
