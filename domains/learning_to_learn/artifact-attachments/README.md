# Learning-To-Learn Artifact Attachments

This directory contains machine-readable attachment registries that bind support assets to graph-native `learning_to_learn` entities.

## Purpose

These registries operationalize the learning-to-learn graph by making support assets addressable against capability nodes rather than leaving them as informal notes or detached examples.

## Files

- `registry.schema.json`: canonical JSON Schema for learning-to-learn artifact attachment registries
- `learning-to-learn-core-support-artifact-registry.yaml`: initial draft registry for upper-band independent-learning support assets

## Claim Boundary

These registry entries define attachment intent and graph linkage. They do not by themselves establish that the referenced support assets are authored, reviewed, or learner-ready.
