# Learning-To-Learn Cross-Domain Attachments

## Purpose

This directory records governed attachment surfaces between subject-domain nodes and `learning_to_learn` nodes.

These attachments exist to make learning-process support explicit in graph form without replacing subject-domain prerequisites or mastery evidence.

## Current Status

The current registry is a draft modeled surface.

It establishes:

- the machine-readable attachment shape
- a first bounded mathematics fraction-slice attachment set for `F1` through `F5`
- a small illustrative extension set across ELA/literacy, science, and social studies
- explicit claim boundaries for future routing logic

## Included Artifacts

- `attachment-registry.yaml`

The first bounded subject-domain application of this registry is described in:

- [fraction-slice-cross-domain-attachment-plan.md](/Volumes/data/development/oc/domains/learning_to_learn/fraction-slice-cross-domain-attachment-plan.md)

## Claim Boundary

These records are draft graph-attachment models. They do not yet claim reviewed cross-domain routing behavior or production-grade adaptive logic.
