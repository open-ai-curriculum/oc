# Learning-To-Learn Operational Graph Export

## Purpose

This export is the first physically unified operational learning-to-learn graph package for external graph tooling and adaptive-system traversal prototypes.

It joins these layers into one importable graph surface:

- learning-to-learn capability nodes and prerequisite topology
- capability families
- support artifact registry and attached artifact nodes
- runtime slice and decision-rule nodes

## Files

- `learning-to-learn-operational-nodes.csv`
- `learning-to-learn-operational-relationships.csv`
- `learning-to-learn-operational-summary.json`
- `import-learning-to-learn-operational-graph.cypher`
