# C1 Intervention Playbook

- For `C1-FM-01`, use one-change-at-a-time routines that restrict revision to one step per attempt.
- For `C1-FM-02`, use error-location prompts that compare intended and actual outcomes side by side.
- For `C1-FM-03`, use short persistence cycles with visible progress markers after each attempt.
- For `C1-FM-04`, use before-and-after checks that force explicit comparison after a revision.
