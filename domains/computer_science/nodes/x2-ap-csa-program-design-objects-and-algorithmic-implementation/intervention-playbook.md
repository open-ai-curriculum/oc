# X2 Intervention Playbook

- For `X2-FM-01`, use program-design analyses that require classes, methods, algorithms, and quality criteria to be named together.
- For `X2-FM-02`, use object-and-algorithm walkthroughs connecting method structure to bounded algorithmic intent.
- For `X2-FM-03`, use implementation review prompts using specification, testing, and maintainability evidence explicitly.
- For `X2-FM-04`, use paired AP CSA synthesis tasks that reapply the same integrated reasoning in a second program context.
