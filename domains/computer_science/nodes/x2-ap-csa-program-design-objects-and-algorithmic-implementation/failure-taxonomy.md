# X2 Failure Taxonomy

- `X2-FM-01` syntax-first AP reasoning: learner treats AP CSA as syntax or API memorization without integrated design and algorithm reasoning.
- `X2-FM-02` object-algorithm disconnect: learner cannot connect class design and object behavior to the algorithm being implemented.
- `X2-FM-03` quality-blind implementation: learner discusses a solution without using specification, testing, or maintainability criteria.
- `X2-FM-04` csa-synthesis transfer weakness: learner can explain one familiar AP CSA pattern only and cannot adapt the same integrated reasoning to another bounded program.
