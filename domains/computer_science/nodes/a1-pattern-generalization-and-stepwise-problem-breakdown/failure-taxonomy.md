# A1 Failure Taxonomy

- `A1-FM-01` one-big-step collapse: learner treats a multi-step task as one vague command or wish.
- `A1-FM-02` pattern-noticing weakness: learner does not recognize repeated structure across similar cases.
- `A1-FM-03` step-boundary confusion: learner produces steps but the boundaries or order are unstable.
- `A1-FM-04` generalization-fragility: learner can solve one example only and cannot extend the pattern to a nearby case.
