# A1 Intervention Playbook

- For `A1-FM-01`, use break-it-down routines that force one large task into named smaller actions.
- For `A1-FM-02`, use same-and-different comparisons across two similar computing tasks.
- For `A1-FM-03`, use step-card sorting activities that stabilize order and step boundaries.
- For `A1-FM-04`, use near-transfer prompts that ask what stays the same when the task changes slightly.
