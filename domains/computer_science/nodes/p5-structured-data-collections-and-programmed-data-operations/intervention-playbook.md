# P5 Intervention Playbook

- For `P5-FM-01`, use collection-meaning routines that label what each position, key, or field stands for before any operation begins.
- For `P5-FM-02`, use stepwise iteration traces that record how the collection changes after each pass or update.
- For `P5-FM-03`, use structure-choice comparisons requiring a reason for selecting one bounded collection form over another.
- For `P5-FM-04`, use paired data-operation tasks that preserve the same pattern across a second dataset or collection type.
