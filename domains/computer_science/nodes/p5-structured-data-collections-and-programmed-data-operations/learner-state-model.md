# P5 Learner State Model

The learner-state for `P5` tracks:

- dependency security on `P4`, `R3`
- direct accuracy on structured data, collections, and programmed data operations tasks
- explanation quality for bounded programmed data-structure reasoning reasoning
- transfer performance across unfamiliar computing contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
