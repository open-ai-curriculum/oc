# I2 Intervention Playbook

- For `I2-FM-01`, use audience-matching routines that compare the same message for different recipients.
- For `I2-FM-02`, use own-work versus borrowed-content sorting tasks with simple attribution examples.
- For `I2-FM-03`, use tone-and-context checks for classroom digital communication.
- For `I2-FM-04`, use share-with-source prompts that require naming where content came from.
