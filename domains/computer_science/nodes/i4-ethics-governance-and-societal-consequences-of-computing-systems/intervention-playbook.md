# I4 Intervention Playbook

- For `I4-FM-01`, use stakeholder-impact mapping routines that require harms, benefits, and affected groups to be named explicitly.
- For `I4-FM-02`, use technical-choice-to-social-consequence prompts tying one bounded design decision to one downstream effect.
- For `I4-FM-03`, use governance scenario analyses comparing policy, review, and accountability mechanisms.
- For `I4-FM-04`, use paired ethics cases using the same stakeholder-and-governance structure in a second computing context.
