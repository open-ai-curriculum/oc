# R1 Node Specification

## Node Identity

- id: `R1`
- name: `symbols_encoding_and_representation_awareness`
- domain: `computer_science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `computer-science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner recognizes that symbols, icons, and simple encodings can stand for information or actions without confusing the representation with the thing represented.

## Dependencies

- `D1`

## Exit Conditions

The learner is secure on `R1` only when all of the following are true:

1. interprets simple symbols, icons, and bounded encodings correctly
2. creates or uses a simple code or representation to communicate information
3. explains that the same information can be represented in more than one way
4. succeeds across more than one simple representation context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- interpret icons, symbols, or simple codes with bounded meaning
- use one representation to stand for information or action intentionally
- compare two simple representations of the same information
- avoid confusing the picture, symbol, or code with the underlying content

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
