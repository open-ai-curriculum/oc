# R1 Failure Taxonomy

- `R1-FM-01` symbol-object collapse: learner treats a representation as the thing itself rather than as a stand-in.
- `R1-FM-02` code-meaning drift: learner loses track of what a simple code or symbol means.
- `R1-FM-03` single-representation lock: learner cannot recognize that the same information can appear in another form.
- `R1-FM-04` representation-context fragility: learner succeeds with one icon or code set only and cannot transfer to another bounded case.
