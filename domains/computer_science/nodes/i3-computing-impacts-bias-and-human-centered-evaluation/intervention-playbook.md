# I3 Intervention Playbook

- For `I3-FM-01`, use people-and-system comparison tasks that identify who is helped or harmed in bounded cases.
- For `I3-FM-02`, use data-choice and design-choice prompts tied to fairness outcomes.
- For `I3-FM-03`, use tradeoff evaluation routines that require one benefit and one risk to be named.
- For `I3-FM-04`, use human-centered design checks that ask who the user is and what conditions matter.
