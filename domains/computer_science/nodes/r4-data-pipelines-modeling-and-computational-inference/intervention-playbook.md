# R4 Intervention Playbook

- For `R4-FM-01`, use pipeline mapping routines that label each stage from input to interpreted output.
- For `R4-FM-02`, use before-and-after comparison tasks connecting one upstream data choice to one downstream inference effect.
- For `R4-FM-03`, use model-limit prompts requiring a statement about assumptions, missingness, or bounded uncertainty.
- For `R4-FM-04`, use paired pipeline analyses that preserve one workflow pattern across a second dataset or context.
