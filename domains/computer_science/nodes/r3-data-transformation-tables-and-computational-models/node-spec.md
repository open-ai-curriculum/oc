# R3 Node Specification

## Node Identity

- id: `R3`
- name: `data_transformation_tables_and_computational_models`
- domain: `computer_science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `computer-science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner transforms and analyzes data using structured tables, rules, or beginner computational models without treating transformed output as meaningful unless the rule and representation are understood.

## Dependencies

- `R2`
- `A2`

## Exit Conditions

The learner is secure on `R3` only when all of the following are true:

1. applies a bounded transformation rule to organized data
2. uses a table or simple model to explain how data changed
3. interprets the resulting representation with reference to the transformation process
4. succeeds across more than one bounded data-transformation context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- track what rule or operation was applied to the data
- explain how a transformed table differs from the original data organization
- connect the output representation back to the transformation process
- avoid treating automated output as self-explanatory

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
