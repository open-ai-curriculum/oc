# R3 Intervention Playbook

- For `R3-FM-01`, use before-and-after table comparisons that label the transformation rule explicitly.
- For `R3-FM-02`, use column-meaning checks requiring each data field to be interpreted in context.
- For `R3-FM-03`, use model-versus-original-data routines that distinguish representation from source information.
- For `R3-FM-04`, use near-transfer transformation tasks that preserve the same rule in a second bounded dataset.
