# N2 Failure Taxonomy

- `N2-FM-01` network-as-magic belief: learner treats network communication as unexplained appearance rather than movement through system parts.
- `N2-FM-02` protocol-purpose weakness: learner cannot explain why shared communication rules matter.
- `N2-FM-03` pathway-fragmentation: learner knows pieces of a networked task but not how the parts relate.
- `N2-FM-04` context-bound networking: learner understands one familiar example only and cannot transfer to another bounded connected-system case.
