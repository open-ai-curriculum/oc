# N2 Intervention Playbook

- For `N2-FM-01`, use send-and-receive pathway diagrams that name system parts and movement steps.
- For `N2-FM-02`, use shared-rule comparison tasks showing what happens when system parts do and do not follow the same rule.
- For `N2-FM-03`, use local-versus-networked scenario sorts with explicit reasoning prompts.
- For `N2-FM-04`, use paired transfer practice across two simple communication-system examples.
