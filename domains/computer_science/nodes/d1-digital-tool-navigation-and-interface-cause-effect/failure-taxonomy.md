# D1 Failure Taxonomy

- `D1-FM-01` random-interface-search: learner explores screens by random clicking without stable action-outcome reasoning.
- `D1-FM-02` control-function confusion: learner confuses what a button, icon, or input action is for.
- `D1-FM-03` input-output disconnect: learner does not connect their action to the resulting change on screen.
- `D1-FM-04` device-context fragility: learner succeeds in one interface only and cannot transfer to another bounded context.
