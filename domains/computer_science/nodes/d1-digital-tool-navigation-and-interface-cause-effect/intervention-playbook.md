# D1 Intervention Playbook

- For `D1-FM-01`, use guided action-and-outcome routines that name one interface action and one resulting change explicitly.
- For `D1-FM-02`, use icon and control matching tasks that connect common interface elements to functions.
- For `D1-FM-03`, use before-and-after screen comparisons that force explicit input-output talk.
- For `D1-FM-04`, use paired practice across two simple device or app contexts with the same action logic.
