# D1 Node Specification

## Node Identity

- id: `D1`
- name: `digital_tool_navigation_and_interface_cause_effect`
- domain: `computer_science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `computer-science-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses basic digital tools and recognizes that specific interface actions produce specific outcomes rather than treating screens and controls as unpredictable magic.

## Dependencies

None

## Exit Conditions

The learner is secure on `D1` only when all of the following are true:

1. uses simple digital interface actions intentionally in bounded tasks
2. predicts what a basic click, tap, drag, or typed input will do in familiar contexts
3. describes simple input-output cause-effect in a digital environment
4. succeeds across more than one device or application context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- use taps, clicks, selections, and typed input with clear intent
- name what changed on screen after an action
- distinguish successful navigation from random trial only
- connect a control or icon to its likely function in bounded cases

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
