# D2 Intervention Playbook

- For `D2-FM-01`, use workflow-mapping routines that label input, processing, storage, and output steps explicitly.
- For `D2-FM-02`, use before-during-after prompts that track where information is at each step.
- For `D2-FM-03`, use save-open-send comparison tasks with concrete classroom examples.
- For `D2-FM-04`, use paired transfer practice across two simple file or device routines.
