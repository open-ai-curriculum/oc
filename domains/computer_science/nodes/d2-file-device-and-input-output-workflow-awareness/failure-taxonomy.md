# D2 Failure Taxonomy

- `D2-FM-01` file-device-role confusion: learner confuses what a file, app, device, or display is doing in the workflow.
- `D2-FM-02` input-output chain break: learner cannot explain how information moves from one step to the next.
- `D2-FM-03` save-open-send collapse: learner treats storage, retrieval, and transmission as interchangeable actions.
- `D2-FM-04` workflow-fragility: learner can complete one familiar routine only and cannot transfer to another bounded workflow.
