# A3 Learner State Model

The learner-state for `A3` tracks:

- dependency security on `A2`, `P4`
- direct accuracy on algorithm analysis, tradeoffs, and formalized problem solving tasks
- explanation quality for bounded algorithm analysis reasoning reasoning
- transfer performance across unfamiliar computing contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
