# A3 Failure Taxonomy

- `A3-FM-01` first-correct-is-best belief: learner treats the first correct-looking algorithm as automatically optimal without comparison.
- `A3-FM-02` tradeoff-blind comparison: learner compares algorithms without acknowledging what is gained or lost.
- `A3-FM-03` constraint-detached analysis: learner analyzes procedures without connecting them to the formalized task constraints.
- `A3-FM-04` analysis-transfer weakness: learner can compare one familiar pair of algorithms only and cannot adapt the same analytic structure to a second bounded problem.
