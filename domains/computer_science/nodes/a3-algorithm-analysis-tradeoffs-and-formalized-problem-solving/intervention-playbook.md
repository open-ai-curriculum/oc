# A3 Intervention Playbook

- For `A3-FM-01`, use algorithm-comparison matrices that require correctness, efficiency, clarity, and resource criteria to be named explicitly.
- For `A3-FM-02`, use tradeoff prompts asking what one bounded design gains and what it sacrifices.
- For `A3-FM-03`, use problem-framing tasks that restate input, output, and constraint conditions before comparing approaches.
- For `A3-FM-04`, use paired analysis work using the same comparison structure across a second algorithm family.
