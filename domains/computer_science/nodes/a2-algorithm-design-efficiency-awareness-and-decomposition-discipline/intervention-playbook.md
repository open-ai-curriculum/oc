# A2 Intervention Playbook

- For `A2-FM-01`, use algorithm comparison tasks with explicit correctness, step-count, and clarity criteria.
- For `A2-FM-02`, use subproblem-mapping routines that require each part to connect to the whole solution.
- For `A2-FM-03`, use why-this-solution-is-better prompts tied to one named criterion.
- For `A2-FM-04`, use near-transfer redesign tasks that preserve structure while changing a bounded task detail.
