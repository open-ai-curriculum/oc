# P2 Failure Taxonomy

- `P2-FM-01` repeat-every-step writing: learner rewrites every repeated action instead of using a bounded repeated structure.
- `P2-FM-02` event-action disconnect: learner does not connect the trigger to the programmed response.
- `P2-FM-03` pattern-without-meaning: learner uses a loop or event block without understanding what repeated or triggered behavior it controls.
- `P2-FM-04` reuse-fragility: learner can use one familiar repeated pattern only and cannot transfer to another bounded case.
