# P2 Intervention Playbook

- For `P2-FM-01`, use repetition spotting tasks that ask which steps are the same and should be grouped.
- For `P2-FM-02`, use event-and-response matching routines in simple interactive programs.
- For `P2-FM-03`, use trace prompts showing what a loop or event actually does over time.
- For `P2-FM-04`, use paired coding tasks that reuse the same pattern in two beginner environments.
