# P3 Node Specification

## Node Identity

- id: `P3`
- name: `variables_conditionals_and_stateful_program_behavior`
- domain: `computer_science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `computer-science-middle-grade-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains how stored values, branching logic, and changing state affect program behavior without treating variables as labels only or conditionals as random forks.

## Dependencies

- `P2`
- `R2`

## Exit Conditions

The learner is secure on `P3` only when all of the following are true:

1. uses variables and conditionals meaningfully in a bounded program
2. explains how a change in stored state changes later program behavior
3. predicts which branch a conditional will follow in a bounded case
4. succeeds across more than one stateful programming context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `28`

## Observable Evidence

The learner can:

- describe what a variable stands for and how it changes
- trace a simple conditional outcome from the stated condition
- connect program state to later outputs or actions
- avoid treating stored values and displayed labels as interchangeable

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
