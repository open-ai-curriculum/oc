# P3 Intervention Playbook

- For `P3-FM-01`, use state-tracing routines that record variable values step by step.
- For `P3-FM-02`, use if-then branch walkthroughs with explicit true and false pathway comparisons.
- For `P3-FM-03`, use before-and-after state tables connecting updates to later outputs.
- For `P3-FM-04`, use paired transfer tasks using the same state-and-branch structure in a second program context.
