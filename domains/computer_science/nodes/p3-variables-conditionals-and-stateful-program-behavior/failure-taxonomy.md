# P3 Failure Taxonomy

- `P3-FM-01` variable-as-name only: learner treats a variable as a label only and not as a changing stored value.
- `P3-FM-02` conditional-branch confusion: learner cannot determine which path a bounded conditional should follow.
- `P3-FM-03` state-change opacity: learner does not connect updated state to later behavior in the program.
- `P3-FM-04` trace-fragility: learner can reason through one familiar stateful example only and cannot transfer to another bounded case.
