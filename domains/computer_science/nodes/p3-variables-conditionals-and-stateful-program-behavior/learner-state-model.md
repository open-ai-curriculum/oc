# P3 Learner State Model

The learner-state for `P3` tracks:

- dependency security on `P2`, `R2`
- direct accuracy on variables, conditionals, and stateful program behavior tasks
- explanation quality for bounded stateful programming reasoning reasoning
- transfer performance across unfamiliar computing contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
