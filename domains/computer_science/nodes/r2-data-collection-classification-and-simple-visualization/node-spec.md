# R2 Node Specification

## Node Identity

- id: `R2`
- name: `data_collection_classification_and_simple_visualization`
- domain: `computer_science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `computer-science-elementary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner gathers, sorts, and represents data for simple computational or informational tasks without treating data as an unstructured pile of facts.

## Dependencies

- `R1`
- `D2`

## Exit Conditions

The learner is secure on `R2` only when all of the following are true:

1. collects or records simple data intentionally in a bounded task
2. sorts or classifies data using a stable rule
3. uses a simple table, chart, or visualization to communicate the organized data
4. succeeds across more than one bounded data context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- record data with stable categories or attributes
- explain the rule used to sort or classify data
- match a simple visualization to the organized information it represents
- avoid mixing unrelated categories or changing the sorting rule mid-task

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
