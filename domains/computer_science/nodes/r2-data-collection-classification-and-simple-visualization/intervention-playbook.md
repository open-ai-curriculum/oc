# R2 Intervention Playbook

- For `R2-FM-01`, use sorting-rule prompts that require naming the classification basis before organizing data.
- For `R2-FM-02`, use data-to-chart matching tasks that link representations back to recorded entries.
- For `R2-FM-03`, use same-data-different-display comparisons across tables and simple charts.
- For `R2-FM-04`, use category-check routines that test whether each data item still fits the chosen rule.
