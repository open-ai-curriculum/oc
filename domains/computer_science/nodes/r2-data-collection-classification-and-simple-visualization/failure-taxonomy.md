# R2 Failure Taxonomy

- `R2-FM-01` data-pile reasoning: learner records or views data without a stable organizing structure.
- `R2-FM-02` category-drift: learner changes the classification rule during the task or mixes categories.
- `R2-FM-03` chart-data disconnect: learner cannot connect the table or chart back to the recorded data.
- `R2-FM-04` representation-choice fragility: learner can use one familiar display only and cannot transfer to another bounded data view.
