# N5 Failure Taxonomy

- `N5-FM-01` tool-list security: learner names security tools or practices without connecting them to a bounded threat model.
- `N5-FM-02` threat-mechanism mismatch: learner cannot explain why a particular defensive mechanism fits the stated risk.
- `N5-FM-03` security-without-tradeoffs: learner discusses security as if stronger controls never introduce bounded costs or constraints.
- `N5-FM-04` threat-model transfer weakness: learner can analyze one familiar attack scenario only and cannot adapt the same reasoning to another bounded system.
