# N5 Intervention Playbook

- For `N5-FM-01`, use threat-model templates that require assets, attacker goals, and likely vulnerabilities to be named explicitly.
- For `N5-FM-02`, use mechanism-to-risk mapping tasks pairing bounded attacks with defenses and explanations.
- For `N5-FM-03`, use security tradeoff prompts that compare protection gains with one cost in usability, performance, or maintenance.
- For `N5-FM-04`, use paired security analyses using the same threat-model structure in a second system context.
