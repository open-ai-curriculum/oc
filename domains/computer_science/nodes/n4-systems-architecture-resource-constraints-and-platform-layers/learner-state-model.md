# N4 Learner State Model

The learner-state for `N4` tracks:

- dependency security on `N3`, `P4`
- direct accuracy on systems architecture, resource constraints, and platform layers tasks
- explanation quality for bounded systems architecture reasoning reasoning
- transfer performance across unfamiliar computing contexts
- delayed retention outcome

`secure` requires all verification gates to pass. Any delayed failure or dependency instability should move the learner to `unstable`.
