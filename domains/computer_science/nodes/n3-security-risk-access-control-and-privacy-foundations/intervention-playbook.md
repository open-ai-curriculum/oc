# N3 Intervention Playbook

- For `N3-FM-01`, use risk-and-protection matching tasks that pair threats with bounded protective practices.
- For `N3-FM-02`, use authorized-versus-unauthorized access scenario sorts with explicit reasoning.
- For `N3-FM-03`, use privacy-boundary prompts that compare public, shared, and private cases.
- For `N3-FM-04`, use why-this-protection helps explanations tied to one concrete digital risk.
