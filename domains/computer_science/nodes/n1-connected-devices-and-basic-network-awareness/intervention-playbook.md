# N1 Intervention Playbook

- For `N1-FM-01`, use system-part naming routines for common classroom technology setups.
- For `N1-FM-02`, use local-versus-connected task comparisons that make the role of the connection explicit.
- For `N1-FM-03`, use simple send-and-receive pathway maps with arrows and bounded explanations.
- For `N1-FM-04`, use paired transfer practice across two common connected-device examples.
