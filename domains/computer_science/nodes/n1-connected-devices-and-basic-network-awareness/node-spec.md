# N1 Node Specification

## Node Identity

- id: `N1`
- name: `connected_devices_and_basic_network_awareness`
- domain: `computer_science`
- support tier: `Incubation`
- current maturity state: `Drafted`
- supported profile: `computer-science-elementary-node-package`
- evidence class: `Synthetic`

## Purpose

The learner explains at a simple level how devices connect and exchange information without treating a networked system as one giant indistinguishable machine.

## Dependencies

- `D2`

## Exit Conditions

The learner is secure on `N1` only when all of the following are true:

1. identifies that more than one device or system part may be involved in a digital task
2. explains in bounded language that information can move between connected devices
3. recognizes when a task depends on a connection rather than one isolated tool only
4. succeeds across more than one simple connected-device context
5. passes a delayed recheck after spacing

## Mastery Criteria

- direct accuracy threshold: `0.90`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- name at least two connected parts in a simple digital system
- describe a bounded information movement between devices or system parts
- distinguish a local action from one that depends on a connection
- avoid describing all network activity as unexplained magic

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
