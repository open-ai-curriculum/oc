# C2 Intervention Playbook

- For `C2-FM-01`, use trace tables that record values and branch choices line by line.
- For `C2-FM-02`, use test-purpose prompts that require naming what a chosen input is supposed to check.
- For `C2-FM-03`, use bug-location-before-fix routines that separate diagnosis from repair.
- For `C2-FM-04`, use after-the-fix checks comparing expected and actual behavior explicitly.
