# X1 Failure Taxonomy

- `X1-FM-01` topic-silo AP reasoning: learner treats programming, data, systems, and impacts as separate talking points rather than one integrated computing account.
- `X1-FM-02` technical-impact disconnect: learner cannot connect a bounded technical mechanism to a social or policy implication.
- `X1-FM-03` artifact-without-abstraction: learner describes a computational artifact without explaining the abstraction, data, or system ideas that support it.
- `X1-FM-04` csp-synthesis transfer weakness: learner can synthesize one familiar AP CSP case only and cannot adapt the same integration to another bounded scenario.
