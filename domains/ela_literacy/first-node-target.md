# First ELA/Literacy Node Target

## Purpose

Identify the first concrete node-package wave for the `ela_literacy` domain.

## Recommendation

The first implementation wave should begin at the true entry point of the literacy system:

1. `O1` joint attention, turn-taking, and response readiness
2. `O2` oral language comprehension for instructions and routines
3. `P1` book handling, print awareness, and directionality
4. `P3` phonological awareness

## Why This Wave

This is the highest-value first build because:

- it starts at pre-kindergarten rather than assuming kindergarten-ready literacy access
- it establishes upstream dependency integrity before formal reading and writing claims
- it supports later `P4`, `F1`, `R1`, and `W1` node work
- it creates the earliest governed learner-state surface for the domain

## Immediate Follow-On Wave

The next node wave is now seeded and should be deepened next:

1. `O3` oral recount, explanation, and sequence language
2. `P2` concept of word, spacing, and text tracking
3. `P4` alphabet knowledge and letter formation
4. `F1` alphabetic principle and sound-symbol mapping

## Next Recommended Wave

The next implementation wave is now seeded and should be deepened next:

1. `W1` drawing, dictation, labeling, and idea capture
2. `R1` literal comprehension of read-aloud and shared text
3. `L1` sentence formation and basic conventions
4. `F2` decoding and encoding simple words

## Following Wave

The next implementation wave is now seeded and should be deepened next:

1. `F3` automatic word recognition and connected text fluency
2. `R2` sentence-level comprehension of supported text
3. `W2` sentence generation and transcription
4. `L2` morphology, word meaning, and academic vocabulary

## Next Frontier

The next implementation wave is now seeded and should be deepened next:

1. `R3` evidence-based response to text
2. `W3` paragraph organization and source-based response
3. `R4` literary structure, theme, and character analysis
4. `R5` informational structure, author claim, and source use

## Following Frontier

The next implementation wave is now seeded and should be deepened next:

1. `W4` multi-paragraph explanation, narrative, and argument
2. `L3` syntax revision and style control
3. `R6` disciplinary literacy, rhetorical analysis, and cross-text synthesis
4. `W5` research synthesis and citation control

## Final Frontier

The remaining implementation wave is now seeded and should be deepened next:

1. `A1` rhetorical analysis and synthesis
2. `A2` literary interpretation, argument, and intertextual analysis

## Next Phase

With the full breadth now seeded from Pre-K through Grade 12, the next phase should shift from adding breadth to deepening package completeness.

Priority targets for deepening should include:

1. learner-state model artifacts
2. verification models
3. failure taxonomies
4. intervention playbooks

## Claim Boundary

This target document defines implementation priority only. It does not mean the listed nodes are already classroom-ready or fully reviewed.
