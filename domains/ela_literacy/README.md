# ELA/Literacy Domain

## Purpose

This domain defines the repository's ELA/literacy knowledge graph and the path to a full pre-K through Grade 12 literacy system.

The domain slug is `ela_literacy`. It includes reading, writing, oral language, language conventions, and disciplinary literacy dependencies that must be modeled together if downstream instructional artifacts are to remain dependency-aware.

## Scope

The current scaffold covers the full intended territory at the family level:

- oral language and listening foundations
- print concepts and early symbol control
- foundational decoding and encoding
- reading comprehension and analysis
- language conventions and vocabulary
- writing and composition
- advanced rhetorical and literary analysis

The intended target scope for this domain is:

- pre-K through Grade 12 ELA/literacy capability modeling
- early-literacy dependency structures
- disciplinary literacy surfaces
- advanced secondary and AP-adjacent reading/writing pathways

## Current Status

Operational domain package. The authoritative graph, full pre-K through Grade 12 node layer, standards-native overlay, and first graph-level runtime and attachment surfaces are now defined at the domain level.

Current maturity posture:

- master graph scaffolded
- Pre-K through Grade 12 spine codified
- all currently seeded node packages operationalized through the advanced-secondary frontier
- standards-augmented graph layer generated for graph-native standards traversal
- graph architecture and graph-level runtime contracts added
- initial artifact-attachment and runtime slice registries added
- export surfaces added for Gephi and Neo4j
