# WC7 Node Specification

## Node Identity

- id: `WC7`
- name: `research_synthesis`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `research-writing-node-package`
- evidence class: `Synthetic`

## Purpose

The learner synthesizes multiple sources into coherent research-based writing before attribution and citation control become the main distinction.

## Dependencies

- `WC6`
- `R6`

## Exit Conditions

The learner is secure on `WC7` only when all of the following are true:

1. synthesizes multiple sources into a coherent response
2. integrates sources around a controlling line of reasoning
3. succeeds across changed source sets or research questions
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- synthesize multiple sources into coherent research writing
- separate source synthesis from citation mechanics
- support advanced rhetorical and research writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
