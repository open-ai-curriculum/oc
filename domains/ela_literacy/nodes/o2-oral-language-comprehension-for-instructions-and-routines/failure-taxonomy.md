# O2 Failure Taxonomy

### O2-FM-01 Routine-Imitation Dependence

- description: learner appears successful only when following familiar routine imitation rather than actual oral-language comprehension

### O2-FM-02 Direction-Sequence Breakdown

- description: learner loses the intended action or order when a spoken direction contains more than one meaningful part

### O2-FM-03 Vocabulary Access Gap

- description: key oral-language meaning breaks because critical words or classroom language are not understood

### O2-FM-04 Context-Bound Comprehension

- description: learner succeeds with one speaker or setting but not when wording, partner, or context changes
