# O2 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `O2`.

## Guidance

- keep oral prompts short and semantically clear before increasing complexity
- do not treat routine imitation as evidence of comprehension
- change wording, speaker, or setting intentionally to test transfer
- separate vocabulary-access issues from attention or turn-taking issues
- escalate to human review when comprehension remains context-bound after repeated bounded support
