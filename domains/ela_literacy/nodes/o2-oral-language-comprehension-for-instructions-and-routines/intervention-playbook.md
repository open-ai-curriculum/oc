# O2 Intervention Playbook

## Purpose

Map `O2` failure modes to bounded intervention responses.

## Intervention Rules

### For O2-FM-01 Routine-Imitation Dependence

Primary interventions:

- vary wording while holding the intended action constant
- use action-selection checks that cannot be solved by copying a peer
- separate familiar routine cues from the oral-language cue

Expected effect:

- learner success becomes tied to comprehension rather than routine imitation

Reverification target:

- one changed-wording oral-direction task

### For O2-FM-02 Direction-Sequence Breakdown

Primary interventions:

- shorten directions to one meaningful action before rebuilding sequence
- gesture or visually segment the parts of a spoken direction
- practice first-then language in small routines

Expected effect:

- learner maintains the intended action or order across spoken directions

Reverification target:

- one bounded multi-part direction task

### For O2-FM-03 Vocabulary Access Gap

Primary interventions:

- preteach the key classroom words in the routine
- pair spoken language with concrete referents before fading support
- repeat the direction using closely related wording after the first attempt

Expected effect:

- learner gains access to the oral-language meaning that was blocking comprehension

Reverification target:

- one oral prompt using the previously blocking vocabulary

### For O2-FM-04 Context-Bound Comprehension

Primary interventions:

- repeat the same direction with a changed speaker or setting
- vary the routine surface while keeping the meaning target stable
- move from highly familiar to slightly less familiar contexts deliberately

Expected effect:

- learner transfers oral-language comprehension beyond one narrow setting

Reverification target:

- one changed-speaker or changed-setting comprehension task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language access, hearing/context factors, and upstream interaction stability.
