# WC3 Node Specification

## Node Identity

- id: `WC3`
- name: `paragraph_structure_and_coherence`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `paragraph-writing-node-package`
- evidence class: `Synthetic`

## Purpose

The learner organizes a coherent paragraph before text support and source integration become the main demand.

## Dependencies

- `WC2`

## Exit Conditions

The learner is secure on `WC3` only when all of the following are true:

1. organizes a paragraph around a coherent main idea
2. maintains logical sentence order and unity
3. succeeds across changed paragraph prompts or topics
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- organize coherent paragraphs without relying on source integration
- maintain unity across multiple sentences
- support later source-based response

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
