# R4 Failure Taxonomy

### R4-FM-01 Summary-Only Response

- description: learner recounts plot or surface events but does not move into literary interpretation

### R4-FM-02 Evidence-To-Interpretation Weakness

- description: learner names a theme, character trait, or structure feature but cannot connect it convincingly to textual evidence

### R4-FM-03 Literary-Element Confusion

- description: learner mixes up literary concepts such as theme, character motivation, or structure in ways that distort the analysis

### R4-FM-04 Text-And-Prompt Fragility

- description: learner succeeds on one familiar literary text or prompt frame but performance collapses on changed texts or prompt types
