# R4 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `R4`.

## Guidance

- keep literary prompts bounded enough that the learner can move beyond summary into supported interpretation
- require textual evidence before accepting broad literary claims
- distinguish literary-concept confusion from evidence-link weakness
- vary text and prompt only after one analysis frame stabilizes
- escalate to human review when interpretive failure cannot be separated from evidence-response weakness
