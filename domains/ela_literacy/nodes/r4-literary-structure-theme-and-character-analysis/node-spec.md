# R4 Node Specification

## Node Identity

- id: `R4`
- name: `literary_structure_theme_and_character_analysis`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `literary-analysis-node-package`
- evidence class: `Synthetic`

## Purpose

The learner analyzes literary structure, theme, character, and interpretation well enough to establish the literary-analysis branch of the literacy graph.

## Dependencies

- `RC6` evidence-based response to text

## Exit Conditions

The learner is secure on `R4` only when all of the following are true:

1. identifies and explains relevant literary elements using text support
2. develops theme or character interpretation with bounded textual grounding
3. distinguishes literary analysis from unsupported reaction or summary alone
4. succeeds across changed literary texts or prompt types
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- analyze theme, character, and structure in literary texts
- support literary claims with relevant textual detail
- move beyond summary into interpretation
- support later advanced literary analysis and intertextual work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
