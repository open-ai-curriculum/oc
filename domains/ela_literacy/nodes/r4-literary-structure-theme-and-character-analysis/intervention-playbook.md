# R4 Intervention Playbook

## Purpose

Map `R4` failure modes to bounded intervention responses.

## Intervention Rules

### For R4-FM-01 Summary-Only Response

Primary interventions:

- have the learner state what the text is saying about a character or theme after summary
- use a simple summary-versus-analysis contrast
- require one interpretive sentence after one recalled detail

Expected effect:

- learner moves from recounting the text to making a supported literary claim

Reverification target:

- one short literary analysis prompt

### For R4-FM-02 Evidence-To-Interpretation Weakness

Primary interventions:

- pair each claim with one cited detail and one why-it-matters explanation
- model the connection between evidence and interpretation before fading support
- keep the interpretive demand bounded to one theme or character idea

Expected effect:

- learner links literary claims to relevant textual evidence more clearly

Reverification target:

- one claim-evidence-explanation task

### For R4-FM-03 Literary-Element Confusion

Primary interventions:

- contrast closely related literary concepts using one short text
- ask which question belongs to theme, which belongs to character, and which belongs to structure
- reduce the number of literary concepts being asked at once

Expected effect:

- learner distinguishes literary elements accurately enough for valid analysis

Reverification target:

- one targeted literary-concept analysis task

### For R4-FM-04 Text-And-Prompt Fragility

Primary interventions:

- rotate to changed literary texts with the same analysis frame
- alternate theme and character prompts while preserving the evidence requirement
- reduce novelty only enough to observe transfer cleanly

Expected effect:

- learner generalizes literary analysis beyond one familiar text or prompt

Reverification target:

- one changed-text or changed-prompt literary-analysis task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of whether `RC6` is actually secure enough, and whether the interpretive demand exceeds the intended `R4` boundary.
