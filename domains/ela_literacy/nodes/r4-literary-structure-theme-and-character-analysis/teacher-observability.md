# R4 Teacher Observability View

Teachers should see:

- current `R4` status
- dependency posture for `RC6`
- direct literary-analysis accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner is analyzing or only summarizing
- recommended intervention

Class-level view should highlight:

- who stay at plot summary
- who can name a theme or trait but cannot support it with evidence
- who lose performance when the literary text or prompt changes
