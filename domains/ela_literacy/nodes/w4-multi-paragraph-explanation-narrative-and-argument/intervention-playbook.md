# W4 Intervention Playbook

## Purpose

Map `W4` failure modes to bounded intervention responses.

## Intervention Rules

### For W4-FM-01 Multi-Paragraph Coherence Collapse

Primary interventions:

- plan the response across paragraphs before drafting
- label the role of each paragraph in the full response
- review whether each paragraph advances the same central line of development

Expected effect:

- learner sustains a coherent response across multiple paragraphs

Reverification target:

- one short multi-paragraph writing task

### For W4-FM-02 Genre-Control Weakness

Primary interventions:

- contrast the structural demands of explanation, narrative, and argument
- keep topic stable while shifting genre expectations
- use one genre frame only as a bridge, not as the end state

Expected effect:

- learner adapts multi-paragraph structure to changed genre demands

Reverification target:

- one changed-genre writing task

### For W4-FM-03 Support-And-Development Thinness

Primary interventions:

- require one clear development move per paragraph such as evidence, elaboration, or event detail
- check paragraph development before adding more paragraphs
- keep support expectations visible during planning and drafting

Expected effect:

- learner develops ideas across paragraphs with enough support to remain meaningful

Reverification target:

- one development-focused multi-paragraph task

### For W4-FM-04 Planning-Transfer Fragility

Primary interventions:

- rotate to changed prompts while preserving a stable planning routine
- gradually remove familiar scaffolds once planning is visible
- distinguish planning dependence from genre misunderstanding

Expected effect:

- learner transfers sustained writing beyond one prompt or scaffold set

Reverification target:

- one changed-prompt multi-paragraph task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of whether `W3`, `R4`, and `R5` are actually secure enough, and whether the sustained-writing demand exceeds the intended `W4` boundary.
