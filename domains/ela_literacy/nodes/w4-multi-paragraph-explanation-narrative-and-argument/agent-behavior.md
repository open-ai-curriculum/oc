# W4 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `W4`.

## Guidance

- keep sustained-writing tasks bounded enough that coherence and development can actually be inspected
- require visible planning before long drafting when the learner is unstable
- distinguish genre weakness from paragraph-organization weakness
- vary genre and prompt only after one sustained-writing routine stabilizes
- escalate to human review when multi-paragraph writing failure cannot be separated from reading-analysis or paragraph-writing dependencies
