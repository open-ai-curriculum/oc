# W4 Failure Taxonomy

### W4-FM-01 Multi-Paragraph Coherence Collapse

- description: learner produces more than one paragraph, but the full response does not sustain a clear line of development

### W4-FM-02 Genre-Control Weakness

- description: learner can write in one genre or response type but loses structure when asked for explanation, narrative, or argument in changed conditions

### W4-FM-03 Support-And-Development Thinness

- description: learner states ideas across paragraphs but does not develop them with enough detail, evidence, or narrative support

### W4-FM-04 Planning-Transfer Fragility

- description: learner succeeds with one familiar planning frame or prompt type but performance collapses on changed prompts or genres
