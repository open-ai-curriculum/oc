# L3 Failure Taxonomy

### L3-FM-01 Surface-Edit Only

- description: learner corrects small errors but does not revise syntax or style in ways that improve meaning or flow

### L3-FM-02 Purpose-And-Audience Blindness

- description: learner revises language without regard to intended audience, task purpose, or rhetorical effect

### L3-FM-03 Sentence-Revision Instability

- description: learner can improve one sentence in isolation but cannot sustain revision control across a passage or changed writing context

### L3-FM-04 Vocabulary-And-Syntax Disconnect

- description: learner attempts more advanced language but revisions reduce clarity because syntax and word choice do not work together
