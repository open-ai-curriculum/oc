# L3 Node Specification

## Node Identity

- id: `L3`
- name: `syntax_revision_and_style_control`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `language-revision-node-package`
- evidence class: `Synthetic`

## Purpose

The learner revises syntax and style with enough control to improve clarity, coherence, and rhetorical effect in later analytical and research writing.

## Dependencies

- `L1` sentence formation and basic conventions
- `L2` morphology, word meaning, and academic vocabulary

## Exit Conditions

The learner is secure on `L3` only when all of the following are true:

1. revises sentence structure or style to improve meaning and coherence
2. makes language choices that fit intended purpose or audience
3. demonstrates control beyond error-correction-only editing
4. succeeds across changed writing contexts or revision prompts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- revise syntax to improve clarity or flow
- choose language for effect, precision, or coherence
- support stronger analytical, argumentative, and research writing
- prepare for advanced rhetorical control

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
