# P4 Failure Taxonomy

### P4-FM-01 Letter-Shape Confusion

- description: learner confuses target letters with visually similar forms or treats letters as unstable visual shapes

### P4-FM-02 Symbol-Boundary Breakdown

- description: learner does not distinguish letters reliably from numbers, marks, or nonletter symbols

### P4-FM-03 Formation Fragility

- description: learner recognizes letters but cannot form them with enough stability for early writing use

### P4-FM-04 Modality Dependency

- description: learner succeeds in one modality or letter set but collapses when the format changes
