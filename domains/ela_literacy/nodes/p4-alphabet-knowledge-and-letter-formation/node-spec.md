# P4 Node Specification

## Node Identity

- id: `P4`
- name: `alphabet_knowledge_and_letter_formation`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `literacy-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner recognizes letters and forms them with enough stability to support early encoding and later alphabetic-principle work.

## Dependencies

- `P1` book handling, print awareness, and directionality
- `P3` phonological awareness

## Exit Conditions

The learner is secure on `P4` only when all of the following are true:

1. recognizes a supported set of letters consistently
2. distinguishes letters from non-letter marks or symbols
3. forms letters with enough control to support early writing and encoding
4. succeeds across changed materials or modalities
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- identify letters in supported contexts
- produce letter forms with usable consistency
- treat letters as meaningful literacy symbols rather than only visual shapes
- support later sound-symbol mapping and early writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
