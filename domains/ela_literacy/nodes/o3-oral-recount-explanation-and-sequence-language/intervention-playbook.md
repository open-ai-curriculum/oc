# O3 Intervention Playbook

## Purpose

Map `O3` failure modes to bounded intervention responses.

## Intervention Rules

### For O3-FM-01 Sequence Collapse

Primary interventions:

- use first-next-last oral frames
- retell one brief event before moving to longer sequences
- support sequence language with visible event cards, then fade them

Expected effect:

- learner preserves event order more reliably in recount

Reverification target:

- one short recount-sequencing task

### For O3-FM-02 Referential Clarity Breakdown

Primary interventions:

- prompt the learner to restate who or what each sentence refers to
- use oral retell routines with explicit noun replacement before pronouns
- rehearse one referent at a time in short recounts

Expected effect:

- learner responses become more interpretable to a listener

Reverification target:

- one oral retell scored for referential clarity

### For O3-FM-03 Explanation Thinness

Primary interventions:

- use because, so, and how prompts tied to familiar events
- model one bounded explanation before asking for an independent response
- rehearse contrast between recount only and recount plus explanation

Expected effect:

- learner adds meaningful explanatory language rather than only listing events

Reverification target:

- one oral explanation prompt

### For O3-FM-04 Topic-Shift Fragility

Primary interventions:

- keep the recount frame stable while changing the topic
- move from familiar routines to slightly less familiar experiences gradually
- practice the same explanation structure across two topics

Expected effect:

- learner transfers oral recount and explanation beyond one familiar topic

Reverification target:

- one changed-topic recount task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of oral-language formulation demands and language-access factors.
