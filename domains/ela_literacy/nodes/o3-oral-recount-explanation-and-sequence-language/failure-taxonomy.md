# O3 Failure Taxonomy

### O3-FM-01 Sequence Collapse

- description: learner cannot preserve event order or sequence reliably in oral recount

### O3-FM-02 Referential Clarity Breakdown

- description: learner speaks, but the listener cannot reliably recover who, what, or where the response refers to

### O3-FM-03 Explanation Thinness

- description: learner recounts events but cannot provide enough why or how explanation to support later literacy tasks

### O3-FM-04 Topic-Shift Fragility

- description: learner succeeds on one familiar recount topic but loses coherence when topic or prompt surface changes
