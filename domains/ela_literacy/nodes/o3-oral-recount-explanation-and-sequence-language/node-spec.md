# O3 Node Specification

## Node Identity

- id: `O3`
- name: `oral_recount_explanation_and_sequence_language`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `literacy-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner recounts, explains, and sequences events orally with enough coherence to support later reading comprehension, evidence response, and writing production.

## Dependencies

- `O2` oral language comprehension for instructions and routines

## Exit Conditions

The learner is secure on `O3` only when all of the following are true:

1. recounts events or experiences in an intelligible sequence
2. explains basic what, why, or how relationships in oral language
3. maintains enough referential clarity to support listener understanding
4. succeeds across changed prompt topics or oral formats
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- retell events in order
- answer oral explanation prompts with relevant meaning
- use oral language to support later text response and writing planning
- provide a stable oral bridge into reading and writing tasks

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
