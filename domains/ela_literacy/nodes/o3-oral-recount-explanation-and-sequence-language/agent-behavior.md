# O3 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `O3`.

## Guidance

- keep oral tasks short enough that sequence and explanation remain visible
- model the recount frame before asking for an independent oral retell
- distinguish retell quality from simple talkativeness
- vary topic only after one recount structure stabilizes
- escalate to human review when explanatory language remains unusually thin despite stable comprehension
