# WC6 Node Specification

## Node Identity

- id: `WC6`
- name: `genre_controlled_explanation_narrative_and_argument`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `extended-composition-node-package`
- evidence class: `Synthetic`

## Purpose

The learner adapts sustained writing to explanation, narrative, and argument rather than producing one generic multi-paragraph form.

## Dependencies

- `WC5`
- `R4`
- `R5`

## Exit Conditions

The learner is secure on `WC6` only when all of the following are true:

1. adapts structure to explanation, narrative, and argument demands
2. maintains genre-appropriate development and reasoning
3. succeeds across changed genre tasks
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- control explanation, narrative, and argument as distinct writing demands
- use reading-analysis knowledge to support genre choices
- support later research and advanced interpretive writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
