# RC1 Failure Taxonomy

### RC1-FM-01 Detail Retrieval Collapse

- description: learner misses or distorts key literal details from the shared text

### RC1-FM-02 Routine-Cue Dependence

- description: learner answers from pattern or picture cues rather than actual text meaning

### RC1-FM-03 Recount Sequence Drift

- description: learner recalls some content but loses order or relevance

### RC1-FM-04 Prompt Interpretation Confound

- description: learner meaning appears stronger than the question-response surface reveals
