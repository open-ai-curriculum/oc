# RC1 Intervention Playbook

## Purpose

Map `RC1` failure modes to bounded intervention responses.

## Intervention Rules

### For RC1-FM-01 Detail Retrieval Collapse

Primary interventions:

- tighten the task around detail so the target is more visible
- keep the text or prompt surface small enough to isolate detail retrieval collapse
- rerun a bounded recheck on a changed but comparable text or prompt

Expected effect:

- learner resolves the dominant `detail retrieval collapse` pattern without changing the core node demand

Reverification target:

- one bounded changed-surface task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of dependency integrity, prompt design, and text-task alignment.
