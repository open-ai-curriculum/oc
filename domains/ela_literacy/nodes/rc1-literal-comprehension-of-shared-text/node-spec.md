# RC1 Node Specification

## Node Identity

- id: `RC1`
- name: `literal_comprehension_of_shared_text`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `early-comprehension-node-package`
- evidence class: `Synthetic`

## Purpose

The learner demonstrates direct literal meaning from shared or read-aloud text well enough to establish the first formal reading-comprehension layer before inference is expected.

## Dependencies

- `O3`
- `P1`

## Exit Conditions

The learner is secure on `RC1` only when all of the following are true:

1. identifies or recounts key literal details from shared or read-aloud text
2. responds to simple text questions with text-relevant meaning
3. maintains comprehension across changed shared texts or prompt formats
4. does not depend only on irrelevant visual or routine cues
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- answer literal who, what, where, or what-happened questions from shared text
- recount simple text events or details
- demonstrate meaning from text before independent decoding is expected

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
