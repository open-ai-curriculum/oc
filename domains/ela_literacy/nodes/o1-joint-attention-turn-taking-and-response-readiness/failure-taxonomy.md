# O1 Failure Taxonomy

### O1-FM-01 Shared Attention Dropoff

- description: learner does not sustain shared attention long enough for a simple interaction routine

### O1-FM-02 Turn-Taking Breakdown

- description: learner does not wait for, take, or return turns with enough stability

### O1-FM-03 Noncontingent Responding

- description: learner responds, but the response is not meaningfully tied to the interaction cue or partner action
