# O1 Intervention Playbook

## Purpose

Map `O1` failure modes to bounded intervention responses.

## Intervention Rules

### For O1-FM-01 Shared Attention Dropoff

Primary interventions:

- use short shared-attention routines around a single clear interaction target
- reduce competing sensory demands during the routine
- repeat the same interaction frame before adding variation

Expected effect:

- learner sustains shared attention long enough to support basic interaction

Reverification target:

- one direct shared-attention routine

### For O1-FM-02 Turn-Taking Breakdown

Primary interventions:

- use explicit turn frames such as my turn and your turn
- shorten the exchange so the learner can complete a full cycle
- reinforce successful waiting and return-turn behavior immediately

Expected effect:

- learner stabilizes the structure of simple reciprocal exchange

Reverification target:

- one structured turn-taking interaction

### For O1-FM-03 Noncontingent Responding

Primary interventions:

- slow the interaction and highlight the cue that requires response
- prompt only one contingent action at a time
- use simple routines where the effect of the learner response is obvious

Expected effect:

- learner responses become meaningfully tied to the interaction event

Reverification target:

- one contingent-response routine

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of attention, communication, and context factors rather than repeating the same routine blindly.
