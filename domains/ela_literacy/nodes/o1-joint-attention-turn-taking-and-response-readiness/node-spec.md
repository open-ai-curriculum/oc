# O1 Node Specification

## Node Identity

- id: `O1`
- name: `joint_attention_turn_taking_and_response_readiness`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner sustains shared attention, participates in simple turn-taking, and produces basic contingent responses that later oral-language and literacy instruction depend on.

## Dependencies

None

## Exit Conditions

The learner is secure on `O1` only when all of the following are true:

1. joins and sustains attention to a shared person, object, or text event
2. participates in simple turn-taking routines
3. produces contingent responses rather than only unrelated reactions
4. succeeds across more than one interaction format
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- orient to a shared interaction target
- wait for and take a turn in a simple exchange
- respond to simple prompts or bids in a contingent way
- support the earliest oral-language routines needed for later literacy work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
