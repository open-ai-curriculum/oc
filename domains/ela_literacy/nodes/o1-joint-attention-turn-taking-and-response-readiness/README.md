# O1 Joint Attention, Turn-Taking, And Response Readiness

## Purpose

This node defines the earliest governed oral-language readiness surface in the `ela_literacy` domain.

It exists so later literacy work does not assume that shared attention, turn-taking, and basic response routines are already stable.
