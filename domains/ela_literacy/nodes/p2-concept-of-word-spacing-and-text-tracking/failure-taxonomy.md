# P2 Failure Taxonomy

### P2-FM-01 Continuous-Print Blur

- description: learner treats a line of print as an undifferentiated visual stream rather than separated word units

### P2-FM-02 Spoken-Print Coordination Breakdown

- description: learner cannot align spoken language with tracked print consistently enough to support shared reading

### P2-FM-03 Layout Dependency

- description: learner tracks one familiar print layout but collapses when spacing, font, or line format changes
