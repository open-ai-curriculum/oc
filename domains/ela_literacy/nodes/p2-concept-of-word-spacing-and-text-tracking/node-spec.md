# P2 Node Specification

## Node Identity

- id: `P2`
- name: `concept_of_word_spacing_and_text_tracking`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `literacy-entry-node-package`
- evidence class: `Synthetic`

## Purpose

The learner tracks print at the word level and recognizes spacing and movement through text as meaningful literacy structure.

## Dependencies

- `P1` book handling, print awareness, and directionality
- `O2` oral language comprehension for instructions and routines

## Exit Conditions

The learner is secure on `P2` only when all of the following are true:

1. tracks text in the intended direction with usable consistency
2. distinguishes individual word units from continuous visual marks
3. coordinates oral language and print tracking in supported shared text
4. succeeds across changed text surfaces or layouts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- follow words across text during shared reading
- notice spaces as boundaries between words
- align spoken language with tracked print in supported contexts
- support later decoding fluency and sentence-level reading

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
