# P2 Intervention Playbook

## Purpose

Map `P2` failure modes to bounded intervention responses.

## Intervention Rules

### For P2-FM-01 Continuous-Print Blur

Primary interventions:

- use short repeated text with explicit finger tracking
- highlight spaces between words during shared reading
- move from one short line to a second only after the first stabilizes

Expected effect:

- learner differentiates word units more reliably in print

Reverification target:

- one concept-of-word tracking item

### For P2-FM-02 Spoken-Print Coordination Breakdown

Primary interventions:

- coordinate one spoken sentence with one tracked line of print
- slow the oral pacing so tracked print can stay aligned
- repeat the same line before changing the sentence

Expected effect:

- learner aligns spoken language with tracked print more reliably

Reverification target:

- one spoken-print alignment task

### For P2-FM-03 Layout Dependency

Primary interventions:

- keep the text familiar while changing the visual layout slightly
- move from large print to smaller print gradually
- practice the same tracking target across two display formats

Expected effect:

- learner transfers text-tracking behavior beyond one visual format

Reverification target:

- one changed-layout tracking task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of visual access, print-density demands, and earlier print-awareness assumptions.
