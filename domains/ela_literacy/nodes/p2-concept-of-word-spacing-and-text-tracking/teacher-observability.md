# P2 Teacher Observability View

Teachers should see:

- current `P2` status
- dependency posture for `P1` and `O2`
- direct text-tracking accuracy
- transfer and retention posture
- dominant failure mode
- whether the learner recognizes word boundaries and coordinates spoken language with print
- recommended intervention

Class-level view should highlight:

- who sees print as one continuous stream
- who loses spoken-print coordination
- who fails when layout or print size changes
