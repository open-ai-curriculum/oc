# WC1 Failure Taxonomy

### WC1-FM-01 Idea-To-Sentence Collapse

- description: learner has an idea but cannot shape it into a complete sentence

### WC1-FM-02 Fragment Instability

- description: learner produces fragments or incomplete thoughts

### WC1-FM-03 Prompt Dependence

- description: learner succeeds only on one familiar sentence frame

### WC1-FM-04 Sentence-Meaning Drift

- description: learner loses alignment between the intended idea and the produced sentence
