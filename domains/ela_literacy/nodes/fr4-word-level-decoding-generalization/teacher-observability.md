# FR4 Teacher Observability View

Teachers should see:

- current `FR4` status
- dependency posture for `FR3`
- direct changed-set word accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether performance survives set and surface changes
- recommended intervention

Class-level view should highlight:

- who is still dependent on one practiced set
- who guesses from cues rather than decoding the word
- who can decode changed targets but cannot encode them
