# FR4 Node Specification

## Node Identity

- id: `FR4`
- name: `word_level_decoding_generalization`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `foundational-reading-node-package`
- evidence class: `Synthetic`

## Purpose

The learner generalizes decoding and encoding control across changed simple-word sets with enough stability to support broader word-level literacy access.

## Dependencies

- `FR3` simple-pattern decoding and encoding

## Exit Conditions

The learner is secure on `FR4` only when all of the following are true:

1. decodes and encodes changed simple-word sets without depending on one practiced list
2. uses pattern reasoning rather than visual recall or cue dependence
3. sustains performance across more than one changed surface
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- carry foundational reading control into new word-level targets
- support the stronger downstream prerequisite for sentence-level reading, vocabulary, and sentence generation
- distinguish generalized control from item rehearsal

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
