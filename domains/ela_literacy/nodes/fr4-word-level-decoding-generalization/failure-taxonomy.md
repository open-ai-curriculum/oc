# FR4 Failure Taxonomy

### FR4-FM-01 Practiced-Set Dependence

- description: learner succeeds on one familiar set but loses control when a new simple-word set is introduced

### FR4-FM-02 Cue-Dependent Word Reading

- description: learner relies on visual, contextual, or positional cues rather than stable word-level decoding logic

### FR4-FM-03 Encoding Lag

- description: learner can generalize more successfully in decoding than in encoding the same changed targets

### FR4-FM-04 Surface-Shift Collapse

- description: learner loses control when the task surface changes even though the underlying word-level demand remains similar
