# FR4 Intervention Playbook

## Purpose

Map `FR4` failure modes to bounded intervention responses.

## Intervention Rules

### For FR4-FM-01 Practiced-Set Dependence

Primary interventions:

- rotate multiple small changed sets sharing the same underlying demand
- revisit old and new sets in alternating sequence
- ask the learner to explain why the same decoding logic still applies

Expected effect:

- learner generalizes beyond one practiced word set

Reverification target:

- one multi-set transfer task

### For FR4-FM-02 Cue-Dependent Word Reading

Primary interventions:

- reduce contextual cues that allow guessing
- require explicit attention to each word element
- use isolated and minimally cued targets before reintroducing richer surfaces

Expected effect:

- learner relies more on word-level decoding logic than contextual cueing

Reverification target:

- one low-cue changed-set task

### For FR4-FM-03 Encoding Lag

Primary interventions:

- pair reading and spelling of new targets
- keep the changed set small enough for full bidirectional application
- require immediate encode-after-decode practice

Expected effect:

- learner generalizes in both directions rather than reading only

Reverification target:

- one paired changed-set decode-encode task

### For FR4-FM-04 Surface-Shift Collapse

Primary interventions:

- vary only one task feature at a time
- keep the underlying word-level demand constant across surfaces
- return to a successful surface after the changed trial

Expected effect:

- learner maintains generalized control across modest task changes

Reverification target:

- one same-demand changed-surface task

## Escalation Rule

If generalized control remains highly cue-dependent after two bounded intervention cycles, escalate to human review.
