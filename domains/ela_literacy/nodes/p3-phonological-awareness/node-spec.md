# P3 Node Specification

## Node Identity

- id: `P3`
- name: `phonological_awareness`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner notices and manipulates spoken-language sound structures well enough to support later alphabetic-principle and early decoding work.

## Dependencies

- `O2` oral language comprehension for instructions and routines

## Exit Conditions

The learner is secure on `P3` only when all of the following are true:

1. detects or manipulates sound patterns in spoken language
2. attends to sound structure rather than only meaning
3. succeeds across changed words or task surfaces
4. demonstrates stable oral-only performance without print dependence
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- notice rhyme, syllables, onset-rime, or simple phoneme-level features
- separate sound structure from word meaning when prompted
- sustain phonological work across changed oral tasks
- support later alphabetic-principle and decoding instruction

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
