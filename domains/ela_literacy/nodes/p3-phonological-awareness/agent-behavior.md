# P3 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `P3`.

## Guidance

- keep the work oral-first rather than introducing print too early
- minimize semantic distraction when the target is sound structure
- change prompt formats only after the learner stabilizes one form
- bound oral-memory load before inferring a core phonological deficit
- escalate to human review when performance remains highly fragile across simple oral tasks
