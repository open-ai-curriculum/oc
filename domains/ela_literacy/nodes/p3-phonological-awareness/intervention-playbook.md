# P3 Intervention Playbook

## Purpose

Map `P3` failure modes to bounded intervention responses.

## Intervention Rules

### For P3-FM-01 Sound-Meaning Confusion

Primary interventions:

- use oral-only contrasts where semantic content is minimized
- explicitly cue the learner to listen for the sound part, not the meaning
- compare pairs that differ in sound structure while holding meaning less central

Expected effect:

- learner attends more reliably to spoken-language sound structure

Reverification target:

- one sound-focused discrimination task

### For P3-FM-02 Segmentation Breakdown

Primary interventions:

- use tapping, clapping, or token movement for syllable or onset-rime segmentation
- reduce task length to one manageable spoken unit at a time
- repeat the same oral pattern before increasing complexity

Expected effect:

- learner separates spoken-language units more reliably

Reverification target:

- one segmentation or blending task

### For P3-FM-03 Prompt-Format Fragility

Primary interventions:

- keep the sound target stable while changing the response format
- move deliberately from imitation to independent response
- practice the same sound pattern in at least two oral task forms

Expected effect:

- learner transfers phonological awareness beyond one task surface

Reverification target:

- one changed-format phonological task

### For P3-FM-04 Oral-Memory Load Boundary

Primary interventions:

- shorten the spoken string or reduce the number of units held at once
- front-load the target sound feature before asking for manipulation
- pause between repetitions to reduce overload

Expected effect:

- learner shows a more valid phonological response when oral-memory load is bounded

Reverification target:

- one reduced-load oral task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language access, auditory processing assumptions, and working-memory demands.
