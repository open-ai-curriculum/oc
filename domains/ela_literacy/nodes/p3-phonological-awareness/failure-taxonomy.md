# P3 Failure Taxonomy

### P3-FM-01 Sound-Meaning Confusion

- description: learner defaults to word meaning or semantic association instead of attending to sound structure

### P3-FM-02 Segmentation Breakdown

- description: learner cannot reliably separate or manipulate relevant spoken-language units

### P3-FM-03 Prompt-Format Fragility

- description: learner succeeds in one oral-task format but collapses when the prompt surface changes

### P3-FM-04 Oral-Memory Load Boundary

- description: performance drops when the spoken-language load exceeds the learner's current holding capacity
