# R2 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `R2`.

## Guidance

- keep sentence tasks short enough that meaning, not endurance, is being observed
- do not confuse successful word reading with successful sentence comprehension
- simplify prompt language before concluding the sentence target itself is not understood
- adjust decoding load when needed, but preserve an actual comprehension demand
- escalate to human review when the learner remains blocked because dependency weakness and sentence-meaning demands cannot be cleanly separated
