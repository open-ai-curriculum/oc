# R2 Intervention Playbook

## Purpose

Map `R2` failure modes to bounded intervention responses.

## Intervention Rules

### For R2-FM-01 Word-Calling Without Meaning

Primary interventions:

- pause after each sentence for a quick meaning check
- ask the learner to paraphrase the sentence with simpler language
- separate sentence reading from sentence meaning and reconnect them explicitly

Expected effect:

- learner treats sentence reading as meaning-making rather than word calling only

Reverification target:

- one short sentence meaning task

### For R2-FM-02 Prompt-Language Confound

Primary interventions:

- simplify the question language while holding the sentence target stable
- use point, choose, or brief oral response modes before longer explanation
- verify whether the failure is in prompt interpretation rather than sentence comprehension

Expected effect:

- learner comprehension becomes visible through an accessible response surface

Reverification target:

- one low-language-load sentence-comprehension task

### For R2-FM-03 Syntax Fragility

Primary interventions:

- keep the sentence meaning target stable while changing word order or sentence frame
- contrast two simple sentence structures with the same semantic relation
- make sentence structure noticeable without turning the task into grammar-only instruction

Expected effect:

- learner carries meaning across more than one familiar sentence pattern

Reverification target:

- one changed-syntax comprehension task

### For R2-FM-04 Decoding-Load Collapse

Primary interventions:

- reduce sentence length or decoding novelty while preserving a comprehension demand
- preteach one unfamiliar word that would otherwise block the meaning check
- diagnose whether the real blocker is still `FR4` rather than `R2`

Expected effect:

- learner shows sentence-level meaning within a supported decoding range

Reverification target:

- one sentence-comprehension task at adjusted decoding load

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of decoding dependency sufficiency, prompt-language mismatch, and whether syntax demands exceed the supported sentence band.
