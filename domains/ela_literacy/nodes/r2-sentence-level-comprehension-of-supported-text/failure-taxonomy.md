# R2 Failure Taxonomy

### R2-FM-01 Word-Calling Without Meaning

- description: learner can decode or approximate the sentence but cannot recover its actual meaning

### R2-FM-02 Prompt-Language Confound

- description: learner comprehension appears weaker because the response prompt or question language exceeds the sentence target

### R2-FM-03 Syntax Fragility

- description: learner understands one familiar sentence frame but loses meaning when sentence structure changes within the supported range

### R2-FM-04 Decoding-Load Collapse

- description: learner meaning breaks down whenever sentence decoding demand rises even slightly above the most controlled level
