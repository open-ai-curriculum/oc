# R2 Node Specification

## Node Identity

- id: `R2`
- name: `sentence_level_comprehension_of_supported_text`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `sentence-comprehension-node-package`
- evidence class: `Synthetic`

## Purpose

The learner derives meaning from supported sentence-level text well enough to bridge from read-aloud comprehension into increasingly independent reading comprehension.

## Dependencies

- `FR4` word-level decoding generalization
- `R1` literal comprehension of read-aloud and shared text

## Exit Conditions

The learner is secure on `R2` only when all of the following are true:

1. reads or processes supported sentences for meaning rather than only word calling
2. answers sentence-level prompts with text-relevant understanding
3. succeeds across changed sentence structures or simple text surfaces
4. demonstrates comprehension that remains valid when decoding load changes within the supported range
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- derive literal meaning from supported written sentences
- connect decoded text to actual comprehension
- answer sentence-level questions with relevant understanding
- support later evidence-based response and paragraph-level reading work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
