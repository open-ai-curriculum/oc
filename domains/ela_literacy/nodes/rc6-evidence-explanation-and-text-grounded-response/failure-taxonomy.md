# RC6 Failure Taxonomy

### RC6-FM-01 Evidence Without Explanation

- description: learner cites a detail but does not connect it to the answer

### RC6-FM-02 Explanation Without Anchor

- description: learner explains a claim without sufficient textual support

### RC6-FM-03 Question-Type Confusion

- description: learner uses the wrong response structure for the prompt

### RC6-FM-04 Transfer Fragility Across Texts

- description: learner succeeds on one familiar text type but not changed text-response surfaces
