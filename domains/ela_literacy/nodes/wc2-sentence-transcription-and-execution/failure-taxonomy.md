# WC2 Failure Taxonomy

### WC2-FM-01 Transcription Bottleneck

- description: learner loses the sentence once writing execution begins

### WC2-FM-02 Encoding-Load Collapse

- description: learner cannot sustain sentence execution when spelling demand rises

### WC2-FM-03 Legibility-Or-Spacing Breakdown

- description: learner execution obscures sentence boundaries or readability

### WC2-FM-04 Prompt-Transfer Fragility

- description: learner succeeds only in one narrow production context
