# WC2 Node Specification

## Node Identity

- id: `WC2`
- name: `sentence_transcription_and_execution`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `sentence-writing-node-package`
- evidence class: `Synthetic`

## Purpose

The learner executes written sentence production with enough control that transcription burden no longer hides sentence knowledge.

## Dependencies

- `WC1`
- `FR4`

## Exit Conditions

The learner is secure on `WC2` only when all of the following are true:

1. writes complete sentences with manageable transcription load
2. preserves sentence meaning during execution
3. succeeds across changed writing prompts or contexts
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- write complete sentences with recoverable meaning
- separate transcription bottlenecks from planning quality
- support the move into paragraph-level writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
