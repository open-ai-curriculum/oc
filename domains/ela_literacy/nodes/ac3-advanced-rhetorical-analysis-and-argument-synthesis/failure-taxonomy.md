# AC3 Failure Taxonomy

### AC3-FM-01 Move-Effect Interpretation Fragility

- description: learner cannot explain how rhetorical choices shape meaning or persuasion

### AC3-FM-02 Argument-Synthesis Collapse

- description: learner cannot sustain a coherent synthesized analytical argument

### AC3-FM-03 Source-Groundedness Drift

- description: learner analysis becomes detached from the actual source set

### AC3-FM-04 Rhetorical-Situation Transfer Fragility

- description: learner succeeds only in one narrow rhetorical context
