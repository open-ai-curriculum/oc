# AC3 Node Specification

## Node Identity

- id: `AC3`
- name: `advanced_rhetorical_analysis_and_argument_synthesis`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `advanced-rhetorical-analysis-node-package`

## Purpose

The learner constructs advanced rhetorical analysis and synthesized argument after disciplinary and rhetorical reading plus mature sourced writing are available.

## Dependencies

- `AC2`
- `WC8`
- `L3`

## Exit Conditions

The learner is secure on `AC3` only when all of the following are true:

1. analyzes rhetoric and synthesizes argument across complex sources
2. maintains coherent source-grounded reasoning in advanced analytical writing
3. succeeds across changed rhetorical situations
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- build advanced rhetorical analysis from coordinated reading and mature sourced writing
- separate high-level rhetorical argument synthesis from earlier source integration
- support capstone rhetorical literacy

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
