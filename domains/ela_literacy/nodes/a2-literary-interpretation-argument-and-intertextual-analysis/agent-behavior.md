# A2 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `A2`.

## Guidance

- keep advanced literary tasks bounded enough that interpretation and intertextual reasoning remain inspectable
- require explicit literary evidence before accepting broad interpretive claims
- distinguish intertextual-comparison breakdown from evidence-link weakness
- vary literary texts and interpretive tasks only after one capstone literary routine stabilizes
- escalate to human review when capstone literary-analysis failure cannot be separated from sustained-writing, literary-analysis, or revision weaknesses
