# L2 Failure Taxonomy

### L2-FM-01 Context-Blind Word Meaning

- description: learner recalls a word loosely but cannot fit its meaning to the actual sentence or text context

### L2-FM-02 Morphology Cue Miss

- description: learner overlooks useful prefixes, suffixes, roots, or word-part clues that could support meaning

### L2-FM-03 Receptive-Expressive Split

- description: learner recognizes the word in comprehension tasks but cannot use it meaningfully in production

### L2-FM-04 Vocabulary-Transfer Fragility

- description: learner succeeds with a small taught set but performance collapses on changed texts or task surfaces
