# L2 Node Specification

## Node Identity

- id: `L2`
- name: `morphology_word_meaning_and_academic_vocabulary`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `vocabulary-and-language-node-package`
- evidence class: `Synthetic`

## Purpose

The learner uses morphology, word meaning, and academic vocabulary well enough to support stronger comprehension and written production.

## Dependencies

- `FR4` word-level decoding generalization
- `RC3` sentence-level comprehension of supported text

## Exit Conditions

The learner is secure on `L2` only when all of the following are true:

1. interprets or uses targeted words with enough contextual accuracy
2. notices meaningful word parts or morphology where appropriate
3. applies vocabulary knowledge across changed texts or tasks
4. supports both comprehension and expressive language use
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- interpret words in supported context
- use morphology or word-part clues where appropriate
- apply academic vocabulary in comprehension or production tasks
- support later sentence revision, evidence-based response, and composition work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
