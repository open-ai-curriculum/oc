# L2 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `L2`.

## Guidance

- keep vocabulary and morphology tied to text meaning, not isolated terminology drills
- use word parts as support for meaning-making, not as an end in themselves
- require both interpretation and bounded use before treating the node as generalized
- vary text contexts once one semantic routine is stable
- escalate to human review when vocabulary failure appears to be driven by deeper comprehension or background-knowledge limits
