# WC4 Failure Taxonomy

### WC4-FM-01 Evidence Drop-In Without Integration

- description: learner inserts source details without linking them to the paragraph purpose

### WC4-FM-02 Source-Relevance Mismatch

- description: learner chooses text support that does not fit the written point

### WC4-FM-03 Paragraph-Coherence Collapse Under Source Load

- description: learner loses organization once source use is added

### WC4-FM-04 Prompt-Type Confusion

- description: learner cannot adapt source integration to changed response demands
