# WC4 Teacher Observability View

Teachers should see:

- current `WC4` status
- dependency posture for `WC3`, `RC6`
- direct performance accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner is showing the node-specific bottleneck rather than an upstream confound
- recommended intervention

Class-level view should highlight:

- who are blocked by the dominant failure modes on `WC4`
- who improve on supported tasks but lose control on changed surfaces
- who appear ready for the next writing or analysis boundary
