# WC4 Node Specification

## Node Identity

- id: `WC4`
- name: `source_integration_in_structured_response`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `paragraph-writing-node-package`
- evidence class: `Synthetic`

## Purpose

The learner integrates relevant textual support into structured writing after paragraph coherence is secure.

## Dependencies

- `WC3`
- `RC6`

## Exit Conditions

The learner is secure on `WC4` only when all of the following are true:

1. integrates relevant source support into a structured response
2. connects evidence to the written point or paragraph purpose
3. succeeds across changed source-based prompts
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- use relevant source support inside coherent structured responses
- separate paragraph coherence from source integration demands
- support later multi-paragraph sourced writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
