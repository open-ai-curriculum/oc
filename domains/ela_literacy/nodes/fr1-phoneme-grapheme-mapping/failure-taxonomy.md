# FR1 Failure Taxonomy

### FR1-FM-01 One-Directional Mapping

- description: learner can produce sound-to-symbol or symbol-to-sound responses, but not both with usable stability

### FR1-FM-02 Naming Without Mapping

- description: learner can name letters or sounds in isolation but does not reliably connect them as correspondences

### FR1-FM-03 Target-Set Fragility

- description: performance appears stable on one micro-set and collapses when a small set change is introduced

### FR1-FM-04 Retrieval Instability

- description: learner has inconsistent correspondence retrieval even when dependencies appear secure
