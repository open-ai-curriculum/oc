# FR1 Intervention Playbook

## Purpose

Map `FR1` failure modes to bounded intervention responses.

## Intervention Rules

### For FR1-FM-01 One-Directional Mapping

Primary interventions:

- alternate sound-to-symbol and symbol-to-sound trials in the same routine
- keep the target set very small
- require immediate reversal of direction on the same correspondence

Expected effect:

- learner stabilizes mapping in both directions

Reverification target:

- one bidirectional correspondence task

### For FR1-FM-02 Naming Without Mapping

Primary interventions:

- contrast letter naming with actual sound-symbol matching
- remove tasks that can be passed by reciting only
- require correspondence use in a structured matching routine

Expected effect:

- learner demonstrates actual mapping rather than isolated naming

Reverification target:

- one structured mapping task with mixed directions

### For FR1-FM-03 Target-Set Fragility

Primary interventions:

- maintain one stable micro-set before introducing one new target
- explicitly contrast known and new correspondences
- revisit prior targets after each addition

Expected effect:

- learner maintains accuracy when the target set changes

Reverification target:

- one changed-set mapping task

### For FR1-FM-04 Retrieval Instability

Primary interventions:

- shorten routines and increase distributed retrieval opportunities
- keep visual clutter low
- confirm dependency strength in `P3` and `P4`

Expected effect:

- learner retrieves supported correspondences more consistently

Reverification target:

- one brief repeated retrieval check

## Escalation Rule

If retrieval remains unstable after two bounded intervention cycles, escalate to human review of dependency integrity and task-format confounds.
