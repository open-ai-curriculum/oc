# FR1 Node Specification

## Node Identity

- id: `FR1`
- name: `phoneme_grapheme_mapping`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `foundational-reading-node-package`
- evidence class: `Synthetic`

## Purpose

The learner maps supported phonemes to graphemes and graphemes to phonemes with enough stability to support later print-encoding logic and controlled word reading.

## Dependencies

- `P3` phonological awareness
- `P4` alphabet knowledge and letter formation

## Exit Conditions

The learner is secure on `FR1` only when all of the following are true:

1. retrieves supported sound-symbol correspondences with usable consistency
2. performs mapping in both directions rather than one-direction naming only
3. remains stable when the supported target set changes slightly
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- connect supported sounds and symbols without collapsing into guessing
- move between sound-to-symbol and symbol-to-sound tasks
- provide the direct correspondence control needed for later foundational-reading nodes

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
