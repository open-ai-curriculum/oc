# AC2 Node Specification

## Node Identity

- id: `AC2`
- name: `disciplinary_and_rhetorical_reading`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `advanced-analysis-node-package`
- evidence class: `Synthetic`

## Purpose

The learner reads complex texts for disciplinary and rhetorical features after cross-text synthesis is available.

## Dependencies

- `AC1`

## Exit Conditions

The learner is secure on `AC2` only when all of the following are true:

1. identifies and interprets disciplinary or rhetorical features in complex texts
2. maintains coherent rhetorical or disciplinary reasoning
3. succeeds across changed reading situations
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- read for rhetorical moves and disciplinary text features
- separate disciplinary and rhetorical reading from general synthesis
- support advanced argument and analysis

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
