# AC2 Failure Taxonomy

### AC2-FM-01 Feature-Spotting Without Interpretation

- description: learner notices features but cannot explain their effect

### AC2-FM-02 Disciplinary-Convention Confusion

- description: learner misreads the text through the wrong disciplinary lens

### AC2-FM-03 Rhetorical-Effect Drift

- description: learner cannot connect rhetorical features to author purpose or effect

### AC2-FM-04 Task-Surface Transfer Fragility

- description: learner succeeds only on one rhetorical or disciplinary prompt type
