# FR6 Intervention Playbook

## Purpose

Map `FR6` failure modes to bounded intervention responses.

## Intervention Rules

### For FR6-FM-01 Tracking Loss

Primary interventions:

- shorten passage length while preserving connected text
- use explicit line and spacing cues
- pair reading with guided finger or visual tracking only as needed

Expected effect:

- learner maintains place through the full text stream

Reverification target:

- one short connected-text tracking task

### For FR6-FM-02 Word-Integrity Breakdown In Passage Reading

Primary interventions:

- return briefly to controlled phrase or line segments
- isolate misread words and then reinsert them into the passage
- confirm that `FR5` remains stable on the affected word set

Expected effect:

- learner preserves full word integrity while reading connected text

Reverification target:

- one passage reread with targeted word monitoring

### For FR6-FM-03 Labored Pace Without Flow

Primary interventions:

- reduce passage length to a manageable fluency window
- rehearse a bounded passage for accuracy first and then smoother pacing
- avoid rewarding speed that sacrifices word integrity

Expected effect:

- learner develops more usable pacing without accuracy collapse

Reverification target:

- one accuracy-first fluent reread task

### For FR6-FM-04 Surface-Change Instability

Primary interventions:

- change one text feature at a time
- preserve comparable difficulty while varying layout or passage content
- alternate familiar and changed connected-text surfaces

Expected effect:

- learner carries connected-text control into modestly changed surfaces

Reverification target:

- one changed-passage or changed-layout task

## Escalation Rule

If connected-text control remains unstable after two bounded intervention cycles, escalate to human review of tracking, visual support needs, and foundational-word stability.
