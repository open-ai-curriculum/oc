# FR3 Node Specification

## Node Identity

- id: `FR3`
- name: `simple_pattern_decoding_and_encoding`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `foundational-reading-node-package`
- evidence class: `Synthetic`

## Purpose

The learner applies print-encoding knowledge to decode and encode highly controlled simple words with enough stability to support early sentence-level literacy access.

## Dependencies

- `FR2` print-encoding principle and bidirectional mapping

## Exit Conditions

The learner is secure on `FR3` only when all of the following are true:

1. decodes supported simple words with sound-symbol reasoning
2. encodes supported simple words with corresponding control
3. demonstrates that performance is not limited to one memorized list
4. explains or otherwise reveals actual pattern reasoning
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- read and spell simple controlled words using explicit pattern logic
- support the first downstream bridge into sentence-level literacy tasks
- distinguish actual pattern application from item-specific memorization

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
