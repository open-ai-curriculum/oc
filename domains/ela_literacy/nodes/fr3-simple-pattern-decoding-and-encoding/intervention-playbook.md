# FR3 Intervention Playbook

## Purpose

Map `FR3` failure modes to bounded intervention responses.

## Intervention Rules

### For FR3-FM-01 Guessing Over Application

Primary interventions:

- use novel but simple targets that reduce memorized response success
- require the learner to point to or verbalize each sound-symbol step
- slow the routine enough to reveal actual pattern use

Expected effect:

- learner relies more on explicit pattern application than guessing

Reverification target:

- one changed-set simple-word task

### For FR3-FM-02 Decoding-Encoding Split

Primary interventions:

- pair reading and spelling of the same controlled target
- move immediately from one direction to the other
- keep the target pattern stable across both forms

Expected effect:

- learner links decoding and encoding within the same simple pattern

Reverification target:

- one paired decode-encode task

### For FR3-FM-03 Memorized-List Dependence

Primary interventions:

- replace part of the familiar list with new items sharing the same pattern
- contrast known and new items explicitly
- require explanation of why the same pattern still works

Expected effect:

- learner generalizes beyond one practiced list

Reverification target:

- one new-list transfer task

### For FR3-FM-04 Pattern-Element Omission

Primary interventions:

- reduce word length while preserving the target pattern
- isolate the dropped element and reinsert it into the whole target
- verify that the learner can track every part of the pattern

Expected effect:

- learner retains the full simple pattern during decoding and encoding

Reverification target:

- one full-pattern integrity task

## Escalation Rule

If guessing or list dependence remains dominant after two bounded intervention cycles, escalate to human review of dependency strength and task design.
