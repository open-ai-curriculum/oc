# FR3 Verification Model

## Gate Set

### G1. Direct Performance Gate

- purpose: confirm decoding and encoding of controlled simple-word patterns
- pass rule: successful application in at least 5 of 6 structured opportunities

### G2. Meaning Gate

- purpose: confirm that the learner is using sound-symbol reasoning rather than item recall alone
- pass rule: correct explanation or demonstration in at least 4 of 5 prompted trials

### G3. Transfer Gate

- purpose: confirm application on a changed simple-pattern set
- pass rule: one successful changed-set task without guessing dominance

### G4. Retention Gate

- purpose: confirm persistence after spacing
- pass rule: stable delayed recheck after about 14 days

## Status Logic

- `blocked`: direct gate fails
- `emerging`: direct gate improves but meaning or transfer still fails
- `provisional`: direct, meaning, and transfer pass while retention is pending
- `secure`: all four gates pass
- `unstable`: previously secure pattern application regresses on later checks
