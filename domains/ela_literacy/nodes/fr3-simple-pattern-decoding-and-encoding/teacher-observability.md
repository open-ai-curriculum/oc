# FR3 Teacher Observability View

Teachers should see:

- current `FR3` status
- dependency posture for `FR2`
- direct simple-pattern accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether performance depends on memorized lists
- recommended intervention

Class-level view should highlight:

- who guesses rather than applies the pattern
- who decodes but cannot encode the same controlled words
- who loses performance on a changed simple-pattern set
