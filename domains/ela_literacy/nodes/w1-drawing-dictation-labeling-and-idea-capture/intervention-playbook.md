# W1 Intervention Playbook

## Purpose

Map `W1` failure modes to bounded intervention responses.

## Intervention Rules

### For W1-FM-01 Meaning-Capture Breakdown

Primary interventions:

- ask the learner to name the intended message before and after producing it
- use one clear target idea rather than an open-ended production prompt
- compare strong versus weak representations side by side

Expected effect:

- produced representations carry more recoverable meaning

Reverification target:

- one bounded idea-capture task

### For W1-FM-02 Oral-to-Representation Split

Primary interventions:

- move directly from oral plan to drawing, dictation, or labeling
- keep the idea stable while changing only the representation act
- rehearse one oral sentence and then capture it immediately

Expected effect:

- learner transfers oral meaning into visible or written form more reliably

Reverification target:

- one oral-to-representation transfer task

### For W1-FM-03 Representation Thinness

Primary interventions:

- prompt for one additional meaningful detail after the first attempt
- use a minimal production frame such as who, what, or happened
- model how one representation can include more than one idea element

Expected effect:

- learner output carries enough content to support later sentence writing growth

Reverification target:

- one extended idea-capture task

### For W1-FM-04 Format Dependency

Primary interventions:

- keep the idea target stable while changing from drawing to dictation to labeling gradually
- practice the same message in two production modes
- separate meaning failure from format-specific production friction

Expected effect:

- learner transfers idea capture across representation formats

Reverification target:

- one changed-format idea-capture task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language formulation, motor demands, and representational access assumptions.
