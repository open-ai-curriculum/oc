# W1 Node Specification

## Node Identity

- id: `W1`
- name: `drawing_dictation_labeling_and_idea_capture`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `early-writing-node-package`
- evidence class: `Synthetic`

## Purpose

The learner captures ideas through drawing, dictation, labeling, and early written marks well enough to establish the first governed writing surface in the literacy system.

## Dependencies

- `O3` oral recount, explanation, and sequence language
- `P4` alphabet knowledge and letter formation

## Exit Conditions

The learner is secure on `W1` only when all of the following are true:

1. externalizes ideas or events through representational or early written production
2. uses labeling, dictation, or marks in a way that carries intended meaning
3. communicates enough content for another observer to recover the basic message
4. succeeds across changed topics or formats
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- represent an idea, object, or event through drawing and/or early writing
- connect oral language to a produced representation
- use labels or dictated language to clarify meaning
- establish the first bridge into later sentence-level writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
