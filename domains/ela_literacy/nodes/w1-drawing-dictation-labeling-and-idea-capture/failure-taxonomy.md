# W1 Failure Taxonomy

### W1-FM-01 Meaning-Capture Breakdown

- description: learner produces marks or drawings, but the intended idea cannot be recovered reliably

### W1-FM-02 Oral-to-Representation Split

- description: learner can say the idea orally but cannot transfer it into drawing, labeling, or dictated representation

### W1-FM-03 Representation Thinness

- description: learner produces extremely limited output that does not carry enough content for later writing growth

### W1-FM-04 Format Dependency

- description: learner succeeds in one production format only and collapses when the representation mode changes
