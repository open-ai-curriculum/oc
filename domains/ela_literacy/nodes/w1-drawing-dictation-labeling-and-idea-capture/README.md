# W1 Drawing, Dictation, Labeling, And Idea Capture

## Purpose

This node defines the earliest written-expression surface in the `ela_literacy` domain.

It exists so later sentence generation and composition work does not assume that the learner can already externalize ideas through representational or early written means.
