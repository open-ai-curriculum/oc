# W1 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `W1`.

## Guidance

- start from a single intended idea rather than open-ended production
- do not confuse enthusiastic drawing or mark-making with meaningful idea capture
- link oral planning and visible production tightly in time
- vary format only after one production mode is stable
- escalate to human review when representational meaning remains too thin after bounded support
