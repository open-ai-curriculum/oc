# R5 Node Specification

## Node Identity

- id: `R5`
- name: `informational_structure_author_claim_and_source_use`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `informational-analysis-node-package`
- evidence class: `Synthetic`

## Purpose

The learner analyzes informational structure, author claims, and source use well enough to establish the informational-analysis branch of the literacy graph.

## Dependencies

- `RC6` evidence-based response to text

## Exit Conditions

The learner is secure on `R5` only when all of the following are true:

1. identifies and explains relevant informational structures or author claims
2. supports analysis with text-grounded detail or source evidence
3. distinguishes informational analysis from unsupported opinion or summary alone
4. succeeds across changed informational texts, source sets, or prompts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- analyze author claim, structure, and source use in informational text
- support informational claims with relevant text detail
- move from literal response into analytic explanation
- support later disciplinary literacy and research synthesis

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
