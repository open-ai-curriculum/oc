# R5 Intervention Playbook

## Purpose

Map `R5` failure modes to bounded intervention responses.

## Intervention Rules

### For R5-FM-01 Summary-Over-Analysis

Primary interventions:

- contrast what the text says with how the text is organized or argued
- require one analytic statement after one recalled fact
- use a simple author-claim or structure frame before fuller response

Expected effect:

- learner moves from topic summary into informational analysis

Reverification target:

- one short informational analysis prompt

### For R5-FM-02 Claim-Evidence Misalignment

Primary interventions:

- pair the identified claim with the text detail or source that supports it
- ask how the source or structure strengthens the author's point
- keep the analysis centered on one claim at a time

Expected effect:

- learner connects claims, evidence, and source roles more accurately

Reverification target:

- one claim-evidence-source linkage task

### For R5-FM-03 Structure-And-Source Confusion

Primary interventions:

- sort text features by structure, claim, evidence, and source role
- use one short text to isolate one informational concept at a time
- reduce concept load before asking for combined analysis

Expected effect:

- learner distinguishes informational-analysis elements accurately enough for valid response

Reverification target:

- one targeted informational-concept task

### For R5-FM-04 Text-And-Source Fragility

Primary interventions:

- rotate to changed informational texts or source sets with the same analysis frame
- vary prompts while preserving the requirement for text-grounded analysis
- reduce novelty only enough to observe transfer cleanly

Expected effect:

- learner generalizes informational analysis beyond one familiar text or source set

Reverification target:

- one changed-text or changed-source informational-analysis task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of whether `RC6` is actually secure enough, and whether the informational-analysis demand exceeds the intended `R5` boundary.
