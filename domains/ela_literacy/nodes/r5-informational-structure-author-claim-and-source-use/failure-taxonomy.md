# R5 Failure Taxonomy

### R5-FM-01 Summary-Over-Analysis

- description: learner repeats informational content or topic facts without analyzing structure, claim, or source use

### R5-FM-02 Claim-Evidence Misalignment

- description: learner identifies a claim or source detail but cannot connect it accurately to the author's reasoning or structure

### R5-FM-03 Structure-And-Source Confusion

- description: learner notices some informational features but mixes up text structure, author purpose, source role, or claim strength

### R5-FM-04 Text-And-Source Fragility

- description: learner succeeds on one familiar informational text or source set but performance collapses when texts, sources, or prompts change
