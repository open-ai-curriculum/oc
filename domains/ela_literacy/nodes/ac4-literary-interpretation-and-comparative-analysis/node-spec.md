# AC4 Node Specification

## Node Identity

- id: `AC4`
- name: `literary_interpretation_and_comparative_analysis`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `advanced-literary-analysis-node-package`
- evidence class: `Synthetic`

## Purpose

The learner produces sustained literary interpretation and comparative analysis before full intertextual literary argument becomes the main distinction.

## Dependencies

- `R4`
- `WC6`
- `L3`

## Exit Conditions

The learner is secure on `AC4` only when all of the following are true:

1. develops sustained literary interpretation and comparative analysis
2. maintains comparative reasoning across changed literary texts
3. succeeds across changed literary prompts or comparisons
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- sustain literary interpretation and comparison in mature written analysis
- separate comparative interpretation from full intertextual argument
- support capstone literary analysis

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
