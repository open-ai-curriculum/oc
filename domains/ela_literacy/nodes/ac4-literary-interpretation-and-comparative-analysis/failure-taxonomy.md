# AC4 Failure Taxonomy

### AC4-FM-01 Interpretation-Without-Comparison

- description: learner can interpret one text but cannot sustain comparison

### AC4-FM-02 Comparison-Without-Claim

- description: learner lists similarities or differences without an interpretive line

### AC4-FM-03 Text-Groundedness Drift

- description: learner comparative claims lose anchor in the literary texts

### AC4-FM-04 Prompt-Transfer Fragility

- description: learner succeeds only on one familiar literary comparison task
