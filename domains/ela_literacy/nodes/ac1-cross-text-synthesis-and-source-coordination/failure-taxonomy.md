# AC1 Failure Taxonomy

### AC1-FM-01 Parallel-Summary Without Synthesis

- description: learner summarizes sources separately without coordination

### AC1-FM-02 Source-Relationship Drift

- description: learner cannot keep the texts in a coherent relationship

### AC1-FM-03 Claim-Source Mismatch

- description: learner synthesis claim does not fit the cited sources

### AC1-FM-04 Source-Set Transfer Fragility

- description: learner succeeds only on one familiar source cluster
