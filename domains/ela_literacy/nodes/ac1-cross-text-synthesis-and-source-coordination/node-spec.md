# AC1 Node Specification

## Node Identity

- id: `AC1`
- name: `cross_text_synthesis_and_source_coordination`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `advanced-analysis-node-package`
- evidence class: `Synthetic`

## Purpose

The learner coordinates and synthesizes across texts before disciplinary and rhetorical reading become the main distinction.

## Dependencies

- `R4`
- `R5`

## Exit Conditions

The learner is secure on `AC1` only when all of the following are true:

1. coordinates relevant relationships across multiple texts
2. maintains a coherent cross-text line of reasoning
3. succeeds across changed source sets or tasks
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- coordinate multiple texts around a coherent synthesis purpose
- separate source coordination from later rhetorical interpretation
- support advanced rhetorical and disciplinary reading

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
