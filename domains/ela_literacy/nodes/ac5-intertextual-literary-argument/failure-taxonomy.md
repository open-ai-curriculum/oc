# AC5 Failure Taxonomy

### AC5-FM-01 Intertextual-Link Fragility

- description: learner cannot sustain a defensible link across texts

### AC5-FM-02 Argument-Structure Collapse

- description: learner has comparisons but no coherent literary argument

### AC5-FM-03 Evidence-Distribution Imbalance

- description: learner overrelies on one text and underuses the other

### AC5-FM-04 Task-Transfer Fragility

- description: learner succeeds only on one familiar intertextual prompt
