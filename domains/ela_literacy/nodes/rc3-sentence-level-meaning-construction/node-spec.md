# RC3 Node Specification

## Node Identity

- id: `RC3`
- name: `sentence_level_meaning_construction`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `sentence-comprehension-node-package`
- evidence class: `Synthetic`

## Purpose

The learner derives meaning from independently read supported sentences with enough stability to separate sentence-level understanding from broader passage integration.

## Dependencies

- `FR4`
- `RC1`

## Exit Conditions

The learner is secure on `RC3` only when all of the following are true:

1. derives meaning from supported independently read sentences
2. connects decoding to sentence meaning rather than word calling only
3. maintains comprehension across changed sentence frames or supports
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- paraphrase the meaning of a supported sentence
- match a sentence to the correct meaning or referent
- show the first independent sentence-reading comprehension layer

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
