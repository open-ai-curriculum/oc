# A1 Failure Taxonomy

### A1-FM-01 Summary-And-Assertion Substitution

- description: learner responds with summary or unsupported argument instead of rhetorical analysis grounded in source choices

### A1-FM-02 Rhetorical-Choice Interpretation Weakness

- description: learner notices rhetorical features but cannot explain their effect or significance convincingly

### A1-FM-03 Complex-Source Synthesis Drift

- description: learner references multiple sources but loses distinctions among claims, perspectives, or rhetorical situations

### A1-FM-04 Situation-And-Source Transfer Fragility

- description: learner succeeds on one familiar rhetorical situation or source set but performance collapses on changed complex materials
