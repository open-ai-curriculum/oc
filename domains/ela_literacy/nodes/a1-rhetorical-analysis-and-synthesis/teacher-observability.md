# A1 Teacher Observability View

Teachers should see:

- current `A1` status
- dependency posture for `W5`, `R6`, and `L3`
- direct advanced rhetorical-analysis accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner is performing rhetorical analysis rather than summary or assertion
- recommended intervention

Class-level view should highlight:

- who summarize instead of analyzing rhetoric
- who flatten distinctions among complex sources
- who lose performance when rhetorical situation or source set changes
