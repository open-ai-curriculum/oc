# A1 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `A1`.

## Guidance

- keep advanced rhetorical tasks bounded enough that analysis and synthesis remain inspectable
- require explicit treatment of rhetorical choices before accepting broader claims
- distinguish complex-source synthesis breakdown from rhetorical-feature interpretation weakness
- vary rhetorical situations and source sets only after one advanced-analysis routine stabilizes
- escalate to human review when capstone rhetorical-analysis failure cannot be separated from research-writing, synthesis, or revision weaknesses
