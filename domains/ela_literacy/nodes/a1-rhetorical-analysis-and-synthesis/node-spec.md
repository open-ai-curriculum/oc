# A1 Node Specification

## Node Identity

- id: `A1`
- name: `rhetorical_analysis_and_synthesis`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `advanced-rhetorical-analysis-node-package`
- evidence class: `Synthetic`

## Purpose

The learner analyzes rhetoric and synthesizes claims across complex sources well enough to establish the advanced-secondary capstone branch for argument and rhetorical reasoning.

## Dependencies

- `W5` research synthesis and citation control
- `R6` disciplinary literacy, rhetorical analysis, and cross-text synthesis
- `L3` syntax revision and style control

## Exit Conditions

The learner is secure on `A1` only when all of the following are true:

1. analyzes rhetorical choices or argument structures in complex texts
2. synthesizes across multiple sources while preserving meaningful distinctions
3. produces source-grounded explanation rather than disconnected summary or unsupported assertion
4. succeeds across changed rhetorical situations or source sets
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- analyze rhetoric in complex source material
- synthesize multiple arguments or perspectives coherently
- support advanced argument and academic analysis
- operate at the capstone rhetorical frontier of the literacy graph

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
