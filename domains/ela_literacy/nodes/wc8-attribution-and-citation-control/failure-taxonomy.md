# WC8 Failure Taxonomy

### WC8-FM-01 Attribution Omission

- description: learner uses source material without adequate attribution

### WC8-FM-02 Citation-Format Instability

- description: learner loses control when citation surface or format changes

### WC8-FM-03 Overformatted-But-Unclear Attribution

- description: learner attempts citation mechanics without clear source linkage

### WC8-FM-04 Mechanics-Overload Interference

- description: citation demands disrupt overall research-writing control
