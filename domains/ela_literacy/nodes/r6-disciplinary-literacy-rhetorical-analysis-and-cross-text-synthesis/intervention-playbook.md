# R6 Intervention Playbook

## Purpose

Map `R6` failure modes to bounded intervention responses.

## Intervention Rules

### For R6-FM-01 Single-Text Drift

Primary interventions:

- require a compare, contrast, or synthesis move across at least two texts
- use a same-source versus cross-source contrast to make synthesis visible
- keep the source set small enough that the synthesis target remains manageable

Expected effect:

- learner moves from isolated single-text treatment into genuine cross-text synthesis

Reverification target:

- one two-text synthesis task

### For R6-FM-02 Cross-Text Evidence Misalignment

Primary interventions:

- use a grid that links one claim to evidence from more than one source
- ask how the sources agree, differ, or complicate each other
- keep the synthesis claim bounded to one analytical focus

Expected effect:

- learner connects multiple sources coherently to a unified analytical claim

Reverification target:

- one claim-with-multiple-sources task

### For R6-FM-03 Rhetorical-Or-Disciplinary Feature Blur

Primary interventions:

- isolate one rhetorical or disciplinary feature at a time
- compare how that feature functions in two sources
- reduce concept load before asking for full cross-text analysis

Expected effect:

- learner distinguishes rhetorical or disciplinary features accurately enough for valid synthesis

Reverification target:

- one feature-focused cross-text analysis task

### For R6-FM-04 Source-Set Transfer Fragility

Primary interventions:

- rotate to changed source sets while preserving the same synthesis frame
- vary discipline or rhetorical situation only after one source routine stabilizes
- reduce novelty only enough to observe transfer cleanly

Expected effect:

- learner generalizes synthesis and rhetorical analysis beyond one familiar source set

Reverification target:

- one changed-source-set synthesis task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of whether `R4` and `R5` are actually secure enough, and whether the cross-text demand exceeds the intended `R6` boundary.
