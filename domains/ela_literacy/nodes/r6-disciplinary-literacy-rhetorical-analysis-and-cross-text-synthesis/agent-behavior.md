# R6 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `R6`.

## Guidance

- keep source sets bounded enough that synthesis remains observable and not just overloaded
- require a genuine cross-text move before treating performance as successful
- distinguish rhetorical-feature confusion from synthesis breakdown
- vary source sets and disciplinary frames only after one cross-text routine stabilizes
- escalate to human review when cross-text failure cannot be separated from upstream literary or informational analysis weakness
