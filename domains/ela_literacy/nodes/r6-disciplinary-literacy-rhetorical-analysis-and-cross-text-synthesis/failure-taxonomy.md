# R6 Failure Taxonomy

### R6-FM-01 Single-Text Drift

- description: learner summarizes or analyzes one text at a time but does not synthesize meaning across sources

### R6-FM-02 Cross-Text Evidence Misalignment

- description: learner gathers details from multiple texts but does not connect them coherently to a unified analytical claim

### R6-FM-03 Rhetorical-Or-Disciplinary Feature Blur

- description: learner notices some rhetorical or disciplinary features but cannot distinguish their function or significance reliably

### R6-FM-04 Source-Set Transfer Fragility

- description: learner succeeds on one familiar source set or disciplinary frame but performance collapses on changed source combinations or contexts
