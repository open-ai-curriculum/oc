# FR5 Failure Taxonomy

### FR5-FM-01 Slow Accurate Decoding Without Consolidation

- description: learner can decode accurately but still relies on full effortful decoding for familiar items

### FR5-FM-02 Mixed-List Recognition Collapse

- description: learner shows automaticity in a narrow set but loses it in mixed or spaced lists

### FR5-FM-03 Accuracy-Speed Tradeoff

- description: learner becomes faster only by sacrificing word accuracy

### FR5-FM-04 Fragile Pattern Recognition

- description: learner consolidates some specific words but not the transferable pattern level beneath them
