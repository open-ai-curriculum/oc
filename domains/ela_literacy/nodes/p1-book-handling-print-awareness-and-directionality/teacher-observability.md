# P1 Teacher Observability View

Teachers should see:

- current `P1` status
- dependency posture for `O1`
- direct print-orientation accuracy
- transfer and retention posture
- dominant failure mode
- whether the learner treats print as meaningful and tracks basic directionality
- recommended intervention

Class-level view should highlight:

- who mishandles books or page progression
- who confuses print with pictures
- who loses orientation when layout changes
