# P1 Intervention Playbook

## Purpose

Map `P1` failure modes to bounded intervention responses.

## Intervention Rules

### For P1-FM-01 Book-Handling Instability

Primary interventions:

- use short modeled book-handling routines with one print object at a time
- rehearse page-turning and shared-orientation actions explicitly
- remove unnecessary materials that compete with the target print object

Expected effect:

- learner stabilizes purposeful handling of literacy materials

Reverification target:

- one direct book-orientation routine

### For P1-FM-02 Print-Picture Confusion

Primary interventions:

- contrast print and pictures side by side in the same book page
- ask which part is for reading and which part shows the image
- repeat the language that print carries the words or message

Expected effect:

- learner distinguishes print as a separate meaningful surface

Reverification target:

- one print-versus-picture discrimination task

### For P1-FM-03 Directionality Breakdown

Primary interventions:

- use finger tracking through a short repeated text line
- rehearse page progression and where reading starts
- keep layout stable before introducing a changed format

Expected effect:

- learner maintains more stable print direction and progression behavior

Reverification target:

- one shared-text tracking task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of visual access, attention stability, and print-exposure assumptions.
