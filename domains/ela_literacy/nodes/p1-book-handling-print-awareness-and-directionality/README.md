# P1 Book Handling, Print Awareness, And Directionality

## Purpose

This node defines the earliest print-access surface in the `ela_literacy` domain.

It exists so later decoding and text-comprehension work does not assume that books, pages, print, and directionality are already meaningful to the learner.
