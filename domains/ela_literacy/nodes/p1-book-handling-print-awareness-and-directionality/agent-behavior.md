# P1 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `P1`.

## Guidance

- keep print tasks highly concrete and shared at first
- do not assume print meaning is obvious to the learner
- separate print-orientation support from later decoding expectations
- vary layout only after stable handling and directionality are present
- escalate to human review when print-access behavior remains unstable after repeated bounded support
