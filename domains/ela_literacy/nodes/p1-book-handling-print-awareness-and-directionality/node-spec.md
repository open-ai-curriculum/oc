# P1 Node Specification

## Node Identity

- id: `P1`
- name: `book_handling_print_awareness_and_directionality`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `upstream-readiness-node-package`
- evidence class: `Synthetic`

## Purpose

The learner treats books and print as meaningful literacy objects and demonstrates the earliest print-orientation behaviors needed for later text tracking and decoding.

## Dependencies

- `O1` joint attention, turn-taking, and response readiness

## Exit Conditions

The learner is secure on `P1` only when all of the following are true:

1. handles books purposefully rather than randomly
2. distinguishes print from pictures or non-print marks
3. demonstrates basic page progression and print direction expectations
4. succeeds across more than one print object or layout
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `false`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- orient to books and pages appropriately
- recognize that print carries meaning
- follow basic print direction and progression routines
- support later text tracking, alphabet knowledge, and reading access

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
