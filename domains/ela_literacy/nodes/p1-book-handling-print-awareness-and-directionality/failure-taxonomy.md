# P1 Failure Taxonomy

### P1-FM-01 Book-Handling Instability

- description: learner does not handle books, pages, or print objects purposefully enough to support shared print routines

### P1-FM-02 Print-Picture Confusion

- description: learner does not reliably distinguish print from pictures or other non-print marks

### P1-FM-03 Directionality Breakdown

- description: learner does not maintain usable page progression or basic print-direction expectations
