# F3 Node Specification

## Node Identity

- id: `F3`
- name: `automatic_word_recognition_and_connected_text_fluency`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `connected-text-fluency-node-package`
- evidence class: `Synthetic`

## Purpose

The learner reads connected text with increasing automaticity, accuracy, and fluency well enough to support stable sentence-level comprehension.

## Dependencies

- `F2` decoding and encoding simple words
- `P2` concept of word, spacing, and text tracking

## Exit Conditions

The learner is secure on `F3` only when all of the following are true:

1. recognizes enough words automatically to sustain connected-text reading
2. reads supported text accurately enough for meaning to remain intact
3. tracks connected text without collapsing on spacing or line progression
4. succeeds across changed text sets or layouts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- read supported connected text with growing automaticity
- maintain accuracy while tracking through text
- preserve meaning while reading, not only pronounce isolated words
- support the move into sentence-level comprehension

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
