# L1 Teacher Observability View

Teachers should see:

- current `L1` status
- dependency posture for `O3` and `P4`
- direct sentence-formation accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether output is sentence-level or fragment-dominant
- recommended intervention

Class-level view should highlight:

- who can say sentences orally but not write them with support
- who loses meaning because conventions collapse
- who only succeeds in one familiar sentence frame
