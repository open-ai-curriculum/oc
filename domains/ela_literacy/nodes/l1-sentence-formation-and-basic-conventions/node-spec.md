# L1 Node Specification

## Node Identity

- id: `L1`
- name: `sentence_formation_and_basic_conventions`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `early-language-node-package`
- evidence class: `Synthetic`

## Purpose

The learner forms complete simple sentences and uses basic conventions well enough to support early written communication and later revision work.

## Dependencies

- `O3` oral recount, explanation, and sequence language
- `P4` alphabet knowledge and letter formation

## Exit Conditions

The learner is secure on `L1` only when all of the following are true:

1. produces complete simple sentences with stable basic structure
2. transfers oral sentence structure into early written production with usable support
3. uses enough basic conventions to preserve intended meaning
4. succeeds across changed prompts or production contexts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- form simple complete sentences
- produce early written sentences that preserve intended meaning
- use basic conventions in a functional way
- support later sentence generation, composition, and revision work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
