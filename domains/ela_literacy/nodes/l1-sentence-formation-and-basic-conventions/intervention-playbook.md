# L1 Intervention Playbook

## Purpose

Map `L1` failure modes to bounded intervention responses.

## Intervention Rules

### For L1-FM-01 Fragment-Dominant Output

Primary interventions:

- model a complete sentence from the learner's fragment and have the learner restate it
- use a constrained who-does-what sentence frame
- fade from oral frame to independent completion

Expected effect:

- learner shifts from labels or fragments toward complete simple sentences

Reverification target:

- one short sentence-generation task

### For L1-FM-02 Oral-Written Transfer Split

Primary interventions:

- capture the oral sentence first, then transcribe it with immediate learner participation
- keep content generation stable while reducing writing demands
- rehearse oral-to-written carryover with one sentence at a time

Expected effect:

- learner transfers known oral sentence structure into supported writing

Reverification target:

- one oral-then-written sentence task

### For L1-FM-03 Convention Load Collapse

Primary interventions:

- narrow the convention target to one or two essentials that preserve meaning
- use spacing, capitalization, and end-mark checks after sentence generation, not before
- separate sentence-structure failure from convention-overload failure

Expected effect:

- learner preserves sentence meaning while stabilizing basic conventions

Reverification target:

- one sentence production task with focused convention check

### For L1-FM-04 Prompt-Bound Sentence Production

Primary interventions:

- vary the prompt while keeping the expected sentence structure stable
- rotate between picture, oral prompt, and experience prompt formats
- use one familiar frame only as a bridge, not as the end state

Expected effect:

- learner generalizes sentence formation beyond a single rehearsed frame

Reverification target:

- one changed-prompt sentence-generation task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of oral-language dependency sufficiency, transcription load, and whether the convention target is masking the sentence-formation target.
