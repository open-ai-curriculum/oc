# L1 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `L1`.

## Guidance

- preserve the sentence-formation target instead of overloading the task with too many convention demands at once
- use oral language as a support bridge, not as a substitute for written sentence production
- fade frames as soon as the learner can sustain a complete simple sentence
- vary prompts once one sentence pattern stabilizes
- escalate to human review when oral-language, transcription, and convention issues cannot be cleanly separated
