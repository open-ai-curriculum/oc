# FR2 Node Specification

## Node Identity

- id: `FR2`
- name: `print_encoding_principle_and_bidirectional_mapping`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `foundational-reading-node-package`
- evidence class: `Synthetic`

## Purpose

The learner understands and applies the principle that print encodes spoken language, using that logic in both decoding-like and encoding-like directions.

## Dependencies

- `FR1` phoneme-grapheme mapping

## Exit Conditions

The learner is secure on `FR2` only when all of the following are true:

1. demonstrates that print units represent spoken sounds
2. applies mapping logic in both reading-like and writing-like directions
3. explains or otherwise reveals the print-to-speech relationship rather than performing by imitation alone
4. succeeds across changed tasks or target sets
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- treat print as an encoding system rather than a visual label set
- shift between decoding-like and encoding-like use of the same mappings
- support later controlled word reading and spelling with explicit print logic

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
