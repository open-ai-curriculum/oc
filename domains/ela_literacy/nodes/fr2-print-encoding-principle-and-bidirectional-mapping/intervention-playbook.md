# FR2 Intervention Playbook

## Purpose

Map `FR2` failure modes to bounded intervention responses.

## Intervention Rules

### For FR2-FM-01 Ritual Without Meaning

Primary interventions:

- vary the prompt format while keeping the target logic constant
- ask what sound or spoken part the print is representing
- require the learner to demonstrate the relationship with a paired example

Expected effect:

- learner reveals actual print-encoding understanding instead of routine imitation

Reverification target:

- one varied-format print-encoding task

### For FR2-FM-02 Directional Split

Primary interventions:

- pair reading-like and encoding-like use of the same target immediately
- keep the spoken and printed forms visible together
- reduce task length while preserving both directions

Expected effect:

- learner stabilizes bidirectional print-speech use

Reverification target:

- one paired bidirectional task

### For FR2-FM-03 Explanation Fragility

Primary interventions:

- prompt the learner to identify what print is standing for
- use a very small set of examples and non-examples
- verify that `FR1` remains secure

Expected effect:

- learner gives evidence of the print-encoding principle beyond routine performance

Reverification target:

- one explanation or demonstration task

### For FR2-FM-04 Changed-Task Collapse

Primary interventions:

- preserve the same target logic across a small changed-set task
- change only one variable at a time
- revisit a known example after the changed example

Expected effect:

- learner carries the print-encoding principle into new but bounded formats

Reverification target:

- one changed-format bidirectional task

## Escalation Rule

If the learner continues to show ritualized performance without evidence of print-encoding understanding after two intervention cycles, escalate to human review.
