# FR2 Failure Taxonomy

### FR2-FM-01 Ritual Without Meaning

- description: learner performs a familiar routine but does not reveal actual print-encoding understanding

### FR2-FM-02 Directional Split

- description: learner succeeds in decoding-like tasks or encoding-like tasks, but not both

### FR2-FM-03 Explanation Fragility

- description: learner can perform some mapping but cannot explain or demonstrate what print is encoding

### FR2-FM-04 Changed-Task Collapse

- description: performance drops when the same mapping logic is requested in a new format or target set
