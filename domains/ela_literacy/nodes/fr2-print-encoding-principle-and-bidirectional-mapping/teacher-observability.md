# FR2 Teacher Observability View

Teachers should see:

- current `FR2` status
- dependency posture for `FR1`
- direct print-encoding performance
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner can use mapping in both decoding-like and encoding-like directions
- recommended intervention

Class-level view should highlight:

- who can perform routines without revealing print-encoding understanding
- who breaks across direction changes
- who cannot explain what print is representing
