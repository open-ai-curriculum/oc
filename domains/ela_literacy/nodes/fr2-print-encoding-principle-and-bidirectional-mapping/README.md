# FR2 Print-Encoding Principle And Bidirectional Mapping

## Purpose

This proposal-state node isolates the learner's understanding that print encodes spoken language and that mapping supports both reading-like and encoding-like action.

It exists to separate direct correspondence retrieval from true bidirectional print-encoding control.
