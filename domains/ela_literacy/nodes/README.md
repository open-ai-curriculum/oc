# ELA/Literacy Nodes

## Purpose

This directory stores governed node packages for the `ela_literacy` domain.

Each node package should eventually include the same operational surfaces already used in maintained mathematics nodes:

- `README.md`
- `node-config.yaml`
- `node-spec.md`
- learner-state model artifacts
- verification model
- failure taxonomy
- intervention artifacts
- observability artifacts

## Current Status

The first Pre-K readiness node packages have been started here as draft package seeds.

The current node wave is intentionally upstream-first.

Wave 1 foundational-reading canonical draft packages now exist for:

- `FR1` phoneme-grapheme mapping
- `FR2` print-encoding principle and bidirectional mapping
- `FR3` simple-pattern decoding and encoding
- `FR4` word-level decoding generalization
- `FR5` orthographic consolidation and automatic word recognition
- `FR6` connected-text fluency and tracking

These packages now represent the canonical foundational-reading branch in the live ELA/literacy graph.

The legacy `F1`, `F2`, and `F3` packages remain on disk as superseded historical packages so earlier planning and migration artifacts preserve explicit lineage.
