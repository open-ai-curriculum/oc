# RC2 Failure Taxonomy

### RC2-FM-01 Inference Without Anchor

- description: learner gives plausible but text-unanchored answers

### RC2-FM-02 Single-Detail Fixation

- description: learner cannot integrate more than one relevant detail

### RC2-FM-03 Overgeneralized Background Knowledge

- description: learner substitutes prior knowledge for text-based reasoning

### RC2-FM-04 Question-Language Confound

- description: learner struggles more with the inferential prompt than with the text itself
