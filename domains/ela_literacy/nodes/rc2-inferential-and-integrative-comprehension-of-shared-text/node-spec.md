# RC2 Node Specification

## Node Identity

- id: `RC2`
- name: `inferential_and_integrative_comprehension_of_shared_text`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `early-comprehension-node-package`
- evidence class: `Synthetic`

## Purpose

The learner makes bounded inferences and integrates simple relationships across shared text after literal meaning is secure.

## Dependencies

- `RC1`

## Exit Conditions

The learner is secure on `RC2` only when all of the following are true:

1. draws bounded inferences from shared or read-aloud text
2. integrates more than one relevant detail to support a simple conclusion
3. maintains inferential meaning across changed texts or prompts
4. does not substitute background guessing for text-grounded integration
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- infer likely meaning or outcome from shared text
- connect two relevant details from the same text
- support the bridge from literal recall to early text reasoning

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
