# W5 Node Specification

## Node Identity

- id: `W5`
- name: `research_synthesis_and_citation_control`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `research-writing-node-package`
- evidence class: `Synthetic`

## Purpose

The learner synthesizes multiple sources and controls citation or attribution well enough to support research-based writing and later advanced rhetorical work.

## Dependencies

- `W4` multi-paragraph explanation, narrative, and argument
- `R6` disciplinary literacy, rhetorical analysis, and cross-text synthesis
- `L3` syntax revision and style control

## Exit Conditions

The learner is secure on `W5` only when all of the following are true:

1. integrates multiple sources into a coherent written response
2. maintains usable attribution or citation control
3. distinguishes synthesis from copied or disconnected source accumulation
4. succeeds across changed research questions or source sets
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `21`

## Observable Evidence

The learner can:

- synthesize source material into coherent writing
- attribute ideas and evidence with usable citation control
- sustain research-based written reasoning across sources
- support later advanced rhetorical synthesis and academic writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
