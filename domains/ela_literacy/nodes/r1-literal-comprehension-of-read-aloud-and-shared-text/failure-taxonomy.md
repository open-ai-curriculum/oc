# R1 Failure Taxonomy

### R1-FM-01 Detail Recall Drift

- description: learner loses key literal details or substitutes unrelated information

### R1-FM-02 Cue-Dependent Responding

- description: learner appears successful only when visual, routine, or question cues make the answer obvious

### R1-FM-03 Oral-Response Constraint

- description: learner may understand text but cannot show the meaning clearly enough through the available oral response

### R1-FM-04 Text-Type Fragility

- description: learner succeeds with one familiar kind of text but loses comprehension when text type changes
