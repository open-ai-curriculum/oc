# R1 Node Specification

## Node Identity

- id: `R1`
- name: `literal_comprehension_of_read_aloud_and_shared_text`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `early-reading-node-package`
- evidence class: `Synthetic`

## Purpose

The learner demonstrates literal meaning from shared and read-aloud text well enough to establish early reading comprehension before independent sentence-level reading is expected.

## Dependencies

- `O3` oral recount, explanation, and sequence language
- `P1` book handling, print awareness, and directionality

## Exit Conditions

The learner is secure on `R1` only when all of the following are true:

1. identifies or recounts key literal details from shared or read-aloud text
2. responds to simple text questions with text-relevant meaning
3. maintains comprehension across changed stories, informational text, or prompt formats
4. does not depend only on irrelevant visual or routine cues
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- answer literal who, what, where, or what-happened questions from shared text
- recount simple text events or details
- demonstrate meaning from text before independent decoding is expected
- support later sentence-level and evidence-based reading work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
