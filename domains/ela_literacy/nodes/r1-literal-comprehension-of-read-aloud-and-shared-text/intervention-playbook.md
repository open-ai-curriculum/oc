# R1 Intervention Playbook

## Purpose

Map `R1` failure modes to bounded intervention responses.

## Intervention Rules

### For R1-FM-01 Detail Recall Drift

Primary interventions:

- reduce the text chunk before asking the question
- ask one literal detail question immediately after the relevant segment
- rehearse what happened using a short oral retell frame

Expected effect:

- learner retains and recovers key literal text details more reliably

Reverification target:

- one short literal-comprehension item

### For R1-FM-02 Cue-Dependent Responding

Primary interventions:

- remove extra cues and reduce answer obviousness
- vary question wording while holding the same literal target stable
- ask the learner to point to or retell the relevant text event rather than guess

Expected effect:

- learner responses become more genuinely text-grounded

Reverification target:

- one low-cue literal-comprehension task

### For R1-FM-03 Oral-Response Constraint

Primary interventions:

- allow a simpler response frame while keeping the meaning target the same
- model the answer structure once before independent response
- separate comprehension failure from expressive-language bottleneck

Expected effect:

- learner shows text meaning more clearly through an accessible response mode

Reverification target:

- one alternate oral-response task

### For R1-FM-04 Text-Type Fragility

Primary interventions:

- keep the comprehension frame stable while changing the text type
- alternate short narrative and informational read-alouds with the same literal prompt type
- make genre change explicit to the learner only after one frame stabilizes

Expected effect:

- learner transfers literal comprehension beyond one preferred text type

Reverification target:

- one changed-text-type literal task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of language-access demands, background-knowledge confounds, and expressive-response constraints.
