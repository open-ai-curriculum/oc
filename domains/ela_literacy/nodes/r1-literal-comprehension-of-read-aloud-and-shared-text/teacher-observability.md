# R1 Teacher Observability View

Teachers should see:

- current `R1` status
- dependency posture for `O3` and `P1`
- direct literal-comprehension accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether success reflects actual text meaning rather than cue-based guessing
- recommended intervention

Class-level view should highlight:

- who misses key details from read-aloud text
- who answers only with heavy cue support
- who breaks when text type changes
