# F2 Teacher Observability View

Teachers should see:

- current `F2` status
- dependency posture for `F1`
- direct simple-word decoding and encoding accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether performance reflects true decoding or guessing
- recommended intervention

Class-level view should highlight:

- who guesses from partial visual cues
- who reads simple words but cannot spell them back
- who loses control when the word set changes
