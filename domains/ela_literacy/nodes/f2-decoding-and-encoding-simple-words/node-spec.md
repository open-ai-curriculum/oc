# F2 Node Specification

## Node Identity

- id: `F2`
- name: `decoding_and_encoding_simple_words`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `early-decoding-node-package`
- evidence class: `Synthetic`

## Purpose

The learner decodes and encodes simple words with enough stability to support later sentence-level reading, fluency, and written production.

## Dependencies

- `F1` alphabetic principle and sound-symbol mapping

## Exit Conditions

The learner is secure on `F2` only when all of the following are true:

1. decodes supported simple words accurately with sound-symbol reasoning
2. encodes supported simple words with corresponding sound-symbol control
3. generalizes across changed word sets rather than memorized item-only performance
4. demonstrates usable bidirectional reading-writing transfer
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- read simple words using applied sound-symbol knowledge
- spell or encode simple words using corresponding mapping knowledge
- distinguish applied decoding from whole-word guessing
- support the move into connected text and sentence-level literacy

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
