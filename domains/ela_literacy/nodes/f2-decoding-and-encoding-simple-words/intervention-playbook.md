# F2 Intervention Playbook

## Purpose

Map `F2` failure modes to bounded intervention responses.

## Intervention Rules

### For F2-FM-01 Guessing-Over-Decoding

Primary interventions:

- remove pictures or obvious context clues during the decoding trial
- require left-to-right sound tracking across every grapheme
- use continuous blending with immediate correction at the point of guessing

Expected effect:

- learner attends to the full sound sequence instead of partial-cue guessing

Reverification target:

- one short decoding set with reduced external cues

### For F2-FM-02 Encoding-Transfer Weakness

Primary interventions:

- pair each decoded word with an immediate spell-it-back task
- use oral segmenting before letter selection or writing
- keep the same sound sequence visible across reading and spelling turns

Expected effect:

- learner carries the same mapping knowledge in both reading and writing directions

Reverification target:

- one short paired decode-and-encode set

### For F2-FM-03 Word-Set Fragility

Primary interventions:

- rotate to novel simple-word sets built from the same taught correspondences
- vary order and format while holding the taught mapping set stable
- separate memorized item success from generalized simple-word control

Expected effect:

- learner generalizes simple-word reading and spelling beyond rehearsed items

Reverification target:

- one changed-set decoding and encoding task

### For F2-FM-04 Sequence-Tracking Breakdown

Primary interventions:

- use sound boxes or finger tracking to preserve phoneme order
- slow the word to phoneme sequence and rebuild it into a full blend
- keep words short while sequence control is being stabilized

Expected effect:

- learner maintains ordered blending and segmenting across the whole word

Reverification target:

- one ordered blend-and-spell task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of dependency sufficiency, phonological-load mismatch, and whether the supported word set is too large for current control.
