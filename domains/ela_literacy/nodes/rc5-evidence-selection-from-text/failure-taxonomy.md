# RC5 Failure Taxonomy

### RC5-FM-01 Answer Without Evidence

- description: learner responds without selecting text support

### RC5-FM-02 Irrelevant Detail Selection

- description: learner selects a true detail that does not answer the prompt

### RC5-FM-03 Evidence Overload

- description: learner cannot decide among multiple possible details

### RC5-FM-04 Oral-Language Explanation Confound

- description: learner can point to evidence but cannot yet state why it matters clearly
