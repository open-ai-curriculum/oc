# W2 Intervention Playbook

## Purpose

Map `W2` failure modes to bounded intervention responses.

## Intervention Rules

### For W2-FM-01 Idea-To-Sentence Collapse

Primary interventions:

- rehearse the sentence orally before writing
- reduce idea-selection demands so the target remains sentence production
- use a brief sentence frame only long enough to establish a complete sentence

Expected effect:

- learner converts an intended idea into a complete written sentence more reliably

Reverification target:

- one oral-to-written sentence task

### For W2-FM-02 Transcription Bottleneck

Primary interventions:

- reduce spelling or copying load while preserving sentence generation
- chunk the sentence into manageable writing units
- separate composition failure from handwriting or encoding overload

Expected effect:

- learner maintains sentence meaning while managing transcription demands

Reverification target:

- one supported sentence transcription task

### For W2-FM-03 Sentence-Control Inconsistency

Primary interventions:

- use short repeated sentence opportunities within one session
- keep prompt type stable while requiring multiple independent sentences
- identify whether breakdown occurs at planning, transcription, or conventions

Expected effect:

- learner sustains sentence-writing performance across more than one opportunity

Reverification target:

- one three-sentence bounded writing set

### For W2-FM-04 Prompt-Transfer Fragility

Primary interventions:

- rotate between picture, question, and experience prompts
- preserve the sentence demand while changing the topic or format
- fade reliance on one favored sentence frame

Expected effect:

- learner generalizes sentence writing across changed prompts

Reverification target:

- one changed-context sentence-writing task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of whether the real bottleneck is still `FR4` or `L1`, and whether motor or transcription constraints require separate support planning.
