# W2 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `W2`.

## Guidance

- preserve sentence-writing as the target rather than collapsing the task into copying only
- use oral rehearsal strategically, then require actual written sentence production
- reduce transcription load when needed, but do not remove the need to communicate meaning in writing
- vary prompts only after one context is stable
- escalate to human review when sentence generation, encoding, and motor demands remain too entangled to diagnose cleanly
