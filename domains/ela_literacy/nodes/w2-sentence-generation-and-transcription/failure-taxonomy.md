# W2 Failure Taxonomy

### W2-FM-01 Idea-To-Sentence Collapse

- description: learner has an intended message but cannot convert it into a complete written sentence

### W2-FM-02 Transcription Bottleneck

- description: writing mechanics consume enough attention that sentence generation breaks down

### W2-FM-03 Sentence-Control Inconsistency

- description: learner sometimes produces a usable sentence but cannot do so reliably across multiple opportunities

### W2-FM-04 Prompt-Transfer Fragility

- description: learner succeeds on one familiar writing prompt or frame but performance falls on changed contexts
