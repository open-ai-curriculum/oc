# F1 Failure Taxonomy

### F1-FM-01 Visual-Memorization Substitution

- description: learner appears to know words or letters visually but does not apply actual sound-symbol mapping

### F1-FM-02 Bidirectional Mapping Breakdown

- description: learner can sometimes map letter-to-sound or sound-to-letter, but not both with usable consistency

### F1-FM-03 Sound-Symbol Set Fragility

- description: learner succeeds with one small supported set but performance collapses when the set changes

### F1-FM-04 Encoding-Reading Split

- description: learner can demonstrate mapping in one direction but not apply it consistently in the paired reading-like or writing-like task
