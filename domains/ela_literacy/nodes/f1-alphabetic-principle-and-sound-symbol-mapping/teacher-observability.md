# F1 Teacher Observability View

Teachers should see:

- current `F1` status
- dependency posture for `P3` and `P4`
- direct mapping accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner is applying actual sound-symbol mapping rather than visual memorization
- recommended intervention

Class-level view should highlight:

- who maps in one direction but not the other
- who loses performance when the target set changes
- who appears to know forms visually without true mapping
