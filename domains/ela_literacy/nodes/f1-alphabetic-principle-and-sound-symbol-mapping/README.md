# F1 Alphabetic Principle And Sound-Symbol Mapping

## Purpose

This node defines the formal entry into decoding for the `ela_literacy` domain.

It exists so later word reading and encoding are grounded in explicit sound-symbol reasoning rather than memorized whole-form performance alone.
