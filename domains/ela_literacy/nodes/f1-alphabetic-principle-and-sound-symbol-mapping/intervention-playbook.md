# F1 Intervention Playbook

## Purpose

Map `F1` failure modes to bounded intervention responses.

## Intervention Rules

### For F1-FM-01 Visual-Memorization Substitution

Primary interventions:

- use novel but simple targets that prevent whole-form memorization
- require both sound-to-letter and letter-to-sound responses
- ask for explanation of what the symbol is doing in the word

Expected effect:

- learner relies more on actual mapping than visual recall alone

Reverification target:

- one changed-set mapping task

### For F1-FM-02 Bidirectional Mapping Breakdown

Primary interventions:

- practice a very small target set in both directions
- alternate reading-like and encoding-like trials with the same target
- keep the oral sound and written symbol visible in a consistent routine

Expected effect:

- learner stabilizes mapping in both directions

Reverification target:

- one paired bidirectional mapping task

### For F1-FM-03 Sound-Symbol Set Fragility

Primary interventions:

- keep one small sound-symbol set stable across several trials
- add only one new target after stability is visible
- contrast old and new targets explicitly

Expected effect:

- learner generalizes mapping knowledge beyond a memorized micro-set

Reverification target:

- one changed-set mapping task

### For F1-FM-04 Encoding-Reading Split

Primary interventions:

- pair reading and spelling of the same supported target
- ask the learner to say the sound while writing or selecting the symbol
- move from one direction to the other immediately in the same routine

Expected effect:

- learner links reading-like and writing-like applications of the same mapping

Reverification target:

- one paired decoding-and-encoding task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of oral-sound discrimination, symbol learning, and task-format confounds.
