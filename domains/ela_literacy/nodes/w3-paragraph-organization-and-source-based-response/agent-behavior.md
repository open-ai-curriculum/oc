# W3 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `W3`.

## Guidance

- keep paragraph tasks bounded enough that organization and source use are observable
- require a response focus before expanding the amount of writing
- distinguish paragraph-organization failure from evidence-use failure
- vary source and prompt only after one short response routine stabilizes
- escalate to human review when writing, reading, and evidence-integration failures remain too entangled to diagnose cleanly
