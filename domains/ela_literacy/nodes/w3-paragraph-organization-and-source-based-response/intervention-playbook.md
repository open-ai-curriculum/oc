# W3 Intervention Playbook

## Purpose

Map `W3` failure modes to bounded intervention responses.

## Intervention Rules

### For W3-FM-01 Sentence-Pile Without Paragraph Control

Primary interventions:

- plan one main idea before sentence drafting
- use a simple paragraph frame with an opening idea and supporting sentences
- review whether each sentence belongs to the same response focus

Expected effect:

- learner organizes sentences into a coherent response unit rather than a loose list

Reverification target:

- one short paragraph-construction task

### For W3-FM-02 Source-Use Weakness

Primary interventions:

- require one relevant detail or cited phrase before elaboration
- identify which text detail best supports the response idea
- keep source integration short and bounded at first

Expected effect:

- learner includes relevant source support instead of topic-only writing

Reverification target:

- one paragraph task with required text support

### For W3-FM-03 Coherence Drift

Primary interventions:

- check each sentence against the paragraph's main point
- use sentence ordering review to restore logical sequence
- cut or revise off-topic sentences rather than adding more content

Expected effect:

- learner maintains sentence-to-sentence coherence within the paragraph

Reverification target:

- one revise-for-coherence paragraph task

### For W3-FM-04 Prompt-And-Source Fragility

Primary interventions:

- rotate between changed prompts and short source texts
- preserve paragraph length expectations while varying topic and source
- reduce novelty only enough to observe transfer

Expected effect:

- learner generalizes paragraph organization and source use across changed tasks

Reverification target:

- one changed-prompt or changed-source paragraph task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of whether the real bottleneck remains `W2` or `RC6`, and whether the response demand exceeds the intended single-paragraph boundary.
