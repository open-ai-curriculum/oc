# W3 Node Specification

## Node Identity

- id: `W3`
- name: `paragraph_organization_and_source_based_response`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `paragraph-writing-node-package`
- evidence class: `Synthetic`

## Purpose

The learner organizes a paragraph and integrates source-based support well enough to bridge from sentence generation into structured written response.

## Dependencies

- `W2` sentence generation and transcription
- `RC6` evidence-based response to text

## Exit Conditions

The learner is secure on `W3` only when all of the following are true:

1. organizes related sentences into a coherent paragraph or bounded response
2. incorporates source-based detail or evidence relevant to the prompt
3. maintains enough internal organization for the response to be interpretable as a unit
4. succeeds across changed prompts or source texts
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- produce a paragraph-length response with a recognizable organizing idea
- integrate text-based detail into writing
- sustain sentence-to-sentence coherence
- support later argument, explanation, and analytical writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
