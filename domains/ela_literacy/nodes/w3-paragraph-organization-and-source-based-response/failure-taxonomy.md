# W3 Failure Taxonomy

### W3-FM-01 Sentence-Pile Without Paragraph Control

- description: learner produces multiple sentences but they do not organize into a coherent paragraph or bounded response unit

### W3-FM-02 Source-Use Weakness

- description: learner writes generally about the topic but fails to include relevant source-based support

### W3-FM-03 Coherence Drift

- description: learner begins with a relevant idea but loses focus or connection from one sentence to the next

### W3-FM-04 Prompt-And-Source Fragility

- description: learner succeeds on one familiar prompt or text but performance collapses on changed sources or response demands
