# RC4 Node Specification

## Node Identity

- id: `RC4`
- name: `short_passage_integration_and_local_coherence`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `passage-comprehension-node-package`
- evidence class: `Synthetic`

## Purpose

The learner integrates meaning across short connected text and maintains local coherence beyond isolated sentence understanding.

## Dependencies

- `RC3`
- `FR6`

## Exit Conditions

The learner is secure on `RC4` only when all of the following are true:

1. maintains meaning across short connected passages
2. connects sentence-level meanings into a coherent local text understanding
3. succeeds across changed short-passage formats or prompts
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- track meaning across more than one sentence in a short passage
- identify locally coherent relations across adjacent sentences
- support the bridge from sentence-level meaning to passage-level evidence work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
