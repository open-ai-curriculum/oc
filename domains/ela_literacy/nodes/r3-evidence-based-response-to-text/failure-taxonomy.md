# R3 Failure Taxonomy

### R3-FM-01 Opinion-Only Responding

- description: learner answers from personal opinion or memory without grounding the response in text evidence

### R3-FM-02 Evidence-Selection Drift

- description: learner selects details from the text, but they are not actually relevant to the prompt

### R3-FM-03 Explanation-Link Weakness

- description: learner can point to evidence but cannot explain how it supports the response

### R3-FM-04 Text-And-Format Fragility

- description: learner succeeds on one text or response format but performance collapses when either changes
