# R3 Node Specification

## Node Identity

- id: `R3`
- name: `evidence_based_response_to_text`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `text-response-node-package`
- evidence class: `Synthetic`

## Purpose

The learner answers text-dependent prompts with relevant evidence and bounded inference well enough to support later literary and informational analysis.

## Dependencies

- `R2` sentence-level comprehension of supported text
- `O3` oral recount, explanation, and sequence language

## Exit Conditions

The learner is secure on `R3` only when all of the following are true:

1. selects or states relevant text evidence in response to a prompt
2. explains the connection between the evidence and the response with usable clarity
3. avoids relying only on unsupported personal opinion or unrelated recall
4. succeeds across changed texts, prompts, or response formats
5. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- answer a question about text using relevant details
- connect evidence to an interpretation or response claim
- sustain bounded inference without leaving the text entirely
- support later paragraph response and analytical reading work

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
