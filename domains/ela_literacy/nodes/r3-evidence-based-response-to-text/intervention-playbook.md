# R3 Intervention Playbook

## Purpose

Map `R3` failure modes to bounded intervention responses.

## Intervention Rules

### For R3-FM-01 Opinion-Only Responding

Primary interventions:

- require the learner to point to or restate a text detail before answering
- use a simple answer-plus-evidence response frame
- contrast text-based answers with unsupported personal reactions

Expected effect:

- learner grounds responses in the text rather than opinion alone

Reverification target:

- one text-dependent response requiring evidence selection

### For R3-FM-02 Evidence-Selection Drift

Primary interventions:

- narrow the prompt and reduce the number of possible details
- ask which part of the text best matches the question and why
- compare a relevant and irrelevant detail from the same text

Expected effect:

- learner chooses evidence that actually addresses the prompt

Reverification target:

- one constrained evidence-selection task

### For R3-FM-03 Explanation-Link Weakness

Primary interventions:

- add a because or this shows frame after the selected evidence
- model one evidence-to-claim explanation, then fade support
- keep the interpretive demand bounded while strengthening the reasoning link

Expected effect:

- learner explains how evidence supports the response, not only which detail was found

Reverification target:

- one evidence-plus-explanation task

### For R3-FM-04 Text-And-Format Fragility

Primary interventions:

- rotate across short narrative and informational texts
- alternate oral and brief written response formats with the same evidence demand
- preserve the evidence target while changing surface format

Expected effect:

- learner transfers evidence-based response beyond one text and one format

Reverification target:

- one changed-text or changed-format response task

## Escalation Rule

If the same dominant failure mode remains blocking after two bounded intervention cycles, escalate to human review of sentence-level comprehension sufficiency, oral-language explanation limits, and whether the inferential demand exceeds the intended `R3` boundary.
