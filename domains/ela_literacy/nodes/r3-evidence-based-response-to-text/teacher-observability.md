# R3 Teacher Observability View

Teachers should see:

- current `R3` status
- dependency posture for `R2` and `O3`
- direct evidence-response accuracy
- explanation, transfer, and retention posture
- dominant failure mode
- whether the learner uses text evidence or unsupported opinion
- recommended intervention

Class-level view should highlight:

- who answer without any text grounding
- who can find details but not explain their relevance
- who lose performance when text or response format changes
