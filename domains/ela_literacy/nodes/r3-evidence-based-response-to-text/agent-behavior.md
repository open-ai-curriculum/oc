# R3 Agent Behavior

## Purpose

Define how an instructional agent should behave when supporting `R3`.

## Guidance

- keep the prompt text-dependent and bounded so the learner is not pushed into full literary analysis too early
- require evidence before elaboration
- distinguish relevant evidence selection from explanation of that evidence
- vary text and response format only after one stable evidence routine exists
- escalate to human review when comprehension, oral explanation, and evidence reasoning cannot be separated cleanly
