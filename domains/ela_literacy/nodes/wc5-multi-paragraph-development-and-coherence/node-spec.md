# WC5 Node Specification

## Node Identity

- id: `WC5`
- name: `multi_paragraph_development_and_coherence`
- domain: `ela_literacy`
- support tier: `Core release`
- current maturity state: `Drafted`
- supported profile: `extended-composition-node-package`
- evidence class: `Synthetic`

## Purpose

The learner sustains coherent development across multiple paragraphs before genre-specific control becomes the main distinction.

## Dependencies

- `WC4`

## Exit Conditions

The learner is secure on `WC5` only when all of the following are true:

1. develops ideas coherently across multiple paragraphs
2. maintains cross-paragraph structure and progression
3. succeeds across changed multi-paragraph tasks
4. remains stable on delayed recheck

## Mastery Criteria

- direct accuracy threshold: `0.85`
- explanation required: `true`
- transfer required: `true`
- retention check days: `14`

## Observable Evidence

The learner can:

- sustain cross-paragraph coherence and development
- separate general multi-paragraph control from genre adaptation
- support later genre-controlled writing

## Claim Boundary

This node package is an operational draft and not a classroom-release claim.
