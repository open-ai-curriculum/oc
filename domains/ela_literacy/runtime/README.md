# ELA/Literacy Runtime

This directory contains graph-runtime artifacts for the ELA/literacy domain.

## Purpose

These artifacts operationalize learner state and navigation against the ELA/literacy graph so bounded slices can be simulated, validated, and later executed in adaptive systems.

## Files

- `runtime-slice.schema.json`: canonical JSON Schema for ELA/literacy runtime slice packages
- `early-literacy-spine-runtime-slice.yaml`: initial runtime slice for the first oral-language and literacy-entry progression
- `reading-composition-spine-runtime-slice.yaml`: runtime slice for reading response, paragraphing, and structured composition
- `advanced-analysis-spine-runtime-slice.yaml`: runtime slice for disciplinary literacy, research writing, and capstone analysis
