# ELA/Literacy Artifact Attachments

This directory contains machine-readable attachment registries that bind curriculum, assessment, intervention, and support artifacts to graph-native ELA/literacy entities.

## Purpose

These registries operationalize the [graph artifact attachment contract](../graph-artifact-attachment-contract.md) by giving the ELA/literacy graph a concrete attachment layer that can later drive curriculum compilation and adaptive navigation.

## Files

- `registry.schema.json`: canonical JSON Schema for ELA/literacy artifact attachment registries
- `early-literacy-spine-attachment-registry.yaml`: initial draft registry for the first oral-language, print, and foundational-literacy slice
- `reading-composition-spine-attachment-registry.yaml`: draft registry for the reading-response, language, and composition middle slice
- `advanced-analysis-spine-attachment-registry.yaml`: draft registry for the advanced disciplinary, research, and capstone analysis slice
