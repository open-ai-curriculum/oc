# ELA/Literacy Capability Graph

## Scope

This document summarizes the initial governed downstream slice for `ela_literacy`.

The slice starts before decoding and writing production because later literacy performance becomes unstable when oral language, print concepts, or phonological processing are assumed rather than verified.

## Initial Maintained Slice

The current slice is:

`oral recount -> print awareness -> phonological awareness -> alphabet knowledge -> phoneme-grapheme mapping -> print encoding -> controlled decoding/encoding -> generalized word control -> sentence comprehension -> sentence generation -> evidence-based response -> paragraph organization`

## Why This Slice

This is the smallest reading/writing segment that can support meaningful downstream system behavior:

- it has explicit upstream dependencies
- it spans both reading and writing
- it exposes language-access and decoding confounds
- it creates a clean entry point for later node-package implementation

## Claim Boundary

This capability graph is a scaffolded planning and governance artifact. It is not yet backed by implemented node packages, item banks, failure taxonomies at node-package depth, or automated audit coverage.
