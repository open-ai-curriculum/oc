# ELA/Literacy Implementation Maturity Tracker

## Purpose

Track the current implementation maturity of the ELA/literacy domain across the same major operational layers used to judge parity with mathematics.

## Scoring Model

Scores use a four-level maturity model:

- `0`: absent
- `1`: present as initial scaffold
- `2`: operational draft
- `3`: materially mature for current modeled scope
