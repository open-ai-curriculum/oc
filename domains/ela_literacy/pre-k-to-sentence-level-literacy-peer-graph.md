# Pre-K To Sentence-Level Literacy Peer Graph

## Purpose

Define the broader early-literacy graph that reaches the peer level of the current literacy-entry and sentence-level frontier.

This document is different from the narrower prerequisite spine in [oral-language-to-f1-literacy-spine.md](oral-language-to-f1-literacy-spine.md).

That document answers:

- what must exist before `FR1` is legitimate

This document answers:

- what other literacy branches should exist at comparable developmental depth even when they do not feed directly into `FR1`

## Design Rule

The repository should not confuse:

- a valid prerequisite path

with:

- a sufficiently broad early-literacy graph

Both are required.

## Graph Scope

The peer-level target here is roughly:

- pre-K oral-language and print readiness
- kindergarten literacy-entry foundations
- Grade 1 to early Grade 3 sentence-level reading and writing

## Core Branches

### A. Oral-Language And Listening Branch

- `O1` joint attention, turn-taking, and response readiness
- `O2` oral language comprehension for instructions and routines
- `O3` oral recount, explanation, and sequence language

### B. Print And Symbol-Control Branch

- `P1` book handling, print awareness, and directionality
- `P2` concept of word, spacing, and text tracking
- `P3` phonological awareness
- `P4` alphabet knowledge and letter formation

### C. Foundational Reading Branch

- `FR1` phoneme-grapheme mapping
- `FR2` print-encoding principle and bidirectional mapping
- `FR3` simple-pattern decoding and encoding
- `FR4` word-level decoding generalization
- `FR5` orthographic consolidation and automatic word recognition
- `FR6` connected-text fluency and tracking

### D. Early Reading Comprehension Branch

- `R1` literal comprehension of read-aloud and shared text
- `R2` sentence-level comprehension of supported text

### E. Early Language Branch

- `L1` sentence formation and basic conventions

### F. Early Writing Branch

- `W1` drawing, dictation, labeling, and idea capture
- `W2` sentence generation and transcription

## Why These Branches Matter Together

This frontier is not legitimate if it contains only decoding.

A peer-level early-literacy graph must also include:

- oral-language access
- print-access conditions
- literal comprehension
- sentence production
- first writing surfaces

Otherwise the graph is only a partial foundational-reading strip rather than a governed literacy frontier.

## Minimal Dependency Logic

`O1 -> O2 -> O3`

`O1 -> P1 -> P2`

`O2 -> P3 -> P4 -> FR1 -> FR2 -> FR3 -> FR4 -> FR5 -> FR6`

`O3 + P1 -> R1 -> R2`

`O3 + P4 -> W1 -> W2`

`O3 + P4 -> L1`

`FR4 + L1 + W1 -> W2`

## Current Repository Judgment

The repository now has this peer-level early-literacy graph materially present, not just the narrow foundational-reading path.

That is a strength relative to earlier states of the ELA/literacy package.

## Claim Boundary

This document defines the broad early-literacy peer graph only. It does not claim reviewed standards alignment or classroom-ready instructional coverage.
