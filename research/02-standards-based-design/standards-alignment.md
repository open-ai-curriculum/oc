# Standards-Based Curriculum Design

## Purpose

This document explains how the repository should approach standards alignment in a way that is instructionally meaningful, reviewable, and eventually interoperable.

## Core Position

Standards alignment should be treated as a design discipline, not a tagging exercise.

That means:

- standards should shape outcomes and evidence
- alignment should be visible in lessons and assessments
- standards references should be structured enough to support later metadata and interoperability work

## Current High-Confidence Principles

### 1. Start with the applicable standards set

For each curriculum artifact, identify:

- the relevant standards framework
- the specific standard identifiers
- the intended level of rigor

### 2. Unpack standards into teachable targets

Broad standards should be translated into:

- learning objectives
- prerequisite knowledge and skills
- likely misconceptions
- evidence of mastery

### 3. Align instruction and assessment to the same target

The most common quality failure in standards-based work is false alignment:

- a lesson claims one target
- activities focus elsewhere
- assessments measure something narrower or different

The repository should explicitly guard against that failure.

### 4. Preserve standards structure in reusable form

The repository should eventually support:

- machine-readable standards references
- consistent tagging across units, lessons, and assessments
- alignment records that can map to CASE or similar exchange models

## Practical Design Process

### Phase 1: Identify standards

- select the standards framework
- identify target standards
- distinguish primary vs supporting standards

### Phase 2: Unpack into objectives

- define what students should know and do
- identify the expected level of rigor
- define evidence of learning

### Phase 3: Build aligned instruction

- plan experiences that plausibly lead to the objective
- design scaffolds without diluting the target
- keep assessment aligned to the same intended learning

### Phase 4: Review alignment quality

- confirm that standards claims are specific
- confirm that lesson and assessment demands match the target
- confirm that tagging is not decorative

## Current Trends Relevant to the Repository

The most relevant current shifts are:

- stronger emphasis on authentic application and reasoning
- increasing need for standards tagging that can support reuse across platforms
- growing expectation that standards-aligned materials also meet accessibility and HQIM expectations
