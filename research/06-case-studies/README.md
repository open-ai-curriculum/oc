# Case Studies in Open Source Curriculum Implementation

## Overview

This directory is reserved for real-world case studies, success stories, and lessons learned from institutions that have implemented open or open-adjacent curriculum approaches.

Current status:

- this section is no longer empty, but it is still early
- initial exemplar notes and a reusable template have been added
- deeper implementation and outcome case studies are still needed

See the [Research Gap Remediation Plan](../RESEARCH_GAP_REMEDIATION_PLAN.md) for the case-study buildout requirements.

## Using These Case Studies

Each case study includes:
- **Context**: Institution background and initial challenges
- **Implementation Process**: Step-by-step adoption journey
- **Outcomes**: Measured results and impacts
- **Lessons Learned**: Key insights and recommendations
- **Replication Guidance**: How others can adapt these approaches

## Contributing Case Studies

Organizations implementing open source curriculum are encouraged to document and share their experiences to benefit the broader educational community. Contact information and submission guidelines are provided in each case study template.
