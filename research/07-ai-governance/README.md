# AI Governance Research

## Purpose

This section will define the governance model for AI-assisted curriculum production in this repository.

It should cover:

- acceptable uses of generative AI
- human review requirements
- provenance and disclosure
- evaluation and revalidation
- bias, safety, and age-appropriateness controls
- academic integrity implications
