# Production Templates

## Purpose

This section converts the research base into practical authoring and review templates for curriculum production.

The templates here are intended to make the repository's standards operational by giving contributors a repeatable structure for:

- unit creation
- lesson creation
- standards mapping
- release review

## Current Templates

- `unit-template.md`
- `lesson-template.md`
- `standards-mapping-template.md`
- `release-review-checklist.md`

## How to Use

1. Start with the unit template when planning a new unit.
2. Use the lesson template for lesson-level buildout.
3. Record standards alignment in the standards mapping template.
4. Use the release review checklist before approving materials for broader use.

## Relationship to Research

These templates should be used with:

- curriculum frameworks
- assessment guidance
- AI governance and human review policy
- accessibility standards and checklist
- HQIM review framework and subject-specific guidance
