# Classroom Materials Baseline

## Purpose

This document defines the end-user instructional artifact set that should accompany repository curriculum packets before they are treated as teacher-ready.

The repository already includes:

- curriculum units and lessons
- standards maps
- review records
- support materials

That is necessary, but not sufficient for real classroom use.

Teachers also need the practical material set that lets them deliver, assign, assess, and review instruction without inventing large missing pieces themselves.

## Required Teacher-Use Artifact Set

Each mature curriculum packet should eventually include the following classroom materials, mapped to the source unit and lesson IDs.

### 1. Lesson narrative for teachers

Purpose:

- give the teacher a fully usable instructional flow, not just a structural lesson outline

Should include:

- opening script or framing language
- modeled examples or teacher moves
- likely student responses
- transition guidance
- closure language

### 2. Classroom task or assignment

Purpose:

- define what students actually do during the lesson

Should include:

- task directions
- materials list
- response format
- expected evidence of learning

### 3. Independent or home practice

Purpose:

- extend or reinforce learning in a realistic way

Should include:

- brief home-appropriate task
- low-burden material assumptions
- family-accessible directions where needed

### 4. Answer key or exemplar

Purpose:

- give teachers a reference for correct, strong, or acceptable responses

Should include one or more of:

- correct answers
- annotated exemplar responses
- common error patterns

### 5. Rubric or scoring guidance

Purpose:

- make evaluation more consistent and usable

Should include:

- criteria aligned to the lesson objective
- performance descriptors or decision rules
- notes on what counts as acceptable evidence

### 6. Quiz, check, or assessment form when appropriate

Purpose:

- support formal or semi-formal evidence collection

Should include:

- assessment items
- directions
- scoring guidance
- accessibility notes

### 7. Assessment key or scoring record

Purpose:

- make grading or interpretation operational

Should include:

- answer key
- exemplar responses where relevant
- scoring notes

## Subject-Specific Adaptation Rules

The classroom artifact set should be adapted by subject, not copied mechanically.

### Literacy

- should include text-dependent tasks, writing prompts, exemplars, and discussion/response guidance

### Mathematics

- should include worked examples, reasoning-oriented answer guidance, and error-analysis notes

### Science

- should include observation sheets, explanation prompts, and evidence-based exemplar responses

### Social studies

- should include source or prompt sets, civic/history response tasks, and framing notes for accuracy and inclusion

### Arts

- should include creation prompts, performance or response guidance, and non-overstandardized criteria

### Health

- should include age-appropriate application tasks and carefully reviewed sensitivity language

### Physical education

- should include movement task cards, participation/safety criteria, and observation-based evidence tools

### Technology / computer science

- should include guided task steps, privacy-safe activity formats, and unplugged alternatives where possible

## Minimum Maturity Levels

### Draft curriculum packet

Required:

- unit
- lesson
- standards map
- review record
- support-material baseline

### Teacher-usable packet

Required:

- all draft packet artifacts
- lesson narrative
- classroom assignment or task sheet
- answer key or exemplar
- scoring guidance

### Classroom-ready packet

Required:

- all teacher-usable artifacts
- home practice where appropriate
- quiz/check/test materials where appropriate
- assessment keys
- completed human review for instructional quality and usability

## Repository Implication

Going forward, the repository should distinguish clearly between:

- structurally complete governed draft packets
- teacher-usable packets
- classroom-ready packets

The current repository has many structurally complete governed drafts.

It does not yet broadly include the full teacher-use classroom-materials set across packets.
