# Research References

## Purpose

This file is the shared reference index for the `research/` directory. It captures the core external sources that currently inform the repository's curriculum, OER, accessibility, interoperability, privacy, assessment, and AI-governance work.

Each major research document should either:

- cite entries from this index, or
- include a local bibliography section that points back to these sources

## Source Use Rules

Use sources in this order of preference:

1. official standards bodies and public guidance
2. government guidance and policy documents
3. peer-reviewed or consensus educational research
4. established field organizations
5. practitioner case evidence

For each source, contributors should note:

- title
- organization
- URL
- source type
- last known publication or update context
- whether the source is stable or time-sensitive

## Maintenance Notes

When a source becomes outdated or superseded:

2. add the newer source
3. record the update in the affected research files
4. revise recommendations where the updated guidance changes project decisions
