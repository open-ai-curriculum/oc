# Civics and History Source and Assessment Guidance

## Purpose

This document provides a practical review lens for source use and assessment design in civics and history materials.

## Core Position

Source quality and assessment quality are tightly linked in civics and history work.

A curriculum can appear rigorous while still being weak if:

- source selection is shallow
- evidence demands are unclear
- tasks reward unsupported opinion

## Source Review Questions

Reviewers and authors should ask:

1. What kind of source is this?
2. Why is this source included?
3. What perspective, evidence, or context does it provide?
4. Does the material help students distinguish source content from later interpretation?
5. Are there bias, omission, or sensitivity concerns?

## Assessment Guidance

Assessment tasks should often ask students to:

- interpret evidence
- compare sources or perspectives
- justify a claim using sources
- distinguish fact from interpretation
- explain civic or historical reasoning

## AI-Era Review Implications

Because AI can generate plausible but false historical or civic content, repository reviewers should be especially alert to:

- invented quotations
- invented events or details
- false balance on well-established issues
- generic prompts answerable without real source engagement

## Common Risks

- source use that is decorative rather than analytical
- prompts that reward unsupported opinion
- historical tasks with weak context
- assessment that treats citation-like behavior as evidence use even when reasoning is thin
