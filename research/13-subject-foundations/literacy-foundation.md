# Literacy and Language Arts Foundation

## Purpose

This document identifies core design considerations for literacy and language arts curriculum development in the repository.

## Key Principles

- treat reading, writing, speaking, listening, and language as connected
- build knowledge as well as skills
- use text selection intentionally
- maintain appropriate attention to vocabulary, comprehension, and evidence use
- design writing and discussion tasks that require real interpretation and support

## Relevant Standards and Source Families

- state ELA standards
- Common Core ELA where applicable
- What Works Clearinghouse literacy practice guides
- National Center on Improving Literacy resources

## Design Implications for This Repository

- text-dependent tasks should require evidence-based reading
- writing prompts should reward reasoning and support, not generic fluency
- accessibility and language support must be handled carefully
- AI-generated reading and writing tasks need review for text quality, bias, and grade fit

## Common Risks

- prompts that are generic rather than text-grounded
- weak text complexity control
- inaccessible reading load unrelated to the goal
- superficial discussion and writing tasks

## Confidence

Medium-high confidence.
