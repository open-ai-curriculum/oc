# ELA/Literacy State Standards Registry

## Purpose

This registry lists the state-level ELA/literacy standards sets currently codified for the `ela_literacy` domain.

It is the literacy counterpart to the mathematics state standards work and exists so later ELA/literacy node mappings can evolve beyond the national CCSS baseline.

## Expansion Rule

A literacy state profile may move from `Drafted` to `Ready for mapping review` only when:

1. its literacy content structure is codified with enough detail for node alignment
2. notable divergences from the CCSS ELA/literacy baseline are stated explicitly
3. adjacent language-development or language-proficiency standards are noted where they materially affect literacy interpretation

Profiles may move from `Drafted` to `Structurally codified` when:

1. the state's public organizational grammar is explicitly modeled
2. repository-relevant crosswalk surfaces are stated for foundational literacy, comprehension, language, writing, and advanced literacy
3. the repository posture toward identifier-level mapping is stated conservatively

## Claim Boundary

This registry defines repository work scope for publicly distributed state literacy standards. It does not by itself assert node-level equivalence or state compliance.
