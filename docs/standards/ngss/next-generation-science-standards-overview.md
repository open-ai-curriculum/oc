# Next Generation Science Standards Overview

## Purpose

This document codifies the Next Generation Science Standards as a repository standards family with explicit source surface, structural identity, and claim boundary.

## Public Source Surface

- NGSS overview page: [nextgenscience.org/get-know-standards](https://www.nextgenscience.org/get-know-standards)
- NGSS structure page: [nextgenscience.org/understanding-standards/understanding-standards](https://www.nextgenscience.org/understanding-standards/understanding-standards)
- NGSS standards search: [nextgenscience.org/search-standards](https://www.nextgenscience.org/search-standards?keys=ngss)
- NGSS appendices: [nextgenscience.org/resources/ngss-appendices](https://www.nextgenscience.org/resources/ngss-appendices)

## Profile Summary

The official NGSS source surfaces describe the standards as integrating the three dimensions from the National Research Council's framework into student performance expectations. The standards were developed through a collaborative, state-led process and published in 2013.

The public NGSS materials also expose:

- front matter
- structure guidance
- searchable performance expectations
- appendices A through M
- DCI arrangements
- topic arrangements

## Relationship To Repository Standards Work

The NGSS package is relevant to future science-domain modeling and to interdisciplinary crosswalk work because the official NGSS appendices explicitly include:

- connections to CCSS-Mathematics
- connections to CCSS-Literacy in Science and Technical Subjects

## Claim Boundary

This document codifies NGSS as a repository standards-family artifact. It does not claim that repository science nodes already exist or are currently mapped to NGSS.
