## Profile Summary

Illinois public standards materials state that Illinois joined the Common Core effort, adopted the Common Core in 2010, and fully implemented the new learning standards during the 2013-14 school year. Current ISBE standards pages describe the Illinois Learning Standards as establishing expectations for what students should know and be able to do in each subject at each grade.

For repository purposes, Illinois is a baseline-aligned profile with an explicit Illinois Learning Standards identity and a strong assessment-resource surface.
