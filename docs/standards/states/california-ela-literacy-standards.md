## Profile Summary

California uses the California Common Core State Standards for English Language Arts and Literacy in History/Social Studies, Science, and Technical Subjects as its core ELA/literacy standards surface. The public California standards materials describe August 2010 adoption of the standards and March 2013 publication modifications tied to California additions and technical changes.

California also treats the English Language Arts/English Language Development framework and the California English Language Development Standards as integral implementation surfaces for literacy instruction.

For repository purposes, California is a baseline-aligned literacy profile with strong state-specific ELD integration.

## Baseline-Aligned Mapping Rules

California should be treated as close to the canonical ELA/literacy baseline, but not as a metadata-free duplicate.

Important repository rules:

1. preserve California additions and technical changes where they materially affect later identifier mapping
2. preserve the `ELA/ELD Framework` and California ELD standards as adjacent interpretation surfaces, not as replacements for the content standards
3. keep language-access and multilingual-learning considerations explicit so literacy difficulty is not misclassified
4. allow the current node map to inherit the canonical baseline posture while deferring reviewed California-specific identifier claims

- reading
- writing

- language

Operational fit with repository node work:

- strong fit for disciplinary literacy, rhetorical analysis, source-based writing, and advanced literacy nodes
