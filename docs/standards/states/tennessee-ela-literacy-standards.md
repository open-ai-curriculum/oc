## Profile Summary

Tennessee's public ELA standards page presents the current Tennessee Academic Standards for English Language Arts in a K-12 vertical chart that emphasizes how reading, writing, speaking and listening, and language spiral across grades.

At the same time, the official Tennessee Academic Standards overview page, reviewed on April 14, 2026, shows that revised English Language Arts standards were proposed to the State Board in February 2026 and are scheduled for classroom implementation in the 2029-2030 school year.

For repository purposes, Tennessee is a baseline-derived literacy profile with an explicit transition horizon.
