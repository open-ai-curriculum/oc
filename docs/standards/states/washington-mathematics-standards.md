## Profile Summary

As of the official OSPI mathematics standards page reviewed on April 14, 2026, Washington identifies the `2025–26 Washington State K-12 Learning Standards for Math` as being in the Initial Adoption and Implementation phase. The page also preserves the `2011–12 Washington State K-12 Learning Standards for Math` and notes that assessments for those standards began in the 2014-15 school year.

The state review timeline further identifies mathematics, including data science, as part of the current standards review cycle.

For repository purposes, Washington is a state-distinct profile with an active transition posture. That makes version tracking especially important.
