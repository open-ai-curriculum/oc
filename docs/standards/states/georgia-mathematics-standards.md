# Georgia Mathematics Standards

## Jurisdiction

- jurisdiction: `Georgia`
- subject: `Mathematics`
- source profile: `Georgia's K-12 Mathematics Standards`
- repository codification status: `Drafted`

## Public Source Surface

- Georgia mathematics standards page: [georgiastandards.org/Georgia-Standards/pages/math.aspx](https://www.georgiastandards.org/Georgia-Standards/pages/math.aspx)

## Profile Summary

Georgia's public mathematics standards page states that the Georgia State Board of Education approved Georgia's K-12 Mathematics Standards on August 26, 2021, with implementation beginning in the 2023-2024 school year.

The same state surface also highlights machine-readable publication in CASE format and provides K-12 Mathematical Practices, K-12 Mathematical Modeling Framework, K-12 Framework for Statistical Reasoning, and transition resources.

For repository purposes, Georgia is a baseline-derived profile with strong state-specific implementation layers.

### Grades 9 Through 12

Primary organizational pattern:

## Notable Divergences Or State-Specific Terminology

- the state surface uses `Georgia's K-12 Mathematics Standards` rather than only the older `GSE` label

- public state resources elevate `Mathematical Practices`, a `Mathematical Modeling Framework`, and a `Framework for Statistical Reasoning`
