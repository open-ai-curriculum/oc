## Profile Summary

Massachusetts uses a Curriculum Framework structure rather than a simple standards-list presentation. The public mathematics framework surface is strongly compatible with Common Core style content organization while retaining Massachusetts framing, guidance, and assessment connections.

The framework was adopted by the Massachusetts Board of Elementary and Secondary Education in March 2017, based on the official public framework materials currently linked by the state.

For repository purposes, Massachusetts is a baseline-derived profile.
