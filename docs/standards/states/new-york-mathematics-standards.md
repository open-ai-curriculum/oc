## Profile Summary

New York uses the Next Generation Mathematics Learning Standards, a revised state mathematics standards set built from a review and revision process rather than a simple one-to-one reuse of older Common Core materials.

For repository purposes, New York is a baseline-derived profile. Its structure remains broadly compatible with a Common Core style content taxonomy, but the state profile should be treated as its own codified standards family.

## Relationship To Canonical Mathematics Content Baseline

- relationship type: `baseline-derived`
- core structure: `P-12 learning standards with grade-band and high-school course surfaces closely related to Common Core style organization`
- notable state distinction: `New York-specific revisions, transitions, and guidance materials should be preserved as state metadata rather than assumed to be identical to baseline CCSS labels`

## Relationship To Repository Performance Baseline

New York publicly exposes mathematics performance-level descriptions for state assessment work. That makes New York a strong candidate for a later state-specific performance profile. For now, the repository should keep content alignment and performance-level alignment separate.

## Grade-Band Structure

### Elementary And Middle Grades

Primary organizational pattern:

- grade-specific learning standards
- domains and content groupings with familiar mathematics progression

Operational fit with repository node work:

- good fit for counting, operations, fractions, expressions, geometry, and statistics nodes
- mapping should preserve the New York state standards family name rather than re-labeling these as generic CCSS-only references

### High School

Primary organizational pattern:

- course-facing high-school mathematics surfaces
- public state resources include transition-sensitive treatment of Algebra I, Geometry, and Algebra II

Operational fit with repository node work:

- algebra, functions, geometry, and statistics nodes can map reasonably well
- timeline-sensitive or course-sensitive references should be documented carefully when later node mappings are added

## Domain Or Strand Structure

New York's codification should preserve:

- grade or course
- domain or content grouping
- anchor statement or identifier
- any transition-sensitive state notes that materially affect later mapping

## Notable Divergences Or State-Specific Terminology

- `Next Generation Mathematics Learning Standards` is the governing state standards family
- New York provides guidance resources and performance-level descriptions that may later justify separate companion artifacts
- transition details around high-school standards should not be collapsed into a timeless generic baseline label

## Mapping Readiness Notes

New York is a strong early profile because it supports both content-standards codification and a plausible later performance-level codification from official state materials.

## Unresolved Review Questions

- how to represent transition-sensitive high-school course references as the state moves fully into next-generation assessment surfaces
- whether New York performance-level descriptions should be codified grade by grade or as a generalized state performance artifact
- whether the repository should preserve crosswalk relationships to older Common Core labels for backward compatibility

## Claim Boundary

This document codifies New York's public mathematics standards surface for repository alignment planning. It does not assert complete equivalence between repository nodes and New York learning standards or assessments.
