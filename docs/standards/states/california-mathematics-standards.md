- jurisdiction: `California`

## Profile Summary

California uses a mathematics standards structure that is strongly aligned to Common Core style organization, with grade-level standards in elementary and course-oriented high-school standards. California also exposes California additions and implementation guidance through its standards and framework materials.

This makes California a baseline-aligned profile with state-specific extensions rather than a completely distinct structure.

## Relationship To Canonical Mathematics Content Baseline

- relationship type: `baseline-aligned`
- core structure: `largely compatible with Common Core style grade, domain, cluster, standard identifier organization`
- notable state distinction: `California additions and state implementation materials should be preserved as explicit metadata rather than collapsed into generic CCSS labels`

## Relationship To Repository Performance Baseline

California's content standards surface is primarily an academic content taxonomy. Repository performance-level mapping should remain separate unless a California-specific performance-descriptor source is later codified for the relevant assessments or reporting levels.

## Grade-Band Structure

### Kindergarten Through Grade 8

Primary organizational pattern:

- grade-specific standards
- domains within grade
- clusters within domain
- individual standard identifiers

Operational fit with repository node work:

- clean mapping surface for early counting, operations, fractions, measurement, geometry, and data nodes
- strong compatibility with node-level alignment by identifier where later needed

### High School

Primary organizational pattern:

- conceptual categories and high-school domains
- course pathways supported through course-specific views or standards placements
- modeling and advanced preparation markers exposed in the standards presentation

Operational fit with repository node work:

- useful for algebra, functions, geometry, statistics, and modeling-oriented node mapping
- requires explicit handling of pathway or course context when a node spans more than one high-school arrangement

## Domain Or Strand Structure

California exposes familiar mathematics content domains that closely match the canonical baseline, including:

- counting and cardinality
- operations and algebraic thinking
- number and operations in base ten
- number and operations with fractions
- expressions and equations
- ratios and proportional relationships
- the number system
- functions
- geometry
- statistics and probability

High-school surfaces also preserve modeling and advanced-course markers.

## Notable Divergences Or State-Specific Terminology

- standards search surfaces identify California additions with a `CA` notation
- modeling and advanced-course markers need to be preserved in any later crosswalk
- the California Mathematics Framework is important implementation guidance but should not be confused with the standards identifiers themselves
