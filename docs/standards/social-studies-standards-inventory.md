# Social Studies Standards Inventory

## Purpose

This document codifies the current anchor-level social studies standards inventory that the repository can validate against the social studies knowledge graph.

The inventory is intentionally conservative:

- it operates at the repository anchor level rather than claiming official identifier-level coverage
- it preserves a direct anchor for each governed family-level node
- it exists so standards-to-node completeness can be validated explicitly rather than inferred from node counts alone
