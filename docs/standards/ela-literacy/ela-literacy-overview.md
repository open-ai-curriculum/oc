# ELA/Literacy Overview

## Purpose

This document codifies the repository's ELA/literacy standards family as a governed package with explicit structure, versioning, and claim boundaries.

## Scope

The current package scope includes:

- foundational reading skills and print concepts
- reading literature
- reading informational text
- writing
- language
- speaking and listening
- disciplinary literacy in history/social studies, science, and technical subjects where relevant

## Relationship To Public Standards Families

The current repository literacy surface is canonically anchored to:

- [Common Core State Standards For ELA And Literacy](../ccss/common-core-state-standards-ela-literacy.md)

State ELA/literacy profiles may remain baseline-aligned, baseline-derived, or structurally distinct, but they should be modeled relative to this package and not as disconnected standalone lists.

## Repository Interpretation Rule

The ELA/literacy standards package is an alignment and codification surface. It does not replace:

- the capability graph
- node-level dependency logic
- verification gates
- failure-mode handling
- intervention rules
- human review requirements

## Claim Boundary

This package codifies the repository literacy standards surface for later mapping work. It does not yet assert identifier-level equivalence for every state, classroom sufficiency, or reviewed learner-impacting approval.
