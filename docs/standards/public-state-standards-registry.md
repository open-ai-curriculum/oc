## Expansion Rule

A registry entry may move from `Planned` to `Drafted` only when a state-specific standards profile exists under the maintained standards surface.

A registry entry may move from `Drafted` to `Ready for mapping review` only when:

1. the state's content structure has been codified with enough detail for node alignment
2. divergences from the canonical baseline are noted
3. claim boundaries remain explicit

## Historical Version Rule

If a drafted or mapping-ready state profile is superseded by a new standards edition, the registry should continue to point to the stable current-path profile, and the prior edition should be preserved under `states/history/` with `standards_track: historical`.

## Claim Boundary

This registry defines repository work scope for public state standards. It does not by itself codify any state's standards content.
