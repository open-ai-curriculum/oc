# AP Human Geography Framework Profile

## Framework

- framework family: `AP Human Geography`
- subject: `Social Studies`
- source profile: `public AP Human Geography course-framework surface`
- repository codification status: `Drafted`

## Public Source Surface

For repository purposes, the important public surfaces are:

- spatial concepts and geographic tools
- population, migration, and human systems
- economic and cultural spatial patterns
- sustainability, policy, and evidence-based argument

## Profile Summary

This framework is a high-value external social studies surface because it aligns strongly to the repository's geography branch and its policy-facing synthesis work.

For repository purposes, it is best treated as a geography-centered course-distinct secondary framework with strong spatial-reasoning and public-argument adjacency.

## Relationship To Repository Social Studies Content Baseline

- relationship type: `course-distinct baseline-adjacent`
- core structure: `spatial reasoning, human-environment systems, migration, global interdependence, and policy application`
- notable framework distinction: `the public course organization is thematic and course-specific rather than a full K-12 geography grammar`

## Relationship To Repository Performance Baseline

The public AP framework informs explanation, evidence use, and argument expectations, but repository performance bands for mastery, transfer, and retention remain separate governance layers unless explicitly codified from reviewed performance materials.

## Organizational Structure

The framework is most useful to the repository when modeled through these strands:

- spatial concepts and geographic tools
- population and migration systems
- human-environment interaction and sustainability
- economic and cultural patterning across space
- policy application and evidence-based public argument

## Repository-Relevant Landing Zones

The strongest current landing zones are:

- `G1` through `G6` for spatial, regional, migration, and policy-facing geography structure
- `Q4` through `Q6` for geographic inquiry, evidence use, and public argument
- `E3` through `E5` where geography and economic patterning intersect
- `A2` where public communication and capstone-style synthesis are required

## Notable Divergences Or Framework-Specific Constraints

- the public AP course surface is narrower and more thematic than the repository's full geography branch
- thematic subunits can exceed the current graph's internal geography granularity even where overall posture is strong
- public course framing and assessment format should not be confused with official repository blueprint verification

## Mapping Readiness Notes

This profile is structurally useful for future mapping because:

- the framework organization is clear enough to identify stable geography landing zones
- the graph already models the core disciplinary territory explicitly
- the remaining gaps are primarily thematic compression rather than missing branch coverage

Current posture:

- structurally useful for framework-to-node review
- not yet sufficient for blueprint-verified equivalence claims
