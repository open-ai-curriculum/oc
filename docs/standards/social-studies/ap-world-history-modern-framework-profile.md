# AP World History: Modern Framework Profile

## Framework

- framework family: `AP World History: Modern`
- subject: `Social Studies`
- source profile: `public AP World History: Modern course-framework surface`
- repository codification status: `Drafted`

## Public Source Surface

For repository purposes, the important public surfaces are:

- historical thinking, comparison, and argument
- global processes, periodization, and change over time
- state building, economy, and exchange
- geography, migration, and human systems

## Profile Summary

This framework is a high-value external social studies surface because it aligns to the repository's history, geography, civics, and economics branches simultaneously, while also exposing where chronological and regional granularity remain compressed.

For repository purposes, it is best treated as a world-history-centered course-distinct secondary framework with strong global-systems and argument adjacency.

## Relationship To Repository Social Studies Content Baseline

- relationship type: `course-distinct baseline-adjacent`
- core structure: `historical thinking, global processes, state systems, migration, exchange, and evidence-based world-history argument`
- notable framework distinction: `the public course organization is more period- and region-specific than the canonical repository history branch`

## Relationship To Repository Performance Baseline

The public AP framework informs evidence use, argument, and historical explanation expectations, but repository performance bands for mastery, transfer, and retention remain separate governance layers unless explicitly codified from reviewed performance materials.

## Organizational Structure

The framework is most useful to the repository when modeled through these strands:

- historical thinking, comparison, and argument
- global processes, periodization, and change over time
- state building, economy, and exchange
- geography, migration, and human systems

## Repository-Relevant Landing Zones

The strongest current landing zones are:

- `H2` through `H6` for historical reasoning, periodization, and argument
- `Q4` through `Q6` for evidence use, evaluation, and argument
- `G2` through `G6` for migration, regional systems, and global interdependence
- `C4` and `C5` for state systems and comparative governance structures
- `E3` through `E5` for exchange, trade, and macro-pattern context
- `A1` and `A2` where synthesis and public communication are required

## Notable Divergences Or Framework-Specific Constraints

- the public AP course surface is more chronologically and regionally specific than the repository's canonical history branch
- some state-building and world-region distinctions exceed the current graph's internal resolution
- public course framing and assessment format should not be confused with official repository blueprint verification

## Mapping Readiness Notes

This profile is structurally useful for future mapping because:

- the framework organization is clear enough to identify stable global-history landing zones
- the graph already models most underlying disciplinary territory explicitly
- the remaining gaps are concentrated in chronology and regional specificity rather than missing branch coverage

Current posture:

- structurally useful for framework-to-node review
