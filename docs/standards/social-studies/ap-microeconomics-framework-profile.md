# AP Microeconomics Framework Profile

## Framework

- framework family: `AP Microeconomics`
- subject: `Social Studies`
- source profile: `public AP Microeconomics course-framework surface`
- repository codification status: `Drafted`

## Public Source Surface

For repository purposes, the important public surfaces are:

- choice, incentives, and microeconomic models
- markets, firms, and decision-making
- government intervention and welfare
- evidence-based argument and application

## Profile Summary

This framework is a high-value external social studies surface because it aligns to the repository's economics, policy, and argument branches, while making the remaining firm-structure and welfare-analysis gaps explicit.

For repository purposes, it is best treated as an economics-centered course-distinct secondary framework with explicitly exploratory course-specificity.

## Relationship To Repository Social Studies Content Baseline

- relationship type: `course-distinct baseline-adjacent`
- core structure: `choice, incentives, market behavior, intervention, welfare, and evidence-based microeconomic application`
- notable framework distinction: `the public course organization expects firm structure, welfare analysis, and microeconomic model detail beyond the canonical repository economics branch`

## Relationship To Repository Performance Baseline

The public AP framework informs explanation, evidence use, and application expectations, but repository performance bands for mastery, transfer, and retention remain separate governance layers unless explicitly codified from reviewed performance materials.

## Organizational Structure

The framework is most useful to the repository when modeled through these strands:

- choice, incentives, and microeconomic models
- markets, firms, and decision-making
- government intervention and welfare
- evidence-based argument and application

## Repository-Relevant Landing Zones

The strongest current landing zones are:

- `E1` through `E6` for incentives, markets, policy tradeoffs, and applied analysis
- `Q3` through `Q6` for explanation, evidence use, evaluation, and argument
- `C4` and `C6` where policy intervention and public reasoning intersect with economics
- `A1` where synthesis and applied economic communication are required

## Notable Divergences Or Framework-Specific Constraints

- the public AP course surface expects more firm-structure and welfare-analysis detail than the current repository economics branch provides
- exploratory status should remain explicit even though the framework grammar is now codified
- public course framing and assessment format should not be confused with official repository blueprint verification

## Mapping Readiness Notes

This profile is structurally useful for future mapping because:

- the framework organization is clear enough to identify stable microeconomics landing zones
- the graph already models the major incentives, markets, and policy structures explicitly
- the remaining gaps are concentrated in firm and welfare detail rather than missing economics coverage

Current posture:

- structurally useful for framework-to-node review
- still exploratory for stronger course-level claims
- not yet sufficient for blueprint-verified or identifier-level equivalence claims

## Unresolved Review Questions

- whether firm structure and welfare-analysis substructure should be decomposed before stronger course-level claims are made
- which microeconomic distinctions should become first-class metadata rather than overlay notes
- how much assessment-format specificity should remain outside the standards package

## Claim Boundary

This document codifies the public AP Microeconomics framework surface for repository alignment planning. It does not claim official blueprint verification, reviewed node-level equivalence, or exam-preparation approval.
