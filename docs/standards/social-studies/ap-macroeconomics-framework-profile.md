# AP Macroeconomics Framework Profile

## Framework

- framework family: `AP Macroeconomics`
- subject: `Social Studies`
- source profile: `public AP Macroeconomics course-framework surface`
- repository codification status: `Drafted`

## Public Source Surface

For repository purposes, the important public surfaces are:

- aggregate measures and macroeconomic foundations
- fluctuation, growth, and stabilization-policy structures
- finance, trade, and global-context organization
- evidence-based argument and application expectations

## Profile Summary

This framework is a high-value external social studies surface because it aligns to the repository's economics, policy, and global-context branches, while also exposing where the current economics topology remains broad relative to formal macro-model detail.

For repository purposes, it is best treated as an economics-centered course-distinct secondary framework with strong policy-and-argument adjacency.

## Relationship To Repository Social Studies Content Baseline

- relationship type: `course-distinct baseline-adjacent`
- core structure: `aggregate reasoning, stabilization policy, global trade and finance, and evidence-based macroeconomic application`
- notable framework distinction: `the public course organization expects stronger macroeconomic measure and model specificity than the canonical repository economics branch currently exposes`

## Relationship To Repository Performance Baseline

The public AP framework informs explanation, evidence use, and application expectations, but repository performance bands for mastery, transfer, and retention remain separate governance layers unless explicitly codified from reviewed performance materials.

## Organizational Structure

The framework is most useful to the repository when modeled through these strands:

- foundations and aggregate measures
- growth, fluctuation, and stabilization policy
- finance, trade, and global context
- evidence-based application and argument

## Repository-Relevant Landing Zones

The strongest current landing zones are:

- `E3` through `E6` for trade, macro-patterns, policy tradeoffs, and applied analysis
- `Q4` through `Q6` for evidence use, evaluation, and argument
- `C4` and `C6` where policy analysis and public reasoning intersect with economics
- `G5` and `G6` where global context and policy application matter
- `A1` and `A2` where synthesis and public communication are required

## Notable Divergences Or Framework-Specific Constraints

- the public AP course surface expects more formal macro-measure and model specificity than the current repository economics branch provides
- public course framing and assessment format should not be confused with official repository blueprint verification
- some macroeconomic distinctions remain compressed inside broader economics and policy nodes

## Mapping Readiness Notes

This profile is structurally useful for future mapping because:

- the framework organization is clear enough to identify stable economics landing zones
- the graph already models the major macro-policy and trade structures explicitly
- the remaining gaps are concentrated in formal macro-model detail rather than missing economics coverage

Current posture:

- structurally useful for framework-to-node review
