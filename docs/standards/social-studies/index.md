# Social Studies Standards Package

## Purpose

This package codifies the repository's social studies standards family as a governed package with explicit structure, versioning, and claim boundaries.

## Current Scope

The current social studies package includes:

- a repository-governed social studies overview
- a repository-governed social studies content architecture
- a codification model for external social studies frameworks and state standards
- an external-framework registry defining non-state framework expansion scope
- a state standards registry defining jurisdiction-level expansion scope
- first external framework profiles for the strongest current AP-facing social studies surfaces
- first drafted state profiles for structurally distinct social studies standards surfaces
- repository-level content and performance baselines
- an anchor-level standards inventory
- a node-to-standards map
- a coverage report

## Current Constraint

The package is currently repository-baseline first.

That means it establishes:

- the shape of social studies alignment work in this repository
- the anchor surface that current social studies nodes map against
- the governance boundary for future crosswalk work

It does not yet establish:

- reviewed official state or national social studies identifier mappings
- external standards completeness claims
- jurisdiction-specific compliance claims
