# Social Studies State Standards Registry

## Purpose

This registry lists the state-level social studies standards sets that are in scope for later codification under the repository social studies standards surface.

The registry tracks repository work posture only. It does not claim that a state's social studies standards set has already been codified in detail.

## Current Registry Posture

The repository does not yet contain jurisdiction-specific social studies profiles under `docs/standards/states/`.

That means every state-level social studies standards set is currently in the `Planned` posture. This is still useful because it defines the work surface needed to move social studies from repository-baseline validation to mathematics-style multi-jurisdiction standards parity.

## Expansion Rule

A social studies state profile may move from `Planned` to `Drafted` only when a jurisdiction-specific standards profile exists under the maintained standards surface.

A profile may move from `Drafted` to `Structurally codified` only when:

1. its inquiry and disciplinary organizational grammar is explicitly modeled
2. its grade-band or course-band structure is stated clearly enough for node alignment
3. its repository claim boundary is conservative about identifier-level equivalence

A profile may move from `Structurally codified` to `Ready for mapping review` only when:

1. node-alignment landing zones are identifiable without hidden assumptions
2. major divergences from the repository social studies baseline are recorded
3. unresolved human-review questions are explicit

## Claim Boundary

This registry defines repository work scope for publicly distributed social studies state standards. It does not by itself assert node-level equivalence, state compliance, or completeness of any jurisdiction's standards content.
