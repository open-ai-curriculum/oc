# Science Standards Inventory

## Purpose

This document codifies the current anchor-level science standards inventory that the repository can validate against the science capability graph.

The inventory is intentionally conservative:

- it operates at the repository anchor level rather than claiming full NGSS performance-expectation identifier coverage
- it preserves the repository rule that capability nodes remain primary and standards mappings remain external alignment metadata
- it exists so science standards-to-node completeness can be validated explicitly rather than inferred informally
